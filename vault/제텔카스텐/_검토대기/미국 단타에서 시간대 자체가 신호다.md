# 미국 단타에서 시간대 자체가 신호다

같은 차트 모양도 프리장 19시에 등장한 것과 본장 9시에 등장한 것은 결과가 다르다. 시간대를 모르면 같은 자리에서도 적중률이 떨어진다.

## 근거

준석TV 데이터로는 하루 급등주의 50%가 프리장(한국 시간 17:00~22:30)에서 발생한다. 본장 시작 직후 30분 동안 첫 움직임이 70% 확률로 반대로 돌아간다. 새벽 3·4·5·6시 정각에 60분봉 전환점에서 급등이 자주 발생한다. 시간대마다 세력의 행동 패턴이 다르기 때문이다.

## 적용

- 17:00 (서머타임 시) 프리장 시작 → 종목 순위 모니터링
- 22:30 본장 시작 → 첫 움직임 반대 베팅
- 23:30 이후 본장 후반 → C자리 진입
- 새벽 매매는 자동 손절 주문 걸고 취침
- 한국 직장인은 프리장 + 자동 주문 조합이 현실적
