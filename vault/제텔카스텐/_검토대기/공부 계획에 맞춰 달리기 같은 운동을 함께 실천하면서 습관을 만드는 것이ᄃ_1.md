# 공부 계획에 맞춰 달리기 같은 운동을 함께 실천하면서 습관을 만드는 것이ᄃ

<0802 방학 #습관 프로젝트 : 학습 습관과 건강한 습관 만들기 7일차>


1. 여러분의 공부계획에 맞춰 공부를 실천하세요!

-> 공부실천 내용은 영상으로 인증
-> 공부 계획은 사진으로 인증! + 공부 결과물도 사진으로 인증

*공부계획 맞게 1시간 30분 정도 실천한 내용을 올립니다.


2. 과제 : 달리기 30분 실천하기 (런데이 앱 활용)

 1시간 30분 공부 계획 안에 "달리기"를 넣어 실천해봅시다. 
(방과후 가는 친구들은 방과후 끝나서~ 달리기 해보는 것은 어떨까요?!)

*달리기 운동을 꼭! 해보세요. 우리는 나눔 마라톤을 할 것입니다. 마라톤 연습이라 생각하고 하면 좋겠어요 ㅎㅎ
