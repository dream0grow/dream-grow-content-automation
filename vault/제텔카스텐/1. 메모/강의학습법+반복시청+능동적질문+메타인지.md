### 생각의 연결고리
분야 : 학습
키워드 : 강의학습법, 반복시청, 능동적질문, 메타인지, 자기주도학습

핵심 메시지 : 강의는 3회 반복 시청하되, 2회차에 궁금증을 먼저 적고 3회차에 스스로 답을 찾아야 진짜 학습이 된다.

근거 : 1회 단순 시청이 아닌 2~4강을 3회 반복 시청하는 구조. 2회차에 이해 안 되는 부분과 궁금한 점을 노트에 적고, 3회차 전에 인터넷으로 답을 먼저 찾은 뒤 다시 영상을 본다.

핵심 메시지 생각을 확장하면? :
- 아이 학습에도 적용: 문제를 먼저 풀어보고 → 모르는 것 표시 → 강의/설명 듣기 순서.
- 메타인지 훈련과 연결: 모르는 것을 인식하는 능력이 학습 효율의 핵심.
