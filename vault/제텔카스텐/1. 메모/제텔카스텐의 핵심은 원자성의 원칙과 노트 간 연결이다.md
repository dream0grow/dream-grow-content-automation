- 제텔카스텐에서 가장 중요한 원칙은 원자성: 하나의 노트에는 하나의 생각만 기록한다.
- 노트와 노트가 연결될 때 가장 큰 파급효과를 얻는다. 연결하는 이유를 반드시 설명해야 생각이 커진다.
- 메모의 7종류: 임시 메모 → 문헌 메모 → 영구 보관 메모 → 키워드 메모 → 의견 메모 → 주장 메모 → 두번째 뇌 메모
- 제텔카스텐에 필요한 4가지 능력: 건축가(지식 구조, 워크플로우), 정원사(아이디어 생성), 사서(수집과 편집), 경찰관(탐색과 연결)
- 제텔카스텐은 장기 프로젝트이며, 메모는 쌓일수록 눈덩이처럼 효과적이다.
- 가장 중요한 이유는 생산성 향상 = 창의성 향상
