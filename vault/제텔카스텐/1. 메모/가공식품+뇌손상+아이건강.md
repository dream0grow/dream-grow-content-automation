### 생각의 연결고리
분야 : 학습
키워드 : 가공식품, 뇌손상, 아이건강, 장건강, 식습관

핵심 메시지 : 편의점 가공식품 속 보존제와 첨가물은 장내 유익균을 파괴하고 뇌세포를 직접 손상시켜 아이의 학습 능력과 정서 발달을 방해한다.

근거 : 가공식품 속 보존제는 장내 유익균을 파괴해 뇌 발달에 필요한 영양소 생산을 차단한다. 프로피온산 같은 첨가물은 뇌세포를 직접 손상시켜 노폐물 처리를 방해하고 신경 연결을 막는다. 설탕이 만드는 혈당 급등락은 짜증과 집중력 저하의 직접적 원인이다.

핵심 메시지 생각을 확장하면? :
- 아이의 학습 능력은 공부법뿐 아니라 식습관에 의해서도 크게 좌우된다. 편의점 음식 관리가 학습 지원의 기본이다.
- 편의점에서도 건강한 선택이 가능하다. 라면 대신 삼각김밥+삶은 계란, 과자 대신 견과류+바나나 같은 대안을 알려주는 콘텐츠로 활용 가능.

이 메시지와 관련있는 메모 :

- [[도파민은 잘못이 없다 - 문제는 순환 구조이다]]

추가로 조사할 내용
- 가공식품 첨가물(프로피온산 등)과 뇌 발달 관계 논문 확인 (수치 출처 검증 필요)

관련된 내 의견 :
