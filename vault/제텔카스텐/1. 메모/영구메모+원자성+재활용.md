인용 : "영구 메모는 다른 글에 재활용 가능해야 하며, 이를 위해 원자적(atomic) 단위로 작성해야 한다."

### 생각의 연결고리
분야 : 크리에이터, 학습
키워드 : 영구메모, 원자성, 재활용, 제텔카스텐, 지식관리

핵심 메시지 : 영구 메모는 하나의 아이디어를 원자적 단위로 담아야 다른 글에 재활용 가능한 형태가 된다.

근거 : 의견이 나뉘는 지점('그러나', '반면에')이 있다면 메모를 2개로 나눠야 한다. 원자적 메모의 핵심 구성요소는 제목(핵심 문장), 상세 내용, 태그, 소스, 연결된 노트다. 단, 하나의 아이디어 덩어리로 인식 가능하면 길이가 길어도 원자성을 인정할 수 있다.

핵심 메시지 생각을 확장하면? :
- 원자성의 실용적 기준은 '재활용 가능성'이다. 다른 맥락에 꽂았을 때 의미가 통하면 원자적이다.
- 제텔카스텐에서 영구 메모로 전환할 때 자신의 언어와 의견을 반드시 추가해야 진짜 지식이 된다.

이 메시지와 관련있는 메모 :

- [[제텔카스텐의 핵심은 원자성과 연결이다]]

추가로 조사할 내용
- 숑케 아렌스 'How to Take Smart Notes'에서 원자성 기준 재확인

관련된 내 의견 : 
- 엄격한 원자성보다 '하나의 아이디어 = 하나의 메모'라는 실용적 기준이 더 유용하다.
