6분 50초 : 사실 의미 듣기는 하고 있느나 공감을 할 때 에너지를 실어서 더 먼저 했으면 좋았을 것 같다. 
아, 계획은 하는데,  정말 답답하겠어요. 에너지는 쓰고 싶은데 쓰지 못하니 말이죠. 
스스로는 자책이 되시겠네요.는 내 생각에서 나온 말 -> 자책이라고 하니 내담자도 ‘자책’이라는 것을 반응함. 
7분 40초, 인정을 한 후 아쉽고 답답할꺼 같아요라고 공감을 함. 공감에 완전히 실리면 어떻게 말을할 수 있었을까? 
8분 40초, 기본적으로 계속 사실의미듣기만 중심으로 함. 지금 내담자가 바라는 것은 공감에 더 가까울 것 같음. 
“수업 준비하는 것이 아이들과 만남에 있어서 여러가지가 엮여있으니 더 잘 준비하고 싶겠어요.”라고 의미 듣기함. 
내담자가 “쉬는 시간에 수업준비가 다 되어있지 않으면 마음이 분주해지는 것 같아요. 그래서 수업준비가 중요하다고 생각합니다.“ 
상담자가 ”그러면 은지님에게…이니 더 수업준비를 하고 싶겠어요.“라고 의미듣기 함. 
내담자가 ”아~ 해야하는데.. 수업 준비를 안 한 것 이라고 생각했는데, 수업준비가 내게 어떤 도움을 주고, 의미 정리가 되네요.” 
은지님이 얼마나 아이들과 같이 잘 지내고 싶은지 느껴지고… 라고 본심을 알아줌. + 칭찬 인정을 함. “주체성이 놀라워요.“ 
여기서 본심만 알아줬으면 어떨까? 
내담자가 ”막상 할려고 하면 쉬고 싶은 맘도 들어요.“라며 칭찬을 받는 것을 어색해 함. 
칭찬할 타이밍이 아니었다는 말.
