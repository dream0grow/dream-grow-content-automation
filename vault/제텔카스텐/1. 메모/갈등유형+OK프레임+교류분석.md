### 생각의 연결고리
분야 : 상담, 학교
키워드 : 갈등유형, OK프레임, 교류분석, 동물비유, 갈등해결스타일

핵심 메시지 : 갈등 해결 유형은 5가지 동물로 나뉘며, I'm OK / You're OK 프레임으로 자기-타인 인정 수준과 결합해 분석할 수 있다.

근거 : 갈등 스타일을 상어(경쟁), 부엉이(협력), 여우(타협), 거북이(회피), 곰(순응) 등 5가지로 분류할 때 가로축과 세로축을 I'm OK / You're OK 교류분석 프레임으로 설정하면 각 스타일의 자기존중-타인존중 수준을 함께 파악할 수 있다.

핵심 메시지 생각을 확장하면? :
- 회복적 생활교육에서 학생의 갈등 유형 진단 도구로 활용 가능
- Thomas-Kilmann 모델과 교류분석 OK 프레임의 연결 가능성
