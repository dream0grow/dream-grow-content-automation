### 생각의 연결고리
분야 : 학습
키워드 : 습관자동화, 환경설계, 버튼모듈, 행동심리학, 마찰력

핵심 메시지 : 습관의 핵심은 반복에 의한 무의식적 자동 실행이며, 이를 가능하게 하는 것은 의지력이 아니라 환경 설계다.

근거 : 습관 = 추진력 - 억제력 +/- 마찰력(환경의 힘). 자동차 운전 시 방향지시등을 틀지 고민하지 않듯 진정한 습관은 '악마의 속삭임'에 영향받지 않는다. 습관을 잘 유지하는 그룹의 비밀은 개인 자제력 차이가 아니라, 습관이 흔들릴 환경 자체를 제거하는 것(과자를 치우고 과일을 눈앞에 두는 '상황 배열')이다. 버튼 모듈: 생리적 신호 또는 반드시 해야 하는 행동에 목표 습관을 연결(청킹)하면 자동 발화 확률이 높아진다.

핵심 메시지 생각을 확장하면? :
- 유튜브 영상 제작도 습관 시스템(환경 배열 + 청킹)으로 설계하면 지속 가능해진다
- 아이 학습 습관 지도 시 '의지력 강화'보다 '공부하기 좋은 환경 배열'이 더 효과적이다
