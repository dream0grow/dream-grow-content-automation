### 생각의 연결고리
분야 : 교과 / 학습
키워드 : 개념기반탐구, 전이가능한지식, 백워드설계, 귀납적사고, 수업설계

핵심 메시지 : 개념기반탐구수업은 사실에서 귀납적으로 개념을 추출하고 이를 연결해 원리를 만들어 실제 상황에 전이 가능한 지식으로 발전시키는 수업 설계다.

근거 : 사실 → 귀납적 공통점 추출 → 개념. 개념 연결 → 원리(일반화된 진술). 원리는 전이 가능하다. 백워드 설계로 탐구를 구조화하며, See-Think-Wonder(근거 기반 발화)와 프레이어 모델(예/비예 분류 → 특징 → 정의)이 대표 전략이다.

핵심 메시지 생각을 확장하면? :
- 제텔카스텐과 구조적으로 동일하다. 원자적 메모(사실) → 연결 → 새로운 통찰(원리).
- 아이들에게 "왜?"를 묻는 것이 개념 추출의 시작임을 부모에게 설명하는 콘텐츠 소재.
