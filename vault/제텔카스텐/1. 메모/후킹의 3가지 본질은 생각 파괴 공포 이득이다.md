# 후킹의 3가지 본질은 생각 파괴 공포 이득이다

곤팀장이 *블로그 상위노출 A TO Z 공략집*과 *컨설팅 첨삭 피드백 모음집*에서 동일하게 제시한 후킹의 3원형.

1. **독자의 기존 생각을 파괴한다**: "전문가한테 물어보면 돈 못 번다" / "부동산으로 돈 못 번다?" → 자신이 알던 것과 정반대 → 호기심 격차 발생 → 클릭.
2. **독자에게 공포를 준다**: "이거 모르면 평생 가난" → 회피 동기 → 클릭.
3. **구체적 이득을 제시한다**: "10분 만에 끝내는 법" → 시간·금액·결과를 구체화 → 가치 인지 → 클릭.

이 3가지는 매체와 무관하다. 블로그 제목, 스레드 도입부, 릴스 첫 3초, 유튜브 썸네일 모두 공통.

Dream_Grow의 'A가 아니라 B이다' 논증 구조는 #1(생각 파괴)과 같은 작동 원리다. 부모가 알던 A를 무너뜨리고 B를 새로 제시하는 흐름.

다만 #2(공포)는 부모 대상 콘텐츠에서 과용 시 신뢰 훼손. "우리 아이 망친다" 식 표현은 톤 충돌. 강도를 낮춰 "많은 부모님이 놓치고 계신 부분"으로 변형해 사용한다.
