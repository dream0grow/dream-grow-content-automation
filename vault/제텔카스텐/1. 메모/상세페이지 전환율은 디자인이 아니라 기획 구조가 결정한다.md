# 핵심 주장

상세페이지 전환율이 안 나오는 이유는 디자인이 촌스러워서가 아니라 **소비자의 구매 흐름이 끊겨 있기 때문**이다.

## 근거

더마르콩 스튜디오 누적 1000+ 상세페이지 작업 결과, 전환이 떨어지는 페이지의 5가지 공통 패턴은 모두 디자인이 아니라 기획 문제였다:

1. 기획 없이 디자인만 한 페이지
2. 고객 문제보다 제품 기능 설명만 많은 페이지
3. 구조가 복잡해 스크롤 이탈 높은 페이지
4. 신뢰 요소(리뷰·인증·근거) 부족
5. 구매 흐름 설계 자체가 없는 페이지

같은 제품, 같은 가격으로 상세페이지만 리뉴얼했을 때 전년 대비 매출 500% 증가 사례가 발생한다.

## 의미
