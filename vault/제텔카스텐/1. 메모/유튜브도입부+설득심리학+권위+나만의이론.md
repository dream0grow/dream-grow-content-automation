저자 : 
URL : 
인용 : 유튜브 도입부를 문제 사례 유인 3가지 요인 대신 설득의 심리학에 나온 요인들로 바꿔보기 ... 이렇게 되면 내 이론으로 만들어서 유튜브 제작할 수 있게 도울 수 있음.

### 생각의 연결고리
분야 : 크리에이터
키워드 : 유튜브도입부, 설득심리학, 권위, 호응, 나만의이론

핵심 메시지 : 유튜브 도입부 공식을 기성 3요인 대신 설득 심리학 원리(권위·호응 등)로 치환하면 나만의 도입부 이론이 된다.

근거 : 기존 도입부 공식(문제·사례·유인)은 누구나 쓰는 패턴이다. 설득의 심리학에서 권위, 호응, 희소성, 사회적 증거, 일관성, 호혜 등의 원리를 도입부 요인으로 바꾸면 채널 고유의 이론으로 차별화된다.

핵심 메시지 생각을 확장하면? :
- Dream_Grow 채널은 권위(교사 경력)와 사회적 증거(교실 사례)를 도입부 메인 요인으로 설정할 수 있다
- 영상 주제별로 어떤 설득 요인을 메인으로 쓸지 선택하는 도입부 매트릭스를 만들면 기획이 빨라진다

관련된 내 의견 : 
- 교사라는 권위와 교실 사례라는 사회적 증거가 Dream_Grow의 가장 강한 도입부 요인이다
