원제: 1. 일상생활에서 느끼는 부담감
변환일: 2026-05-06
단계: 1단계 메모
---

1. 일상생활에서 느끼는 부담감
[ ] 시험이나 과제 제출 때문에 마음이 무거워요 
[ ] 학원 시간표를 맞추느라 힘들어요
[ ] 부모님이나 선생님의 기대가 부담스러워요
[ ] 매일매일 할 일이 너무 많아서 피곤해요


2. 건강과 관련된 걱정
[ ] 내 키나 몸무게 때문에 스트레스에요.
[ ] 운동할 때 잘 못할까봐 걱정돼요 
[ ] 아프거나 다친 적이 있어서 걱정이에요 
[ ] 가족 중에 아픈 분이 계셔서 마음이 불편해요


3. 친구관계에서 느끼는 어려움
[ ] 친구들이 내 말을 잘 안 들어주는 것 같아요 
[ ] 친구들의 모임에 끼지 못한 적이 있어요 
[ ] 단짝 친구와 싸웠거나 멀어졌어요 
[ ] 새로운 친구를 사귀기가 어려워요


4. 자신감이 떨어지는 경험
[ ] 발표할 때 실수해서 창피했어요 
[ ] 친구들이 나를 놀리거나 무시한 적이 있어요 
[ ] 내가 못난 것 같다는 생각이 들어요 
[ ] 다른 친구들보다 잘하지 못하는 것 같아요


5. 학업과 관련된 걱정
[ ] 공부가 어려워서 따라가기 힘들어요
[ ] 열심히 해도 성적이 오르지 않아요 
[ ] 특정 과목이 너무 어려워요 
[ ] 수행평가나 발표가 걱정돼요


6. 친구관계의 변화
[ ] 친한 친구가 전학을 가서 슬퍼요 
[ ] 옛날처럼 친하지 않은 친구가 생겼어요 
[ ] 친구들과 자주 다투게 돼요 
[ ] 반에서 혼자인 것 같은 느낌이 들어요


7. 가정에서의 변화
[ ] 이사나 전학의 걱정이 있어요 
[ ] 부모님이 자주 다투셔서 걱정이에요 
[ ] 가족 관계(이혼, 재혼, 동생 태어남 등) 가 변화되어서 적응이 어려워요 
[ ] 집에서 편하게 쉬기 어려워요


8. 미래에 대한 걱정
[ ] 중학교 가는 게 걱정돼요 
[ ] 나중에 뭐가 되어야 할지 모르겠어요 
[ ] 부모님의 바람과 내 꿈이 달라요 
[ ] 미래에 대한 걱정이 많아요


9. 실수와 실패
[ ] 중요한 시험/대회에서 실수한 적이 있어요 
[ ] 반장/임원 선거에서 떨어졌어요 
[ ] 열심히 준비했는데 잘 안된 적이 있어요 
[ ] 실수한 것이 자꾸 생각나요


10. 큰 변화나 사건
[ ] 학교나 학원의 갑작스러운 변화가 있었어요 
[ ] 가족 상황(이혼, 가족의 죽음 등) 에 큰 변화가 있었어요 
[ ] 예상치 못한 일(나 또는 가족의 병, 사고 등) 이 생겨서 당황스러워요 
[ ] 주변 환경(이사, 전학, 학원 변경 등)이 많이 바뀌어서 적응이 어려워요
