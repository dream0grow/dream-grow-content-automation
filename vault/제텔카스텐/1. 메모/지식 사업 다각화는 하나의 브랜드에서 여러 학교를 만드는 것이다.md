지식 사업을 다각화하려면 하나의 브랜드(예: 붐랩) 아래 여러 '학교'를 만드는 방식이 효과적이다. 해빗스쿨(습관), 페런츠스쿨(부모교육), 머니스쿨(재테크), 소울스쿨(영적 성장), 커뮤스쿨(커뮤니케이션)처럼 주제별로 나누고, 각 사업마다 부캐(부캐릭터)와 의상을 다르게 설정한다. 이렇게 하면 각 분야에서 독립적인 전문성을 보여줄 수 있으면서도 하나의 브랜드로 시너지를 만들 수 있다.

생각의 확장 :
- 부캐 전략: 해빗스쿨=운동복(해빗츠), 패런츠스쿨=정장(웃음결), 머니스쿨=명품같은 짝퉁(리마), 소울스쿨=스님(영혼21)
- 각 학교는 독립적 콘텐츠 파이프라인을 가질 수 있다
