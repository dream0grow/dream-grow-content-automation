# O - 아이의 학습 실패는 능력의 문제가 아니라 순서의 문제이다

## 근거
수학에서 CPA 순서를 건너뛰면 구구단은 외우지만 문장제를 못 세운다. 독서에서 선택권 없이 읽히면 책이 과제가 된다. 공부에서 감정 인정 없이 "열심히 해"부터 말하면 내적 동기가 꺾인다. 세 영역 모두 동일한 패턴을 보인다. 아이가 실패하는 지점에는 항상 '건너뛴 단계'가 있다.

구체물 -> 그림 -> 추상(수학), 선택 -> 반복 -> 습관(독서), 감정 인정 -> 경계 설정 -> 대안 제시(훈육). 이 순서를 지키면 아이는 스스로 해낸다. 순서를 무시하고 결과만 요구하면 아이는 무너진다.

## 반론과 한계
모든 학습 실패를 '순서의 문제'로 환원하면, 학습 장애나 발달 차이 같은 개인 변수를 간과할 수 있다. 또한 순서를 지키더라도 충분한 시간과 반복이 뒷받침되지 않으면 효과가 제한적이다. 순서는 필요조건이지 충분조건은 아니다.
