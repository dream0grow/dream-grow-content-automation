# 노트 메타데이터 스키마 (v2.1)

모든 신규 노트의 frontmatter 표준. 자동 파이프라인도 이 스키마로 쓴다.

```yaml
---
title: 명제형 제목 (한 문장)
author: 이한결 | 이한결(구술) | AI
source_type: experience | own_script | book | paper | youtube | own_content
source_author:            # 지식의 원천인 사람 (책 저자, 크리에이터 등)
출처: plaud:<file_id>      # 또는 raw/ 경로. 중복 처리 방지 로그 겸용
recorded: 2026-07-06      # 녹음/원본 날짜
created: 2026-07-06
verbatim: true            # 원문 발화를 그대로 옮긴 경우
links: []
---
```

핵심 구분: **author = 문장을 쓴 사람 / source_author = 지식의 원천인 사람.**

- 플라우드 녹음 발췌 → `author: 이한결(구술)`, `verbatim: true`
- AI가 구조화·요약한 노트 → 파일명 접두사 `_ai` + `author: AI`
- `_ai` 노트는 4단계(주장)·5단계(글감)에 인용 금지 — 승격 의식을 거친 것만 벽돌이 된다
