생각의 확장 :
- 한교육은 홍익인간 이념을 바탕으로 관계성을 키우는 교육이며, 핵심 역량은 자기 사랑, 문제해결, 공감적 대화이다. 그러나 철학만으로는 확산이 어렵다. IB교육처럼 미션-비전-핵심역량-기술의 구조를 갖추면 복제 가능한 교육 시스템이 된다.
- 개념기반탐구수업(사실 -> 개념 -> 원리 -> 전이)은 한교육의 철학을 수업 레벨에서 구현하는 구체적 방법론이다. 학생이 사실에서 개념을 추출하고 전이하는 과정 자체가 비판적 사고와 창의성을 훈련하는 것이다.
- 교사 성장의 5단계(기술 숙련 -> 관점 전환 -> 실행 -> 존재 -> 예술)와 감수성 훈련이 교사 양성 프로그램의 핵심이 되고, 퍼실리테이션과 학급회의가 학급 운영의 기본 구조가 되면, 한교육 모델 학교의 구체적 모습이 그려진다.
