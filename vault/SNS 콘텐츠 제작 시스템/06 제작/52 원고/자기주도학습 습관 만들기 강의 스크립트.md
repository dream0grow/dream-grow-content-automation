# [[C_자기주도학습|자기주도학습]] 습관 만들기 강의 스크립트

## 1강: [[C_성장|성장]]의 시작 - 목표 설정과 감정의 균형 잡기

### 핵심 내용
1. **목표란?** 내가 어떻게 변했으면 하는 기대, 이상향
2. **현실과의 거리**: 이 간극이 도전 의식을 만들고 성장시키는 원동력
3. **목표 달성 시**: 스스로 축하해야 함 → 다음 목표의 원동력
4. **목표 미달성 시**: 더 중요한 순간 → 성찰의 기회

### 성찰하는 방법
1. 기대(목표)를 낮출 것인가?
2. 어제보다 나은 오늘의 현실을 찾아볼 것인가?
- 주의: 자기 합리화에 빠지지 않도록 주의

### 영상 원고 포함
- 오프닝부터 맺음말까지 전체 스크립트 완성됨
- 초등학생부터 성인까지 대상
