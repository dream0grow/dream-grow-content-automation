# 왜 먹히는가
- 복잡한 문제를 한 가지 행동으로 줄여준다
- 지금 당장 해볼 수 있을 것 같은 느낌을 준다
- 선택 피로를 줄여준다

# 어떤 타겟에게 먹히는가
- 시간이 부족한 사람
- 빠른 실행법을 원하는 사람
- 너무 많은 정보를 싫어하는 사람

# 내 타겟에 맞춘 변형
- 초등 저학년이라면 이것 1가지만
- 성적보다 먼저 이것 1가지만
- 부모라면 먼저 이것 1가지만
