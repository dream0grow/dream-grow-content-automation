# 스레드 본문 후킹 구조 분석

## 패턴 1: 권위 → 충격 → 공감 → 솔루션 예고 → 결핍 유도
- 예시: "초등교사입니다. 초등 수학 공부, 문제지만 풀려서는 안 됩니다."
- 핵심: 권위("초등교사")로 신뢰 → 역발상으로 궁금증 → "큰 그림 로드맵을 주겠다"로 결핍 유도

## 패턴 2: 권위 → 충격 → 공감 → 궁금증 → 반전 → 과학적 근거 → 경각심
- 예시: "18년차 초등교사입니다. 제발, 아이를 좀 심심하게 키우세요."
- 핵심: 상식 뒤집기("심심하게 키워라") → 진짜 핵심("도파민 과잉으로 뇌 보상회로 망가짐")

## 패턴 3: 자기 서사형 (고난→전환→[[C_성장|성장]]→솔루션)
- 예시: 감사일기 700일 + 두 번의 이혼 통보 → "감사일기는 나를 회복하지만, 관계는 '감사 표현'으로 회복된다"
- 핵심: 진정성 있는 자기 이야기로 공감 → 구체적 전환점 제시

## 사람들이 끝까지 보게 되는 공통 요소
1. 권위감 (신뢰 확보)
2. 역발상 후킹 (충격)
3. 현실적 장면 (공감/몰입)
4. 긴장 조성 (궁금증)
5. 과학적/경험적 근거 (전문성)
6. 감정 자극 마무리 (경각심/행동 유도)
