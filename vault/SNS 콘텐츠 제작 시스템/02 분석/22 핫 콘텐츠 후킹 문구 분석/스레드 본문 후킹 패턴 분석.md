# 스레드 본문 후킹 패턴 분석

참고: https://www.instagram.com/min_is__/

## 패턴 1: 권위 - 충격 - 공감 - 솔루션 예고 - 결핍 유도
- 예시: "초등교사입니다. 초등 수학 공부, 문제지만 풀려서는 안 됩니다."
- 분석: 권위("초등교사") → 역발상("문제집 풀면 안 된다") → 두려움(수포자) → 대안(로드맵) → 결핍(다음 글 유도)

## 패턴 2: 권위 - 충격 - 공감 - 궁금증 - 반전 - 과학적 근거 - 경각심
- 예시: "18년차 초등교사입니다. 제발, 아이를 좀 심심하게 키우세요."
- 분석: 상식 뒤집기("심심하게 키워라") → 현실적 장면("딸깍 클릭") → 반전("유해 콘텐츠가 핵심이 아니라 도파민 과잉") → 뇌과학 근거

## 패턴 3: 자기 이야기 - 문제 직면 - 깨달음 - 해결 - 증명
- 예시: 감사일기 700일 → 이혼 통보 → "감사일기는 나를 회복하지만 관계는 감사 표현으로 회복된다"
- 구조: 실패 경험 → 핵심 깨달음 → 행동 변화 → 결과 증명

## 핵심 요소 정리
1. 권위감 (신뢰 확보)
2. 역발상 충격 (상식 뒤집기)
3. 현실 공감 (구체적 장면)
4. 과학적 근거 (전문성)
5. 결핍/경각심 (행동 유도)
