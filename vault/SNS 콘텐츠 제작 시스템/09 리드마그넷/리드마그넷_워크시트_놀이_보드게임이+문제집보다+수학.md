# 학년별 수학 보드게임 추천 가이드

**대상:** 초등 1~6학년 자녀를 둔 부모 | **유형:** 워크시트 (아이와 함께 작성)

---

## [1] 우리 아이 수학 감정 먼저 확인하기

아래 질문을 아이에게 읽어주고, 아이가 고른 답에 동그라미를 쳐주세요.

| 질문 | 그렇다 | 보통 | 아니다 |
|------|--------|------|--------|
| 수학 시간이 기다려진다 | 3점 | 2점 | 1점 |
| 어려운 문제를 보면 일단 시도한다 | 3점 | 2점 | 1점 |
| 수학 문제를 틀려도 괜찮다고 느낀다 | 3점 | 2점 | 1점 |
| 숫자를 가지고 노는 게 재밌다 | 3점 | 2점 | 1점 |
| 내가 수학을 잘할 수 있다고 생각한다 | 3점 | 2점 | 1점 |

**합산 점수:** ______ / 15점

- 12~15점: 수학 자신감이 높습니다. 보드게임으로 사고력을 확장해 주세요.
- 8~11점: 보통 수준입니다. 보드게임으로 "수학이 재밌다"는 경험을 쌓아주세요.
- 5~7점: 수학에 대한 부정적 감정이 있습니다. 쉬운 게임부터 시작해 성공 경험을 만들어 주세요.

## [2] 학년별 추천 보드게임

아이 학년에 해당하는 칸을 확인하고, 집에 있거나 해볼 게임에 표시하세요.

### 저학년 (1~2학년)

| [ ] | 게임 이름 | 연결되는 수학 영역 | 한 줄 설명 |
|-----|----------|------------------|-----------|
| | 할리갈리 | 수 세기, 덧셈 | 같은 과일 5개를 빠르게 세어 종을 치는 게임 |
| | 루미큐브 (주니어) | 수 배열, 규칙 찾기 | 숫자 타일을 규칙에 맞게 나열하는 게임 |
| | 다빈치 코드 | 수의 크기 비교, 추론 | 상대 숫자를 추리하며 논리력을 키우는 게임 |
| | 도블 | 관찰, 분류 | 같은 그림을 빠르게 찾는 패턴 인식 게임 |

### 중학년 (3~4학년)

| [ ] | 게임 이름 | 연결되는 수학 영역 | 한 줄 설명 |
|-----|----------|------------------|-----------|
| | 루미큐브 (정식) | 덧셈, 뺄셈, 조합 | 숫자 조합을 바꿔가며 전략을 세우는 게임 |
| | 블로커스 | 공간 감각, 도형 | 다양한 모양의 블록을 많이 놓는 전략 게임 |
| | 스플렌더 | 계산, 자원 관리 | 보석을 모아 점수를 계산하는 경제 전략 게임 |
| | 셋 (SET) | 분류, 패턴 인식 | 색, 모양, 수, 무늬 4가지 속성을 조합하는 게임 |

### 고학년 (5~6학년)

| [ ] | 게임 이름 | 연결되는 수학 영역 | 한 줄 설명 |
|-----|----------|------------------|-----------|

| | 카탄 | 확률, 자원 교환, 비율 | 주사위 확률을 계산하며 자원을 교환하는 게임 |
| | 블로커스 트라이곤 | 도형, 넓이, 공간 추론 | 삼각형 기반 블록으로 난이도를 높인 버전 |
| | 프라임 클라임 | 소수, 약수, 배수 | 소수를 활용해 점수를 얻는 수학 특화 게임 |

## [3] 게임 후 던지는 수학적 사고 질문

게임이 끝난 뒤, 아래 질문 중 1~2개만 골라서 가볍게 물어보세요. 정답을 요구하는 것이 아니라, 아이가 자기 생각을 말로 표현하게 하는 것이 목적입니다.

| 번호 | 질문 | 키우는 능력 |
|------|------|-----------|
| 1 | "어떻게 그 수를 찾았어?" | 수학적 추론 |
| 2 | "다른 방법도 있었을까?" | 다양한 풀이 탐색 |
| 3 | "이번에는 어떤 작전을 썼어?" | 전략적 사고 |
| 4 | "그때 왜 그 타일을 골랐어?" | 의사결정 설명 |
| 5 | "다음에 하면 뭘 다르게 해볼 거야?" | 메타인지 |
| 6 | "그 규칙은 왜 그렇게 정해졌을까?" | 규칙 이해 |
| 7 | "몇 점 차이로 이겼어(졌어)?" | 뺄셈, 비교 |
| 8 | "이기려면 최소 몇 점이 더 필요했을까?" | 역산, 계획 |
| 9 | "이 게임에서 수학이 어디에 숨어 있었어?" | 수학적 연결 |
| 10 | "친구한테 이 게임을 설명한다면 뭐라고 할 거야?" | 수학 언어화 |

주의: "정답이 뭐야?" "왜 틀렸어?"와 같은 질문은 피해 주세요. 게임 후 대화의 목적은 평가가 아니라 사고의 언어화입니다.

## 기억해 주세요

보드게임이 수학 자신감을 키우는 이유는 단순합니다. 문제집은 틀리면 빨간 줄이 남지만, 보드게임은 틀려도 다음 판이 있습니다. "다시 해보자"라는 마음이 곧 성장 마인드셋의 시작입니다.

교실에서 수학을 싫어하던 아이가 루미큐브를 시작한 뒤 수학 시간에 처음으로 손을 들었습니다. 변화는 문제를 더 많이 푸는 데서 오지 않았습니다. 수학이 즐겁다는 경험에서 왔습니다.

주 2회, 30분이면 충분합니다. 그리고 절대 일부러 져주지 마세요. 진짜 이겼을 때의 성취감이 수학 자신감의 씨앗이 됩니다.
