# 우리집 놀이 규칙 만들기 워크시트

**대상: 5~12세 자녀를 둔 부모**
**활용 시간: 부모-자녀 함께 약 20분**

---

## 시작하기 전에

규칙을 잘 지키는 아이는 규칙을 많이 들은 아이가 아니라, 규칙을 직접 만들어 본 아이입니다.

비고츠키(L. Vygotsky)는 어른이 아이의 발달을 돕는 방식을 '비계(scaffolding)'에 비유했습니다. 건물을 지을 때 비계를 세웠다가 건물이 완성되면 거두는 것처럼, 부모는 규칙의 틀을 잡아 주되 아이가 스스로 설 수 있게 점차 손을 떼야 합니다.

이 워크시트는 보드게임, 몸놀이, 디지털 게임 등 가정에서 자주 하는 놀이에 적용할 수 있도록 설계되었습니다. 오늘 저녁 놀이 시간 전에 아이와 함께 한 장씩 채워 보세요.

---

## 1장. 우리 가족 놀이 약속 만들기

### 활동 1. 어떤 놀이를 함께 할까요?

아이와 마주 앉아, 함께 하고 싶은 놀이를 적어 봅니다. 부모가 먼저 정하지 말고, 아이가 먼저 말하게 합니다.

| 놀이 이름 | 누가 함께 할까요? | 언제 할까요? |
|-----------|------------------|--------------|
|           |                  |              |
|           |                  |              |
|           |                  |              |

### 활동 2. 규칙을 만들어 봅시다

아이가 먼저 한 줄 쓰고, 부모가 한 줄 씁니다. 번갈아 가며 채웁니다.
**중요: 부모가 정한 규칙도 부모가 지켜야 합니다.**

| 번호 | 규칙 | 누가 정했나요? (아이 / 부모) |
|------|------|------------------------------|
| 1    |      |                              |
| 2    |      |                              |
| 3    |      |                              |
| 4    |      |                              |
| 5    |      |                              |

### 활동 3. 규칙을 어겼을 때는 어떻게 할까요?

벌이 아니라 '복구'를 정합니다. 규칙을 깬 사람이 무엇을 하면 다시 놀이로 돌아올 수 있는지 함께 정해 봅니다.

규칙을 어긴 사람은: ______________________________________________________

다시 놀이로 돌아오기 위해서는: ____________________________________________

## 3장. 연령별 규칙 협상 대화 스크립트

아이의 발달 단계에 따라 규칙을 만드는 방식이 달라야 합니다. 부모가 곧바로 따라 말할 수 있게 정리했습니다.

### 5~7세 (선택지를 좁혀 주는 단계)

> 부모: "오늘 블록 놀이는 몇 시까지 할까? 30분이 좋을까, 1시간이 좋을까?"
> 아이: "한 시간."
> 부모: "좋아. 그럼 한 시간이 끝나면 어떻게 알려 줄까? 알람을 쓸까, 엄마가 말해 줄까?"

이 시기에는 무한한 선택권보다 두세 개의 선택지가 더 잘 작동합니다.

### 8~10세 (스스로 제안하게 하는 단계)

> 부모: "이번 보드게임에서 우리만의 규칙을 하나 추가해 보자. 너는 어떤 규칙이 있으면 더 재밌을 것 같아?"
> 아이: "한 번 진 사람이 다음 판 카드 섞기."
> 부모: "그 규칙, 엄마도 지킬게. 나도 지면 카드 섞을게."

부모가 규칙에 함께 묶이는 것을 분명히 말로 표현해 주는 단계입니다.

### 11~12세 (규칙 자체를 협상하는 단계)

> 부모: "이 규칙이 너무 불공평하다고 생각해? 그럼 어떻게 바꿔야 공평할까? 네 의견을 듣고 싶어."
> 아이: "동생은 어리니까 한 칸 더 가게 해 주자."
> 부모: "그럼 그 규칙은 언제까지 적용할까? 동생이 몇 살이 되면 바꿀까?"

이 시기에는 '공정함'에 대한 감각이 자라기 때문에, 규칙의 근거까지 함께 따져 보는 대화가 가능합니다.

## 4장. 일주일 점검표

규칙을 만든 다음 날부터 일주일 동안, 매일 잠자기 전 1분씩 함께 체크합니다. 부모가 혼자 평가하지 말고, **아이에게 먼저 묻고 아이가 표시하게 합니다.**

| 요일 | 오늘 우리가 만든 규칙을 지켰나요? | 어려웠던 순간이 있었나요? | 내일 바꾸고 싶은 규칙이 있나요? |
|------|----------------------------------|--------------------------|-------------------------------|
| 월   |  [ ] 네  [ ] 아니오               |                          |                               |
| 화   |  [ ] 네  [ ] 아니오               |                          |                               |
| 수   |  [ ] 네  [ ] 아니오               |                          |                               |
| 목   |  [ ] 네  [ ] 아니오               |                          |                               |
| 금   |  [ ] 네  [ ] 아니오               |                          |                               |
| 토   |  [ ] 네  [ ] 아니오               |                          |                               |
| 일   |  [ ] 네  [ ] 아니오               |                          |                               |

일주일이 끝나면 다시 마주 앉아, 규칙을 수정하거나 새로 만듭니다. 규칙은 한 번 정해서 끝나는 것이 아니라, 함께 다듬어 가는 것입니다.

## 부모용 가이드 (예시 답안)

워크시트를 처음 활용할 때 막막할 수 있어, 한 가족의 작성 예시를 함께 싣습니다.

**활동 2 예시 (8세 아이와 작성)**

| 번호 | 규칙 | 누가 정했나요? |
|------|------|----------------|
| 1 | 게임 시작 전에 모두 핸드폰 멀리 두기 | 아이 |
| 2 | 진 사람한테 놀리지 않기 | 아이 |
| 3 | 한 판 끝나면 정리는 다 같이 | 부모 |
| 4 | 30분이 지나면 한 번 쉬기 | 부모 |
| 5 | 규칙 바꾸고 싶을 때는 손을 들기 | 아이 |

**활동 3 예시**

규칙을 어긴 사람은: 게임을 잠깐 멈추고, 무엇을 어겼는지 한 줄로 말한다.
다시 놀이로 돌아오기 위해서는: 다음 판에서 자기 차례를 한 번 양보한다.

처벌이 아닌 '복구'로 설계할 때, 아이는 규칙을 두려움이 아니라 약속으로 인식하게 됩니다.

## 자주 묻는 질문

**Q. 아이가 자기에게 너무 유리한 규칙만 제안해요.**
부모가 거부하지 마시고, "그 규칙으로 한 판 해보고, 한 판 끝나면 다시 의논하자"라고 말해 보세요. 직접 해 보면 아이도 불공평함을 스스로 깨닫는 경우가 많습니다. 깨닫지 못하면, 부모가 그 규칙을 똑같이 적용해 주십시오. 규칙이 약자에게만 적용되는 것이 아니라는 점을 몸으로 배웁니다.

**Q. 규칙을 만들었는데 다음 날 까먹어요.**
규칙을 적은 종이를 놀이 도구 옆에 함께 두십시오. 매번 놀이를 시작할 때 아이가 규칙을 한 번 소리 내어 읽게 하면 좋습니다. 부모가 읽어 주는 것과 아이가 읽는 것은 효과가 다릅니다.

**Q. 형제자매가 함께 있을 때는 어떻게 하나요?**
나이가 다른 형제자매가 있으면, 큰아이의 의견에 더 많은 비중이 실리기 쉽습니다. 규칙을 정할 때 작은아이가 먼저 말하게 하고, 큰아이가 그 위에 의견을 더하는 순서로 진행해 보세요.
