# 초등 분수 개념지도

## 분수가 무너지면 중학수학 어디까지 무너지는가

초등 1학년부터 중학 3학년까지, 분수 한 개념이 어디로 뻗어가는지 한눈에 보는 자료입니다.

---

## 1. 분수의 본질 (가장 먼저 잡아야 할 뿌리)

```
                  [분수의 본질]
                       |
        "전체를 똑같이 나눈 것 중의 일부"
                       |
        ┌──────────────┼──────────────┐
        |              |              |
   똑같이 나누기   전체와 부분    부분의 개수
   (등분의 개념)   (단위분수)     (분자의 의미)
```

- 이 세 가지가 머릿속에 그림으로 들어와야 분수입니다.
- 계산 공식을 먼저 가르치면, 아이는 분수를 '숫자 쓰는 규칙'으로만 외웁니다.
- 브루너의 학습 순서: **직접 해보기 → 그림 → 기호**. 이 순서를 거꾸로 가면 무너집니다.

---

## 2. 학년별 분수 학습 흐름

```
[2학년]                [3학년]               [4학년]
똑같이 나누기    →    분수의 도입       →    분수의 종류
"반으로"        →    1/2, 1/3, 1/4     →    진분수, 가분수, 대분수
"넷으로"             단위분수 개념            분모가 같은 덧셈/뺄셈
   |                     |                       |
   ▼                     ▼                       ▼
[5학년]                [6학년]               [중학교]
약수와 배수      →    분수의 곱셈       →    유리수
통분, 약분            분수의 나눗셈           일차방정식
이분모 덧셈/뺄셈      소수와 분수 변환        함수의 기울기
                                              비례식, 닮음비
```

- 학년이 올라갈수록 **'본질 그림'을 기억하는 아이만** 따라갑니다.
- 5학년 통분, 6학년 분수의 나눗셈에서 가장 많이 무너집니다.

---

## 3. 분수 개념의 가지치기 (한 뿌리에서 뻗어나가는 모든 단원)

```
                    [분수의 본질]
                          │
        ┌─────────────────┼─────────────────┐
        │                 │                 │
   [연산 가지]        [표현 가지]       [관계 가지]
        │                 │                 │
   ┌────┼────┐       ┌────┼────┐       ┌────┼────┐
   덧셈  곱셈  나눗셈  소수  백분율  비   비례  비율  닮음
   (4학) (5학) (6학) (4학) (6학) (6학) (6학) (6학) (중2)
        │                 │                 │
        ▼                 ▼                 ▼
   [중1 정수와         [중1 유리수        [중2 일차함수
    유리수의           의 소수 표현]      기울기, 변화율]
    사칙연산]
                            │
                            ▼
                       [중3 제곱근,
                        무리수까지]
```

- 분수는 '하나의 단원'이 아니라 **중학수학 전체의 뿌리**입니다.
- 함수의 기울기(y의 증가량 / x의 증가량)도 결국 분수입니다.
- 닮음비, 확률, 삼각비 모두 분수의 사고가 깔려 있습니다.

---

## 4. 선수학습 ↔ 후속학습 관계도

| 무너지면 | 직접 무너지는 단원 | 결국 무너지는 단원 |
|----------|------------------|------------------|
| 똑같이 나누기 (2학년) | 단위분수 (3학년) | 통분 (5학년) |
| 단위분수 (3학년) | 진분수/가분수 (4학년) | 분수의 곱셈 (5학년) |
| 약수와 배수 (5학년) | 약분, 통분 (5학년) | 분수의 나눗셈 (6학년) |
| 분수의 나눗셈 (6학년) | 비와 비율 (6학년) | 일차방정식 (중1) |
| 비례식 (6학년) | 일차함수 (중2) | 닮음비, 삼각비 (중2~중3) |

- 한 칸이 비면, 그다음 모든 칸이 무너집니다.
- 아이가 막힌 단원의 **두 학년 앞**으로 돌아가 다시 잡아야 합니다.

---

## 5. '분수가 잡혔다'의 기준

아이가 다음 세 가지를 **그림과 말로 설명할 수 있어야** 분수가 잡힌 것입니다.

```
[기준 1] "왜 1/2이 1/3보다 커?"
         └─ 같은 크기의 빵을 2조각 낸 것과 3조각 낸 것 비교

[기준 2] "1/2 + 1/3은 왜 2/5가 아니야?"
         └─ 단위가 다르면 그대로 더할 수 없다는 것

[기준 3] "1/2 ÷ 1/4는 왜 2야?"
         └─ "1/2 안에 1/4이 몇 개 들어가?"로 설명
```

- 계산만 맞히는 것은 분수를 아는 게 아닙니다.
- 그림으로 설명하지 못하면, 그 분수는 **아직 아이의 것이 아닙니다**.

---

## 6. 가정에서 잡는 분수 (단계별)

```
[1단계] 구체물로 나누기
   피자, 김밥, 과일, 종이 → 손으로 직접 나눠보기
        ↓
[2단계] 그림으로 표현하기
   동그라미, 네모, 막대 → 종이에 그려서 색칠하기
        ↓
[3단계] 말로 설명하기
   "이건 전체를 4등분한 것 중의 1조각"
        ↓
[4단계] 기호로 옮기기
   비로소 1/4 이라는 분수 기호 도입
        ↓
[5단계] 비교하고 연산하기
   다른 분수와 크기 비교, 덧셈/뺄셈
```

- 많은 가정이 1, 2단계를 건너뛰고 4단계부터 시작합니다.
- 그래서 아이는 분수를 '암호'로 외우게 됩니다.

---

## 7. 학년별 가정 점검 체크포인트

| 학년 | 부모가 물어볼 질문 |
|------|------------------|
| 2~3학년 | "이 사과를 너랑 동생이랑 똑같이 나누려면 어떻게 할까?" |
| 3~4학년 | "1/4은 전체를 몇 등분한 것 중의 몇 조각이야?" |
| 4~5학년 | "1/2이랑 2/4는 같아? 왜?" |
| 5~6학년 | "1/2 + 1/3을 그림으로 그려서 설명해줄래?" |
| 6학년~ | "1/2 ÷ 1/4가 왜 2가 되는지 그림으로 보여줄래?" |

- 학년이 올라갈수록 **말과 그림으로 설명하는 능력**을 점검합니다.
- 답을 맞히는 것보다, 설명할 수 있는지가 중요합니다.

---

## 8. 핵심 메시지

분수는 빠른 계산이 아니라, **나눠본 경험과 그림으로 설명하는 능력**으로 잡힙니다.

중학수학이 무너지는 아이는 머리가 나쁜 게 아니라, 초등 3~4학년 때 **'직접 해보기' 단계를 건너뛴 아이**입니다.

지금 아이가 5학년, 6학년이어도 늦지 않습니다. 두 학년 전으로 돌아가, 종이 한 장과 가위만 있으면 됩니다.
