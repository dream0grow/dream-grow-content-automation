# 오늘부터 시작하는 스마트폰 사용 규칙 설계 가이드

**대상:** 유튜브/스마트폰에 빠진 초등학생 자녀를 둔 부모

---

## 왜 빼앗으면 안 되는가

스마트폰을 빼앗으면 문제가 해결될 것 같지만, 실제로는 더 강한 집착을 만듭니다. 교실에서 보면 부모가 강하게 통제하는 아이일수록 친구 폰을 빌려 보거나, 몰래 보는 기술이 늘어납니다.

문제의 본질은 스마트폰이 아니라 **도파민 순환 구조**입니다. 짧은 영상은 즉각적인 보상을 주는데, 초등학생의 전두엽은 아직 충동 조절 기능이 미성숙한 상태입니다. 그래서 금지가 아니라, 아이 스스로 조절하는 힘을 길러주는 방향이 맞습니다.

핵심 원칙은 하나입니다. **부모가 정한 규칙은 통제이고, 아이가 정한 규칙은 연습입니다.**

---

## Step 1. 아이의 미디어 세계를 먼저 이해한다

**무엇을:** 아이가 유튜브에서 무엇을, 왜 보는지 파악합니다.

**어떻게:** 비난 없이 대화를 시작합니다. 아래 질문을 자연스럽게 던져보세요.

- "요즘 유튜브에서 뭐가 제일 재밌어?"
- "그 채널에서 뭘 배우는 것 같아?"
- "친구들은 어떤 거 봐?"

**왜:** 아이가 무엇에 끌리는지 알아야 대체 활동을 설계할 수 있습니다. 또한 아이는 자기 관심사를 존중받았다고 느낄 때 대화에 열립니다.

**주의점:** "또 그런 거 봐?"와 같은 평가는 금물입니다. 아이가 입을 닫으면 다음 단계로 넘어갈 수 없습니다.

---

## Step 2. 아이와 함께 시청 규칙을 정한다

**무엇을:** 하루 시청 시간과 시청 가능한 시간대를 아이가 직접 정하게 합니다.

**어떻게:** 아래 대화 흐름을 참고하세요.

> 부모: "스마트폰 보는 시간을 네가 정해볼래? 하루에 몇 분이 적당할 것 같아?"
>
> 아이: (시간을 말한다)
>
> 부모: "좋아. 그러면 언제 보는 게 좋을까? 숙제 끝나고? 저녁 먹고?"
>
> 아이: (시간대를 말한다)
>
> 부모: "그거 종이에 적어서 냉장고에 붙여놓자. 네가 정한 거니까 네가 지키는 거야."

**왜:** 아이가 직접 정한 규칙은 자기 결정으로 인식합니다. 자기 결정은 위반했을 때 스스로 책임감을 느끼게 합니다. 교실에서도 학급 규칙을 아이들이 직접 정하면 지키는 비율이 눈에 띄게 올라갑니다.

**주의점:** 아이가 비현실적인 시간(예: 하루 5시간)을 제시하면 "왜 그만큼 필요한지" 물어보고, 협상합니다. 일방적으로 깎으면 다시 통제가 됩니다.

---

## Step 3. 규칙표를 만들어 눈에 보이는 곳에 붙인다

**무엇을:** 아이와 합의한 규칙을 종이에 적어 냉장고나 책상 앞에 붙입니다.

**어떻게:** 아래 템플릿을 인쇄하거나 직접 따라 그려서 작성하세요.

```
┌─────────────────────────────────────────────┐
│         우리 집 스마트폰 규칙표              │
│                                             │
│  이름: _______________                      │
│                                             │
│  하루 시청 시간: ______분                    │
│                                             │
│  시청 가능 시간대: ___시 ~ ___시             │
│                                             │
│  시청 가능 장소: _______________             │
│  (예: 거실 소파 / 식탁)                      │
│                                             │
│  잠자기 전 스마트폰 반납 시간: ___시          │
│                                             │
│  디지털 쉬는 날: 매주 ___요일                │
│                                             │
│  정한 사람: _____________ (아이 서명)        │
│  함께한 사람: _____________ (부모 서명)       │
│  만든 날: ____년 ____월 ____일               │
│                                             │
└─────────────────────────────────────────────┘
```

**왜:** 문서화는 약속의 무게를 만듭니다. 머릿속 약속은 흐려지지만, 눈에 보이는 약속은 기준이 됩니다. 서명을 넣는 이유는 아이에게 계약의 경험을 주기 위해서입니다.

**주의점:** 규칙표는 벌칙 목록이 아닙니다. "안 지키면 일주일 금지" 같은 벌칙을 적으면 규칙표 자체가 두려움의 대상이 됩니다. 규칙표는 약속이지, 처벌 도구가 아닙니다.

---

## Step 4. 대체 활동을 설계한다

**무엇을:** 스마트폰이 채우던 시간을 다른 활동으로 채웁니다. 특히 일주일에 하루, "디지털 쉬는 날"을 만들어 보세요.

**어떻게:** 아래 목록에서 아이와 함께 3~5개를 골라 주간 계획에 넣습니다.

| 활동 유형 | 구체적 활동 예시 |
|-----------|-----------------|
| 만들기 | 레고, 블록, 종이접기, 그림 그리기 |
| 몸 움직이기 | 공원 산책, 자전거, 줄넘기, 배드민턴 |
| 함께하기 | 보드게임, 카드게임, 요리, 베이킹 |
| 읽기 | 만화책, 그림책, 학습만화, 도감 |
| 관찰하기 | 식물 키우기, 곤충 관찰, 날씨 기록 |

**왜:** "스마트폰 하지 마"만으로는 빈 시간이 생길 뿐입니다. 지루함을 견디는 힘은 중요하지만, 처음부터 아무것도 없이 견디라고 하면 실패합니다. 대체 활동은 지루함에서 재미를 발견하는 다리 역할을 합니다.

**주의점:** 대체 활동도 아이가 고르게 하세요. 부모가 "오늘은 독서"라고 정하면, 아이에게는 스마트폰 금지 + 독서 강제라는 이중 통제가 됩니다.

---

## Step 5. 2주 단위로 점검하고 조정한다

**무엇을:** 규칙이 현실적인지 아이와 함께 돌아봅니다.

**어떻게:** 2주에 한 번, 아래 세 가지를 물어보세요.

1. "규칙 지키기 어려웠던 적 있어? 언제?"
2. "시간을 좀 더 늘리거나 줄이고 싶은 부분 있어?"
3. "디지털 쉬는 날에 뭐 하면 재미있었어?"

잘 지켰다면 구체적으로 인정해 주세요. "네가 정한 시간 잘 지켰구나. 스스로 멈추는 연습이 된 거야."

**왜:** 규칙은 한 번 정하고 끝이 아닙니다. 아이의 생활 패턴은 계속 바뀌고, 규칙도 함께 자라야 합니다. 점검 자체가 아이에게 "이건 벌이 아니라 연습"이라는 메시지를 줍니다.

**주의점:** 규칙을 어겼을 때 "그것 봐, 못 지키잖아"라고 하면 아이는 규칙 자체를 포기합니다. "어떤 부분이 어려웠어?"라고 물어야 다음 시도가 가능해집니다.

## 자주 묻는 질문

**Q1. 아이가 규칙을 정하면서 "하루 3시간"이라고 하면 어떡하나요?**

"왜 3시간이 필요한지" 물어보세요. 대부분 구체적인 이유는 없습니다. "그러면 2시간으로 시작해보고, 2주 뒤에 다시 이야기하자"로 협상합니다. 처음부터 완벽한 규칙을 만들 필요 없습니다. 아이가 협상 과정 자체를 경험하는 것이 중요합니다.

**Q2. 규칙을 정해도 몰래 보면 어떡하나요?**

몰래 보는 것 자체를 문제 삼기보다, "왜 몰래 봤는지"를 물어보세요. 규칙이 너무 빡빡했거나, 규칙을 어겼을 때의 반응이 무서웠을 수 있습니다. 몰래 보는 행동은 규칙의 신호이지 아이의 결함이 아닙니다. 규칙을 조정하는 기회로 삼으세요.

**Q3. 형제자매가 있으면 규칙을 통일해야 하나요?**

학년과 발달 수준이 다르므로 개별 규칙이 원칙입니다. 단, "디지털 쉬는 날"처럼 가족 전체가 함께하는 규칙은 통일하는 것이 효과적입니다. 교실에서도 모둠 규칙과 개인 목표를 분리할 때 아이들이 더 잘 따릅니다.
