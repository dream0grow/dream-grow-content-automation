# 초등 수학 사고력 개념지도

## 부모가 알아야 할 '도와주는 법'의 과학적 구조

대상: 초등 1~6학년 자녀를 둔 부모
활용: 아이가 수학 문제 앞에서 막혔을 때, 어디서 멈추고 어디서 도와야 할지 한눈에 파악

---

## 1. 중심 개념: 근접발달영역(ZPD)

소비에트 심리학자 비고츠키(Vygotsky, 1896~1934)가 제시한 개념입니다.
수학 사고력이 자라는 정확한 위치를 알려주는 지도입니다.

```
                    [아이의 수학 능력 3영역]

  ┌─────────────────────────────────────────────┐
  │                                             │
  │   영역 A: 혼자서 할 수 있는 것              │
  │   (이미 학습 완료, 자동화된 계산)           │
  │                                             │
  ├─────────────────────────────────────────────┤
  │                                             │
  │   영역 B: 근접발달영역(ZPD)                 │  ← 사고력이
  │   혼자선 못 하지만,                         │     자라는
  │   적절한 도움이 있으면 해낼 수 있는 것      │     단 하나의
  │                                             │     구간
  ├─────────────────────────────────────────────┤
  │                                             │
  │   영역 C: 아직 할 수 없는 것                │
  │   (선수학습 부족, 도와줘도 불가능)          │
  │                                             │
  └─────────────────────────────────────────────┘
```

핵심: 부모가 식을 대신 써주면 영역 B가 영역 A로 가는 게 아니라, **그냥 사라집니다.**
아이는 영역 A의 계산만 반복하고, B에서 일어나야 할 사고 확장이 멈춥니다.

```
                  [아이가 막힌 순간]
                         │
            ┌────────────┴────────────┐
            │                         │
       [개입 A: 풀이]            [개입 B: 질문]
        식을 대신 써줌            사고를 여는 물음
            │                         │
       단기 효과 ○               단기 효과 △
       장기 효과 ✗               장기 효과 ◎
            │                         │
       "이해한 척"                "스스로 발견"
       비슷한 문제                비슷한 문제
       또 못 풂                   응용 가능
            ↓                         ↓
       영역 B 소멸                영역 B → 영역 A
                                  (사고력 확장)
```

## 3. 학년별 수학 사고 연결 지도

각 학년에서 형성된 사고가 다음 학년의 ZPD를 결정합니다.
한 단계가 비면, 다음 학년에서 아무리 도와줘도 ZPD가 형성되지 않습니다.

```
[1학년]               [2학년]              [3학년]
수 감각              두 자리 수            곱셈 구구
가르기/모으기   →    덧셈/뺄셈      →     나눗셈 도입
                     자릿값 이해          (곱셈의 역연산)
    │                    │                    │
    └────────────────────┴────────────────────┘
                         ↓
[4학년]               [5학년]              [6학년]
분수 개념            약수/배수            비와 비율
혼합 계산       →    분수 통분      →     비례식
나머지 있는          (3학년 나눗셈        분수 나눗셈
나눗셈                기반)               (5학년 약분 기반)
```

### 단계별 막힘이 가르쳐주는 신호

| 막히는 단원 | 실제 부족한 선수학습 | 부모가 던질 질문 |
|---|---|---|
| 2학년 받아올림 | 1학년 가르기/모으기 | "10을 어떻게 만들 수 있어?" |
| 3학년 나눗셈 | 2학년 곱셈 구구 | "곱하면 어떻게 되는지 먼저 볼까?" |
| 4학년 분수 덧셈 | 3학년 분수 개념 | "1을 몇 조각으로 나눈 거야?" |
| 5학년 약분 | 4학년 곱셈/나눗셈 | "둘 다 나눠지는 수가 뭐가 있을까?" |
| 6학년 비례식 | 5학년 분수 곱셈 | "이 둘이 같은 크기라는 게 무슨 뜻일까?" |

## 4. 질문 사다리: 사고를 여는 4단계

풀이를 보여주지 않고, 질문만으로 ZPD를 작동시키는 순서입니다.

```
[1단계] 문제 이해
   "이 문제는 뭘 묻고 있어?"
   "한 줄로 다시 말해볼래?"
              ↓
[2단계] 정보 분류
   "아는 건 뭐고, 모르는 건 뭐야?"
   "필요 없는 정보는 뭐야?"
              ↓
[3단계] 전략 탐색
   "전에 비슷한 문제 풀어본 적 있어?"
   "그림으로 그려볼 수 있을까?"
              ↓
[4단계] 검증
   "왜 이렇게 풀었어?"
   "답이 말이 되는지 어떻게 확인할까?"
```

각 단계 사이에 **5분의 침묵**을 허용하는 것이 핵심입니다.
침묵은 게으름이 아니라, 영역 B에서 사고가 작동하는 시간입니다.

## 5. 교실에서 본 두 아이

같은 3학년, 같은 나눗셈 단원에서 출발했습니다.

**A 아이**: 집에서 부모가 늘 식을 대신 써줌. 학교에서 비슷한 유형은 풀지만, 문장제가 나오면 "이거 더하기예요, 빼기예요?"부터 물음. 5학년 분수 통분에서 완전히 무너짐.

**B 아이**: 집에서 부모가 "왜 그렇게 생각했어?"만 물음. 처음엔 진도가 느렸지만, 5학년부터 한 번도 본 적 없는 유형도 스스로 그림 그려가며 접근함.

차이는 머리가 아니라, **막혔을 때 견디는 힘**이었습니다.

---

## 6. 부모의 자가 점검

지금 우리 집은 어디에 가까운가요?

- [ ] 아이가 막히면 5분 이상 기다린다
- [ ] 풀이보다 질문을 먼저 던진다
- [ ] "왜 그렇게 풀었어?"를 정답 여부와 상관없이 묻는다
- [ ] 틀린 답에 화내지 않고 사고 과정을 묻는다
- [ ] 아이가 스스로 발견할 때까지 답을 말하지 않는다

3개 이상 체크된다면, 아이의 ZPD가 잘 작동하고 있습니다.
