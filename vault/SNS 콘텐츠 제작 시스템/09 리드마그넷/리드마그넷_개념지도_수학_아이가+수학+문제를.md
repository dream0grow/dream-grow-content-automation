## 한눈에 보는 핵심 구조

```
                    [ 부모의 질문 한 단어 ]
                              |
              ┌───────────────┴───────────────┐
              |                               |
         "왜 틀렸어?"                    "어떻게 생각했어?"
        (결과 추궁형)                    (과정 복원형)
              |                               |
        ↓ 작동 회로 ↓                   ↓ 작동 회로 ↓
              |                               |
      [편도체 활성화]                   [전두엽 활성화]
      방어/회피 모드                     관찰/분석 모드
              |                               |
        풀이는 사라지고                   풀이가 말로
        변명이 남는다                     꺼내지며 드러난다
              |                               |
       ↓ 누적 결과 ↓                   ↓ 누적 결과 ↓
              |                               |
       수학 회피 강화                    메타인지 성장
       오답 = 위협                      오답 = 데이터
```

## 학년별 적용 지도: 메타인지 질문이 어디서 작동하는가

```
[1~2학년: 수 감각 형성기]
  덧셈/뺄셈/한 자리 곱셈
        |
  핵심 질문: "어떤 그림이 떠올랐어?"
  ※ 구체물·반구체물 단계. 머릿속 이미지 확인.
        |
        ↓
[3~4학년: 연산 확장기]
  곱셈구구 → 나눗셈 → 분수 도입
        |
  핵심 질문: "이 식을 말로 풀어보면 어떻게 돼?"
  ※ 식과 상황을 연결하는 언어화 훈련.
        |
        ↓
[5학년: 추상화 분기점]      ← 수포자가 시작되는 학년
  약수/배수, 분수의 사칙연산, 비와 비율
        |
  핵심 질문: "어디까지는 이해됐고
              어디서부터 막혔어?"
  ※ 막힌 지점을 스스로 짚는 능력이 결정적.
        |
        ↓
[6학년: 통합 응용기]
  비례식, 분수·소수 혼합 연산, 도형의 부피
        |
  핵심 질문: "비슷한 문제를 본 적 있어?
              어떻게 풀었지?"
  ※ 과거 풀이를 현재로 연결하는 메타인지.
        |
        ↓
[중학교 수학으로 연결]
  문자와 식 → 함수 → 방정식
  ※ 초등에서 메타인지가 형성된 아이만
     "왜 이렇게 풀리지?"를 묻기 시작한다.
```

## 상황별 질문 지도: 언제, 무엇을 물을 것인가

```
                  [ 채점 직후 ]
                        |
        ┌───────────────┼───────────────┐
        |               |               |
    [오답일 때]     [막혔을 때]     [정답일 때]
        |               |               |
        ↓               ↓               ↓
  "이 문제          "여기서 어떤      "이거 어떻게
   한 번             생각이            맞췄어?
   설명해줄래?"      들었어?"          말해줄 수 있어?"
        |               |               |
        ↓               ↓               ↓
  풀이를 말로       막힌 지점을       정답도 우연인지
  꺼내며 빠진       스스로 짚는다     실력인지
  단계가 드러남                       구분된다
        |               |               |
        └───────────────┴───────────────┘
                        |
                  [공통 효과]
              아이가 자기 사고를
              관찰자 시점으로 본다.
              = 메타인지 작동 순간
```

## 절대 피해야 할 질문 회로

```
"왜 틀렸어?"
     ↓
아이 머릿속: "어떻게 답해야 안 혼나지?"
     ↓
[변명 생성 회로 작동]
"몰랐어요" / "실수예요" / "어려워요"
     ↓
풀이 과정은 영구히 사라짐
     ↓
다음에도 같은 오류 반복
     ↓
"왜 또 틀렸어?" 재발
     ↓
[수학 = 혼나는 과목] 학습
     ↓
회피 → 수포자
```

이 회로는 학년이 올라갈수록 강화된다. 5학년 분수 단원에서 무너지는 아이들의 공통점은 4학년까지 "왜 틀렸어?"만 들어온 경험이다.

## 교실에서 관찰한 두 아이

같은 5학년 분수 단원에서 똑같이 틀린 두 아이.

- **A 부모**: "왜 또 틀렸어. 어제도 알려줬잖아."
  → A는 다음 시간 분수 단원에서 손을 들지 않는다. 모르겠다는 말도 줄어든다.

- **B 부모**: "이 문제 풀 때 머릿속에 뭐가 떠올랐어?"
  → B는 "통분을 하긴 했는데 분자만 더했어요"라고 답한다. 자기 오류를 자기가 발견한 순간이다.

차이는 지능이 아니라 부모 질문의 회로다.

## 가정 적용 체크 (오늘부터)

| 상황 | 바꿀 말 | 새로운 말 |
|------|---------|-----------|
| 오답 발견 | "왜 틀렸어" | "이 문제 한 번 설명해줄래" |
| 막혀 있음 | "그것도 몰라" | "어디까지는 됐고 어디서 막혔어" |
| 정답 맞춤 | "잘했네" | "어떻게 그렇게 풀 생각을 했어" |
| 반복 오류 | "또 이거야" | "지난번이랑 어떤 점이 비슷해" |
| 시간 부족 | "빨리 풀어" | "지금 어디쯤 가고 있어" |

## 사용법

1. 이 개념지도를 출력해 책상 옆이나 냉장고에 붙입니다.
2. 채점 전 한 번 보고 호흡을 정리합니다.
3. 한 주에 한 가지 질문만 의식적으로 바꿔봅니다.
4. 일주일 후 아이의 말 길이가 달라지는지 관찰합니다.

질문 한 단어를 바꾸는 것은 작은 일이지만, 6년이 누적되면 아이의 수학 회로 자체가 달라집니다. 아이가 수학을 즐기며 자라길 바랍니다.
