## 2. 연령별 "규칙 만들기" 놀이 목록

아이와 함께 아래 표를 보며, 해보고 싶은 놀이에 동그라미를 치세요. 이번 주에 하나만 골라서 실천합니다.

### 저학년 (1~3학년)

| 놀이 | 방법 | 아이가 만드는 규칙 예시 |
|------|------|----------------------|
| 이불 위 보물찾기 | 방 안에 물건 5개를 숨기고 찾는 놀이 | "찾으면 박수 3번 치기" |
| 동물 술래잡기 | 술래가 동물 이름을 외치면 그 동물처럼 움직여야 함 | "토끼는 두 발로만 뛰기" |
| 카드 탑 쌓기 | 번갈아 카드를 올리며 탑을 쌓음 | "무너뜨린 사람이 다음 판 순서 정하기" |
| 그림자 놀이 | 손전등으로 벽에 그림자를 만들고 맞추기 | "3초 안에 못 맞추면 교대" |
| 신문지 공 야구 | 신문지를 구겨 공을 만들고, 빗자루 방망이로 치기 | "벽에 닿으면 홈런" |

### 고학년 (4~6학년)

| 놀이 | 방법 | 아이가 만드는 규칙 예시 |
|------|------|----------------------|
| 나만의 보드게임 | 도화지에 게임판을 직접 그려서 만듦 | 칸별 미션, 벌칙을 아이가 설계 |
| 단어 릴레이 배틀 | 주제를 정하고 번갈아 단어를 말함. 3초 초과 시 탈락 | "같은 글자로 시작하는 단어 금지" |
| 실내 볼링 | 빈 페트병 10개 + 공 하나로 볼링 | 점수 체계를 아이가 직접 설계 |
| 역할극 법정 놀이 | 가상 상황을 주고 판사, 변호사, 증인 역할 배분 | "발언 시간 1분 제한" |
| 미션 임파서블 | 복도에 테이프로 레이저 장애물을 만들고 통과하기 | 난이도별 점수를 아이가 결정 |

## 4. 놀이 중 갈등, 부모는 언제 개입할까?

아이들이 놀다 보면 반드시 갈등이 생깁니다. 그 갈등이야말로 사회성 훈련의 핵심입니다. 아래 기준표를 냉장고에 붙여 두세요.

| 상황 | 부모의 행동 |
|------|-----------|
| 목소리가 커지고 언성이 높아질 때 | 지켜봅니다. 아이들은 감정을 조절하는 연습 중입니다. |
| 서로 규칙이 다르다고 주장할 때 | 지켜봅니다. "그럼 어떻게 하면 좋을까?"라고만 물어봅니다. |
| 한쪽이 울기 시작할 때 | 가까이 가되, 바로 해결하지 않습니다. "무슨 일이야?" 한마디만 합니다. |
| 물리적 충돌 (밀기, 때리기) | 즉시 개입합니다. 행동을 멈추게 한 뒤 각자 이야기를 듣습니다. |
| 한 아이가 계속 소외될 때 | 놀이가 끝난 뒤 따로 이야기합니다. 놀이 중간에 끼어들지 않습니다. |

**핵심 원칙:** 말싸움은 지켜보고, 몸싸움은 개입한다.
