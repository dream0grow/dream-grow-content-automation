# 우리 집 자유놀이 환경 만들기 워크시트

**대상:** 초등 1~6학년 자녀를 둔 부모 (아이와 함께 작성)
**소요 시간:** 20분
**준비물:** 연필, 지우개

---

## 사용 안내

이 워크시트는 부모와 아이가 함께 작성합니다. 빈칸을 채우고, 선을 잇고, 분류하면서 우리 집에 맞는 자유놀이 환경을 직접 설계해 봅니다. 정답은 마지막 장에 있지만, 먼저 가족만의 답을 찾아보길 권합니다.

| 화   |          |                      |             |                      |
| 수   |          |                      |             |                      |
| 목   |          |                      |             |                      |
| 금   |          |                      |             |                      |
| 토   |          |                      |             |                      |
| 일   |          |                      |             |                      |

### 1-2. 합계 계산

- 일주일 학원/숙제 총 시간: **(        )시간**
- 일주일 자유놀이 총 시간: **(        )시간**

### 1-3. 진단

자유놀이 시간이 **하루 평균 1시간 미만**이라면, 아이의 뇌가 스스로 정돈할 시간이 부족한 상태입니다.

체크: 우리 아이는 (   ) 충분하다 / (   ) 부족하다

## 부록: 예시 답안

### PART 2 분류 정답

- **O (자유놀이를 여는 멘트):** 2, 4, 5, 7, 9, 10
- **X (자유놀이를 닫는 멘트):** 1, 3, 6, 8

X 멘트의 공통점은 부모가 답을 먼저 주거나, 다른 과제로 빈 시간을 메운다는 점입니다. O 멘트는 아이의 내면을 들여다보게 하거나, 심심함 자체를 머무를 자리로 인정합니다.

### PART 3 환경 설계 참고 예시

**거실에 더하면 좋은 것**
- 종이, 색연필, 가위, 테이프 같은 단순한 재료
- 천 한 장 (아지트, 망토, 텐트로 변신 가능)
- 빈 박스 (어떤 용도든 가능)

**잠시 치우면 좋은 것**
- 정해진 사용법이 있는 완성된 장난감
- TV 리모컨, 태블릿이 늘 보이는 위치

**아이 방에 더하면 좋은 것**
- 책장 한 칸을 비우기 (전시 공간)
- 큰 스케치북 또는 화이트보드

### PART 4 약속 작성 참고

- "하루 1시간"을 채우기 어렵다면 "하루 30분"부터 시작합니다.
- "심심해"의 답은 한 가지로 통일하지 않아도 됩니다. "지금 어떤 느낌이야?" "오늘은 뭐가 떠올라?" 등 가족만의 문장을 만듭니다.
- 부모가 옆에 있되 주도하지 않는 것이 핵심입니다. "뭐 하고 있어?"라고 물어볼 뿐, "이거 해보자"라고 말하지 않습니다.
