# 초등 자기주도학습 습관 체크리스트

**받아쓰기 100점보다 중요한 것: 점수가 아니라 습관입니다.**

현직 초등교사가 교실에서 관찰한 '공부를 잘하게 되는 아이'의 진짜 기준을 정리했습니다.
이 체크리스트는 점수 대신 **과정**을 보는 부모를 위한 자가 진단 도구입니다.

---

## Part 1. 우리 아이 학습 습관 진단 (아이 관찰용)

아이의 평소 모습을 떠올리며 체크해 주세요.

### A. 질문력 - "모르는 걸 모른다고 말할 수 있는가"

- [ ] 수업이나 공부 중 모르는 것이 생기면 질문한다
- [ ] "이게 무슨 뜻이야?"라고 스스로 물어본 적이 있다
- [ ] 책을 읽다가 이해 안 되는 부분에서 멈추고 다시 읽는다
- [ ] "왜?"라는 질문을 귀찮아하지 않는다
- [ ] 정답을 맞혀도 "이게 왜 맞아?"라고 생각해 본다

### B. 재시도 - "틀린 후에 다시 해보는가"

- [ ] 틀린 문제를 다시 풀어보려고 한다
- [ ] 받아쓰기에서 틀린 글자를 스스로 다시 써본다
- [ ] 실수했을 때 "난 못해"가 아니라 "다시 해볼게"라고 말한다
- [ ] 어려운 문제를 만나도 바로 포기하지 않는다
- [ ] 지우개로 지우고 다시 쓰는 것을 자연스럽게 여긴다

### C. 자기 시작 - "스스로 공부를 시작하는가"

- [ ] 부모가 말하기 전에 숙제를 꺼내 놓은 적이 있다
- [ ] 공부 시작 시간을 아이 스스로 정한 적이 있다
- [ ] 무엇을 먼저 할지 순서를 정하려고 시도한다
- [ ] 하루 중 10분이라도 스스로 선택한 공부를 한다
- [ ] "오늘 뭐 공부해?"라는 질문에 구체적으로 답할 수 있다

## 진단 결과 해석

체크한 항목 수를 영역별로 세어 보세요.

| 영역 | 체크 수 | 해석 |
|------|---------|------|
| **질문력** | 4~5개 | 배움에 대한 호기심이 살아 있습니다. 질문을 환영해 주세요. |
| | 2~3개 | 질문할 수 있는 안전한 분위기를 만들어 주면 좋습니다. |
| | 0~1개 | "틀려도 괜찮아"라는 메시지를 일상에서 자주 전해 주세요. |
| **재시도** | 4~5개 | 실패를 배움의 과정으로 받아들이고 있습니다. |
| | 2~3개 | 틀렸을 때 반응을 점검해 보세요. 과정을 격려하면 달라집니다. |
| | 0~1개 | 점수보다 "다시 해본 것 자체"를 칭찬하는 연습이 필요합니다. |
| **자기 시작** | 4~5개 | 자기주도학습의 씨앗이 자라고 있습니다. |
| | 2~3개 | 작은 선택권(순서, 시간, 분량)을 아이에게 넘겨 보세요. |
| | 0~1개 | "하루 10분, 네가 정하는 공부 시간"부터 시작해 보세요. |

**핵심:** 세 영역 중 가장 약한 부분 하나만 먼저 집중하세요. 한꺼번에 바꾸려 하면 부모도 아이도 지칩니다.

## Part 2. 부모 대화 습관 점검 (부모 자가 진단용)

아이에게 평소 하는 말을 떠올리며 체크해 주세요.

### "점수 대신 과정을 묻는 말"을 하고 있나요?

- [ ] "몇 점 맞았어?" 대신 "오늘 새로 알게 된 거 있어?"라고 묻는다
- [ ] "왜 틀렸어?" 대신 "어디서 헷갈렸는지 같이 보자"라고 말한다
- [ ] 100점일 때 "잘했어!"만이 아니라 "어떤 부분이 제일 어려웠어?"도 묻는다
- [ ] 틀린 문제에 대해 혼내기보다 다시 풀어보도록 기다려 준다
- [ ] "공부해!"보다 "오늘은 뭐부터 할 거야?"라고 선택지를 준다

| 체크 수 | 해석 |
|---------|------|
| 4~5개 | 과정 중심 대화를 잘 실천하고 계십니다. |
| 2~3개 | 의식적으로 질문 방식을 바꾸면 아이의 반응이 달라집니다. |
| 0~1개 | 오늘부터 딱 한 가지만 바꿔 보세요. 아래 대화 전환표를 참고하세요. |

---

## Part 3. 대화 전환표

| 기존 (결과 중심) | 전환 (과정 중심) |
|-----------------|-----------------|
| 몇 점 맞았어? | 오늘 새로 알게 된 거 있어? |
| 왜 틀렸어? | 어디서 헷갈렸는지 같이 보자 |
| 1등이 누구야? | 지난번보다 나아진 부분이 있어? |
| 빨리 해 | 오늘은 뭐부터 하고 싶어? |
| 그것도 모르니? | 어디까지는 알겠어? 거기서부터 해보자 |

## Part 4. 4주 실천 기록표: "하루 10분, 스스로 시작하는 시간"

매일 10분, 아이가 **스스로 정한 공부**를 하는 시간을 만들어 보세요.
부모는 옆에서 지켜보되, 무엇을 할지는 아이가 정합니다.

| 3주차 | _ | _ | _ | _ | _ | _ | _ |

**기록 방법:**
- 스스로 시작했으면 O
- 한 번 말한 후 시작했으면 △
- 시작하지 못했으면 빈칸 (X를 쓰지 마세요. 못한 날을 벌주는 기록이 아닙니다.)

**4주 후 돌아보기:**
- O가 점점 늘어나고 있나요? 그렇다면 습관의 씨앗이 자라고 있는 것입니다.
- △가 많아도 괜찮습니다. "한 번 말하면 시작한다"는 것도 성장입니다.
