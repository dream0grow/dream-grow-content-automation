# 자기 설명 학습 습관 8주 로드맵
### 초등 자녀를 둔 부모를 위한 내재적 동기 코칭 가이드

---

## 시작하기 전에

아이의 공부 습관은 **칭찬에서 만들어지지 않습니다. 자기 설명에서 만들어집니다.**

"잘했어" "똑똑해"라는 말은 아이에게 짧은 기쁨을 줍니다. 그러나 평가의 도구가 된 칭찬은 외부의 손에 의지하는 동기, 즉 외재적 동기를 키웁니다. 부모가 칭찬을 멈추는 순간, 아이의 책상도 함께 멈춥니다.

교실에서 같은 문제를 푼 두 아이를 일주일 간격으로 다시 만나본 적이 있습니다. 풀이를 자기 말로 설명한 아이는 같은 유형의 문제를 다시 풀어냈고, 답만 맞췄던 아이는 풀이 과정을 기억하지 못했습니다. 차이는 머리가 아니었습니다. **자기 입으로 한 번 더 정리했는가**의 차이였습니다.

이 로드맵은 8주에 걸쳐 가정에서 '자기 설명'을 자연스러운 습관으로 자리잡게 만드는 단계별 안내입니다. 매주 한 가지씩 늘려나가시면 됩니다.

## 8주 로드맵

| 시기 | 목표 | 구체적 활동 | 확인 |
|------|------|------------|------|
| **1주차** | 채점 후 1문제, 자기 설명 시작 | 채점이 끝난 문제 중 맞은 문제 1개를 골라 "이건 어떻게 풀었어?"라고 묻기. 답은 듣기만 하기. | [ ] 주 3회 이상 실행 |
| **2주차** | 칭찬을 관찰 언어로 전환 | "잘했어" 대신 "이 문제는 단위를 먼저 맞췄네"처럼 보이는 행동을 그대로 말로 옮기기. | [ ] 칭찬 → 관찰 변환 5회 |
| **3주차** | 틀린 문제에도 자기 설명 적용 | 틀린 문제 1개를 골라 "여기까진 어떻게 생각했어?"로 풀이 흐름을 묻기. 정답 알려주지 않기. | [ ] 주 3회, 야단치지 않음 |
| **4주차** | 독서에 자기 설명 확장 | 책을 읽은 뒤 "가장 기억나는 장면 하나만 말해줄래?"로 짧게 요약시키기. 1분 이내. | [ ] 주 3회 실행 |
| **5주차** | '왜?' 질문 1일 1회 | 일상에서 아이의 선택에 "왜 그렇게 정했어?"를 부드럽게 묻기. 평가 아님을 톤으로 전달. | [ ] 매일 1회 실행 |
| **6주차** | 아이가 부모를 가르치는 시간 | 아이가 배운 단원 하나를 골라 "엄마(아빠)한테 가르쳐줘"로 5분 미니 수업 진행. | [ ] 주 2회 진행 |
| **7주차** | 학습 기록을 아이가 직접 쓰기 | 그날 푼 문제 중 한 문제의 풀이 과정을 아이가 직접 노트에 한두 줄로 쓰기. | [ ] 주 4회 기록 |
| **8주차** | 자기 평가 루틴 정착 | 하루 공부 마무리에 "오늘 가장 잘 이해된 부분은?" / "헷갈렸던 건?" 두 질문 묻기. | [ ] 매일 실행 |

## 단계별 부모의 한 줄 대본

각 주차에서 가장 자주 쓸 표현입니다. 외우지 마시고, 식탁이나 책상 옆에 붙여두세요.

| 주차 | 추천 문장 | 피해야 할 문장 |
|------|-----------|---------------|
| 1주차 | "이건 어떻게 풀었어?" | "정답 맞네, 잘했어." |
| 2주차 | "이 문제는 단위부터 적었네." | "역시 똑똑해." |
| 3주차 | "여기까진 어떻게 생각한 거야?" | "이걸 또 틀려?" |
| 4주차 | "가장 기억나는 장면 하나만." | "다 읽었어? 몇 권 읽었어?" |
| 5주차 | "왜 그걸로 정했어?" | "그건 이상한 선택이야." |
| 6주차 | "엄마한테 가르쳐줄래?" | "엄마가 알려줄게." |
| 7주차 | "한 줄로만 적어볼래?" | "풀이 다시 적어." |
| 8주차 | "오늘 가장 잘 된 건 뭐였어?" | "오늘 몇 점 받았어?" |

## 진도 점검 체크포인트

### 2주차 끝
- [ ] 아이가 "이건 어떻게 풀었어?"라는 질문에 거부감 없이 입을 연다
- [ ] 부모가 칭찬 대신 관찰을 말하는 데 어색함이 줄었다

### 4주차 끝
- [ ] 틀린 문제 앞에서 아이가 변명 대신 풀이 과정을 말한다
- [ ] 부모가 정답을 즉시 알려주는 습관이 줄었다

### 6주차 끝
- [ ] 아이가 부모에게 자발적으로 무언가를 설명하는 장면이 한 번이라도 있었다
- [ ] "왜?" 질문이 아이에게 위협이 아니라 대화로 인식된다

### 8주차 끝
- [ ] 아이가 공부를 마치고 스스로 잘된 점과 헷갈린 점을 말한다
- [ ] 부모가 결과 점수보다 과정 언어를 더 자주 사용한다

## 흔히 무너지는 지점과 회복법

**3주차 정체 구간.** 틀린 문제 앞에서 부모의 인내가 무너지는 시기입니다. 정답을 즉시 알려주고 싶은 충동이 생깁니다. 이때는 답을 말하는 대신 "여기까진 맞아"라고 풀이의 옳은 지점까지만 짚어주세요. 나머지는 아이가 잇습니다.

**5주차 저항 구간.** 아이가 "몰라" "그냥"으로 답하는 시기입니다. 이건 거부가 아니라 익숙해지는 과정입니다. 다그치지 마시고 "그래, 천천히 생각해도 돼"로 한 박자 쉬어주세요.

**7주차 피로 구간.** 부모도 아이도 지칩니다. 이때는 활동을 늘리지 마시고 가장 효과가 좋았던 한 가지만 남기세요. 무리한 확장보다 한 가지의 지속이 습관을 만듭니다.

## 이 로드맵의 핵심 원칙

**평가가 아니라 과정을 비추는 일입니다.** 아이가 스스로 자기 사고를 들여다보게 만드는 거울이 부모의 질문입니다. 거울은 평가하지 않습니다. 그저 비춥니다.

**칭찬을 줄이라는 말이 아닙니다.** 평가의 칭찬을 줄이고 관찰의 언어를 늘리라는 뜻입니다. "잘했어"는 결과를 비추지만, "이 문제는 단위를 먼저 맞췄네"는 과정을 비춥니다.

**8주가 끝나도 끝이 아닙니다.** 이 로드맵은 시작점입니다. 자기 설명이 가정의 공기처럼 자리잡으면, 아이는 부모가 묻지 않아도 스스로 자기 사고를 정리하기 시작합니다. 그때부터가 진짜 공부 습관입니다.
