# 부모와 함께 노는 아이가 혼자서도 잘 노는 이유

---

> "혼자 있을 수 있는 능력은 누군가와 함께 있는 경험에서 발달한다."
> — 도널드 위니콧(발달심리학자)

혼자 잘 노는 아이를 만들고 싶다면, 먼저 충분히 함께 놀아야 합니다.

## 2. 연령별 "따라가기 놀이" 목록

아이가 주도하고, 부모가 따라가는 놀이입니다.
아이의 연령대에 해당하는 칸에서 이번 주 해볼 놀이 3가지를 골라 동그라미 치세요.

### 5~6세 (유치원~초등 1학년)

| 놀이 | 부모의 역할 | 선택 |
|------|-----------|------|
| 소꿉놀이 | 아이가 정한 역할을 맡는다 (손님, 아기 등) | O / X |
| 블록 쌓기 | 아이가 시키는 블록을 건네주는 조수가 된다 | O / X |
| 그림 그리기 | 옆에서 같이 그리되, 아이 그림에 대해 묻기만 한다 | O / X |
| 숨바꼭질 | 아이가 정한 규칙대로 한다 | O / X |
| 인형놀이 | 아이가 배정한 인형의 대사를 읽는다 | O / X |

### 7~8세 (초등 2~3학년)

| 놀이 | 부모의 역할 | 선택 |
|------|-----------|------|
| 보드게임/카드게임 | 아이가 규칙을 설명하게 하고, 그대로 따른다 | O / X |
| 레고/조립 | 아이의 설계를 듣고 부품을 찾아주는 역할을 한다 | O / X |
| 만들기 (종이접기, 공작) | 아이가 가르쳐 주는 대로 따라 접는다 | O / X |
| 역할놀이/상황극 | 아이가 만든 시나리오에서 조연을 맡는다 | O / X |
| 실외 놀이 (공놀이, 자전거) | 아이가 정한 게임 방식을 따른다 | O / X |

### 9~10세 (초등 4~5학년)

| 놀이 | 부모의 역할 | 선택 |
|------|-----------|------|
| 전략 보드게임 | 아이에게 전략을 물어보며 함께 고민한다 | O / X |
| 요리/베이킹 | 아이가 레시피를 고르고 지시하면 따른다 | O / X |
| 실험/탐구 놀이 | 아이의 궁금증에서 출발해 같이 찾아본다 | O / X |
| 스포츠/신체 놀이 | 아이가 정한 규칙과 팀 편성을 따른다 | O / X |
| 이야기 만들기 | 아이가 시작한 이야기에 한 문장씩 보탠다 | O / X |
