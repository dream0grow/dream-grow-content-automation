# 자기주도학습 부모 실천 로드맵

### 학원 줄이기가 아닙니다. 아이에게 '선택'을 돌려주는 8주 로드맵입니다.

초등교사 15년차, 수백 명의 아이를 봐왔습니다.
학원을 3개, 4개 다니는 아이들의 시험 성적은 나쁘지 않았습니다.
그런데 "오늘 뭐 공부할래?" 하고 물으면 대답을 못 합니다.
뭘 공부할지, 언제 할지, 어떻게 할지를 스스로 정해본 적이 없기 때문입니다.

이 로드맵은 학원을 당장 끊으라는 이야기가 아닙니다.
아이가 '공부의 주인'이 되는 환경을 가정에서 만들어가는 8주 실천 계획입니다.

## 8주 자기주도학습 전환 로드맵

### 1~2주차: 관찰하기 -- "지금 우리 아이의 하루를 봅니다"

| 구분 | 내용 |
|------|------|
| **목표** | 아이의 현재 학습 패턴을 파악한다 |
| **구체적 활동** | (1) 아이의 일주일 시간표를 함께 적어본다 (학원, 숙제, 자유시간 구분) (2) "이번 주에 네가 직접 정한 공부가 있었어?" 하고 물어본다 (3) 아이가 학원 없이 보내는 시간에 무엇을 하는지 관찰한다 |
| **부모 대화법** | "학원에서 오늘 뭐 했어?" (X) --> "오늘 공부하면서 재미있었던 거 있어?" (O) |
| **주의점** | 이 시기에 학원을 줄이거나 바꾸지 마세요. 관찰만 합니다 |
| **확인** | [ ] 아이의 주간 시간표를 적어보았다 / [ ] 자유시간 활용 패턴을 파악했다 |

### 3~4주차: 30분의 자율 -- "하루 30분, 네가 정한다"

| 구분 | 내용 |
|------|------|
| **목표** | 아이가 스스로 학습 내용을 선택하는 경험을 만든다 |
| **구체적 활동** | (1) 하루 중 30분을 '자기 공부 시간'으로 정한다 (2) "이 30분 동안 뭘 공부할지 네가 정해봐"라고 말한다 (3) 아이가 정한 내용이 무엇이든 존중한다 (만화책 읽기도 괜찮습니다) (4) 끝나면 "오늘 뭘 했는지 한 줄로 적어볼래?" 하고 기록을 남긴다 |
| **부모 대화법** | "그걸 왜 골랐어?" (X) --> "그걸 골랐구나. 해보니까 어땠어?" (O) |
| **주의점** | 아이가 고른 내용을 평가하지 마세요. 선택 자체가 연습입니다. 처음에는 "몰라요"가 당연합니다. 3가지 선택지를 주고 고르게 해도 좋습니다 |
| **확인** | [ ] 매일 30분 자기 공부 시간을 실행했다 / [ ] 아이가 스스로 내용을 골랐다 |

---

### 5~6주차: 계획 세우기 -- "이번 주는 네가 짜보자"

| 구분 | 내용 |
|------|------|
| **목표** | 아이가 주간 단위로 학습 계획을 세워본다 |
| **구체적 활동** | (1) 일요일 저녁, 아이와 함께 앉아 다음 주 자기 공부 시간 계획을 세운다 (2) 아이가 직접 요일별로 무엇을 할지 적는다 (아래 템플릿 활용) (3) 금요일에 함께 돌아보며 "이번 주 계획대로 된 건 뭐야?"라고 묻는다 (4) 못한 것을 지적하지 말고, 한 것을 먼저 확인한다 |
| **부모 대화법** | "왜 이것밖에 못 했어?" (X) --> "3개 중에 2개나 했네. 못한 1개는 다음 주에 넣어볼까?" (O) |
| **주의점** | 계획이 완벽할 필요 없습니다. '세우고 돌아보는 과정'이 핵심입니다. 부모가 대신 짜주면 학원과 같아집니다 |
| **확인** | [ ] 아이가 직접 주간 계획을 작성했다 / [ ] 주말에 함께 돌아보기를 했다 |

---

### 7~8주차: 과정에 반응하기 -- "어떻게 풀었어?"

| 구분 | 내용 |
|------|------|
| **목표** | 결과가 아닌 과정에 반응하는 대화 습관을 만든다 |
| **구체적 활동** | (1) 시험 결과가 나오면 점수 대신 과정을 묻는다: "어떻게 공부했어?" "어디가 어려웠어?" (2) 틀린 문제를 함께 보며 "왜 틀렸다고 생각해?"라고 묻는다 (3) 아이 스스로 "다음에는 이렇게 해볼래요"라고 말하면 그것을 기록해둔다 (4) 학원 성적표보다 아이가 직접 쓴 자기 공부 기록에 더 관심을 보인다 |
| **부모 대화법** | "100점이네!" (X) --> "이 문제 어떻게 풀었어? 방법이 궁금해" (O) |
| **주의점** | 과정 질문은 '추궁'이 아닙니다. 진심으로 궁금해하는 톤이 중요합니다. 아이가 부담스러워하면 질문 수를 줄이세요 |
| **확인** | [ ] 점수 대신 과정을 묻는 대화를 했다 / [ ] 아이가 스스로 다음 계획을 말했다 |

## 8주 이후: 유지와 확장

8주를 마치면 아이에게 작은 변화가 보이기 시작합니다.
"오늘 이거 하려고요"라고 먼저 말하거나, 계획표를 스스로 꺼내는 순간이 옵니다.
그때가 진짜 시작입니다.

**유지를 위한 3가지 원칙:**

1. **자기 공부 시간은 계속 유지한다** -- 30분이 익숙해지면 45분, 1시간으로 천천히 늘린다
2. **학원 조정은 아이와 상의한다** -- "이 학원이 너한테 도움이 되고 있다고 느껴?" 하고 아이의 의견을 듣는다
3. **월 1회 돌아보기를 한다** -- 한 달에 한 번, 아이와 함께 "이번 달에 스스로 잘한 것 3가지"를 적어본다

## 자주 묻는 질문

**Q. 학원을 줄여야 하나요?**
당장 줄이지 않아도 됩니다. 이 로드맵의 핵심은 학원 수를 줄이는 것이 아니라, 아이에게 '선택의 경험'을 돌려주는 것입니다. 자기 공부 시간이 자리 잡으면 아이 스스로 "이 학원은 필요 없는 것 같아요"라고 말하는 날이 옵니다.

**Q. 자기 공부 시간에 아이가 놀기만 하면요?**
처음에는 자연스러운 반응입니다. 자유 시간을 '공부 시간'으로만 써본 적이 없으니까요. 2주 정도는 지켜봐 주세요. 선택지를 3개 제시하고 고르게 하는 것도 방법입니다. 중요한 것은 아이가 '직접 골랐다'는 경험입니다.

**Q. 중학교 가면 결국 학원이 필요하지 않나요?**
중학교에서도 학원은 도구입니다. 문제는 도구에 의존하는 것이지, 도구를 쓰는 것 자체가 아닙니다. 초등학교 때 자기 공부 습관이 잡힌 아이는 중학교에서 학원을 다니더라도 '내가 부족한 부분을 채우려고 간다'는 목적의식을 갖습니다. 그 차이가 고등학교에서 갈립니다.
