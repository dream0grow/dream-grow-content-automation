# 초등수학 영역별 개념지도

## 우리 아이가 어디서 막혔는지, 한눈에 보이는 지도

초등수학은 5개 영역이 6년 동안 나선형으로 쌓이는 구조입니다. 한 학년에서 빠진 개념은 다음 학년에서 반드시 구멍으로 돌아옵니다. 분수를 못 하는 6학년은 6학년 분수를 못 하는 게 아니라 3학년 '1' 개념에서 막힌 경우가 대부분입니다. 이 자료는 어디서 막혔는지 부모가 직접 확인할 수 있도록 만든 영역별 개념지도입니다.

---

## 1. 다섯 가지 영역, 한 장 요약

```
                   [초등수학]
                       │
   ┌───────┬───────┬───┴───┬───────┬───────┐
   │       │       │       │       │       │
[수와연산] [도형] [측정] [규칙성] [자료와가능성]
   ★         ☆     ☆      ☆         ☆
 (★ = 구멍 가장 잘 나는 영역)
```

분수, 소수, 비와 비율은 모두 '수와 연산' 영역에서 나옵니다. 학부모가 가장 먼저 점검해야 할 영역입니다.

---

## 2. 영역 1. 수와 연산 — 분수 중심 개념지도

```
[1학년]                  [2학년]                    [3학년]
한 자릿수 덧셈/뺄셈   →   곱셈구구              →   나눗셈 도입
100까지 수          →   1000까지 수            →   10000까지 수
                                                       │
                                                       ▼
                                            ★ 분수의 도입 (3-1)
                                            "전체를 똑같이 나눈
                                             것 중의 몇"
                                                       │
                                                       ▼
                                            ★ 분수의 종류 (3-2)
                                            진분수, 가분수, 대분수
                                                       │
[4학년]                                                ▼
분수의 덧셈/뺄셈 (동분모) — "분모가 같으면 분자끼리"
소수의 덧셈/뺄셈
                                                       │
                                                       ▼
[5학년]
약수와 배수  →  약분과 통분  →  분수의 덧셈/뺄셈 (이분모)
                                          │
                                          ▼
                              분수의 곱셈, 소수의 곱셈
                                          │
                                          ▼
[6학년]
분수의 나눗셈, 소수의 나눗셈  →  비와 비율  →  비례식
```

### 분수 단원의 선수학습 ↔ 후속학습 관계

| 막혔다면 | 진짜 못 하는 것은 | 다시 가르쳐야 할 시점 |
|---|---|---|
| 5학년 통분 | 3학년 '1'의 개념, 동치분수 | 3-1 분수 단원 다시 |
| 5학년 분수 곱셈 | 3-2 분수의 종류 (가분수/대분수) | 3-2 다시 |
| 6학년 분수 나눗셈 | 5학년 분수 곱셈, 역수 개념 | 5-2 다시 |
| 6학년 비와 비율 | 4학년 분수의 의미 (비교) | 4-1 다시 |

### 분수에서 가장 흔한 구멍 세 가지

첫째, 기준이 되는 '1'이 무엇인지 모릅니다. 1/2의 1이 피자 한 판인지 한 조각인지 흔들립니다. 둘째, 분수를 '나눈 결과'로만 알고 '비교의 도구'로는 모릅니다. 1/2이 1/3보다 큰 이유를 설명하지 못하는 이유입니다. 셋째, 가분수와 대분수의 의미가 분리되어 있습니다. 7/3과 2와 1/3이 같은 양임을 손으로 만져본 적이 없는 거예요.

## 3. 영역 2. 도형

```
[1~2학년]              [3~4학년]              [5~6학년]
모양 인식      →    평면도형의 성질      →    합동과 대칭
입체/평면도형        삼각형, 사각형 분류        직육면체, 정육면체
                    각도, 평면도형 이동         각기둥, 각뿔, 원기둥
```

도형은 누적이 비교적 단순한 영역입니다. 다만 4학년 '각도'와 '평면도형의 이동'이 5-6학년 합동·대칭의 기초이므로 이 부분이 흔들리면 6학년 입체도형 학습 전체가 어려워집니다.

## 4. 영역 3. 측정

```
[1~2학년]              [3~4학년]              [5~6학년]
비교하기, 시각      →   cm/m, mm/km          →   둘레와 넓이
시간 읽기              들이와 무게               부피와 겉넓이
                       각도 측정                 원의 넓이
```

측정 영역의 핵심 구멍은 '단위 환산'입니다. 1m가 100cm라는 것을 외우는 것과, 1m 안에 100cm가 들어간다는 것을 손으로 아는 것은 다릅니다. 5-6학년 넓이·부피 단원에서 cm²와 m² 환산이 어려운 아이는 3-4학년 단위 학습으로 돌아가야 합니다.

## 5. 영역 4. 규칙성

```
[1~2학년]              [3~4학년]              [5~6학년]
규칙 찾기        →    규칙과 대응          →    비례식과 비례배분
                                                정비례와 반비례
```

규칙성은 중학교 함수의 뿌리입니다. 4학년 '규칙과 대응'이 흔들리면 6학년 비례식이 무너지고, 중학교 1학년 함수에서 다시 막힙니다. '한 변수가 변하면 다른 변수가 어떻게 변하는가'를 표로 직접 채워본 경험이 핵심입니다.

## 6. 영역 5. 자료와 가능성

```
[1~2학년]              [3~4학년]              [5~6학년]
분류하기, 표      →    그림그래프           →    평균과 가능성
                       막대그래프                띠그래프, 원그래프
                       꺾은선그래프
```

이 영역은 다른 영역과 달리 '읽기'와 '해석'이 중심입니다. 분수와 비율을 못 하면 6학년 띠그래프·원그래프에서 막히므로, 수와 연산 영역의 보강이 우선입니다.

## 7. 부모가 활용하는 법 — 5단계

**1단계.** 아이가 현재 막힌 단원을 적습니다. (예: "5학년 통분이 어렵다")

**2단계.** 위의 '선수학습 ↔ 후속학습 표'에서 진짜 구멍이 어느 학년인지 찾습니다.

**3단계.** 해당 학년 교과서를 다시 펼치고, 기호가 아닌 그림과 구체물로 그 개념을 복원합니다. (분수라면 종이접기, 측정이라면 계량컵과 자)

**4단계.** 아이에게 '이게 무엇의 1/2일까'처럼 기준을 묻는 질문을 합니다. 답을 가르치지 말고 기준을 가리키게 하세요.

**5단계.** 같은 개념을 1주일 간격으로 두 번 더 꺼내서 손으로 다시 다룹니다. 한 번에 끝나는 개념은 없거든요.

## 8. 자주 묻는 질문

**Q. 학년을 거슬러 올라가면 진도가 늦어지지 않나요?**

거슬러 올라가는 것이 가장 빠른 길입니다. 막힌 개념 위에 새 단원을 쌓으면 단원이 늘어날수록 구멍은 깊어집니다. 한 주 거슬러 올라가서 두 달의 헛수고를 줄이는 게 합리적입니다.

**Q. 학원에 다니는데도 이 작업을 따로 해야 하나요?**

학원은 진도를 나가는 곳입니다. 막힌 개념을 거슬러 점검하는 일은 학원에서 잘 해주지 않거든요. 가정에서 종이 한 장, 계량컵 하나로 충분히 보완할 수 있습니다.

**Q. 수학을 싫어하는 아이에게 어떻게 시작하나요?**

기호로 시작하지 마세요. 종이를 접고, 물을 붓고, 피자를 자르는 것에서 시작합니다. 수학은 손에서 머리로 올라가는 학문이거든요. 아이가 수학을 즐기며 자라길 바랍니다.
