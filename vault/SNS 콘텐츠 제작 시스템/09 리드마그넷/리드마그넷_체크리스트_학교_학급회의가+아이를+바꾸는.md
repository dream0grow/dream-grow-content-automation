# 우리 집 가족회의 실천 체크리스트

**초등 자녀를 둔 부모를 위한 가족회의 진행 가이드**

---

## 이 자료는 이렇게 활용하세요

가족회의는 거창한 행사가 아닙니다.
"내 말이 들린다"는 경험을 아이에게 주는 가장 작은 단위의 민주주의입니다.
아래 체크리스트를 하나씩 점검하며, 우리 가족만의 회의 문화를 만들어 보세요.

---

## Part 1. 회의 전 - 환경 만들기

| | 항목 | 설명 |
|---|------|------|
| [ ] | TV, 스마트폰을 모두 끄고 시작한다 | 화면이 켜져 있으면 아이는 "대충 하는 시간"으로 인식합니다. |
| [ ] | 온 가족이 같은 높이에 앉는다 | 소파에 누운 부모, 바닥에 앉은 아이. 이 배치만으로 위계가 생깁니다. 식탁에 함께 앉으세요. |
| [ ] | "오늘 회의 주제"를 미리 한 가지 정한다 | 주제 없이 시작하면 잡담으로 끝납니다. 냉장고에 메모지를 붙여두고 한 주간 안건을 모으는 방법도 좋습니다. |
| [ ] | 발언 시간을 연령에 맞게 정한다 | 저학년(1~2학년): 1인당 1분 / 중학년(3~4학년): 1인당 2분 / 고학년(5~6학년): 1인당 3분 |
| [ ] | "틀린 말은 없다"는 규칙을 매번 확인한다 | 회의가 익숙해져도 이 선언은 생략하지 마세요. 교실에서도 매 회의 첫 마디로 이 말을 합니다. |

---

## Part 2. 회의 중 - 대화 방식 점검

| | 항목 | 설명 |
|---|------|------|
| [ ] | 아이가 말할 때 끝까지 듣고 나서 반응한다 | 중간에 "그건 아니지"가 나오면, 아이는 다음 회의에서 입을 닫습니다. |
| [ ] | 반대 의견에 "왜?"가 아니라 "어떤 마음이야?"로 묻는다 | "왜?"는 논리를 요구합니다. 아이에게 필요한 것은 감정을 말할 기회입니다. |
| [ ] | 부모도 자기 의견을 "제안"으로 말한다 | "이렇게 하자"가 아니라 "나는 이런 생각인데, 어떻게 생각해?"로 바꿔보세요. |
| [ ] | 결론이 나지 않아도 "오늘은 여기까지"로 마무리할 수 있다 | 결론을 강제하면 회의가 아니라 통보가 됩니다. 다음 회의로 넘기는 것도 민주주의입니다. |
| [ ] | 한 사람이 대화를 독점하지 않는지 살핀다 | 말이 많은 형/누나가 있다면, 순서를 정해 돌아가며 말하는 방식이 효과적입니다. |

---

## Part 3. 회의 후 - 실천과 연결하기

| | 항목 | 설명 |
|---|------|------|
| [ ] | 회의에서 정한 내용을 한 문장으로 적어둔다 | 아이가 직접 적으면 더 좋습니다. "이번 주말에는 공원에서 자전거를 타기로 했다." |
| [ ] | 다음 회의에서 지난 결정을 돌아본다 | "지난번에 정한 건 어땠어?"로 시작하면, 아이는 자기 말이 진짜 반영된다고 느낍니다. |
| [ ] | 잘 진행된 점을 구체적으로 말해준다 | "오늘 OO이가 동생 의견도 들어보자고 한 게 정말 좋았어." 행동을 짚어주세요. |
| [ ] | 회의 주기를 일정하게 유지한다 | 매주 같은 요일, 같은 시간이 가장 좋습니다. 격주도 괜찮지만, 한 달에 한 번은 너무 뜸합니다. |
| [ ] | 아이가 안건을 직접 제안하면 반드시 다룬다 | 아이가 "이것도 회의에서 얘기하면 돼?"라고 물으면, 가족회의가 뿌리를 내리기 시작한 것입니다. |

---

## 점수 해석 가이드

체크한 항목 수를 세어 보세요.

**13~15개: 이미 훌륭한 가족회의를 하고 계십니다.**
아이가 "우리 집은 내 말을 들어준다"고 느끼고 있을 가능성이 높습니다. 지금 방식을 꾸준히 유지해 주세요.

**9~12개: 기본 틀은 잡혀 있습니다.**
체크하지 못한 항목을 한 주에 하나씩만 추가해 보세요. 한꺼번에 바꾸려 하면 부모도 아이도 부담을 느낍니다.

**5~8개: 시작이 반입니다.**
환경 만들기(Part 1)부터 하나씩 갖춰가면 됩니다. 처음에는 5분짜리 짧은 회의로 시작하는 것을 권합니다.

**0~4개: 오늘이 가장 좋은 시작점입니다.**
이번 주말, 딱 한 가지 주제로 10분만 앉아 보세요. "이번 주말 뭐 할까?"처럼 가벼운 주제가 좋습니다. 완벽한 회의는 없습니다. 앉아서 서로 얼굴을 보는 것 자체가 첫걸음입니다.

## "왜?" 대신 쓸 수 있는 말 5가지

| 기존 표현 | 전환 표현 |
|-----------|-----------|
| "왜 반대야?" | "어떤 마음에서 그렇게 생각했어?" |
| "왜 싫어?" | "어떤 부분이 마음에 안 들어?" |
| "왜 그렇게 하고 싶어?" | "그걸 하면 어떤 기분일 것 같아?" |
| "왜 안 된다고 생각해?" | "걱정되는 게 있어?" |
| "왜 말을 안 해?" | "지금 말하기 어려우면 다음에 해도 괜찮아." |

## 주제별 가족회의 템플릿

**[주말 계획 정하기]**
"이번 주말에 하고 싶은 것 한 가지씩 말해볼까?" → 각자 제안 → 겹치는 부분 찾기 → 하나 선택 또는 절충

**[갈등 해결하기]**
"요즘 불편했던 일이 있으면 이야기해볼까?" → 감정 먼저 말하기 → 상대 입장 들어보기 → 함께 해결 방법 찾기

**[가족 규칙 정하기]**
"우리 집에서 이건 꼭 지키자, 하는 게 있을까?" → 각자 제안 → 합의된 것만 규칙으로 → 냉장고에 붙이기
