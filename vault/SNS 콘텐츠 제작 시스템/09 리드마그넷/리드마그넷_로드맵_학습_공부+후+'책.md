# 집에서 시작하는 자기주도학습 실천 가이드

### 4주 인출 연습 로드맵 — "책 덮고 설명하기"로 공부 주인 만들기

---

## 시작하기 전에

아이가 공부를 안 하는 것이 아닙니다. **꺼내는 연습**이 부족한 것입니다.

Roediger와 Karpicke(2006)의 인출 연습(Retrieval Practice) 연구는 한 가지 사실을 분명히 보여줍니다. 같은 내용을 다시 읽는 학생보다, 책을 덮고 떠올려 설명한 학생의 기억이 더 오래, 더 정확하게 남았다는 것입니다.

이 자료는 그 원리를 **가정에서 4주 동안** 단계적으로 정착시키는 로드맵입니다. 하루 5분, 식탁 옆 빈 시간이면 충분합니다.

---

## 전체 로드맵 한눈에 보기

| 주차 | 단계 | 핵심 목표 | 하루 소요 시간 |
|------|------|-----------|----------------|
| 1주차 | 적응기 | 책 덮고 떠올리는 감각 익히기 | 5분 |
| 2주차 | 확장기 | '왜?'와 '어떻게?'로 깊이 들어가기 | 7분 |
| 3주차 | 연결기 | 과목 간 연결 설명하기 | 10분 |
| 4주차 | 자립기 | 부모 없이 스스로 설명·점검하기 | 10분 |

## 1주차 — 적응기: "책을 덮는 것"부터

| 시기 | 목표 | 구체적 활동 | 확인 |
|------|------|--------------|------|
| 월 | 규칙 합의 | "공부 끝나면 5분만 엄마(아빠)한테 설명" 약속 | [ ] |
| 화 | 국어 1단원 설명 | 오늘 배운 글 내용을 30초 안에 말하기 | [ ] |
| 수 | 수학 개념 설명 | 오늘 푼 문제 중 1개를 처음부터 풀이 설명 | [ ] |
| 목 | 사회/과학 설명 | 가장 기억나는 사실 3가지 말하기 | [ ] |
| 금 | 자유 설명 | 아이가 고른 과목 하나를 자유롭게 | [ ] |
| 토 | 주간 정리 | 이번 주 가장 어려웠던 개념 1개 다시 설명 | [ ] |

**부모 역할**: "잘했어/못했어" 평가하지 않습니다. **모르는 척 듣기**가 전부입니다.

**주의점**: 첫 주에 막히는 것은 정상입니다. "원래 처음엔 그래"라고만 말해주세요.

## 2주차 — 확장기: 질문으로 깊이 만들기

| 시기 | 목표 | 구체적 활동 | 확인 |
|------|------|--------------|------|
| 월 | '왜?' 도입 | 아이 설명 중간에 "왜 그렇게 되는 거야?" 1회 | [ ] |
| 화 | '어떻게?' 도입 | "어떻게 그걸 알았어?" 1회 | [ ] |
| 수 | '예를 들면?' 도입 | "예를 들면 어떤 거야?" 1회 | [ ] |
| 목 | 일부러 틀리기 | 부모가 일부러 잘못 이해한 척 → 아이가 정정 | [ ] |
| 금 | 그림 설명 | 종이에 그림 그리며 설명하기 | [ ] |
| 토 | 가족 발표 | 가족 앞에서 1분 미니 발표 | [ ] |

**부모가 던지면 좋은 질문 6선**
1. "왜 그렇게 되는 거야?"
2. "어떻게 그걸 알게 됐어?"
3. "예를 들면 어떤 게 있을까?"
4. "그럼 이 경우엔 어떻게 돼?"
5. "엄마(아빠)는 이게 헷갈리는데, 다시 설명해줄래?"
6. "이거랑 비슷한 게 또 있을까?"

**피해야 할 NG 질문**
- "이거 시험에 나와?"
- "이것도 몰라?"
- "선생님이 그렇게 가르쳤어?"

| 시기 | 목표 | 구체적 활동 | 확인 |
|------|------|--------------|------|
| 월 | 수학 → 일상 | 오늘 배운 수학을 마트/요리 상황으로 설명 | [ ] |
| 화 | 과학 → 일상 | 배운 과학 개념의 일상 사례 1개 찾기 | [ ] |
| 수 | 국어 → 경험 | 글의 주제를 본인 경험과 연결해 말하기 | [ ] |
| 목 | 사회 → 뉴스 | 배운 내용과 비슷한 뉴스/사건 찾기 | [ ] |
| 금 | 과목 간 연결 | 두 과목에서 비슷한 점 1개 찾기 | [ ] |

**주의점**: 연결이 어색해도 괜찮습니다. 연결하려는 시도 자체가 뇌의 **출력 회로**를 강화합니다.

## 4주차 — 자립기: 부모 없이 스스로

| 시기 | 목표 | 구체적 활동 | 확인 |
|------|------|--------------|------|
| 월 | 셀프 녹음 | 휴대폰에 1분 설명 녹음, 다시 들어보기 | [ ] |
| 화 | 빈 종이 테스트 | 책 덮고 빈 종이에 배운 것 다 적기 | [ ] |
| 수 | 막힌 곳 표시 | 적다가 막힌 부분에 별표 → 그것만 복습 | [ ] |
| 목 | 거울 설명 | 거울 앞에서 자기 자신에게 설명 | [ ] |
| 금 | 동생/인형 강의 | 동생, 인형에게 선생님처럼 설명 | [ ] |
| 토 | 한 달 정리 | 한 달 중 가장 자신 있는 단원 5분 강의 | [ ] |

**자립기 핵심**: 4주차가 되면 부모는 **청중**에서 **관찰자**로 한 걸음 물러섭니다. 아이가 막힌 지점을 스스로 알아채는 것이 자기주도학습의 시작입니다.

## 부모가 꼭 기억할 3가지

**1. 평가하지 않습니다.**
설명이 어수룩해도 "그래서 어떻게 됐는데?"로 이어주세요. 평가는 학교 시험이 합니다.

**2. 막히는 지점이 보석입니다.**
아이가 멈춘 곳, 얼버무린 곳이 **진짜 복습 포인트**입니다. 거기만 다시 보면 됩니다.

**3. 매일 5분이 한 달 2시간보다 강합니다.**
인출 연습의 효과는 **분산**될 때 가장 큽니다. 주말 몰아 하지 말고 매일 짧게 하세요.

## 자주 묻는 질문

**Q. 아이가 설명을 너무 짧게 끝내면요?**
"한 문장만 더 해줄래?"라고 짧게 요청하세요. 길게 강요하면 회피하게 됩니다.

**Q. 설명을 틀리게 하면 바로 고쳐줘야 할까요?**
아니요. 끝까지 듣고 "엄마는 이 부분이 좀 헷갈렸어, 다시 한 번 볼까?"로 책을 함께 펴세요. 정답을 부모가 주면 인출 효과가 사라집니다.

**Q. 학원 숙제만으로도 벅찬데 이걸 또 해야 하나요?**
숙제가 끝난 후 **5분**이면 됩니다. 양을 더하는 게 아니라, 끝낸 것을 **단단히 굳히는** 시간입니다.
