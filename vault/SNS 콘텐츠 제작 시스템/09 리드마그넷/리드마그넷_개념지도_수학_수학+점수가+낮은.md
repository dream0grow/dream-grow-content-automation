```

            ┌──────────────────────┼──────────────────────┐
            │                      │                      │
            ▼                      ▼                      ▼
      [마인드셋]             [채점 경험]            [부모 언어]

```

수학 자신감은 타고나는 것이 아닙니다. 마인드셋(어떻게 인식하는가), 채점 경험(어떻게 평가받는가), 부모 언어(어떤 말을 듣는가) 세 가지가 만나는 자리에서 만들어집니다.

## 2. 세 가지 축의 분기 구조

### [축 1] 마인드셋 — 같은 오답, 다른 해석

```
                  오답 한 문제
                       │
        ┌──────────────┴──────────────┐
        ▼                             ▼
   고정 마인드셋                  성장 마인드셋
   (Fixed)                        (Growth)
        │                             │
        ▼                             ▼
  "나는 못 한다"                 "아직 모른다"
        │                             │
        ▼                             ▼
   회피·포기                      재시도·질문
        │                             │
        ▼                             ▼
   문제량 ↑ = 좌절 ↑            문제량 ↑ = 실력 ↑
```

**핵심:** 같은 오답이 어떤 아이에게는 '내 한계의 증거'가 되고, 어떤 아이에게는 '다음 학습의 출발점'이 됩니다. 갈라지는 지점은 문제집 두께가 아니라 해석의 방향입니다.

---

### [축 2] 채점 경험 — 빨간 X에서 물음표로

```
   기존 채점                          전환된 채점
   ─────────                          ─────────
   빨간 X                             물음표 (?)
       │                                  │
       ▼                                  ▼
   "끝났다"                           "다시 본다"
       │                                  │
       ▼                                  ▼
   감정: 수치심                       감정: 호기심
       │                                  │
       ▼                                  ▼
   행동: 노트 덮기                    행동: 다시 펴기
```

**연결고리:** 채점 표시는 단순한 기호가 아니라 아이의 다음 행동을 결정하는 신호입니다. 'X'는 끝맺음의 신호이고, '?'는 이어짐의 신호입니다.

---

### [축 3] 부모 언어 — 결과 반응 vs 과정 반응

```
   결과 반응 (Closed)                 과정 반응 (Open)
   ──────────────                     ──────────────
   "왜 또 틀렸어"                     "여기까진 알았네"
   "이것도 몰라?"                     "어디서 헷갈렸을까"
   "집중해서 풀어"                    "다시 한 번 같이 보자"
       │                                  │
       ▼                                  ▼
   대화 종료                          대화 시작
       │                                  │
       ▼                                  ▼
   아이: 입 닫음                      아이: 설명 시도
       │                                  │
       ▼                                  ▼
   메타인지 ↓                         메타인지 ↑
```

**연결고리:** 부모의 언어는 아이가 자기 사고를 들여다보는 거울입니다. 결과만 비추는 거울에서는 자기 사고 과정이 보이지 않습니다.

## 3. 학년별 적용 시기 (선수학습 → 후속학습)

```
[1~2학년]              [3~4학년]              [5~6학년]              [중학교]
──────────             ──────────             ──────────             ──────────

수 감각 형성     →     연산 정확도      →     개념 추상화      →     논리적 증명
                                                                        
틀려도 칭찬      →     과정 질문 도입   →     자기 채점 시작   →     오답 노트 자율
받는 경험              ("어떻게 풀었어?")     (스스로 ?)             관리

부모 채점        →     공동 채점        →     자기 채점        →     자기 평가
                       (옆에서 같이 보기)      (부모는 코칭만)         (부모는 신뢰)


[필수 토대]            [전환 시점]            [자율 단계]            [내면화]
"틀려도 안전하다"      "왜 틀렸는지              "내가 다시              "오답이 곧
                       함께 본다"               풀어본다"               배움이다"
```

**핵심:** 5~6학년에 갑자기 자기 주도 학습을 요구하면 무너집니다. 1~2학년부터 '틀려도 안전한 경험'이 누적되어야 자율 단계로 넘어갑니다. 늦었다고 판단되면 한 단계 앞으로 돌아가 공동 채점부터 다시 시작하면 됩니다.

## 4. 통합 흐름도 — 한 장으로 보는 회복 경로

```
        [현재 상태]
   "아이가 펜을 안 든다"
            │
            ▼
   ┌────────────────────┐
   │ 진단: 능력 문제 X  │
   │       감정 문제 O  │
   └────────┬───────────┘
            │
            ▼
   ┌─────────────────────────────────────────┐
   │  3가지 환경 동시 전환                   │
   │                                         │
   │  채점:  X  →  ?                         │
   │  언어:  결과 반응  →  과정 반응         │
   │  태도:  평가자  →  동행자               │
   └────────┬────────────────────────────────┘
            │
            ▼
   ┌────────────────────┐
   │ 1~2주: 펜이 움직임 │
   └────────┬───────────┘
            │
            ▼
   ┌────────────────────┐
   │ 3~4주: 질문이 생김 │
   └────────┬───────────┘
            │
            ▼
   ┌────────────────────┐
   │ 6~8주: 점수 변화   │
   └────────────────────┘
```

**주의:** 점수가 가장 마지막에 움직입니다. 펜이 움직이고, 질문이 생기고, 그다음에 결과가 따라옵니다. 순서가 거꾸로 되면 다시 멈춥니다.

## 5. 잘못된 인과 vs 올바른 인과

```
   [흔한 오해]
   문제풀이 양 ↑  →  점수 ↑

   [실제 작동]
   안전한 경험 ↑  →  자신감 ↑  →  시도 ↑  →  실력 ↑  →  점수 ↑
        △
        │
        └── 채점 방식 + 부모 언어가 만드는 환경
```

수학 점수가 낮은 아이에게 필요한 것은 더 많은 문제집이 아니라, 틀려도 괜찮다는 경험을 충분히 쌓을 시간입니다. 양은 자신감이 회복된 다음에 늘려도 늦지 않습니다.

## 6. 오늘부터 부모가 할 수 있는 3가지

| 영역 | 오늘 바꿀 것 | 다음 주 확인할 것 |
|------|------------|----------------|
| 채점 | 빨간 X 대신 연필 ? | 아이가 노트를 다시 여는가 |
| 언어 | "왜 틀렸어" → "어디서 헷갈렸어" | 아이가 자기 풀이를 설명하는가 |
| 태도 | 옆자리에 같이 앉기 | 아이가 모른다고 말할 수 있는가 |
