# 학부모 상담 완전 가이드

## 담임선생님이 내 아이를 제대로 보고 있는지 확인하는 질문 + 상담 준비 체크리스트

---

현직 초등교사가 드리는 솔직한 이야기입니다.

학부모 상담 15분.
대부분의 부모님이 성적을 물어봅니다.
그런데 성적은 가정통신문에 이미 나와 있습니다.

상담에서만 들을 수 있는 이야기가 따로 있습니다.
교실에서 내 아이가 어떤 사람인지.
그건 부모가 절대 볼 수 없는 장면입니다.

이 자료는 상담 전에 준비하고, 상담 중에 질문하고, 상담 후에 실천하는 전 과정을 담았습니다.

---

## [1] 상담 전 체크리스트: 우리 아이 관찰 메모

상담에 들어가기 전, 가정에서 본 아이의 모습을 먼저 정리하세요.
교사가 교실에서 본 모습과 비교할 수 있어야 상담이 깊어집니다.

| 확인 | 항목 | 메모란 |
|:---:|------|--------|
| [ ] | 요즘 아이가 자주 하는 말이나 표현이 있다 | |
| [ ] | 학교 이야기를 스스로 꺼내는 편이다 / 물어봐야 말한다 | |
| [ ] | 특정 친구 이름을 자주 언급한다 (누구: ) | |
| [ ] | 숙제나 준비물을 스스로 챙기는 편이다 / 도움이 필요하다 | |
| [ ] | 최근 감정 변화가 눈에 띈 적이 있다 (언제: ) | |
| [ ] | 학교 가기 싫다는 말을 한 적이 있다 (빈도: ) | |
| [ ] | 가정에서 걱정되는 행동이나 습관이 있다 | |
| [ ] | 아이가 요즘 빠져 있는 관심사가 있다 | |

이 메모를 상담 시작할 때 교사에게 먼저 공유하세요.
"집에서는 이런 모습인데, 교실에서는 어떤가요?"
이 한 마디가 상담의 질을 완전히 바꿉니다.

---

## [2] 상담 중 체크리스트: 영역별 핵심 질문 15가지

성적이 아니라 존재를 물어야 합니다.
아래 질문 중 상황에 맞는 5~7개를 골라 메모해 가세요.

### 또래 관계 (교실에서의 사회성)

| 확인 | 질문 | 이 질문이 알려주는 것 |
|:---:|------|---------------------|
| [ ] | "우리 아이, 교실에서 어떤 친구인가요?" | 또래 속에서 아이의 위치와 역할 |
| [ ] | "쉬는 시간에 주로 누구와, 어떻게 보내나요?" | 자발적 관계 형성 능력 |
| [ ] | "모둠 활동에서 어떤 역할을 맡는 편인가요?" | 협력 능력과 리더십/팔로워십 |
| [ ] | "친구와 갈등이 생기면 어떻게 해결하나요?" | 문제 해결 방식과 감정 조절력 |

### 학습 태도 (점수가 아닌 과정)

| 확인 | 질문 | 이 질문이 알려주는 것 |
|:---:|------|---------------------|
| [ ] | "어떤 순간에 가장 집중하나요?" | 아이의 강점 영역과 몰입 패턴 |
| [ ] | "수업 중 모르는 게 생기면 어떻게 하나요?" | 도움 요청 능력, 학습 자세 |
| [ ] | "발표나 의견 나누기에 참여하는 편인가요?" | 자기표현 능력과 자신감 수준 |
| [ ] | "과제를 끝까지 마무리하는 편인가요?" | 과제 지속력과 책임감 |

### 감정과 태도 (교실에서의 정서)

| 확인 | 질문 | 이 질문이 알려주는 것 |
|:---:|------|---------------------|
| [ ] | "교실에서 아이의 표정이 주로 어떤가요?" | 학교생활 만족도와 정서 상태 |
| [ ] | "힘들어 보이거나 위축된 적이 있었나요?" | 교사가 포착한 정서적 신호 |
| [ ] | "칭찬받을 때와 지적받을 때 반응이 어떤가요?" | 자존감 수준과 피드백 수용력 |
| [ ] | "선생님께 먼저 다가와서 이야기하는 편인가요?" | 어른과의 관계 형성 방식 |

### 가정 연계 (부모가 도울 수 있는 것)

| 확인 | 질문 | 이 질문이 알려주는 것 |
|:---:|------|---------------------|
| [ ] | "가정에서 어떤 부분을 도와주면 좋을까요?" | 교사가 본 아이의 현재 필요 |
| [ ] | "아이에게 꼭 전해주고 싶은 말이 있으신가요?" | 교사의 관심 깊이와 관찰력 |
| [ ] | "2학기(다음 학기)까지 함께 지켜볼 부분이 있을까요?" | 장기적 성장 방향 |

## [3] 상담 후 체크리스트: 실천 계획 기록

상담이 끝난 뒤 24시간 안에 정리하세요.
기억은 빠르게 흐려집니다. 기록은 1년을 갑니다.

### 교사에게 들은 핵심 이야기

| 확인 | 항목 | 기록 |
|:---:|------|------|
| [ ] | 교실에서 아이의 강점 | |
| [ ] | 교사가 걱정하거나 지켜보는 부분 | |
| [ ] | 가정에서 해주면 좋겠다고 한 것 | |
| [ ] | 가정의 모습과 교실의 모습이 달랐던 점 | |

### 가정 실천 계획

| 확인 | 항목 | 구체적 계획 |
|:---:|------|------------|
| [ ] | 이번 주부터 시작할 것 1가지 | |
| [ ] | 아이에게 전할 교사의 긍정 피드백 | |
| [ ] | 다음 상담 때 다시 확인할 질문 | |

---

## 상담을 망치는 3가지 vs. 살리는 3가지

### 이렇게 하면 15분이 아깝습니다

1. "성적이 왜 이런가요?" -- 교사를 방어적으로 만듭니다. 성적 원인은 함께 찾는 것이지 추궁하는 것이 아닙니다.
2. 다른 아이와 비교하는 질문 -- "반에서 몇 등인가요?"는 교사가 답할 수 없고, 답해서도 안 되는 질문입니다.
3. 준비 없이 "별 말씀 없으시면..."으로 끝내기 -- 교사도 준비한 이야기가 있습니다. 질문이 없으면 꺼내기 어렵습니다.

### 이렇게 하면 15분이 1년을 바꿉니다

1. "집에서는 이렇습니다" 먼저 공유하기 -- 교사가 더 솔직하게 이야기할 수 있는 분위기가 만들어집니다.
2. 아이의 존재를 묻는 질문하기 -- "어떤 친구인가요?"라는 질문을 받은 교사는 생각합니다. '이 부모님은 진심이구나.'
3. 상담 후 실천하고, 다음 상담에서 공유하기 -- "지난번 말씀대로 해봤는데 이렇게 변했어요." 이 한 마디가 교사와 부모 사이에 신뢰를 만듭니다.

---

## 학년별 상담 포인트

모든 학년에 위의 질문을 쓸 수 있지만, 학년에 따라 특히 중요한 영역이 있습니다.

| 학년 | 특히 확인할 영역 | 이유 |
|------|----------------|------|
| 1~2학년 | 또래 관계, 학교 적응 | 처음 단체생활을 시작하는 시기. 친구 사귀는 방식이 형성됩니다. |
| 3~4학년 | 학습 태도, 감정 표현 | 교과가 어려워지고, 또래 비교가 시작되는 시기입니다. |
| 5~6학년 | 자기 표현, 교우 갈등 | 사춘기 초입. 가정과 교실의 모습이 달라지기 시작합니다. |
