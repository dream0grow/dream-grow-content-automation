# 초등 분수 개념학습 로드맵

## 학년별 분수 개념 발달 지도

부모가 한눈에 보는, 분수가 어디서 시작해서 어디로 이어지는지

---

## 1. 분수 학습의 전체 흐름

```
[1~2학년]                [3학년]              [4학년]
전체와 부분 ────────▶  단위분수 ────────▶ 분모 같은
(똑같이 나누기)         (1/2, 1/3)         분수 덧뺄셈
                                            가분수/대분수
                                                │
                                                ▼
[6학년]                 [5학년]
분수의 나눗셈   ◀───── 약분, 통분
비와 비율                분모 다른 분수 덧뺄셈
백분율, 비례식           분수의 곱셈
```

**읽는 법**: 왼쪽이 무너지면 오른쪽도 무너집니다. 6학년 비와 비율에서 책을 덮는 아이는 사실 3학년 단위분수에서 막힌 경우가 많습니다.

---

## 2. 핵심 개념 지도: '분수란 무엇인가'

```
                    분수 (Fraction)
                         │
        ┌────────────────┼────────────────┐
        ▼                ▼                ▼
    [부분-전체]      [몫(나눗셈)]      [비(비율)]
   3학년 도입        4~5학년 확장      6학년 연결
        │                │                │
        ▼                ▼                ▼
   피자 1/4조각      3 ÷ 4 = 3/4      남:여 = 2:3
                                       = 2/5와 3/5
```

**부모가 알아야 할 점**: 분수는 한 가지 뜻이 아닙니다. 학년이 올라가면서 같은 1/2도 의미가 달라집니다. "1/2이 뭐야?"라는 질문에 아이가 한 가지 대답만 할 수 있다면, 다음 단계에서 막힙니다.

---

## 3. 단계별 선수학습 ↔ 후속학습 연결도

### 3학년 — 분수의 첫 만남

```
[선수학습]                              [후속학습]
                                        
똑같이 나누기 ──────▶ 단위분수 ──────▶ 진분수
(1~2학년)            1/2, 1/3, 1/4     2/3, 3/4
                          │
                          ▼
                    분모/분자 용어
```

**무너지는 신호**: "1/2과 1/3 중 뭐가 더 커?"에 1/3이라고 답함 (숫자 크기로 판단)

**가정 활동**: 종이 한 장을 반으로 접고, 한 장을 3등분으로 접어서 직접 비교

---

### 4학년 — 분수의 확장

```
[선수학습]                              [후속학습]
                                        
단위분수 ──────────▶ 가분수/대분수 ──▶ 분모 같은
진분수               5/4, 1과 1/4       분수 덧뺄셈
(3학년)                  │              2/5 + 1/5
                         ▼
                    분수의 크기 비교
```

**무너지는 신호**: 1과 1/2을 3/2로 못 바꿈. 1/2 + 1/3 = 2/5라고 적음

**가정 활동**: 피자/케이크를 자르며 "1판하고 반 더는 몇 조각?" 물어보기

---

### 5학년 — 분수의 핵심 고비

```
[선수학습]                              [후속학습]
                                        
분모 같은 ─────────▶ 약분, 통분 ─────▶ 분모 다른
분수 덧뺄셈          (공약수, 공배수    분수 덧뺄셈
가분수/대분수         가 선수)           분수의 곱셈
(4학년)                   │
                          ▼
                    크기가 같은 분수
                    (2/4 = 1/2)
```

**무너지는 신호**: 분모가 다른 분수를 더할 때 무엇부터 해야 할지 모름. 약분을 절차로만 외움

**가정 활동**: 종이 띠를 잘라 1/2 = 2/4 = 3/6이 같은 길이임을 눈으로 확인

---

### 6학년 — 분수의 통합

```
[선수학습]                              [후속학습]
                                        
분모 다른 분수 ─────▶ 분수의 ──────▶ 비와 비율
덧뺄셈, 곱셈          나눗셈            백분율
(5학년)                   │              비례식
                          ▼              (중학 수학)
                    분수 = 비율
                    1/4 = 25%
```

**무너지는 신호**: 비와 비율에서 책을 덮음. "왜?"를 못 설명함

**가정 활동**: 할인율, 시청률, 야구 타율 등 일상 속 비율을 분수로 바꿔보기

## 4. 무너지는 지점 진단표

| 시기 | 핵심 개념 | 무너지면 보이는 신호 | 돌아가야 할 곳 |
|------|-----------|---------------------|----------------|
| 3학년 | 단위분수 | 1/3 > 1/2라고 답함 | 1~2학년 똑같이 나누기 |
| 4학년 | 가분수/대분수 | 5/4를 못 그림 | 3학년 단위분수 |
| 5학년 | 통분 | 분모 다르면 멈춤 | 4학년 분수 크기 비교 |
| 5학년 | 분수 곱셈 | "왜 곱하는데 작아져?" | 3학년 부분-전체 의미 |
| 6학년 | 분수 나눗셈 | 뒤집어 곱하기만 외움 | 5학년 약분/통분 의미 |
| 6학년 | 비와 비율 | 책을 덮음 | 3~5학년 전체 분수 의미 |

**활용법**: 아이가 막힌 학년이 아니라, 그 위 학년에서 무너졌을 가능성이 큽니다. 한 학년 아래로 돌아가서 손으로 만지고 그림으로 그려보세요.

## 5. Bruner의 학습 3단계와 분수

인지심리학자 Jerome Bruner는 학습이 세 단계로 진행된다고 했습니다.

```
[1단계] 행동적 표상 (Enactive)
       └─ 손으로 만지기
          종이 접기, 피자 자르기, 레고 나누기
                  │
                  ▼
[2단계] 영상적 표상 (Iconic)
       └─ 그림으로 보기
          분수막대, 원그림, 수직선
                  │
                  ▼
[3단계] 상징적 표상 (Symbolic)
       └─ 숫자로 쓰기
          1/2, 2/3, 5/4
```

**문제는 여기**: 대부분 가정에서는 1단계와 2단계를 건너뛰고 3단계부터 시작합니다. 그러면 절차는 외워도 개념은 비어 있게 됩니다.

**원칙**: 새로운 분수 개념이 나오면 반드시 1단계 → 2단계 → 3단계 순서를 지키세요. 어른에게는 빨라 보여도 아이에게는 이 순서가 학습의 골격입니다.

## 6. 부모가 가정에서 확인할 수 있는 3가지 질문

아이가 분수를 진짜 이해했는지 확인하는 질문입니다. 답을 못 하면 다시 1단계로 돌아가야 합니다.

**질문 1.** "1/2을 그림으로 그려볼래?"
- 정답: 원, 사각형, 막대 등 다양하게 그릴 수 있어야 함
- 무너진 신호: 숫자 1/2만 다시 적음

**질문 2.** "1/2 + 1/3은 왜 2/5가 아닐까?"
- 정답: "조각의 크기가 다르니까"라고 설명
- 무너진 신호: "그냥 통분해야 해서"라고만 답함

**질문 3.** "왜 그렇게 풀었어?"
- 정답: 자기 말로 과정을 설명
- 무너진 신호: "선생님이 그렇게 하랬어"

## 7. 오늘부터 시작하는 4주 실천 가이드

| 주차 | 활동 | 목표 |
|------|------|------|
| 1주 | 종이 접기 (1/2, 1/4, 1/8) | 단위분수 감각 |
| 2주 | 피자/케이크 그림 그리기 | 부분-전체 시각화 |
| 3주 | 같은 크기 분수 찾기 (1/2=2/4) | 통분의 기반 |
| 4주 | "왜?" 질문으로 자기설명 | 절차에서 개념으로 |

하루 10분이면 충분합니다. 문제집을 한 권 더 푸는 것보다, 종이 한 장을 접는 것이 더 멀리 갑니다.
