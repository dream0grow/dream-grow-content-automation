# 초등수학 영역별 로드맵

## 무너지지 않는 순서로 다지는 6년의 지도

---

### 들어가며

초등 수학에서 아이가 헤매는 순간은 능력의 문제가 아니라 **계열의 문제**일 때가 많습니다. 5학년 약수에서 무너진 아이를 거슬러 가보면 3학년 곱셈에 구멍이 있고, 6학년 분수 나눗셈을 못 푸는 아이는 4학년 분수의 덧셈에서 이미 흐릿해진 경우가 대부분입니다.

이 자료는 초등 수학 5개 영역(수와 연산 / 도형 / 측정 / 규칙성 / 자료와 가능성)을 학년별 계열 지도로 정리한 것입니다. 우리 아이가 지금 어디에 서 있고, 무엇을 먼저 다져야 다음 칸으로 넘어갈 수 있는지 한눈에 보실 수 있습니다.

---

## 1. 수와 연산 — 가장 길고, 가장 자주 무너지는 영역

```
[1학년]                [2학년]                [3학년]
한 자리 수 ─────▶ 두 자리 수 ──────▶ 세 자리·네 자리 수
한 자리 덧·뺄셈 ─▶ 받아올림 덧셈 ───▶ (가로/세로 셈)
                  곱셈구구 시작 ────▶ 곱셈(두 자리×한 자리)
                                     나눗셈 도입(÷)
                                     분수 처음 만남(1/2, 1/3)

[4학년]                [5학년]                [6학년]
큰 수(억·조) ───────▶ 약수와 배수 ──────▶ 분수의 나눗셈
곱셈(세 자리×두 자리) 최대공약수·최소공배수  비와 비율
나눗셈(세 자리÷두 자리)분수의 곱셈·나눗셈    백분율
분수의 덧·뺄셈 ─────▶ 소수의 곱셈 ──────▶ (앞 단계 모두 합쳐짐)
소수의 덧·뺄셈
```

**선수학습 → 후속학습 연결**

- 곱셈구구(2학년) → 곱셈·나눗셈(3·4학년) → 약수와 배수(5학년) → 분수 통분(5학년) → 분수 나눗셈(6학년)
- 분수 도입(3학년) → 분수의 크기·덧뺄셈(4학년) → 분수의 곱셈·나눗셈(5·6학년) → 비와 비율(6학년)

**여기서 무너지면 다음이 무너진다**

| 무너지는 지점 | 영향받는 다음 학습 |
|---|---|
| 곱셈구구가 자동화 안 됨 | 나눗셈 전체, 약수, 통분 |
| 두 자리 곱셈이 흐릿함 | 큰 수 곱셈, 분수의 곱셈 |
| 분수가 '두 숫자'로 보임 | 통분, 비와 비율, 중1 유리수 |

## 2. 도형 — 직관에서 정의로, 정의에서 측정으로

```
[1~2학년]              [3~4학년]              [5~6학년]
입체·평면도형 직관 ─▶ 변·꼭짓점·각 ────▶ 다각형의 둘레·넓이
사각형·삼각형·원       각도(도, °)            합동과 대칭
                      삼각형의 종류          원의 넓이
                      사각형의 종류          직육면체·정육면체
                                            원기둥·원뿔·구
                                            겉넓이·부피
```

**선수학습 → 후속학습 연결**

- 도형 직관(1·2학년) → 도형 정의(3·4학년) → 도형 측정(5·6학년) → 중1 평면도형의 성질
- 각도(4학년) → 삼각형 내각의 합 → 다각형 내각 → 중학교 도형 증명

**놓치기 쉬운 포인트**

- 4학년 '각도' 단원은 도형의 정의로 들어가는 관문입니다. 여기서 '예각·둔각'이 헷갈리면 5학년 다각형 분류에서 다시 막힙니다.
- 도형은 손으로 그려본 만큼 압니다. 자·각도기·컴퍼스 사용 경험이 부족하면 6학년 '입체도형의 겉넓이'에서 갑자기 어려워집니다.

## 3. 측정 — 일상에서 자라야 하는 영역

```
[1학년]            [2학년]              [3학년]
길이 비교 ───────▶ cm, m ────────────▶ mm, km
                  시각·시간 도입         들이(L, mL)·무게(g, kg)
                                       시간 계산(시·분·초)

[4학년]            [5학년]              [6학년]
각도(°) ─────────▶ 다각형의 넓이 ─────▶ 원의 넓이
                  (직사각형·평행사변형·  원주율
                   삼각형·사다리꼴·       직육면체의 부피
                   마름모)               원기둥의 부피
```

**선수학습 → 후속학습 연결**

- 길이 단위(2·3학년) → 넓이 단위(5학년) → 부피 단위(6학년) → 중학교 단위환산 전체
- 시간 계산(3학년) → 비와 속력(6학년) → 중1 일차방정식 활용(거리·속력·시간)

**가장 자주 막히는 단원**

5학년 1학기 '다각형의 넓이'. 이유는 단원 자체가 어려워서가 아니라, 곱셈(연산 영역)·도형(도형 영역)·단위(측정 영역) 세 영역이 한꺼번에 만나기 때문입니다. 한 영역만 흐릿해도 무너집니다.

## 4. 규칙성 — 작지만 중학 수학의 뿌리

```
[1~3학년]              [4~5학년]              [6학년]
규칙 찾기 ──────────▶ 규칙과 대응 ────────▶ 비례식과 비례배분
(수·도형의 배열)       (□와 △의 관계식)       비의 성질
                                            비례 관계
```

**선수학습 → 후속학습 연결**

- 규칙과 대응(5학년) → 비례식(6학년) → 중1 함수의 시작 → 중2 일차함수
- 이 영역은 분량이 적어서 가볍게 보이지만, **중학교 함수 개념의 뿌리**입니다.

## 5. 자료와 가능성 — 정보를 읽는 힘

```
[1~3학년]              [4학년]                [5~6학년]
표 만들기 ────────▶ 막대그래프 ────────▶ 평균(5학년)
간단한 그래프         꺾은선그래프            비율그래프(6학년)
                                            (띠그래프·원그래프)
                                            가능성(6학년)
```

**선수학습 → 후속학습 연결**

- 그래프 읽기(4학년) → 평균(5학년) → 비율그래프(6학년) → 중1 통계의 시작
- 가능성(6학년) → 중2 확률

**현장에서 본 것**

이 영역은 시험 비중이 작아 부모도 아이도 가볍게 넘기지만, 6학년 후반의 '비율그래프'에서 분수·소수·백분율·비율·그래프 읽기가 한꺼번에 요구되어 갑자기 어려워집니다. 평소 신문·뉴스의 그래프를 같이 읽어두면 큰 자산이 됩니다.

## 영역 간 연결망 — 한 영역만 따로 자라지 않는다

```
                    [수와 연산]
                   /     |     \
                  /      |      \
            [측정]──────[도형]
                  \      |      /
                   \     |     /
                  [규칙성]─[자료와 가능성]
```

- **수와 연산**은 다른 모든 영역의 도구입니다. 곱셈이 흐리면 도형의 넓이도, 측정의 단위환산도, 규칙성의 대응도 흔들립니다.
- **도형 ↔ 측정**: 5학년 '다각형의 넓이'에서 만나고, 6학년 '입체도형의 부피'에서 다시 만납니다.
- **규칙성 ↔ 수와 연산**: 6학년 비와 비율은 분수·소수·백분율의 종합 시험장입니다.

## 가정에서 다지는 활동 — 영역별 한 줄 가이드

| 영역 | 일상에서 쌓는 활동 |
|---|---|
| 수와 연산 | 식탁·마트에서 수 세기, 묶어 세기, 나눠 담기. "피자 8조각 중 3조각 남았네"가 분수의 시작입니다. |
| 도형 | 종이접기, 블록쌓기, 도형 찾기 산책. 입체감각은 화면이 아니라 손에서 자랍니다. |
| 측정 | 요리 계량, 키재기, 장보기 가격 비교, 시계 보고 약속 잡기. |
| 규칙성 | "엘리베이터 버튼 5층 다음은?", 빨래 개키며 짝 맞추기, 동요의 박자 세기. |
| 자료와 가능성 | 뉴스·날씨 그래프 함께 읽기, 일주일 용돈 기록표 만들기. |

## 무너짐 신호 체크 — 학년별 점검 한 가지

- **1~2학년**: 받아올림·받아내림에서 손가락이 사라지지 않는다면, 수의 합성·분해(10 만들기)부터 다시.
- **3~4학년**: 두 자리 곱셈에서 자릿값이 어긋난다면, 곱셈구구 자동화부터 다시.
- **5학년**: 약수와 배수가 흐릿하다면, 곱셈·나눗셈을 거꾸로 보는 연습부터 다시.
- **6학년**: 비와 비율에서 막힌다면, 분수의 의미(전체와 부분)부터 다시.
