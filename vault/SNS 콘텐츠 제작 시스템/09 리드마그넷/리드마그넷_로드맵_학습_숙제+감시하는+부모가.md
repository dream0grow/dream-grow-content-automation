# 자기주도학습 부모 조력자 로드맵

**초등 1~6학년 · 감시자에서 조력자로 전환하는 12주 실천 가이드**

---

## 시작하며

숙제 옆에 붙어 틀린 문제를 바로 지적하는 것은 성실한 양육이 아니라, 아이의 내재동기를 소모하는 행위입니다. 데시와 라이언의 자기결정성이론은 내재동기가 **자율성·유능감·관계성** 세 축에서 나온다고 말합니다. 감시는 이 세 축을 동시에 무너뜨립니다.

자기주도학습은 '혼자 하는 공부'가 아닌 **'함께 설계하되, 실행은 맡기는 공부'**입니다. 이 로드맵은 부모의 위치를 단계적으로 뒤로 물리는 과정을 담았습니다.

---

## 1부. 학년별 자기주도학습 로드맵

학년이 올라갈수록 부모의 물리적·심리적 거리는 **멀어져야** 합니다. 아래 표는 학년별로 부모가 '얼마만큼 물러설지'를 기준으로 설계했습니다.

| 학년 | 부모의 위치 | 아이가 스스로 하는 것 | 부모가 아직 돕는 것 | 피해야 할 것 |
|------|-------------|----------------------|---------------------|--------------|
| 1학년 | 같은 책상 옆 | 준비물 꺼내기, 연필 깎기, 숙제 시작하기 | 문제 읽어주기, 전체 흐름 안내 | 연필 쥐는 법까지 교정하기 |
| 2학년 | 같은 책상 맞은편 | 오늘 할 일 순서 정하기, 시간 재기 | 모르는 단어 설명 | 틀린 답 즉시 지적 |
| 3학년 | 같은 공간 다른 자리 | 답지로 스스로 채점, 틀린 문제 표시 | 틀린 이유 함께 분석 | 채점을 부모가 대신하기 |
| 4학년 | 옆방 또는 거실 반대편 | 주간 학습 계획 세우기, 우선순위 정하기 | 계획 타당성 질문하기 | 부모가 계획표 만들어 주기 |
| 5학년 | 눈에 보이지 않는 거리 | 과목별 공부법 선택, 오답노트 작성 | 공부법 사례 공유 | 참고서 종류까지 정해주기 |
| 6학년 | 요청 시 등장 | 학습 목표 설정, 진도 점검, 자기평가 | 중학 준비 방향 의논 | "엄마 없으면 공부 안 하지?" 발언 |

**핵심 원칙:** 학년이 바뀌어도 아이가 이전 단계를 해내지 못한다면, 학년 표가 아닌 **아이가 실제로 도달한 단계**에서 다시 시작합니다. 나이는 숫자일 뿐 자율성은 경험의 산물입니다.

---

## 2부. 12주 감시자→조력자 전환 로드맵

어제까지 옆에 붙어 채점하던 부모가 갑자기 사라지면 아이는 불안합니다. 12주에 걸쳐 **단계적 후퇴**를 연습하세요.

### 1단계 (1~3주차): 환경 재배치

| 주차 | 목표 | 구체적 활동 | 완료 확인 |
|------|------|-------------|-----------|
| 1주 | 부모의 물리적 위치 1m 뒤로 | 아이 옆자리 → 같은 방 다른 의자로 이동. 부모는 책을 편다. | [ ] 3일 이상 실천 |
| 2주 | 채점 권한을 아이에게 이양 | 답지를 아이에게 건네고 "다 풀면 네가 먼저 맞춰봐"라고 말한다. | [ ] 주 3회 이상 성공 |
| 3주 | 실시간 개입 금지 | 문제 푸는 중 말 걸지 않기. 다 끝난 뒤에만 대화한다. | [ ] 하루도 개입 안 한 날 3일 |

### 2단계 (4~6주차): 질문으로 전환

| 주차 | 목표 | 구체적 활동 | 완료 확인 |
|------|------|-------------|-----------|
| 4주 | 지적을 질문으로 바꾸기 | "이거 틀렸어" → "이 문제는 어떻게 풀었어?" | [ ] 질문형 전환 주 10회 |
| 5주 | 오답을 함께 분석하되 답은 아이가 | 아이가 설명하게 하고, 부모는 "그래서 다음엔?" 만 묻는다. | [ ] 아이가 먼저 원인을 말한 경험 3회 |
| 6주 | 칭찬을 과정으로 이동 | "100점이네" → "끝까지 앉아 있었네", "어제보다 빨라졌네" | [ ] 결과 칭찬 대신 과정 칭찬 일주일 |

### 3단계 (7~9주차): 계획권 이양

| 주차 | 목표 | 구체적 활동 | 완료 확인 |
|------|------|-------------|-----------|
| 7주 | 오늘 할 일 목록을 아이가 작성 | A4 한 장에 오늘 할 일 쓰기. 부모는 "도와줄 거 있어?" 만 묻는다. | [ ] 아이가 직접 쓴 목록 7장 |
| 8주 | 주간 계획표를 아이 손으로 | 일요일 저녁 10분, 한 주 계획을 함께 앉아 '아이가' 채운다. | [ ] 2주 연속 아이 주도 작성 |
| 9주 | 계획 실패를 경험하게 두기 | 계획 어긋나도 구조하지 않는다. 주말에 같이 원인을 돌아본다. | [ ] 실패→복기 대화 1회 이상 |

### 4단계 (10~12주차): 거리 두기 완성

| 주차 | 목표 | 구체적 활동 | 완료 확인 |
|------|------|-------------|-----------|
| 10주 | 같은 공간 벗어나기 | 부모는 거실, 아이는 방. "필요하면 불러"라고 말한다. | [ ] 다른 공간 공부 주 4회 |
| 11주 | 자기평가 도입 | 숙제 끝난 뒤 아이가 스스로 별점(1~5)과 이유를 쓴다. | [ ] 자기평가 기록 7회 |
| 12주 | 부모의 역할 재정의 선언 | "앞으로 엄마/아빠는 네가 부를 때만 갈게"를 말로 약속한다. | [ ] 실제로 선언하고 3일 지킴 |

## 3부. 매일 쓰는 조력자 점검표

하루 3분, 잠들기 전 스스로 체크하세요.

- [ ] 오늘 아이 숙제에 손대지 않았다
- [ ] 틀린 문제를 먼저 지적하지 않았다
- [ ] 답지는 아이가 먼저 보게 했다
- [ ] 속도를 재촉하지 않았다
- [ ] 결과가 아닌 과정에 대해 한 마디 했다
- [ ] 아이가 먼저 도움을 청할 때만 개입했다
- [ ] 같은 공간에서 나는 내 일(독서, 업무)을 했다

**5개 이상:** 조력자 단계 안정 진입  
**3~4개:** 전환 중. 한 가지만 더 붙잡아 보세요  
**2개 이하:** 아직 감시자 단계. 1단계 환경 재배치부터 다시 시작

## 4부. 전환기 자주 겪는 상황

**Q. 숙제를 안 하고 버티면 어떻게 하나요?**  
개입하지 않습니다. 안 해간 결과를 교실에서 경험하는 것도 학습입니다. 단, 학습 자체가 어려워서 회피하는 경우라면 난이도를 낮춰 '할 수 있는 경험'을 먼저 만들어 주세요. 유능감이 자율성의 바탕입니다.

**Q. 틀린 걸 빤히 보는데 가만히 있어야 하나요?**  
채점 단계까지 기다리세요. 푸는 중 지적은 자율성을 무너뜨리고, 채점 후 대화는 메타인지를 키웁니다. 같은 정보라도 타이밍이 효과를 결정합니다.

**Q. 아이가 계획을 엉성하게 세워요.**  
완벽한 계획보다 **스스로 만든 계획**이 중요합니다. 일주일 실행 후 함께 돌아보며 아이가 직접 수정하게 하세요. 계획력은 실패를 해봐야 자랍니다.
