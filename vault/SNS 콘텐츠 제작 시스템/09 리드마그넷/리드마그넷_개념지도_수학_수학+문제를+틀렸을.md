# 초등 수학 메타인지 질문 개념지도
## 틀린 문제 앞에서 부모가 던져야 할 질문의 지도

---

## 1. 시작하기 전에

틀린 문제는 실패가 아닙니다. 아이 머릿속에서 길이 어긋난 지점을 보여주는 **지도**입니다. 부모의 역할은 채점이 아니라 그 지도를 함께 읽는 것입니다.

이 자료는 "왜 틀렸어"를 대체할 메타인지 질문이, 어떤 사고 과정과 연결되고, 학년에 따라 어떻게 발달하는지 한 장에 담은 개념지도입니다.

---

## 2. 중심 개념지도 — 질문의 두 갈래

```
                  ┌───────────────────────┐
                  │  틀린 문제 = 사고의 지도  │
                  └───────────┬───────────┘
                              │
              ┌───────────────┴───────────────┐
              ▼                               ▼
       [추궁형 질문]                     [탐색형 질문]
       평가 신호                          탐색 신호
              │                               │
       사고 차단                         사고 확장
              │                               │
       방어 반응                         되짚기 반응
              │                               │
       정답 베끼기                       오류 자가 발견
              │                               │
       고정 마인드셋 강화                 성장 마인드셋 강화

      "왜 틀렸어?"                     "어디까지 맞다고 생각했어?"
      "이걸 왜 몰라?"                  "이 문제가 뭘 묻고 있을까?"
      "다시 풀어."                     "다른 길은 없을까?"
```

핵심: 같은 시험지를 보더라도, **질문이 평가로 들리느냐 탐색으로 들리느냐**에 따라 아이의 다음 행동이 달라집니다.

---

## 3. 사고 3단계 지도 — 어디서 길을 잃었나

수학 문제 풀이는 세 단계의 사고로 구성됩니다. 틀린 지점이 어느 단계인지 찾아야 다음 질문이 정확해집니다.

```
       [1단계] 이해                [2단계] 풀이                [3단계] 검토
       독해 사고                   논리 사고                   검증 사고
          │                           │                           │
   "무엇을 묻고 있나"           "어떻게 식을 세웠나"          "답이 말이 되나"
          │                           │                           │
   ▼ 부모 질문                  ▼ 부모 질문                  ▼ 부모 질문
   "이 문제 한 번             "어디까지 맞다고            "지금 답이 너한테
   다른 말로 바꿔             생각했어?"                  말이 돼?"
   설명해줄래?"                                            
   "여기서 묻는 게             "그 다음에 뭐가              "단위가 맞아?"
   딱 한 가지가 뭐야?"         헷갈렸어?"                  "어림셈으로 비슷해?"
   "여기서 핵심                "이 식 말고                  "친구한테 설명한다면
   단어가 뭐야?"               다른 방법은?"                어떻게?"
          │                           │                           │
   ↓ 막히면                    ↓ 막히면                    ↓ 막히면
   문제 자체를                 사고 과정을                 정답을 다시
   안 읽었다는 신호            중간에 잃었다는 신호        의심하지 않았다는 신호
```

자주 틀리는 아이의 80%는 1단계에서 길을 잃습니다. **계산이 아니라 독해 단계에서 무너지는 것**입니다.

## 4. 학년별 질문 발달 지도

질문은 학년에 따라 깊어져야 합니다. 너무 빨리 추상적인 질문을 던지면 아이는 입을 닫고, 너무 단순한 질문에 머물면 사고가 자라지 않습니다.

```
[1~2학년] 인지 시작기
   질문 키워드: "보여줘"
   ────────────────────
   "이 문제 그림으로 그려볼래?"
   "손가락으로 한 번 세어볼까?"
   "어디까지 맞다고 생각해?"
                │
                │  (선수: 구체물 조작)
                ▼
[3~4학년] 사고 형성기
   질문 키워드: "왜 그렇게 했어"
   ────────────────────
   "이 식을 어떻게 세운 거야?"
   "이 문제가 너한테 뭘 묻고 있을까?"
   "다른 방법으로도 풀 수 있을까?"
                │
                │  (선수: 자기 풀이 설명)
                ▼
[5~6학년] 메타인지 확장기
   질문 키워드: "스스로 점검"
   ────────────────────
   "이 문제, 어떤 유형의 문제 같아?"
   "어디서 막힐 줄 미리 알았어?"
   "답이 나왔는데, 그 답이 합리적이야?"
                │
                │  (후속: 중학교 서술형/증명)
                ▼
[중학교로 연결]
   "근거를 말로 풀어 쓰기" 단계로 이어짐
```

**선수-후속 관계**: 1~2학년의 "그림으로 그려볼래"가 5~6학년의 "어떤 유형 같아"로 자랍니다. 단계를 건너뛰면 사고의 토대가 비어버립니다.

## 6. 추궁 → 탐색 변환 지도 (실전 카드)

| 추궁형 (사고 차단) | 탐색형 (사고 확장) | 어떤 사고를 여는가 |
|---|---|---|
| 왜 틀렸어 | 어디까지 맞다고 생각했어 | 자기 풀이 되짚기 |
| 이걸 왜 몰라 | 이 문제가 너한테 뭘 묻고 있을까 | 문제 독해 |
| 집중 안 했지 | 어디서부터 헷갈리기 시작했어 | 사고 단절 지점 찾기 |
| 다시 풀어 | 같은 문제를 다른 방법으로 풀 수 있을까 | 풀이 다양화 |
| 이런 것도 몰라 | 비슷한 문제 본 적 있어 | 유형 인식 |
| 빨리 풀어 | 천천히, 한 줄씩 말로 풀어볼래 | 사고 외현화 |
| 답이 뭐야 | 너의 답이 말이 된다고 생각해 | 검증 사고 |
| 이거 쉬운 거야 | 이 문제에서 어려운 게 뭐였어 | 난이도 자기 인식 |
| 엄마가 가르쳐줄게 | 너라면 이걸 어떻게 설명할 거야 | 설명을 통한 정리 |
| 시험 못 보면 어떡해 | 오늘 이 문제로 뭘 새로 알게 됐어 | 학습 의미화 |

## 7. 교실에서 본 풍경

6학년 담임 시절 일입니다. 분수 나눗셈을 틀린 아이에게 "왜 틀렸어?"라고 물었더니, 그 아이는 답만 슥 지우고 정답을 옮겨 적었습니다. 사고 과정은 사라지고 숫자만 바뀐 것입니다.

다음 날 같은 유형의 다른 문제에서 그 아이는 또 틀렸습니다. 당연한 결과였습니다. **틀린 답을 고친 게 아니라, 틀린 답을 가렸을 뿐**이었으니까요.

그래서 질문을 바꿨습니다. "이 문제가 너한테 뭘 묻고 있는 것 같아?"  
아이는 한참 문제를 들여다보더니 말했습니다. "어, 나누는 게 아니라 곱하는 건가?"  
그 순간이 이 자료의 출발점입니다.

---

## 8. 부모가 자주 빠지는 함정

- **함정 1**: 탐색형 질문을 던지고도 답을 기다리지 못하고 곧바로 설명으로 넘어감
- **함정 2**: 질문 후 아이의 답이 틀렸을 때 "그게 아니지"로 받아쳐 다시 추궁이 됨
- **함정 3**: 한 번 질문해보고 효과 없으면 "역시 안 통한다"고 포기
- **함정 4**: 부모가 모르는 단원에서 질문이 멈춤 (아는 사람만 좋은 질문을 할 수 있다고 착각)

핵심: 좋은 질문은 부모가 답을 알기 때문에 던지는 게 아닙니다. **답을 모를 때도 던질 수 있는 질문**이 메타인지 질문입니다.
