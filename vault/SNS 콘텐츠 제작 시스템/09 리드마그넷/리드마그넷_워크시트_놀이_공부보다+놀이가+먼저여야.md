# 놀이 x 내재동기 워크시트: 우리 아이는 놀이에서 무엇을 채우고 있을까?

**대상:** 초등학생 자녀를 둔 부모 (자녀와 함께 작성)

---

## 이 워크시트를 시작하기 전에

아이가 학원에서 돌아와 지친 표정으로 소파에 누울 때, 부모는 생각합니다. "그래도 안 보내면 뒤처지니까." 15년 넘게 교실에서 아이들을 보면서 확인한 사실이 있습니다. 공부가 먼저인 아이는 금방 지치고, 놀이가 먼저인 아이는 스스로 앉습니다.

심리학자 데시(Deci)와 라이언(Ryan)의 자기결정성이론에 따르면, 사람이 스스로 움직이려면 세 가지가 채워져야 합니다.

| 요소 | 뜻 | 놀이에서 채워지는 순간 |
|------|-----|----------------------|
| **자율성** | 내가 선택하고 결정한다는 느낌 | "오늘 술래잡기 하자!" 아이가 직접 놀이를 정할 때 |
| **유능감** | 내가 해낼 수 있다는 느낌 | "나 오늘 한 번도 안 잡혔어!" 성취를 경험할 때 |
| **관계성** | 누군가와 연결되어 있다는 느낌 | 친구와 규칙을 정하고, 부모와 눈을 맞추며 놀 때 |

이 세 가지가 놀이 속에서 자연스럽게 채워진 아이는, 시키지 않아도 스스로 책상 앞에 앉습니다. 학원이 아니라 놀이가 내재동기의 출발점입니다.

이 워크시트는 부모와 아이가 함께 작성하며, 우리 아이의 놀이 속에 자율성/유능감/관계성이 얼마나 채워지고 있는지 점검하는 활동지입니다.

## [활동 2] 자율성/유능감/관계성 체크하기

활동 1에서 "제일 재미있었던 놀이"를 기준으로 아래 질문에 답해주세요. 아이가 직접 O/X로 표시합니다.

### 자율성 (내가 정했다는 느낌)

| 질문 | O / X |
|------|:-----:|
| 이 놀이는 내가 하고 싶어서 시작했다 | |
| 놀이 규칙을 내가 정하거나 바꿀 수 있었다 | |
| 하기 싫으면 그만둘 수 있었다 | |
| 어른이 "이렇게 해"라고 시키지 않았다 | |

O 개수: \_\_\_\_ / 4

### 유능감 (내가 해냈다는 느낌)

| 질문 | O / X |
|------|:-----:|
| 놀이 중에 "나 잘한다!"고 느낀 순간이 있었다 | |
| 어려운 부분을 스스로 해결한 적이 있다 | |
| 처음보다 더 잘하게 된 것이 있다 | |
| 누군가에게 "잘했어"라는 말을 들었다 | |

O 개수: \_\_\_\_ / 4

### 관계성 (함께한다는 느낌)

| 질문 | O / X |
|------|:-----:|
| 이 놀이를 같이 한 사람이 있었다 | |
| 놀이 중에 웃거나 즐거운 감정을 나눴다 | |
| 서로 도와주거나 양보한 적이 있다 | |
| 놀이가 끝난 뒤에도 "또 하자"고 말했다 | |

O 개수: \_\_\_\_ / 4

## [활동 3] 우리 아이 내재동기 지도 그리기

활동 2의 O 개수를 아래에 옮겨 적고, 해당하는 칸까지 색칠해주세요.

```
자율성  |______|______|______|______|
         1      2      3      4

유능감  |______|______|______|______|
         1      2      3      4

관계성  |______|______|______|______|
         1      2      3      4
```

### 결과 해석

**3~4개 채워진 영역:** 놀이 속에서 이 요소가 충분히 채워지고 있습니다. 지금처럼 유지해주세요.

**1~2개 채워진 영역:** 이 요소가 부족합니다. 아래 활동 4에서 보충 방법을 확인하세요.

**0개인 영역:** 아이의 놀이 환경을 점검해볼 필요가 있습니다. 부모가 놀이를 주도하고 있지는 않은지, 아이가 놀이 자체를 즐기고 있는지 대화해보세요.

## [활동 4] 부족한 요소 채우는 10분 놀이 처방전

활동 3에서 부족한 영역에 해당하는 놀이를 골라, 이번 주에 하나만 시도해보세요.

### 자율성이 부족할 때

| 놀이 | 방법 | 핵심 |
|------|------|------|
| "네가 선생님" 놀이 | 아이가 부모에게 무엇이든 가르치는 놀이. 주제, 방법, 순서 모두 아이가 정한다 | 부모는 배우는 역할에만 충실한다 |
| 규칙 바꾸기 게임 | 이미 알고 있는 보드게임/카드게임의 규칙을 아이가 새로 만든다 | "이상한 규칙"이어도 따라간다 |

### 유능감이 부족할 때

| 놀이 | 방법 | 핵심 |
|------|------|------|
| 기록 깨기 도전 | 줄넘기, 종이비행기 날리기, 컵 쌓기 등 측정 가능한 놀이를 반복하며 자기 기록을 갱신한다 | 다른 아이와 비교하지 않는다. 어제의 나와 비교한다 |
| 미션 클리어 놀이 | 아이가 할 수 있는 수준의 미션 5개를 적어두고 하나씩 완료 표시한다 | 미션 난이도는 "살짝 도전적"이 적당하다 |

### 관계성이 부족할 때

| 놀이 | 방법 | 핵심 |
|------|------|------|
| 합동 작품 만들기 | 레고, 그림, 요리 등을 부모와 아이가 역할을 나눠 함께 완성한다 | 결과물보다 과정에서의 대화가 목적이다 |
| 이야기 이어가기 | "옛날에 고양이 한 마리가..."로 시작해 한 문장씩 번갈아 이야기를 만든다 | 정답이 없으므로 아이가 어떤 말을 해도 받아준다 |

## [활동 5] 놀이 중 부모의 말 점검표

놀이의 효과는 부모의 반응에 달려 있습니다. 아래 표를 냉장고에 붙여두세요.

| 피해야 할 말 (놀이가 수업이 됩니다) | 효과적인 반응 (놀이가 내재동기가 됩니다) |
|----------------------------------|--------------------------------------|
| "이건 이렇게 하는 거야" | "어떻게 하고 싶어?" |
| "이거 해볼래?" (부모가 정한 놀이) | "오늘 뭐 하고 놀까?" (아이가 정하는 놀이) |
| "잘했어!" (평가) | "와, 아까보다 더 높이 쌓았네!" (구체적 관찰) |
| "조심해", "안 돼" (통제) | "어떻게 하면 안전할까?" (아이가 생각하게) |
| "이제 그만하고 공부해" (전환 강요) | "5분 뒤에 마무리하자, 마지막으로 뭐 하고 싶어?" |
