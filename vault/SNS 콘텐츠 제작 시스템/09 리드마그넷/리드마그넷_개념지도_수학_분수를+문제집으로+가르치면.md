# 초등수학 양감 로드맵
## 분수·소수·비율을 한 줄로 잇는 개념지도

**대상:** 초등 3~6학년 자녀를 둔 부모
**목적:** 우리 아이가 어느 단계에 있는지, 어디서 무너지는지 한눈에 확인

---

## 이 자료를 보는 법

초등 3학년 분수 도입부터 중학교 1학년 함수까지, 모든 양감 개념은 한 줄로 이어져 있습니다.
어느 한 단계에서 양감이 비면, 그 다음 학년에서 반드시 무너집니다.
지금 우리 아이가 어느 지점에 서 있는지 확인하고, 부족한 단계를 채워주세요.

---

## 모든 것의 뿌리: 양감(量感)

```
                    [ 양감 (量感) ]
                          │
            수와 양을 몸으로 아는 감각
                          │
        ┌─────────────────┼─────────────────┐
        ↓                 ↓                 ↓
     크기 비교         절반·등분         단위 환산
   (많다·적다)      (똑같이 나누기)    (mL ↔ L)
```

양감이 없으면 1/2 + 1/3은 풀어도 "주스 1L의 절반"에서 멈춥니다.
양감이 있으면 계산 전에 "이 정도일 거야"라고 어림할 수 있습니다.

---

## 양감을 만드는 3단계 (브루너 학습 이론)

```
   [ 1단계 ] 행동 (Enactive)
       ↓   사과를 손으로 반으로 자른다
       ↓   주스를 컵에 따라본다
       ↓   피자를 6명이 나눈다
       │
       ▼
   [ 2단계 ] 영상 (Iconic)
       ↓   원을 그려 6등분한다
       ↓   막대 그림에 1/2을 표시한다
       ↓   수직선 위에 위치를 찍는다
       │
       ▼
   [ 3단계 ] 기호 (Symbolic)
       ↓   1/2, 0.5, 50%, 2:3
       ↓   분수의 덧셈·뺄셈·곱셈·나눗셈
       ↓   비례식, 함수
```

문제집은 **3단계 도구**입니다.
1·2단계가 비어 있으면, 3단계를 아무리 반복해도 6학년에서 무너집니다.

## 학년별 개념 연결 지도

### [3학년] 분수의 시작
```
  [선수: 2학년 곱셈, 등분 경험]
              │
              ▼
   전체와 부분 ──→ 똑같이 나누기 ──→ 분수의 도입
                                    (1/2, 1/3, 1/4)
              │
              ▼
   [후속: 4학년 분수의 크기 비교]
```
**핵심 양감 활동:** 종이 접기, 사과·빵 나누기, 물 따르기
**무너지는 신호:** "1/2이 1/3보다 크다"를 그림으로 설명하지 못함

---

### [3~4학년] 분수와 소수의 만남
```
   분수 (1/2, 1/4)  ←──→  소수 (0.5, 0.25)
         │                       │
         ▼                       ▼
   분모·분자 이해           자릿값 이해
         │                       │
         └────── 같은 양의 ──────┘
                두 가지 표현
```
**놓치면 위험:** 0.5 = 1/2이라는 감각이 없으면 5학년 약분·통분에서 무너집니다.
**확인 질문:** "1/2과 0.5는 같은 거야?"

---

### [5학년] 분수 연산의 본격화
```
   [선수: 4학년 곱셈, 등분 양감]
              │
              ▼
   약수·배수 ──→ 약분·통분 ──→ 분수의 덧셈·뺄셈
              │                      │
              └──→ 분수의 곱셈 ──→ 분수의 나눗셈
                                      │
                                      ▼
                          [후속: 6학년 비와 비율]
```
**놓치면 위험:** 분수의 나눗셈을 "뒤집어서 곱하기" 공식으로만 외우면 6학년에서 무너집니다.
**확인 질문:** "1/2 ÷ 1/4을 그림으로 설명해줄래?"

---

### [6학년] 비와 비율 — 가장 많이 무너지는 지점
```
   분수 감각 ─┐
              ├──→ 비 (2:3) ──→ 비율 (2/5) ──→ 백분율 (40%)
   소수 감각 ─┘                       │
                                      ▼
                              정비례 · 반비례
                                      │
                                      ▼
                          [후속: 중1 함수]
```
**놓치면 위험:** 비율을 공식으로 외운 아이는 중학교 함수와 그래프에서 모두 무너집니다.
**확인 질문:** "20%는 1/4보다 큰가, 작은가?"

---

### [중1] 함수와 비례
```
   6학년 비례 ──→ 정비례·반비례 ──→ 일차함수 ──→ 그래프
                                                     │
                                                     ▼
                                          [후속: 중2 함수, 중3 이차함수]
```
**무너지는 신호:** "y = 2x의 그래프를 그려봐"에서 멈춤
**원인 추적:** 거의 모두 6학년 비율, 5학년 분수, 3학년 양감으로 거슬러 올라갑니다.

## 부모가 확인할 5가지 연결고리

| 단계 | 확인 질문 | 통과 신호 |
|---|---|---|
| 1 | "사과 한 개의 절반은?" | 손으로 가르며 "이거"라고 답함 |
| 2 | "1/2과 0.5는 같아?" | "같은 양이야"라고 답함 |
| 3 | "1L의 절반은 몇 mL?" | 500mL라고 어림함 |
| 4 | "20%는 1/4보다 큰가, 작은가?" | "작아"라고 답함 |
| 5 | "2:3과 2/5는 무슨 관계?" | "같은 비율의 다른 표현"이라 답함 |

3번에서 멈추면 → 5학년 분수에서 무너집니다.
5번에서 멈추면 → 중학교 함수에서 무너집니다.

## 학년별 식탁 양감 대화 12선

**[3~4학년: 행동 단계]**
1. "빵의 1/3을 잘라줘."
2. "사과를 4명이 똑같이 나누면 한 사람 몫은?"
3. "물을 컵의 절반만 따라줘."

**[4~5학년: 영상 단계]**
4. "1L 우유의 1/4은 몇 mL일까?"
5. "피자를 6명이 나누면 2명이 먹는 양은?"
6. "1시간의 1/3은 몇 분?"

**[5~6학년: 기호 단계]**
7. "주스 2/5는 1/2보다 많아, 적어?"
8. "30%는 분수로 어떻게 써?"
9. "30% 할인된 만원짜리 옷, 얼마야?"

**[6학년~중1: 비례 단계]**
10. "물 2컵에 가루 3숟갈. 물 4컵이면 가루는?"
11. "지도에서 1cm가 실제 100m라면, 5cm는?"
12. "30분에 2km 걸으면 1시간엔 몇 km?"

## 무너지는 신호 vs 살아있는 신호

| 무너지는 신호 | 살아있는 신호 |
|---|---|
| 1/2 + 1/3은 풀지만 "주스 절반"에서 멈칫 | 계산 전에 "이 정도"라고 어림함 |
| 분수의 나눗셈을 공식으로만 푼다 | 그림으로 설명할 수 있다 |
| 0.5와 1/2이 같은지 답하지 못한다 | 분수·소수·백분율을 자유롭게 오간다 |
| 비례식은 풀지만 일상 적용이 안 된다 | "이 비율이면 얼마?"를 어림한다 |

## 핵심 정리

분수는 연산이 아니라 **양에 대한 감각**입니다.
양감이 자란 아이는 6학년 비와 비율을 자연스럽게 받아들이고,
중학교 함수와 그래프로 무리 없이 넘어갑니다.

문제집은 기호 단계의 도구입니다.
행동과 영상 단계가 비어 있으면, 문제집 100권을 풀어도 6학년에서 무너집니다.

오늘 저녁 식탁에서, 사과 한 알이 첫 시작입니다.
