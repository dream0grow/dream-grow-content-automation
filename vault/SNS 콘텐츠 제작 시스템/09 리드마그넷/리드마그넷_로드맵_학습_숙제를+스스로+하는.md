# 숙제를 스스로 하는 아이로 키우는 4주 로드맵

**초등 자녀를 둔 부모를 위한 단계별 실천 가이드**

---

## 이 자료를 읽기 전에

숙제를 감시할수록 아이의 자기주도성은 무너집니다.
교실에서 보면 혼자 문제를 풀어보는 아이와 한 글자도 못 쓰는 아이의 차이는 머리가 아닙니다.
집에서 틀렸을 때 어떤 반응을 받았는지의 차이입니다.

자기결정성 이론(Deci & Ryan)에 따르면, 아이가 스스로 움직이려면 세 가지가 필요합니다.

| 요소 | 의미 | 숙제 장면에서 |
|------|------|-------------|
| 자율성 | 내가 선택했다는 느낌 | 언제, 어떤 순서로 할지 스스로 정함 |
| 유능감 | 내가 해낼 수 있다는 느낌 | 틀려도 괜찮다는 경험이 쌓임 |
| 관계성 | 나를 믿어주는 사람이 있다는 느낌 | 결과가 아니라 과정을 봐주는 부모 |

부모가 숙제를 감시하고 오답을 지적하면, 이 세 가지가 동시에 무너집니다.
이 로드맵은 4주에 걸쳐 부모의 개입을 줄이고, 아이의 자기주도성을 키우는 구체적인 실천 계획입니다.

---

## 1주차: 말 바꾸기 (지시형 -> 선택형)

**목표:** 숙제 관련 대화에서 지시를 선택으로 바꾼다.

| 날짜 | 구체적 활동 | 확인 |
|------|-----------|------|
| 1일 | "숙제 다 했어?"를 "오늘 숙제 언제 할 거야?"로 바꿔본다 | [ ] |
| 2일 | "이거 틀렸잖아"를 "여기 다시 한번 읽어볼래?"로 바꿔본다 | [ ] |
| 3일 | "빨리 해"를 "몇 시까지 끝낼 계획이야?"로 바꿔본다 | [ ] |
| 4일 | "왜 이것도 못해?"를 "어디서 막혔어?"로 바꿔본다 | [ ] |
| 5일 | 숙제 시간에 지시형 언어를 몇 번 썼는지 스스로 세어본다 | [ ] |
| 6~7일 | 한 주간 가장 잘 바뀐 말 하나를 골라 메모한다 | [ ] |

**핵심 원리:** 말을 바꾸면 관계가 바뀝니다. "해라"에서 "어떻게 할래?"로 바뀌는 순간, 아이는 수동적 수행자에서 능동적 결정자가 됩니다.

**주의할 점:** 말만 바꾸고 표정과 어조가 여전히 감시하는 톤이면 아이는 금방 알아챕니다. 진짜로 아이의 대답을 기다려야 합니다.

---

## 2주차: 손 놓기 (물리적 거리두기)

**목표:** 숙제 시간에 부모가 옆에 붙어있지 않는 연습을 한다.

| 날짜 | 구체적 활동 | 확인 |
|------|-----------|------|
| 1일 | 아이가 숙제를 시작하면 같은 공간에서 부모도 자기 할 일을 한다 | [ ] |
| 2일 | 숙제하는 동안 아이 노트를 들여다보지 않는다 (10분) | [ ] |
| 3일 | 10분을 15분으로 늘린다 | [ ] |
| 4일 | 아이가 "다 했어"라고 말할 때까지 먼저 확인하지 않는다 | [ ] |
| 5일 | 아이가 도움을 요청할 때만 가서 봐준다 | [ ] |
| 6~7일 | 한 주간 아이가 스스로 한 시간을 기록한다 (분 단위) | [ ] |

**핵심 원리:** 부모가 물리적으로 떨어져야 아이에게 자율성의 공간이 생깁니다. 같은 방에 있되, 아이의 숙제가 아닌 자기 할 일에 집중하는 모습을 보여주세요. 아이는 부모의 행동을 보고 배웁니다.

**주의할 점:** 처음에는 아이가 더 자주 부를 수 있습니다. 이것은 실패가 아니라, 아이가 새로운 방식에 적응하는 과정입니다.

---

## 3주차: 오답 내려놓기 (결과에서 과정으로)

**목표:** 틀린 답을 지적하지 않고, 스스로 발견하게 돕는 습관을 만든다.

| 날짜 | 구체적 활동 | 확인 |
|------|-----------|------|
| 1일 | 숙제를 다 한 후 오답을 발견해도 지적하지 않는다 | [ ] |
| 2일 | 대신 "스스로 한번 검토해볼래?"라고만 말한다 | [ ] |
| 3일 | 아이가 검토 후에도 못 찾으면 "몇 번 문제 다시 읽어볼래?"로 범위만 좁혀준다 | [ ] |
| 4일 | 숙제 내용이 아닌 과정에 대해 말한다: "오늘 집중 잘 했네" | [ ] |
| 5일 | 틀린 문제를 아이가 다음 날 학교에서 스스로 해결하도록 맡긴다 | [ ] |
| 6~7일 | 아이에게 "이번 주 숙제하면서 어떤 느낌이었어?"라고 물어본다 | [ ] |

**핵심 원리:** 오답 지적은 유능감을 파괴합니다. 틀려도 괜찮다는 경험이 쌓여야, 아이는 도전하는 사람이 됩니다. 오답 수정은 교사의 역할입니다. 부모의 역할은 시도 자체를 인정해주는 것입니다.

**주의할 점:** "틀려도 괜찮아"라고 말하면서 한숨을 쉬면 아이는 말이 아니라 한숨을 기억합니다.

---

## 4주차: 루틴 만들기 (아이 주도 시스템 정착)

**목표:** 아이가 스스로 숙제 시간과 방법을 정하는 루틴을 만든다.

| 날짜 | 구체적 활동 | 확인 |
|------|-----------|------|
| 1일 | 아이와 함께 "숙제 시간"을 정한다 (부모가 정하는 것이 아님) | [ ] |
| 2일 | 아이가 정한 시간에 스스로 시작하는지 지켜본다 (재촉하지 않는다) | [ ] |
| 3일 | 시작이 늦어져도 결과가 아닌 "스스로 시작했다"는 사실을 인정한다 | [ ] |
| 4일 | 아이가 숙제 순서도 스스로 정하게 한다 (쉬운 것부터/어려운 것부터) | [ ] |
| 5일 | 숙제 후 "오늘 스스로 한 것 중 뭐가 제일 뿌듯해?"라고 물어본다 | [ ] |
| 6~7일 | 4주간의 변화를 아이와 함께 이야기한다 | [ ] |

**핵심 원리:** 루틴은 부모가 만들어주는 것이 아니라, 아이가 스스로 만들어야 지속됩니다. 부모의 역할은 루틴을 짜주는 것이 아니라, 아이가 루틴을 짤 수 있도록 기다려주는 것입니다.

**주의할 점:** 4주 만에 완벽한 변화를 기대하지 마세요. 3주차쯤 아이가 다시 도움을 요청하거나 늦어지는 건 자연스럽습니다. 후퇴가 아니라 정착 과정입니다.

## 자주 묻는 질문

**Q. 아이가 숙제를 안 하면 어떡하나요?**
자연적 결과를 경험하게 하세요. 숙제를 안 가져가면 학교에서 선생님께 이야기를 듣습니다. 그 경험이 부모의 잔소리보다 훨씬 강력한 동기가 됩니다. 단, 시험 기간처럼 결과가 큰 상황에서는 "내일 시험인데 계획이 있어?"라고 선택형으로 물어보세요.

**Q. 저학년(1~2학년)도 혼자 할 수 있나요?**
1~2학년은 완전한 자기주도보다 "함께 시작하고 혼자 마무리하는" 단계가 적절합니다. 처음 5분만 옆에서 방향을 잡아주고, 나머지는 스스로 하게 두세요. 학년이 올라갈수록 함께하는 5분을 줄여가면 됩니다.

**Q. 배우자가 여전히 감시형이면 어떡하나요?**
이 자료를 함께 읽어보시길 권합니다. 한쪽은 기다려주고 한쪽은 감시하면 아이는 혼란을 느낍니다. 부모가 먼저 "우리 아이 숙제에 대해 어떻게 할지" 대화를 나누는 것이 첫 번째 단계입니다.
