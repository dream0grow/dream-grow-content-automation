# 연령별 놀이 상호작용 가이드

**우리 아이와 함께 노는 법, 오늘부터 달라집니다**

## Part 1. 우리 집 놀이 현황 점검표

| 질문 | 그렇다 | 아니다 |
|------|:------:|:------:|
| 1. 아이가 하루 중 혼자 노는 시간이 함께 노는 시간보다 길다 | [ ] | [ ] |
| 2. 아이가 게임이나 놀이에서 지면 울거나 화를 낸다 | [ ] | [ ] |

| 4. 아이와 함께 논 시간이 이번 주에 30분 미만이다 | [ ] | [ ] |

**결과 해석**

- "그렇다" 0~2개: 놀이 상호작용이 잘 이루어지고 있습니다. Part 3의 대화 문장으로 더 깊은 놀이를 시도해 보세요.
- "그렇다" 3~5개: 함께 노는 경험을 조금 더 늘려야 합니다. Part 2의 연령별 가이드를 참고해 오늘 10분부터 시작하세요.
- "그렇다" 6개 이상: 놀이의 방향을 점검할 시기입니다. 이 워크시트를 처음부터 끝까지 따라가 보세요.

**이 시기 추천 놀이:** 보드게임(할리갈리, 루미큐브), 카드게임, 간단한 요리 함께 하기

### 9~10세 (초등 4~5학년)

**이 시기 추천 놀이:** 협력 보드게임(팬데믹, 코드네임), 퍼즐, 야외 스포츠, 함께 만들기 프로젝트

## 예시 답안 (Part 3 반응 문장)

| 상황 | 놀이를 살리는 말 |
|------|----------------|
| 블록을 엉뚱하게 쌓을 때 | "우와, 이건 어떤 건물이야?" |
| 규칙을 자기 맘대로 바꿀 때 | "재밌는 규칙이다. 그러면 이건 어떻게 되는 거야?" |
| 게임에서 졌을 때 | "아깝다. 이번 판은 어떤 작전으로 해볼까?" |
| 그림을 이상하게 그릴 때 | "이 부분이 재밌다. 여기는 뭘 그린 거야?" |
| 놀이 도중 포기할 때 | "잠깐 쉬었다 해도 돼. 아까 여기까지 한 거 멋졌어." |

핵심은 하나입니다. **평가하지 않고, 호기심으로 반응하는 것.**
"잘했어"도, "틀렸어"도 아닌, "그건 뭐야?"가 놀이를 이어가게 합니다.
