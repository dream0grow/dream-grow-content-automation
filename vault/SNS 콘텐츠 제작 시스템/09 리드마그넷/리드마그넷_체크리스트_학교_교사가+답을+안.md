# 아이 사고력을 키우는 부모의 되묻기 질문 30선

**대상:** 초등 자녀를 둔 부모
**활용법:** 일주일간 아이와 대화하며 직접 사용해본 질문에 체크하세요. 어떤 상황에서 되묻기가 어려웠는지, 어떤 질문이 아이의 사고를 깨웠는지 기록해두면 됩니다.

---

## 들어가며: 왜 '되묻기'인가

교실에서 교사가 답을 참는 건 방치가 아니라 **아이의 사고를 기다리는 시간**입니다.
교사는 두 개의 모자를 번갈아 씁니다.

- **발표자 모자**: 답을 전달할 때 (개념 설명, 시범)
- **진행자 모자**: 답을 알면서도 물을 때 (탐구, 토론)

두 모자를 동시에 쓰면 아이는 **'정답 맞히기 게임'**을 시작합니다.
부모도 마찬가지입니다. "이거 뭐예요?"에 즉답하지 않고 한 박자 늦추는 것, 그것이 사고력 교육의 시작입니다.

## PART 1. 상황별 되묻기 질문 30선

### A. 아이가 질문해 올 때 (10개)

답을 묻는 순간이 가장 좋은 사고 훈련 기회입니다.

- [ ] 1. "너는 어떻게 생각해?"
   *가장 기본. 즉답 대신 던지는 첫 질문.*
- [ ] 2. "왜 그게 궁금해졌어?"
   *호기심의 출발점을 짚어주면 사고가 확장됩니다.*
- [ ] 3. "비슷한 걸 본 적 있어?"
   *기존 경험과 연결하는 유추적 사고를 끌어냅니다.*
- [ ] 4. "만약 ○○이 없다면 어떻게 될까?"
   *조건을 바꾸는 가정적 사고 훈련.*
- [ ] 5. "한 단어로 표현하면 뭘까?"
   *추상화 능력을 키우는 압축 질문.*
- [ ] 6. "그 생각 어디서 시작됐어?"
   *생각의 출처를 되짚는 메타인지 질문.*
- [ ] 7. "다르게 설명할 수도 있을까?"
   *한 가지 답에 머물지 않게 하는 확장 질문.*
- [ ] 8. "엄마(아빠)도 모르겠는데, 같이 찾아볼까?"
   *부모도 모를 수 있다는 정직한 모델링.*
- [ ] 9. "그게 사실인지 어떻게 알 수 있을까?"
   *근거와 출처를 묻는 비판적 사고 질문.*
- [ ] 10. "지금 답이 떠오르지 않아도 괜찮아. 천천히 생각해볼래?"
    *침묵을 허용하는 안전한 사고 공간 제공.*

### B. 아이가 숙제할 때 (10개)

답을 알려주고 싶어도 참아야 하는 순간입니다.

- [ ] 11. "문제에서 묻고 있는 게 뭐야?"
    *문제 해석부터 다시. 가장 자주 빠뜨리는 단계.*
- [ ] 12. "아는 것부터 말해볼래?"
    *모르는 것이 아니라 아는 것에서 출발하기.*
- [ ] 13. "전에 비슷한 문제 풀어봤어? 그땐 어떻게 했어?"
    *과거 학습과 연결하는 전이 질문.*
- [ ] 14. "그림으로 그려보면 어떨까?"
    *추상을 시각으로 바꾸는 사고 도구.*
- [ ] 15. "어디까지 풀 수 있었어?"
    *막힌 지점을 본인이 인식하게 하는 질문.*
- [ ] 16. "이 답이 맞다고 생각하는 이유는?"
    *답보다 근거를 묻기. 정답이어도 묻습니다.*
- [ ] 17. "다른 방법으로 풀 수 있을까?"
    *해법이 하나가 아니라는 사고 유연성.*
- [ ] 18. "이 문제에서 가장 어려운 부분이 어디였어?"
    *어려움을 언어화하는 메타인지 훈련.*
- [ ] 19. "이게 왜 정답이 아니라고 생각해?"
    *오답에서 배우는 추론 질문.*
- [ ] 20. "선생님께 뭐라고 여쭤보면 좋을까?"
    *질문을 만드는 능력 자체를 키우는 질문.*

### C. 아이와 갈등이 생겼을 때 (10개)

감정과 사고를 동시에 다루는 가장 어려운 영역입니다.

- [ ] 21. "지금 기분이 어떤지 한 단어로 말해볼래?"
    *감정의 정확한 명명이 사고의 시작.*
- [ ] 22. "어떤 일이 있었는지 처음부터 말해줄래?"
    *서사적 재구성으로 거리두기.*
- [ ] 23. "그때 어떤 마음이었을까?"
    *상대방이 아닌 본인의 마음부터.*
- [ ] 24. "친구는 어떤 마음이었을 것 같아?"
    *조망 수용(타인 입장)으로 확장.*
- [ ] 25. "다시 그 순간으로 돌아간다면 어떻게 하고 싶어?"
    *과거를 사고 자료로 바꾸는 질문.*
- [ ] 26. "네가 원하는 결과는 뭐야?"
    *목표 지향적 사고로 전환.*
- [ ] 27. "그러려면 무엇이 필요할까?"
    *목표와 수단을 분리하는 사고.*
- [ ] 28. "이 일에서 네가 배운 게 있다면?"
    *경험을 학습으로 전환하는 질문.*
- [ ] 29. "엄마(아빠)가 어떻게 도와주면 좋겠어?"
    *주도권을 아이에게 돌려주는 질문.*
- [ ] 30. "다음에 비슷한 일이 생기면 어떻게 해볼래?"
    *미래 계획으로 마무리하는 닫는 질문.*

## PART 2. '정답 맞히기 게임'을 막는 부모 언어 7가지

아이가 부모 눈치를 보며 정답을 추측하기 시작하면 사고는 멈춥니다.
다음 7가지를 의식적으로 사용해보세요.

- [ ] 1. **"틀려도 괜찮아"** — 오답을 사고 과정의 일부로 인정.
- [ ] 2. **"엄마(아빠)도 잘 모르겠어"** — 부모의 무지를 정직하게 보이기.
- [ ] 3. **"흥미로운 생각이네"** — 정답/오답 판정 대신 사고 자체를 인정.
- [ ] 4. **"왜 그렇게 생각했어?"** — 결론보다 과정에 주목.
- [ ] 5. **"한 번 더 생각할 시간 줄게"** — 침묵을 허용.
- [ ] 6. **"네 답이 맞을 수도 있어, 같이 확인해보자"** — 권위가 아닌 검증.
- [ ] 7. **"답보다 네가 어떻게 푸는지가 더 궁금해"** — 결과보다 과정 가치화.

## PART 3. 아이 답에 반응하는 3단계 후속 질문 흐름표

아이가 답을 했다고 끝이 아닙니다. 한 번의 답을 사고로 키우는 3단계입니다.

| 단계 | 목적 | 질문 예시 |
|------|------|-----------|
| **1단계: 받아주기** | 답을 평가하지 않고 인정 | "그렇게 생각하는구나." / "아, 그런 관점이 있네." |
| **2단계: 깊이 파기** | 근거와 과정을 물음 | "왜 그렇게 생각했어?" / "어떻게 그 답에 도달했어?" |
| **3단계: 확장하기** | 다른 가능성으로 열기 | "다르게 볼 수도 있을까?" / "반대로 생각하면 어떨까?" |

세 단계를 모두 거치면 한 번의 답이 5분짜리 사고 훈련이 됩니다.
처음에는 어색해도 일주일만 의식하면 자연스러워집니다.

## 자가 점검 가이드

이번 주에 사용해본 질문에 체크한 후, 개수를 세어보세요.

| 체크 개수 | 해석 |
|-----------|------|
| **0~5개** | 시작 단계입니다. 하루에 1~2개만 골라 시도해보세요. 부담 없이. |
| **6~15개** | 되묻기 습관이 자라고 있습니다. 어려운 상황(갈등, 숙제)에서도 시도해보세요. |
| **16~25개** | 가정에 사고력 문화가 정착 중입니다. 아이의 답이 길어졌다면 신호입니다. |
| **26~30개** | 부모가 이미 '진행자 모자'를 자유자재로 씁니다. 이제 아이가 부모에게 되묻는지 관찰해보세요. |

**중요:** 개수보다 **꾸준함**이 핵심입니다. 매일 1개씩, 100일이면 충분합니다.

오늘 아이가 "이거 뭐예요?"라고 물어오거든, 한 박자 멈추고 되물어보세요.
