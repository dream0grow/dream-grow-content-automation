# 초등 수학 개념지도
## 분수를 중심으로 본 1~6학년 연결 구조

**대상:** 초등 1~6학년 자녀를 둔 부모
**활용법:** 인쇄해서 책상 옆에 붙여두고, 아이가 막힐 때마다 "지금 어디쯤 있는지" 함께 확인하세요.

---

## 1. 왜 '분수'가 뿌리인가

초등 수학은 단원이 따로 노는 것처럼 보이지만, 실제로는 **하나의 큰 줄기**로 연결되어 있습니다. 그 줄기의 뿌리가 분수입니다.

> **분수가 흔들리면 — 소수, 비율, 백분율, 비례식, 중학 함수까지 차례로 무너집니다.**

분수는 단순한 '한 단원'이 아니라, 초등 수학 후반부 전체의 **언어**입니다.

## 2. 핵심 개념지도 (한눈에 보기)

```
                    [전체를 똑같이 나눈 부분]
                              │
                              ▼
        ┌─────────────────[분수]─────────────────┐
        │                     │                    │
        ▼                     ▼                    ▼
    [소수]               [비와 비율]            [나눗셈]
        │                     │                    │
        ▼                     ▼                    ▼
    [백분율]            [비례식·비례배분]      [약수·배수]
        │                     │                    │
        └─────────┬───────────┘                    │
                  ▼                                │
            [중학 함수·통계]◀───────────────────────┘
```

**핵심 원리:** 위쪽 개념이 흔들리면 아래쪽은 반드시 무너집니다. 아래에서 막혔다면, **위쪽 뿌리부터 다시** 봐야 합니다.

## 3. 학년별 등장 시기와 연결고리

| 학년 | 단원/개념 | 뿌리가 되는 개념 | 막히면 영향받는 후속 학습 |
|------|-----------|------------------|---------------------------|
| 1학년 | 가르기·모으기, 10의 보수 | (수의 양감) | 받아올림, 받아내림 |
| 2학년 | 곱셈구구, 똑같이 나누기 | 가르기·모으기 | 나눗셈, 분수의 분모 개념 |
| 3학년 | **분수의 의미, 나눗셈** | 똑같이 나누기 | 분수 연산, 소수, 비율 |
| 4학년 | 분수의 덧셈·뺄셈, 소수 | 분수의 의미 | 분수 곱셈·나눗셈, 소수 연산 |
| 5학년 | **약수·배수, 분수의 곱셈, 통분** | 곱셈구구, 분수 | 분수의 나눗셈, 비율 |
| 6학년 | **분수의 나눗셈, 비와 비율, 백분율, 비례식** | 분수, 약수·배수 | 중학 일차함수, 통계 |

**굵게 표시된 학년**이 가장 흔히 막히는 지점입니다.

## 4. '분수' 줄기 따라가기 — 6단계 흐름

```
[1단계] 똑같이 나누기 (2학년)
   "사과 6개를 3명이 똑같이 = 2개씩"
                ▼
[2단계] 분수의 의미 (3학년)
   "전체를 똑같이 나눈 부분 중 몇 개"
   2/3 = 전체를 3등분 한 것 중 2개
                ▼
[3단계] 분수의 크기 비교·동치분수 (4학년)
   "왜 1/2 = 2/4 = 4/8 일까?"
                ▼
[4단계] 통분과 분수 연산 (4~5학년)
   "분모가 다르면 같게 만들고 더한다"
                ▼
[5단계] 분수의 곱셈·나눗셈 (5~6학년)
   "분수 ÷ 분수 = 분수 × 역수"
                ▼
[6단계] 비·비율·백분율 (6학년)
   "비율은 결국 분수다. 3:4 = 3/4 = 0.75 = 75%"
```

**부모가 기억할 것:** 아이가 5단계에서 막혔다면, **3단계의 동치분수 개념**이 비어있을 가능성이 높습니다.

## 5. 옆가지 — 분수와 함께 자라는 개념

### 소수 줄기
```
[10의 자리값 이해] → [소수 첫째자리] → [소수 연산] → [분수↔소수 변환]
       (2학년)         (3~4학년)        (4~5학년)        (6학년)
```
**연결점:** 0.1 = 1/10 이라는 사실을 '외우는' 게 아니라 **그릴 수 있어야** 합니다.

### 비율 줄기
```
[똑같이 나누기] → [분수] → [비와 비율] → [백분율] → [비례식]
   (2학년)       (3학년)   (6학년)      (6학년)    (6학년)
```
**연결점:** "반 학생 25명 중 10명이 안경" = 10/25 = 2/5 = 40%. **모두 같은 말**임을 아이가 손으로 그려 확인해야 합니다.

### 나눗셈 줄기
```
[똑같이 나누기] → [나눗셈] → [약수·배수] → [공약수·공배수] → [통분] → [분수 연산]
   (2학년)       (3학년)     (5학년)        (5학년)        (5학년)   (5학년)
```
**연결점:** 통분을 못하는 아이는 대부분 **공배수 개념이 비어있습니다.**

아이가 분수에서 막혔다면, 학년에 상관없이 아래 질문부터 던져보세요.

| 번호 | 질문 | 못 하면 돌아갈 단계 |
|------|------|---------------------|
| 1 | "사과 한 개를 똑같이 4조각으로 그려볼래?" | 2학년 - 똑같이 나누기 |
| 2 | "그중에 3조각을 색칠하면 분수로 어떻게 써?" | 3학년 - 분수의 의미 |

| 4 | "1/2 과 2/4 는 같을까 다를까?" | 4학년 - 동치분수 |
| 5 | "1/2 + 1/3 은 왜 2/5 가 아닐까?" | 5학년 - 통분 |

**1번부터 차례로** 물어보세요. 멈추는 지점이 아이가 '진짜 막힌 곳'입니다.

## 8. 부모가 기억할 3가지 원칙

**원칙 1. 문제집은 진단 도구이지, 해결책이 아니다**
틀린 문제 = 어디서 막혔는지 알려주는 신호. 같은 유형을 더 풀게 하는 게 아니라, **위 뿌리 단계로 돌아가야** 합니다.

**원칙 2. 손끝의 경험이 기호보다 먼저다 (CPA 순서)**
**C**oncrete(구체물) → **P**ictorial(그림) → **A**bstract(기호). 이 순서를 건너뛰면 반드시 무너집니다. 분수는 사과부터, 비율은 음료수 농도부터.

**원칙 3. 설명하게 하라**
아이가 푼 문제를 다시 **부모에게 설명하게** 하세요. 설명할 수 있으면 안 거고, 못 하면 모르는 겁니다. 이건 정답률보다 정확한 진단입니다.

## 9. 자주 묻는 질문

**Q. 학년이 한참 지났는데 이제 와서 뿌리로 돌아가도 될까요?**
A. 됩니다. 오히려 그게 가장 빠른 길입니다. 무너진 위에 또 쌓는 것보다, 단단한 바닥부터 다시 다지는 편이 훨씬 빠릅니다.

**Q. 학원을 보내야 할까요?**
A. 학원은 '진도'를 빼주지만, '뿌리'는 부모와의 일상에서 자랍니다. 하루 10분 부엌에서의 대화가, 한 시간의 문제 풀이보다 강력할 때가 많습니다.

**Q. 아이가 너무 싫어해요.**
A. 분수를 '문제집'으로 만나면 누구나 싫어합니다. 음식, 시간, 돈으로 만나면 다릅니다. 수학을 책상 위가 아닌 식탁 위로 옮겨주세요.
