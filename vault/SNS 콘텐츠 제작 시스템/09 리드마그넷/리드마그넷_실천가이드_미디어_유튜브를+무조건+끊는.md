# 우리 집 디지털 쉼표 설계 가이드

### 유튜브를 끊지 않고도 도파민 회로의 방향을 바꾸는 법

**대상**: 초등 자녀의 스마트폰·유튜브 사용을 고민하는 부모

---

## 시작하기 전에

스마트폰을 압수해도 아이의 멍한 눈빛은 사라지지 않습니다. 문제는 도파민이 아니라, 도파민이 흐르는 회로입니다. 같은 회로라도 유튜브에 연결되면 중독이 되고, 학습과 몰입에 연결되면 성취가 됩니다.

이 가이드는 **끊는 방식이 아니라 전환하는 방식**으로, 가정에서 5단계로 실천할 수 있도록 정리한 자료입니다. 한 단계당 일주일씩, 5주에 걸쳐 천천히 적용해 보시기를 권합니다.

---

## STEP 1. 우리 집 사용 현황 진단하기

**무엇을**: 일주일 동안 아이의 스마트폰 사용 시간과 상황을 기록합니다.

**어떻게**:
- iOS '스크린타임' 또는 안드로이드 '디지털 웰빙'에서 일별 사용 시간 확인
- 아래 표에 시간대별 패턴 기록

| 요일 | 총 사용시간 | 가장 많이 본 앱 | 주로 보는 시간대 |
|------|------------|----------------|-----------------|
| 월   |            |                |                 |
| 화   |            |                |                 |
| 수   |            |                |                 |
| 목   |            |                |                 |
| 금   |            |                |                 |
| 토   |            |                |                 |
| 일   |            |                |                 |

**왜**: 회로의 방향을 바꾸려면 현재 회로가 언제, 어떤 자극으로 켜지는지 알아야 합니다. 측정 없이 바꾸려 하면 부모와 아이 모두 감정싸움으로 끝납니다.

**주의점**: 진단 단계에서는 잔소리하지 않습니다. 기록 자체가 목적입니다.

---

## STEP 2. 가족 디지털 쉼표 약속 정하기

**무엇을**: 일주일 중 하루, 가족 전원이 스마트폰을 멈추는 날을 정합니다.

**어떻게**:
- 토요일 오전 또는 일요일 저녁 등 가족이 모이는 시간으로 선택
- 부모도 동일하게 참여 (가장 중요한 조건)
- 현관 또는 거실에 '쉼표 바구니'를 두고 식구 모두 핸드폰을 넣어둠
- 시간은 4시간부터 시작, 익숙해지면 8시간으로 확장

**왜**: 혼자 끄면 처벌이지만, 같이 끄면 의식이 됩니다. 6학년 아이도 부모가 함께 참여하는 약속에는 반발하지 않습니다. 압수의 본질은 권력의 위계지만, 쉼표의 본질은 가족의 합의입니다.

**주의점**: 부모가 한 번이라도 예외를 만들면 약속은 무너집니다. 업무 통화가 필요하면 미리 가족에게 말하고 다른 공간에서 짧게 처리합니다.

---

## STEP 3. 작은 보상 회로 설계하기

**무엇을**: 학습 행동 직후에 짧고 빠른 보상을 매칭합니다.

**어떻게**: 아래 매칭표 중 우리 아이에게 맞는 조합을 3가지만 골라 시작합니다.

| 학습 행동 | 보상 (15분 이내) |
|----------|-----------------|
| 책 30분 읽기 | 좋아하는 간식 한 조각 |
| 영어 단어 10개 외우기 | 보드게임 한 판 |
| 수학 문제 1쪽 풀기 | 부모와 5분 산책 |
| 일기 한 편 쓰기 | 좋아하는 음악 한 곡 듣기 |
| 학교 숙제 완료 | 가족과 카드게임 한 판 |

**왜**: 도파민 회로는 '행동 → 즉각 보상'의 짧은 간격에서 강화됩니다. "한 달 동안 잘하면 게임기 사줄게"는 회로에 박히지 않습니다. 유튜브가 강력한 이유는 길이가 짧고, 보상이 즉각적이고, 매번 다르기 때문입니다. 이 구조를 학습에 옮겨 적용하는 것이 핵심입니다.

**주의점**: 보상은 반드시 학습 직후에 줍니다. "이따 줄게"는 효과가 절반으로 줄어듭니다. 또한 돈·게임 시간을 보상으로 쓰지 않습니다. 더 큰 자극에 의존하는 회로가 만들어집니다.

---

## STEP 4. 압수가 아닌 전환 대화로 바꾸기

**무엇을**: 스마트폰을 빼앗는 말 대신, 회로를 옮기는 말로 바꿉니다.

**대화 스크립트 예시**:

> **이전**: "유튜브 그만 봐. 지금 당장 꺼."
> **전환**: "10분 뒤에 같이 간식 먹으려고 하는데, 그때까지 보고 끌까?"

> **이전**: "또 보고 있어? 압수한다."
> **전환**: "지금 본 영상 중에 제일 재밌었던 거 한 개만 엄마한테 설명해줄래?"

> **이전**: "숙제부터 해. 그게 끝나야 폰 줘."
> **전환**: "수학 한 쪽 끝내고 나면 우리 둘이 보드게임 한 판 어때?"

**왜**: 압수는 아이의 회로를 끄지 못하고, 부모를 향한 적대감을 추가할 뿐입니다. 전환은 같은 시간에 다른 회로를 켜는 방식입니다. 6학년 아이의 멍한 눈빛은 자극이 사라져서가 아니라, 자극을 옮길 출구가 없어서 생깁니다.

**주의점**: 전환 대화의 핵심은 '대안을 부모가 함께한다'는 것입니다. "보드게임 해"가 아니라 "같이 한 판 어때"가 작동합니다.

---

## STEP 5. 실패했을 때 회복 루틴 갖기

**무엇을**: 약속이 깨졌을 때 죄책감 없이 다시 시작하는 방식을 미리 정해둡니다.

**어떻게**:
1. 약속이 깨진 날, 그날은 재시도하지 않고 그대로 흘려보냄
2. 다음 날 아침 식사 자리에서 단 한 문장만 공유: "어제는 우리 다 같이 어려웠지. 오늘은 다시 해보자."
3. 이번 주 약속 시간을 30분 줄여서 다시 시작 (난이도 낮추기)
4. 일주일 뒤 다시 원래 시간으로 복귀

**왜**: 실패에 처벌이 따라붙으면 아이는 다음 시도를 두려워합니다. 회복 루틴이 있는 약속만이 6주, 6개월, 6년을 갑니다. 끊는 게 목표가 아니라 회로의 방향을 바꾸는 게 목표라면, 실패는 데이터일 뿐입니다.

**주의점**: 부모가 약속을 어긴 경우에도 똑같이 적용합니다. 부모의 회복 모습이 아이의 회복력을 만듭니다.

## 자주 묻는 질문

**Q1. 아이가 디지털 쉼표 자체를 거부하면 어떻게 하나요?**
첫 주는 30분으로 시작해도 됩니다. 시간보다 '가족 전원 동참'이 핵심입니다. 부모가 먼저 핸드폰을 바구니에 넣는 모습을 일주일만 보여주면, 대부분의 아이는 따라옵니다.

**Q2. 보상으로 간식을 자주 주면 식습관이 망가지지 않을까요?**
간식은 한 조각, 한 입 단위입니다. 양보다 '즉각성'이 회로를 만듭니다. 견과류, 과일 한 조각, 좋아하는 음료 한 모금으로도 충분합니다.

**Q3. 부모가 일 때문에 핸드폰을 못 끄면 어떻게 하나요?**
업무 시간과 쉼표 시간을 분리합니다. 쉼표 4시간이 어렵다면 저녁 식사 1시간부터 시작합니다. 길이보다 일관성이 중요합니다.
