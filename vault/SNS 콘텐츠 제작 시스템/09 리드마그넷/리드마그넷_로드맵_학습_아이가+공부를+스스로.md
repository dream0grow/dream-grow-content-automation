# 자기주도학습 부모 언어 가이드

**4주 완성 로드맵 — 평가자에서 동행자로**

초등 자녀 부모를 위한 단계별 실천 자료

---

## 시작하기 전에

"공부해라"라고 말하지 않으면 아이가 책상에 앉지 않습니다. 칭찬 스티커를 붙여도, 보상을 약속해도, 잠깐일 뿐입니다.

문제는 아이의 의지가 아니라 **구조**입니다. 결정하는 사람이 부모로 고정되어 있을 때, 아이는 '내가 선택한 일'이라는 감각을 잃어버리거든요. 공부하는 아이를 만드는 것은 칭찬이 아니라 **선택권**입니다.

이 자료는 4주 동안 부모의 언어를 단계적으로 바꾸는 로드맵입니다. 한 번에 모든 것을 바꾸려 하지 마시고, 한 주에 하나씩 천천히 옮겨가시면 됩니다.

---

## 교실에서 본 작은 변화

3학년을 맡았을 때 학습 의욕이 가장 낮았던 한 아이가 있었습니다. 어머니께 한 가지만 부탁드렸어요. "오늘 숙제 다 했어?"를 "오늘 숙제 중에 뭐가 제일 헷갈렸어?"로 바꿔달라고요.

3주 뒤 아이는 처음으로 자기 입으로 "이 부분 다시 배우고 싶어요"라고 말했습니다. 칭찬이 아니라 **질문의 방향**이 아이를 깨운 순간이었습니다.

---

## 4주 로드맵 한눈에 보기

| 주차 | 핵심 목표 | 부모가 바꾸는 것 | 아이에게 일어나는 변화 |
|------|----------|----------------|--------------------|
| 1주차 | 결정권 이양 | 명령어 → 선택 질문 | "내가 골랐다"는 감각 회복 |
| 2주차 | 평가어 줄이기 | 결과 평가 → 과정 질문 | 점수에서 배움으로 시선 이동 |
| 3주차 | 메타인지 깨우기 | 정답 확인 → 과정 복기 | 자기 학습을 설명할 수 있음 |
| 4주차 | 동행자 자리 잡기 | 감독자 → 옆자리 | 스스로 계획하고 시작함 |

## 1주차 | 결정권 이양

**목표:** 하루 한 번, 부모가 정하던 것을 아이가 선택하게 둡니다.

| 요일 | 구체적 활동 | 부모 멘트 예시 | 확인 |
|------|-----------|--------------|------|
| 월 | 공부 순서 선택 | "수학과 독서 중 뭐 먼저 할래?" | [ ] |
| 화 | 시작 시간 선택 | "5시랑 6시 중 언제 시작할까?" | [ ] |
| 수 | 분량 선택 | "두 페이지 할래, 세 페이지 할래?" | [ ] |
| 목 | 장소 선택 | "책상에서 할래, 식탁에서 할래?" | [ ] |
| 금 | 휴식 선택 | "10분 쉬고 시작할까, 바로 할까?" | [ ] |
| 토 | 도구 선택 | "연필로 풀래, 색연필로 풀래?" | [ ] |
| 일 | 주간 회고 선택 | "이번 주 어떤 공부가 제일 재미있었어?" | [ ] |

**주의점:** 두 가지 선택지를 모두 부모가 받아들일 수 있는 범위 안에서 제시하세요. "공부할래, 안 할래?"는 선택이 아닙니다.

**1주차 완료 기준:** 7번 중 5번 이상 아이가 망설임 없이 선택했다.

---

## 2주차 | 평가어 줄이기

**목표:** 결과를 묻는 말 대신 과정을 묻는 말로 바꿉니다.

| 평소 자주 쓰는 말 | 바꿀 말 | 사용 횟수 |
|----------------|--------|---------|
| "몇 점 받았어?" | "어떤 부분이 어려웠어?" | [ ] [ ] [ ] |
| "다 풀었어?" | "어디까지 해봤어?" | [ ] [ ] [ ] |
| "잘했어" | "어떻게 푼 거야? 설명해 줄래?" | [ ] [ ] [ ] |
| "틀렸잖아" | "다시 보면 어디가 헷갈려?" | [ ] [ ] [ ] |
| "이거 쉬운 거야" | "처음 봤을 때 어떤 느낌이었어?" | [ ] [ ] [ ] |
| "왜 이걸 몰라?" | "이 부분은 누가 알려주면 좋을까?" | [ ] [ ] [ ] |
| "빨리 해" | "지금 어디쯤 가고 있어?" | [ ] [ ] [ ] |

**주의점:** 칭찬을 모두 없애지 않아도 됩니다. "잘했어"보다 "이 문제 푸는 방법이 신기하네, 어떻게 생각한 거야?"가 같은 따뜻함을 더 깊은 자국으로 남깁니다.

**2주차 완료 기준:** 하루에 평가어 3회 이하, 과정 질문 3회 이상.

---

## 3주차 | 메타인지 깨우기

**목표:** 아이가 자기 공부를 **설명할 수 있게** 만듭니다.

| 시점 | 묻는 질문 | 기대하는 답변 | 확인 |
|------|---------|------------|------|
| 공부 시작 전 | "오늘은 뭐부터 할 계획이야?" | 순서를 말로 표현 | [ ] |
| 공부 시작 전 | "이거 끝내는 데 시간 얼마나 걸릴 것 같아?" | 시간을 가늠해봄 | [ ] |
| 공부 중 | "지금 막힌 부분이 어디야?" | 어려운 지점을 짚음 | [ ] |
| 공부 중 | "이 문제는 어떤 단원이랑 비슷해?" | 연결고리 인식 | [ ] |
| 공부 후 | "오늘 새로 알게 된 건 뭐야?" | 핵심을 한 줄로 요약 | [ ] |
| 공부 후 | "내일은 어디부터 하면 좋을 것 같아?" | 다음 계획을 스스로 세움 | [ ] |
| 주말 | "이번 주에 가장 잘 이해한 건 뭐야?" | 자기 성장을 인식 | [ ] |

**대화 예시 — 식탁에서**
- 부모: "오늘 수학 중에 어떤 부분이 어려웠어?"
- 아이: "분수 더하는 거…"
- 부모: "어떻게 어렵던?"
- 아이: "분모가 다른 게 헷갈려요."
- 부모: "그러면 내일은 그 부분만 다시 보고 싶어, 아니면 새로운 거 나가고 싶어?"
- 아이: "다시 보고 싶어요."

이 대화가 **메타인지의 시작**입니다.

**3주차 완료 기준:** 일주일에 5번 이상, 아이가 스스로 자기 학습 상태를 한 문장으로 설명한다.

---

## 4주차 | 동행자 자리 잡기

**목표:** 부모가 감독자가 아니라 **옆자리에 앉은 사람**이 됩니다.

| 단계 | 활동 | 구체적 행동 | 확인 |
|------|------|-----------|------|
| 1 | 위치 옮기기 | 아이 맞은편이 아니라 옆에 앉기 (감시 → 동행) | [ ] |
| 2 | 침묵 연습 | 아이가 문제를 풀 때 5분간 말 걸지 않기 | [ ] |
| 3 | 질문 기다리기 | 먼저 알려주지 않고, 아이가 물을 때까지 기다리기 | [ ] |
| 4 | 함께 모르기 | "엄마/아빠도 이거 헷갈리네, 같이 찾아볼까?" | [ ] |
| 5 | 계획 위임 | 다음 주 공부 계획을 아이가 직접 종이에 쓰게 하기 | [ ] |
| 6 | 실수 허용 | 잘못된 선택의 결과를 그대로 경험하게 두기 | [ ] |
| 7 | 자기평가 도입 | "이번 주 너 자신에게 점수를 준다면 몇 점이야?" | [ ] |

**주의점:** 4주차에는 아이가 잠깐 흔들릴 수 있습니다. 부모의 통제가 사라진 자리를 아이가 채우기까지 며칠이 걸립니다. 이때 다시 개입하면 1주차로 돌아갑니다. 기다리세요.

**4주차 완료 기준:** 부모의 지시 없이 아이가 책상에 앉아 시작하는 날이 일주일에 3일 이상.

## 자주 묻는 질문

**Q. 선택권을 줬는데 자꾸 쉬운 것만 고릅니다.**
A. 처음 2~3주는 그래도 됩니다. 선택의 경험 자체가 중요한 시기입니다. 4주차쯤 "두 가지 다 어려운 것" 안에서 선택지를 좁혀가세요.

**Q. 과정 질문을 하니 아이가 "몰라요"라고만 합니다.**
A. "몰라요"는 거부가 아니라 **언어가 없다는 뜻**입니다. 한 달간 부모가 시범을 보여주세요. "엄마는 이 부분이 헷갈렸어. 너는 어땠어?"

**Q. 학원 숙제는 분량이 정해져 있어 선택권이 없습니다.**
A. 분량은 못 바꿔도 **순서, 시간, 장소**는 바꿀 수 있습니다. 선택권이 있는 영역부터 시작하세요.
