# 책 읽어주기 3단계 로드맵
## 침묵·기다림·한 질문으로 키우는 아이의 독해력

**대상:** 4~8세 자녀를 둔 부모  
**기간:** 4주 (28일) 실천 프로그램  
**준비물:** 그림책, 타이머(스마트폰), 이 가이드

---

## 왜 이 로드맵인가

부모가 책을 많이 읽어주는데도 아이의 독해력이 늘지 않는다면, 책을 읽는 양이 아니라 **읽는 동안의 말의 양**을 점검해야 합니다.

심리학자 Nelson Cowan의 연구에 따르면 사람의 작업기억(working memory)은 한 번에 약 4개의 정보 단위만 담을 수 있습니다. 부모가 던지는 질문 하나하나가 그 자리를 차지하면, 아이가 이야기를 머릿속에 그릴 공간이 사라집니다.

또한 교육학자 Mary Budd Rowe의 '대기시간(wait time)' 연구는 교사가 질문 후 3초만 기다려도 아이의 답이 더 깊고 길어진다는 사실을 보여줍니다.

**책 읽기는 아이가 듣는 시간이 아니라, 머릿속에서 이야기를 직접 짓는 시간입니다.**

이 로드맵은 그 공간을 되돌려주는 4주 훈련입니다.

---

## 1주차 — 침묵 적응기

**목표:** 부모가 '말하고 싶은 충동'을 인식하고 멈추는 연습

| 일차 | 활동 | 구체적 실천 | 확인 |
|------|------|-------------|------|
| 1일 | 현재 상태 점검 | 평소처럼 책을 읽어주며 내가 던진 질문 횟수를 세어본다 | [ ] |
| 2일 | 무질문 읽기 | 한 권을 읽는 동안 단 한 마디도 덧붙이지 않고 글자만 읽는다 | [ ] |
| 3일 | 무질문 읽기 | 같은 방식으로 다른 책 1권 | [ ] |
| 4일 | 한 페이지 끝 3초 멈춤 | 페이지를 넘기기 전 속으로 "하나, 둘, 셋" 센다 | [ ] |
| 5일 | 한 페이지 끝 3초 멈춤 | 같은 방식 반복 | [ ] |
| 6일 | 아이 반응 관찰 | 아이가 먼저 입을 여는 순간을 메모한다 | [ ] |
| 7일 | 1주차 점검 | 아래 자가체크 작성 | [ ] |

**1주차 자가체크**
- [ ] 책을 읽다가 멈추고 싶은 순간을 알아챘다
- [ ] 한 권을 끝까지 질문 없이 읽을 수 있었다
- [ ] 아이가 먼저 말을 건 적이 한 번 이상 있었다

---

## 2주차 — 한 질문 훈련기

**목표:** 책을 다 덮은 뒤 단 하나의 열린질문 던지기

| 일차 | 활동 | 구체적 실천 | 확인 |
|------|------|-------------|------|
| 8일 | 닫힌질문 끊기 | "이거 뭐야? 주인공 이름 기억나?" 같은 정답 확인 질문 금지 | [ ] |
| 9일 | 열린질문 1개 | 책을 덮고 "어디가 제일 기억에 남아?" 한 번만 묻는다 | [ ] |
| 10일 | 열린질문 1개 | "주인공이 어떤 기분이었을 것 같아?" | [ ] |
| 11일 | 열린질문 1개 | "너라면 어떻게 했을 것 같아?" | [ ] |
| 12일 | 침묵으로 기다리기 | 질문 후 아이가 답할 때까지 5초 이상 기다린다 | [ ] |
| 13일 | 답 늘리지 않기 | 아이의 답에 부모가 "맞아, 그리고 또..."로 덧붙이지 않는다 | [ ] |
| 14일 | 2주차 점검 | 아래 자가체크 작성 | [ ] |

**활용 가능한 열린질문 10개**
1. 어디가 제일 기억에 남아?
2. 주인공이 어떤 기분이었을 것 같아?
3. 너라면 어떻게 했을 것 같아?
4. 이 이야기에서 가장 신기했던 건 뭐야?
5. 만약 결말이 달랐다면 어땠을까?
6. 주인공한테 한 마디 해준다면?
7. 이 책에서 마음에 안 드는 장면이 있었어?
8. 다음에 어떤 일이 생길 것 같아?
9. 너랑 닮은 사람이 있었어?
10. 이 이야기를 누구한테 들려주고 싶어?

**2주차 자가체크**
- [ ] 닫힌질문(정답 확인용)을 거의 던지지 않았다
- [ ] 책을 덮은 뒤에만 질문했다
- [ ] 아이의 답을 끝까지 들었다

---

## 3주차 — 연령별 정착기

**목표:** 우리 아이 연령에 맞는 리듬 찾기

**연령별 기준표**

| 연령 | 1회 독서 시간 | 한 권당 질문 수 | 멈춤 시간 |
|------|---------------|-----------------|-----------|
| 4세 | 10~15분 | 0~1개 | 페이지마다 3초 |
| 6세 | 15~20분 | 1개 | 페이지마다 3초 |
| 8세 | 20~30분 | 1~2개 | 챕터 끝 5초 |

| 일차 | 활동 | 구체적 실천 | 확인 |
|------|------|-------------|------|
| 15일 | 우리 아이 기준 정하기 | 위 표에서 우리 아이 행에 형광펜 표시 | [ ] |
| 16일 | 시간 맞춰 읽기 | 타이머로 적정 시간 확인 | [ ] |
| 17일 | 시간 맞춰 읽기 | 같은 방식 반복 | [ ] |
| 18일 | 아이 주도 책 선택 | 아이가 직접 고른 책을 그대로 읽어준다 | [ ] |
| 19일 | 같은 책 두 번 읽기 | 처음엔 그냥, 두 번째는 아이가 멈춰달라 할 때만 멈춘다 | [ ] |
| 20일 | 아이가 읽어주기 | 부모가 청자 역할로 바꾼다 (8세 이상) / 그림 보고 이야기 짓기 (4~6세) | [ ] |
| 21일 | 3주차 점검 | 아래 자가체크 작성 | [ ] |

**3주차 자가체크**
- [ ] 우리 아이 연령에 맞는 시간을 지켰다
- [ ] 아이가 책을 직접 골랐다
- [ ] 아이가 책에 대해 먼저 이야기를 꺼냈다

---

## 4주차 — 자율 독서기

**목표:** 부모의 도움이 줄어도 아이가 책을 즐기는 단계로 넘어가기

| 일차 | 활동 | 구체적 실천 | 확인 |
|------|------|-------------|------|
| 22일 | 책 읽고 그림 그리기 | 다 읽은 뒤 가장 인상적인 장면을 그린다 | [ ] |
| 23일 | 책 읽고 다른 결말 짓기 | "만약 ~했다면?"을 함께 상상한다 | [ ] |
| 24일 | 부모가 한 줄 일기 쓰기 | 오늘 아이가 한 말 중 가장 인상 깊었던 한 줄 기록 | [ ] |
| 25일 | 도서관 함께 가기 | 아이가 직접 책을 고르게 한다 | [ ] |
| 26일 | 아이만의 책 추천 | "친구한테 어떤 책 추천하고 싶어?" | [ ] |
| 27일 | 4주 회고 | 1주차와 비교해 무엇이 달라졌는지 적어본다 | [ ] |
| 28일 | 다음 한 달 계획 | 아래 다음 단계 참고 | [ ] |

**4주차 자가체크**
- [ ] 아이가 책을 다 읽고 자기만의 감상을 표현했다
- [ ] 아이가 다음 책을 스스로 가져왔다
- [ ] 부모로서 책 읽어주는 시간이 더 편안해졌다

## 4주 후 다음 단계

- **유지하기:** 매일 또는 격일로 '침묵·기다림·한 질문' 리듬 이어가기
- **확장하기:** 같은 작가의 다른 책, 같은 주제의 다른 책으로 시리즈 독서
- **기록하기:** 아이가 한 말 중 인상 깊은 한 줄을 한 달에 5개씩 모으기
- **비우기:** 한 주에 한 번은 '아무 질문도 없는 날' 유지

## 자주 하는 실수

1. **3초 멈춤이 어색해서 결국 말을 걸어버린다** — 머릿속으로 천천히 "하나, 둘, 셋"을 세는 동안 아이의 표정만 봅니다.
2. **아이의 답이 짧아서 추가 질문을 던진다** — 짧은 답도 완성된 답입니다. "그렇구나" 한 마디로 충분합니다.
3. **읽다가 모르는 단어를 미리 설명한다** — 아이가 먼저 물을 때까지 기다립니다. 묻지 않으면 그 단어는 아이에게 아직 필요 없는 단어입니다.
4. **잘 읽었는지 확인하려 한다** — 이해도 점검은 독해력을 키우지 않습니다. 즐거움이 키웁니다.
