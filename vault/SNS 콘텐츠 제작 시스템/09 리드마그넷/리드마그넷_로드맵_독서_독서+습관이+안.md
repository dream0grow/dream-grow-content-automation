# 초등 독서 4단계 로드맵

## 책을 강요하지 않고 스스로 읽는 아이로 키우는 단계별 가이드

---

## 시작하며

"우리 아이는 왜 책을 안 읽을까요?"

많은 부모님이 의지나 성격 문제로 생각하시지만, 사실은 **순서의 문제**입니다.

발달심리학자 비고츠키는 아이의 능력에 두 영역이 있다고 보았습니다. 혼자서도 할 수 있는 영역과, 어른의 도움을 받으면 할 수 있는 영역입니다. 독서도 마찬가지입니다. 아직 '함께 읽기' 단계인 아이에게 '혼자 읽기'를 요구하면, 아이는 책 자체를 싫어하게 됩니다.

이 로드맵은 교실에서 책을 사랑하게 된 아이들이 거쳤던 4단계를 정리한 자료입니다. 우리 아이가 지금 어느 단계에 있는지 확인하고, 그 단계에 맞는 활동부터 시작해 보세요.

---

## 우리 아이 현재 단계 자가진단

아래 항목 중 우리 아이에게 해당되는 것에 체크해 보세요.

| 체크 | 항목 |
|:---:|---|
| [ ] | 책을 보면 도망가거나 거부 반응을 보인다 |
| [ ] | 책을 펴도 1분을 못 버틴다 |
| [ ] | 책 읽어주면 듣지만 혼자서는 안 본다 |
| [ ] | 만화책이나 그림책만 본다 |
| [ ] | 부모가 권한 책은 안 읽는다 |
| [ ] | 책을 골라주면 읽지만 스스로 고르진 않는다 |
| [ ] | 짧은 글은 읽지만 긴 글은 못 읽는다 |
| [ ] | 좋아하는 분야의 책은 혼자 읽는다 |

**해석**
- 1~2번 체크: **1단계** (책과 친해지기)
- 3~4번 체크: **2단계** (함께 읽기)
- 5~6번 체크: **3단계** (선택권 주기)
- 7~8번 체크: **4단계** (혼자 읽기)

## 단계별 로드맵

### 1단계: 책과 친해지기

| 항목 | 내용 |
|---|---|
| **목표** | 책은 즐거운 것이라는 정서 형성 |
| **기간** | 보통 1~3개월 (개인차 큼) |
| **부모 개입** | 매우 높음 (90%) |
| **추천 도서 유형** | 사운드북, 팝업북, 촉감책, 짧은 그림책, 만화책 |
| **핵심 활동** | 하루 5~10분, 부모가 재미있게 읽어주기 |
| **하지 말 것** | "읽어라" 강요, 독후감 요구, 지식 확인 질문 |
| **확인 신호** | 아이가 먼저 "책 읽어줘"라고 말한다 |

**구체적 활동**
- 잠자기 전 10분, 같은 책을 반복해서 읽어줘도 괜찮습니다
- 만화책이라도 책상에 자연스럽게 놓아둡니다
- 부모가 자신의 책을 읽는 모습을 보여줍니다
- 도서관/서점 나들이를 '재미있는 외출'로 만듭니다

---

### 2단계: 함께 읽기

| 항목 | 내용 |
|---|---|
| **목표** | 책 내용을 이해하고 반응하는 경험 쌓기 |
| **기간** | 보통 3~6개월 |
| **부모 개입** | 높음 (70%) |
| **추천 도서 유형** | 그림책, 짧은 동화, 학습만화, 시리즈 동화 초입 |
| **핵심 활동** | 한 페이지씩 번갈아 읽기, 책 내용으로 대화하기 |
| **하지 말 것** | 발음/속도 지적, "이게 무슨 뜻이야?" 시험 |
| **확인 신호** | 책 내용을 자기 말로 이야기한다 |

**구체적 활동**
- 부모가 한 페이지, 아이가 한 페이지 번갈아 읽기
- "이 다음에 어떻게 될 것 같아?" 예측 질문하기
- "엄마/아빠는 이 부분이 슬펐어" 감정 공유하기
- 같은 작가/같은 시리즈로 확장하기

---

### 3단계: 선택권 주기

| 항목 | 내용 |
|---|---|
| **목표** | 자기가 좋아하는 책을 찾는 안목 기르기 |
| **기간** | 보통 6개월~1년 |
| **부모 개입** | 중간 (40%) |
| **추천 도서 유형** | 시리즈 동화, 학습만화, 관심 분야 도감, 짧은 소설 |
| **핵심 활동** | 도서관에서 아이가 절반 이상 직접 고르기 |
| **하지 말 것** | "이건 너무 쉬워", "그건 만화잖아" 평가 |
| **확인 신호** | 좋아하는 작가/장르가 생긴다 |

**구체적 활동**
- 도서관 방문 시 5권 중 3권은 아이가 자유 선택
- 만화/학습만화도 정식 책으로 인정
- "왜 그 책 골랐어?" 호기심 질문 (시험 아닌)
- 아이가 고른 책을 부모가 먼저 읽고 이야기 나누기

---

### 4단계: 혼자 읽기

| 항목 | 내용 |
|---|---|
| **목표** | 책을 일상의 도구로 사용하기 |
| **기간** | 지속적 유지 |
| **부모 개입** | 낮음 (20%) |
| **추천 도서 유형** | 장편 동화, 청소년 소설, 비문학, 분야별 전문서 |
| **핵심 활동** | 매일 정해진 시간에 혼자 읽기, 독서 환경 유지 |
| **하지 말 것** | 권장 도서 강요, 독서 시간 측정/감시 |
| **확인 신호** | 부모가 말하지 않아도 책을 집어 든다 |

**구체적 활동**
- 가족 독서 시간 (예: 저녁 8시~8시 30분)
- 침대 옆, 식탁 위 등 책이 손에 닿는 환경
- 도서관 회원증 직접 관리하게 하기
- 한 달에 한 번 '인생 책' 이야기 나누기

## 4주 실천 계획표

처음 시작하는 분을 위한 단계 무관 공통 계획입니다.

| 주차 | 목표 | 구체적 활동 | 확인 |
|---|---|---|---|
| 1주 | 환경 만들기 | 거실 책장 정리, 아이 눈높이에 책 배치 | [ ] |
| 1주 | 도서관 가기 | 주말에 도서관 방문, 회원증 만들기 | [ ] |
| 2주 | 함께 읽기 시작 | 매일 10분 잠자기 전 책 읽어주기 | [ ] |
| 2주 | 선택권 주기 | 아이가 고른 책 1권 함께 읽기 | [ ] |
| 3주 | 대화 추가 | "어떤 장면이 좋았어?" 한 가지만 묻기 | [ ] |
| 3주 | 부모 모범 | 부모도 자신의 책 매일 10분 읽기 | [ ] |
| 4주 | 점검과 조정 | 자가진단표 다시 체크, 단계 확인 | [ ] |
| 4주 | 다음 단계로 | 우리 아이 단계에 맞춰 활동 조정 | [ ] |

## 자주 묻는 질문

**Q1. 만화책만 봐도 괜찮나요?**
A. 1~2단계에서는 만화책도 훌륭한 독서 도구입니다. 글을 읽는 즐거움을 먼저 익혀야 줄글로 자연스럽게 넘어갑니다. 만화를 보지 못하게 막으면 책 자체에서 멀어집니다.

**Q2. 또래보다 늦은 것 같아요.**
A. 단계는 나이가 아니라 경험으로 결정됩니다. 5학년이라도 1단계가 필요한 아이가 있고, 1학년이라도 3단계에 있는 아이가 있습니다. 학년이 아닌 우리 아이의 현재 위치에서 시작하세요.

**Q3. 학습만화 너무 좋아하는데 줄글로 어떻게 넘어가나요?**
A. 학습만화의 주제를 줄글 책으로 확장하세요. 과학만화를 좋아하면 어린이 과학 잡지로, 역사만화를 좋아하면 인물 동화로 연결합니다. 갑작스러운 변화가 아니라 자연스러운 확장입니다.

**Q4. 다음 단계로 언제 넘어가나요?**
A. '확인 신호'가 꾸준히 보일 때입니다. 한두 번이 아니라 2~3주 이상 지속되어야 안정된 것입니다. 조급하게 넘기면 다시 앞 단계로 돌아옵니다.

## 마치며

독서는 강요로 만들어지지 않습니다. 부모가 단계에 맞는 환경을 제공할 때, 아이는 스스로 책을 사랑하게 됩니다.

오늘부터 우리 아이의 현재 단계를 확인하고, 그 단계에 맞는 활동 하나만 시작해 보세요. 한 단계씩 차근차근 올라가는 길이 가장 빠른 길입니다.

아이가 책을 사랑하는 사람으로 자라길 바랍니다.
