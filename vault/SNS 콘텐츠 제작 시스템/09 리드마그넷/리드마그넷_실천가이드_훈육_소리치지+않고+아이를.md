# 오늘부터 시작하는 단호한 훈육 3단계
## 소리치지 않고 아이를 움직이는 부모의 실천 가이드

> 단호한 훈육은 데시벨이 아니라 구조입니다.
> 감정을 인정하고, 경계를 분명히 하고, 대안을 제시하는 순서.
> 이 순서가 아이의 전두엽을 켭니다.

---

## 시작하기 전에

훈육은 아이를 굴복시키는 일이 아니라, 스스로 선택할 수 있는 회로를 만들어주는 일입니다. 큰 소리는 그 순간 행동을 멈추게 하지만, 다음 행동을 가르치지는 못합니다. 부모가 일관되게 보여주는 3단계 구조가 결국 아이의 뇌에 새겨집니다.

준비물은 단 하나입니다. **평소에 미리 설계해 둔 문장**.

---

## STEP 1. 감정 인정 — "그 마음, 봤어"

### 무엇을
아이가 보인 1차 감정에 이름을 붙여 돌려줍니다. 행동을 평가하지 않고, 감정만 먼저 인정합니다.

### 어떻게
- 아이의 눈높이로 내려앉습니다.
- 목소리 톤을 평소보다 한 단계 낮춥니다.
- "~구나" "~었구나" 어미를 활용해 감정을 비춰줍니다.

| 상황 | 감정 인정 예문 |
|------|---------------|
| 동생이 장난감을 가져갔을 때 | "동생이 갑자기 가져가서 화났구나." |
| 학원 가기 싫다고 떼쓸 때 | "오늘은 정말 가기 싫었구나." |
| 친구가 안 놀아준다고 울 때 | "혼자 남은 것 같아서 서운했구나." |
| 시험에서 틀려 짜증낼 때 | "맞을 줄 알았는데 속상했구나." |
| 형이 먼저 양보 안 해줄 때 | "형이 양보해줄 거라 기대했었구나." |
| 잠자기 싫다고 버틸 때 | "더 놀고 싶은 마음이 컸구나." |

### 왜
감정을 부정당한 아이는 행동을 고치기 전에 자기 마음부터 의심합니다. 인정받은 감정은 빠르게 가라앉습니다. 흥분 상태의 뇌는 어떤 가르침도 받아들이지 못합니다.

### 주의점
- "그래도 동생을 때리면 안 되지"처럼 곧바로 평가를 붙이지 않습니다.
- "화내지 마"는 감정 인정이 아니라 감정 차단입니다.
- 인정은 30초면 충분합니다. 길게 설명할수록 흐려집니다.

---

## STEP 2. 경계 설정 — "사람은 때리지 않아"

### 무엇을
가정의 변하지 않는 기준을 짧고 분명한 문장으로 알려줍니다.

### 어떻게
- **8자 이내 문장**으로 자릅니다.
- "안 돼" 대신 **무엇이 기준인지** 말합니다.
- 같은 상황에서는 **항상 같은 문장**을 씁니다.

#### 8자 경계 설정 문장 모음

| 영역 | 경계 문장 |
|------|----------|
| 신체 | 사람은 때리지 않아. |
| 신체 | 몸은 다치게 하지 않아. |
| 언어 | 욕은 쓰지 않아. |
| 언어 | 사람을 놀리지 않아. |
| 물건 | 물건은 던지지 않아. |
| 물건 | 책은 찢지 않아. |
| 약속 | 약속은 지키는 거야. |
| 안전 | 길에서는 손을 잡아. |
| 식사 | 밥은 앉아서 먹어. |
| 수면 | 9시에는 불을 꺼. |

#### 부모 톤 가이드
- 빠르기: 평소의 0.7배 속도
- 높이: 가슴 아래에서 나오는 낮은 톤
- 시선: 아이의 눈을 1초 이상 응시
- 손: 어깨에 가볍게 얹거나 무릎 위
- 거리: 한 팔 이내

### 왜
경계는 길어질수록 협상이 됩니다. 짧은 문장은 규칙임을 분명히 하고, 부모 자신도 흔들리지 않게 합니다. 매번 같은 문장을 듣는 아이는 결국 그 문장을 자기 안에 내면화합니다.

### 주의점
- 화가 났을 때 만든 새로운 규칙은 효력이 없습니다.
- 한 상황에 두세 개의 경계를 동시에 들이밀지 않습니다.
- 부모가 어겼다면 "엄마도 방금 어겼네, 미안해"라고 회복합니다.

---

## STEP 3. 대안 제시 — "이렇게 해보자"

### 무엇을
하면 안 되는 것 대신 **무엇을 하면 되는지** 구체적으로 알려줍니다. 핵심은 **평온할 때 미리** 정해두는 것입니다.

### 어떻게
- 폭풍이 지나간 뒤, 또는 평온한 주말에 가족이 함께 약속표를 만듭니다.
- 한 감정에 한 가지 대체 행동만 정합니다.
- 종이에 적어 잘 보이는 곳에 붙여둡니다.

#### 가족 대체 행동 약속표 (템플릿)

| 이런 마음이 들 때 | 이렇게 해본다 |
|-------------------|---------------|
| 화가 머리끝까지 날 때 | 베개를 세 번 친다. |
| 동생이 미울 때 | 내 방으로 가서 문을 닫는다. |
| 짜증이 날 때 | 물 한 컵을 천천히 마신다. |
| 울고 싶을 때 | 엄마 무릎에 5분 앉아 있는다. |
| 소리 지르고 싶을 때 | 화장실에서 셋까지 센다. |
| 던지고 싶을 때 | 쿠션을 소파에 던진다. |
| 거부하고 싶을 때 | "잠깐만"이라고 먼저 말한다. |

#### 직접 만드는 우리 집 약속표

| 이런 마음이 들 때 | 이렇게 해본다 |
|-------------------|---------------|
|                   |               |
|                   |               |
|                   |               |

### 왜
폭풍 속에서 새로운 행동을 발명하는 뇌는 없습니다. 작동하는 건 평소에 설계해 둔 규칙뿐입니다. 대안이 있을 때만 아이는 멈출 수 있습니다.

### 주의점
- "그러면 안 돼"로 끝내면 아이는 갈 곳이 없어집니다.
- 대체 행동은 **아이가 실제로 할 수 있는 수준**이어야 합니다.
- 약속을 지켰을 때 즉시 알아채주는 것이 핵심입니다. "베개 쳤구나, 잘 참았어."

## 3단계 통합 대화 예시

**상황: 동생을 때린 아이**

부모: (옆에 앉아, 낮은 톤으로)
"동생이 네 블록을 무너뜨려서 정말 화났구나." → STEP 1

(2초 침묵)
"그런데 사람은 때리지 않아." → STEP 2

(아이의 눈을 보며)
"다음에 이렇게 화가 나면 베개를 세 번 치기로 했었지. 같이 가볼까." → STEP 3

## 자주 묻는 질문

**Q1. 매번 이렇게 길게 말할 시간이 없어요.**
처음에만 길게 느껴집니다. 익숙해지면 세 단계가 30초 안에 끝납니다. 그리고 30초 투자가 30분 잔소리를 줄여줍니다.

**Q2. 감정만 인정해주면 버릇이 나빠지지 않을까요?**
감정 인정은 행동 허용이 아닙니다. STEP 2에서 분명히 경계를 긋기 때문에 버릇이 흐려지지 않습니다. 오히려 감정이 막힌 아이가 더 큰 행동으로 표현하게 됩니다.

**Q3. 아이가 약속표를 지키지 않아요.**
처음 몇 주는 부모가 옆에서 같이 해주어야 합니다. "지금 그 마음이지, 약속표 어디 있더라" 하고 함께 찾아주세요. 혼자 작동하기까지 보통 6~8주가 걸립니다.

## 한 장 요약

| 단계 | 한 문장 | 핵심 동작 |
|------|---------|----------|
| 1. 감정 인정 | "~구나" | 눈높이, 낮은 톤, 평가 금지 |
| 2. 경계 설정 | 8자 이내 문장 | 항상 같은 문장, 협상 없음 |
| 3. 대안 제시 | "이렇게 해보자" | 평온할 때 미리 약속, 함께 작성 |

좋은 훈육은 점점 필요 없어지는 구조입니다. 부모가 만든 3단계가 결국 아이가 스스로 자기를 다루는 방식이 됩니다. 아이가 자기 마음을 믿으며 자라길 바랍니다.
