# 채널 정의

이 시스템은 같은 아이디어를 여러 채널로 변환하기 위한 구조입니다.

---

## YouTube 롱폼 (주력 채널)
긴 호흡의 설명, 사례, 근거, 전개에 강합니다.
현재 운영 중인 주요 콘텐츠:
- 부모 교육 (초등 수학, 자기주도학습, 훈육, 식습관과 뇌)
- 유튜브 성장/수익화 노하우
- 자기계발 (습관, 미루기, 디지털 디톡스)
- 교사 대상 콘텐츠 (노션, 회복적 생활교육)

영상 원고 구조: 유인제시 -> 문제 제시 -> 유인(해결책) 제시
보관 폴더: 05 제작/, 05 제작/52 원고/

## YouTube 쇼츠 / 숏폼
1분 이내, 초반 5초 안에 호기심 확보, 기승전결 완결.
현재 운영 중인 시리즈:
- 숏폼 부자 시리즈 (영화/만화 캐릭터로 배우는 부의 원리)
- 쇼포츠 (스포츠 감동/반전 숏폼)
- 부모 교육 릴스 (분수, 수학, 개학 준비 등)

보관 폴더: 05 제작/

## Instagram Reels (활발 사용)
짧고 강한 훅, 빠른 템포, 시각적 전환이 중요합니다.
첫 문장과 첫 화면의 긴장감이 핵심입니다.
현재 운영 중인 콘텐츠:
- 부모 교육 릴스 (드림그로우): 수학, 공부습관, 자녀 관계
- 스레드+릴스 조합 시리즈: 훈육, 설 연휴 질문 등
- 원고 형식: 00초~00초 단위의 타임스탬프 기반 스크립트

보관 폴더: 05 제작/52 원고/ (원고_릴스_ 접두어)

## Threads (활발 사용)
대화체, 관찰, 짧은 단락 전개가 잘 맞습니다.
공감과 생각 전환이 중요합니다.
현재 운영 중인 콘텐츠:
- 초등교사 시리즈 (교육 인사이트)
- 유튜브 성장 시리즈 (영상 원고를 스레드로 리믹스)
- 부모 교육 시리즈 (훈육, 다정함, 통지표)
- 자기계발/마인드셋

후킹 패턴: 권위-충격-공감-솔루션 예고-결핍 유도 (상세: 07 스레드/스레드 본문 후킹 구조 분석)
보관 폴더: 07 스레드/

## Instagram Cardnews
한 장씩 넘기며 읽게 만드는 문장 구조가 중요합니다.
페이지별 핵심 메시지와 흐름 설계가 필요합니다.
스레드 글을 7장 슬라이드로 변환하는 방식이 효율적입니다.

## 블로그
영상 원고를 텍스트 기반으로 재편집하여 발행합니다.
기존 발행 콘텐츠: 유튜브 시작 4개월 이야기, 유튜브 수익의 진실 등.

## 뉴스레터 (신규)
주 1회 발행. 스레드/영상 콘텐츠를 심화하여 구독자 전용 가치를 제공합니다.
이메일 구독자를 리드마그넷과 연결하여 충성 독자층을 구축합니다.

목적:
- 스레드/릴스로 유입된 팔로워를 이메일 구독자로 전환
- 소셜미디어 알고리즘에 의존하지 않는 직접 소통 채널 확보
- 리드마그넷 자료를 구독자에게 정기 전달

구조:
- 제목: 후킹형 한 문장 (열어보게 만드는)
- 인사말: 간결한 인사 + 이번 주제 예고 (2~3줄)
- 본문 1: 이번 주 핵심 인사이트 (스레드 심화 버전, 교실 에피소드 포함)
- 본문 2: 실천법 (가정에서 바로 할 수 있는 것 2~3가지)
- 큐레이션: 이번 주 스레드/릴스/영상 중 인기글 1~2개 링크
- 마무리: "아이가 건강하게 자라길 바랍니다." + 다음 호 예고
- 서명: "아이와 부모의 꿈을 키웁니다. -Dream_Grow-"

톤: 스레드보다 조금 더 편안한 대화체. 구독자에게 보내는 편지 느낌.
분량: 본문 800~1200자 (읽기 3~5분)
CTA: 리드마그넷 다운로드, 릴스 댓글 참여, 유튜브 시청 유도

보관 폴더: 05 제작/52 원고/ (원고_뉴스레터_ 접두어)

## 운영 팁

하나의 핵심 메시지를 먼저 만들고, 이후 각 채널로 변환합니다.
채널별로 새 아이디어를 만드는 것보다 하나의 원본 메시지를 여러 포맷으로 풀어내는 방식이 효율적입니다.

### 실제 리믹스 패턴 (현재 사용 중)
1. 영상 원고(롱폼) -> 스레드 6~8단락으로 압축
2. 스레드 글 -> 릴스 35~45초 스크립트로 변환
3. 스레드 글 -> 인스타 카드뉴스 7장으로 변환
4. 강의 스크립트 -> 영상 원고 + 숏폼 스크립트로 분리
5. 스레드 인기글 + 릴스 인기글 -> 뉴스레터 심화 버전으로 편집

### 채널별 CTA 패턴
- YouTube: 구독 + 알림 설정, 다음 영상 예고
- Reels: 저장 + 팔로우, PDF 자료 댓글 안내
- Threads: 다음 글 유도(결핍), 저장 + 공유 요청
- Cardnews: 마지막 장 CTA (저장/공유/댓글)
- Newsletter: 리드마그넷 다운로드 + 릴스/영상 링크 + 다음 호 예고
