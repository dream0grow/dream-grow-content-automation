# SNS 콘텐츠 제작 시스템 시작하기

이 폴더는 초생산 하위에서 SNS 콘텐츠를 수집, 분석, 벤치마킹, 제작, 발행, 회고까지 연결하는 작업 공간입니다.

핵심 원칙은 3가지입니다.

1. 수집 노트와 분석 노트를 분리합니다.
2. 재사용할 수 있는 훅, 구조, 메시지는 라이브러리로 따로 빼둡니다.
3. 실제 제작물은 프로젝트 단위로 묶고, 발행 후 성과와 피드백까지 이어서 기록합니다.

전체 흐름은 아래와 같습니다.

1) 내 타겟들이 좋아하는 콘텐츠 수집
2) 좋아하는 이유 분석
3) 플랫폼별 핫 콘텐츠 수집
4) 핫 콘텐츠가 뜨는 이유 분석
5) 핫 콘텐츠의 후킹 문구를 내 타겟에 맞게 적용
6) 잘된 구조와 핵심 내용 추출
7) 타겟 선호 콘텐츠의 핵심 내용과 댓글 추출
8) 본문에 반드시 들어갈 내용 정리
9) 영상/카드뉴스/글 초안 작성
10) 채널별 변환
11) 성과 기록
12) 셀프 피드백

권장 운영 방식

- 새로 발견한 콘텐츠는 먼저 01 수집에 넣습니다.
- 분석이 끝난 패턴은 03 라이브러리로 축약합니다.
- 실제로 만들 콘텐츠는 05 제작 안의 프로젝트 노트에서 관리합니다.
- 발행 후에는 반드시 06 운영까지 기록해 학습 루프를 닫습니다.
