# 파일명 규칙

## 핵심 원칙

1. `_` (언더바): **카테고리별 분류** (단계_형식_주제영역)
2. `+` (플러스): **키워드별 분류** (핵심단어+핵심단어)
3. `-` (하이픈): **부제 연결** (대주제 - 소주제)
4. 제텔카스텐 원칙: **한 파일 = 한 가지 핵심 주장** (원자성)

## 카테고리 코드표

### 단계 (첫번째 `_` 앞)
| 코드  | 의미        | 예시             |
| --- | --------- | -------------- |
| 수집  | 01 수집 단계  | 수집_YT_롱폼_...   |
| 분석  | 02 분석 단계  | 분석_후킹_...      |
| 벤치  | 04 벤치마킹   | 벤치_구독자100만_... |
| 원고  | 05 제작 원고  | 원고_릴스_...      |
| 변환  | 53 채널별 변환 | 변환_스레드→릴스_...  |

### 형식 (두번째 `_` 앞)
| 코드 | 채널/형식 |
|------|-----------|
| 스레드 | Threads 글 |
| 릴스 | Instagram Reels (35~60초) |
| 숏폼 | YouTube Shorts / 기타 숏폼 |
| YT롱폼 | YouTube 롱폼 (10~30분) |
| 블로그 | 블로그 글 |
| 카드뉴스 | Instagram 카드뉴스 |
| 책 | 책 출판 원고 |
| 뉴스레터 | 이메일 뉴스레터 |

### 주제 카테고리 (세번째 `_` 앞)
| 카테고리 | 포함 주제 |
|----------|-----------|
| 훈육 | 엄한 훈육, 안전교육, 경계설정, 비폭력대화 |
| 수학 | 연산, 분수, 나눗셈, CPA이론, 수감각 |
| 독서 | 책추천, 독서습관, 그림책, 국어 |
| 미디어 | 스마트폰, AI, 디지털 디톡스, 유해콘텐츠 |
| 놀이 | 놀이학습, 창의성, 장난감, 체험 |
| 감정 | 자존감, 감정코칭, 부모스트레스, 칭찬/비교 |
| 학습 | 공부습관, 메타인지, 자기주도학습, 학원 |
| 학교 | 개학준비, 통지표, 입학, 학교생활 |
| 크리에이터 | 유튜브, 스레드 성장, 콘텐츠 전략, 수익화 |

### 카테고리 2개 연결 규칙
주제가 2개 카테고리에 걸칠 때: **주 카테고리를 앞에, 보조 카테고리를 뒤에** 배치
- `원고_릴스_훈육+감정_한숨+자존감+뇌발달` (훈육이 주, 감정이 보조)
- `원고_스레드_수학+학습_구구단+수감각+CPA이론` (수학이 주, 학습이 보조)

## 폴더별 파일명 패턴

### 01 수집/
```
수집_[채널코드]_[형식]_[제목키워드]_[키워드+키워드]
```
예시:
- `수집_YT_롱폼_1시간 끝내기_엑셀+총정리`
- `수집_YT_롱폼_1시간 끝내기_GPT+왕초보`
- `수집_YT_롱폼_1시간 끝내기_인간관계+고수`
- `수집_YT_영상_분수_분수+나눗셈+쉽게 푸는 법`

### 02 분석/
```
분석_[분석대상]_[키워드+키워드]
```
예시:
- `분석_후킹_권위자+훅+구조`
- `분석_타겟선호_초등저학년+부모+이유`
- `분석_도입부_YT롱폼+1시간끝내기`

### 04 벤치마킹/
```
벤치_[대상]_[키워드+키워드]
```
예시:
- `벤치_구독자100만_채널+분석`
- `벤치_후킹패턴_도입부+유튜브`
- `벤치_인스타성장_부자공장+마인드피봇`

### 05 제작/52 원고/
```
원고_[형식]_[카테고리]_[키워드+키워드]
```
예시:
- `원고_릴스_훈육_한숨+자존감+뇌발달`
- `원고_릴스_수학_나눗셈+뺄셈+공부법`
- `원고_릴스_감정_세뱃돈+경제교육`
- `원고_YT롱폼_학습_메타인지+성적+공부법`
- `원고_YT롱폼_크리에이터_도입부후킹+5가지공식`
- `원고_블로그_훈육_훈육의과학+바움린드+경계`
- `원고_블로그_수학_문제집+수감각+CPA이론`
- `영상원고 - [주제]` (기존 패턴도 허용, 점진적 전환)

### 07 스레드/
하위 카테고리 폴더로 분류. 파일명은 **핵심 주장 한 문장** (원자성).
```
[카테고리]/[핵심 주장이 담긴 제목].md
```
예시:
- `훈육/아이 앞에서 한숨 쉬지 마세요.md`
- `수학/구구단 외우기 전에 반드시 해야할 것.md`
- `독서/교실에서 본 독서하는 아이.md`
- `미디어/스마트폰이 아이 뇌에 미치는 영향.md`
- `감정/칭찬하면 안되는 이유.md`
- `학습/공부 잘하는 아이는 머리가 좋은 게 아닙니다.md`
- `크리에이터/유튜브 100만 구독자의 비밀 - 떡상 영상 공식.md`

07 스레드 하위 카테고리 폴더:
- 훈육/
- 수학/
- 독서/
- 미디어/
- 놀이/
- 감정/
- 학습/
- 학교/
- 크리에이터/
- 초안/

### 08 리뷰/
원본 파일명 유지. frontmatter에 `상태: 리뷰대기` 표시.
AI가 새로 생성한 콘텐츠는 여기서 사람이 최종 검토 후 해당 폴더로 이동.
