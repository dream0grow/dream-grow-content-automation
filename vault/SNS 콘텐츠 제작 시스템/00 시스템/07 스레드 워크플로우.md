# SNS 콘텐츠 제작 워크플로우

## 전체 프로세스

```
01-아이디어-수집 → 02-주제-분류 → 03-초안-작성 → 04-검토-수정 → 05-브랜딩-완성 → 06-업로드-대기 → 07-게시-완료
```

---

## 01. 아이디어 수집
- 일상에서 떠오른 교육 관련 생각, 메모, 키워드를 던져놓는 공간
- 형식 없음. 한 줄이든 키워드든 OK
- 파일명: 자유 (예: `메모-오늘수업.md`, `아이디어-독서습관.md`)

## 02. 주제 분류
- 01에서 모인 아이디어를 주제별로 정리
- 파일명 규칙: `T-{주제키워드}` (예: `T-공부싫어하는아이-대처법`, `T-어림셈-중요성`)
- 주제 카테고리:
  - 수학/연산
  - 영어
  - 독서/읽기
  - 습관/루틴
  - 뇌발달/심리
  - 훈육/지도
  - 학부모소통
  - 공부법/학습전략
  - 동기부여/마인드셋

## 03. 초안 작성
- T-파일을 기반으로 스레드 초안 작성
- 구조:
  - 훅(Hook): 권위 + 호기심 유발 첫 문장
  - 본문: 번호(1/, 2/, 3/...) 구분, 과학적 근거 포함
  - 각 파트 280자 이내
  - 5~10개 파트 구성

## 04. 검토 수정
- 작성자(한결 선생님)가 직접 읽고 수정하는 단계
- 톤 확인: 전문적이면서 친근한 말투
- 과학적 근거 정확성 확인
- 불필요한 부분 삭제, 보강

## 05. 브랜딩 완성
- Dream_Grow 서명 추가
- 마지막 파트에 아래 내용 포함:

```
과학적으로 건강하게 성장하는 법을 돕습니다.
"아이와 부모의 꿈을 키웁니다."
-Dream_Grow-
```

- CTA(행동유도) 확인: 팔로우, 리포스트, 저장 유도 문구

## 06. 업로드 대기
- 최종 완성된 글을 여기로 이동
- 작성자가 "업로드" 신호를 보내면 게시 실행
- 파일명: `T-{주제키워드}.md` 유지

## 07. 게시 완료
- 게시된 글은 여기로 이동
- 파일명에 날짜 추가: `T-{주제키워드}_2026-04-09.md`

## 성과 기준 (CSV 데이터 기반)

- 조회수 TOP 글 공통점: "초등교사입니다." 권위 훅 시작
- 평균 조회수 높은 카테고리: 훈육/지도, 뇌발달/심리, 공부법
- 스레드 최적 길이: 5~8파트
- 핵심 성공 패턴:
  1. 권위 훅 ("초등교사입니다", "초등 2학년 담임입니다")
  2. 반전/호기심 ("절대 아닙니다", "함정입니다", "속지 마세요")
  3. 과학적 근거 (뇌과학, 교육학 연구)
  4. 실천 가능한 팁 (구체적 행동 제안)
  5. 감정 공감 ("ㅠㅜ", "하...", 솔직한 경험 공유)
