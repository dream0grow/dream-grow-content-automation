# 08 리뷰

AI가 생성하거나 수정한 콘텐츠를 사람이 최종 확인하는 폴더.

## 운영 규칙

1. AI가 새로 작성하거나 대폭 수정한 글은 이 폴더에 먼저 배치
2. 사람이 검토 후 승인하면 해당 폴더(07 스레드/, 05 제작/)로 이동
3. 수정이 필요하면 파일 안에 `리뷰메모:` 프론트매터를 추가해 피드백 기록

## frontmatter 규칙

```yaml
---
주제: 글 제목
상태: 리뷰대기 | 리뷰완료 | 수정필요
출처: AI생성 | AI수정 | AI리믹스
원본: (원본 파일 링크, 있을 경우)
생성일: YYYY-MM-DD
리뷰메모: (검토자 피드백)
---
```

## 워크플로우 위치

흐름 A/B에서 Step 10(채널별 변환) 이후, Step 11(발행 계획) 이전에 위치.

```
... → 채널별 변환 → [08 리뷰] → 발행 계획 → ...
```
