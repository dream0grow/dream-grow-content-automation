# 릴스 스크립트: 규칙을 만드는 아이

## 1. 릴스 스크립트

**[0~3초] 후킹**
"규칙을 잘 지키는 아이는, 규칙을 많이 배운 아이가 아니에요."
(화면: 보드게임 앞에 앉은 아이의 진지한 얼굴 클로즈업)

**[3~10초] 문제 공감**
"'떠들지 마' 백 번 말해도 안 되잖아요.
저도 교실에서 매년 봐요.
백 번 말한 반은 끝까지 시끄럽거든요."
(화면: 시끄러운 교실, 손드는 선생님)

**[10~25초] 핵심 인사이트**
"근데 학기 초에 아이들이 직접 학급 규칙을 만든 반은요,
1년 내내 조용해요.
비고츠키의 비계 이론이 그래요.
지원을 세웠다가, 거두는 겁니다.
타인이 만든 규칙은 어기지만,
자기가 만든 규칙은 자기가 지키거든요."
(화면: 칠판에 규칙 적는 아이들, 손글씨로 채워지는 종이)

**[25~38초] 실천법**
"오늘 저녁 보드게임 할 때요,
'이게 룰이야' 말고 '우리만의 규칙 만들어볼까?'로 시작해보세요.
아이가 정한 규칙은 부모도 지켜야 해요.
규칙이 약자에게만 적용되는 게 아니라는 걸,
아이는 몸으로 배우거든요."
(화면: 아이와 부모가 마주 앉아 게임 카드 정리, 종이에 규칙 쓰는 손)

**[38~45초] 마무리 + CTA**
"훈육은 길들이는 게 아니에요. 비계를 세웠다 거두는 일입니다.
이외에도 놀이로 자기 조절력 키우는 법 더 알고 싶으신 분들은
아무 댓글이나 남겨주세요.
'우리집 놀이 규칙 만들기 가이드' 자료 보내드릴게요.
아이가 건강하게 자라길 바랍니다."
(화면: 리드마그넷 미리보기 PDF 표지 + "댓글 남기면 무료 자료 전송" 텍스트)

---

## 2. 리드마그넷 제안

- **이름**: 우리집 놀이 규칙 만들기 가이드
- **유형**: PDF 워크북 (A4, 8페이지)
- **내용 3줄**:
  - 연령별(5~12세) 보드게임/몸놀이 규칙 협상 대화 스크립트 12종
  - 규칙이 깨졌을 때 잔소리 대신 쓰는 질문 5문장 카드
  - 가족 규칙 함께 적는 워크시트 (인쇄용 1장)

## 3. B-roll 장면 목록

| 타임코드 | 장면 설명 | 영문 검색 키워드 | 대체 옵션 |
|---|---|---|---|
| 0~3초 | 보드게임 앞 아이의 진지한 표정 클로즈업 | `child playing board game close up`, `kid focused face` | 게임 카드 손에 든 아이 손 클로즈업 |
| 3~7초 | 시끄러운 교실, 떠드는 학생들 | `noisy classroom kids talking`, `elementary school classroom chaos` | 손드는 교사 실루엣 |
| 7~10초 | 곤란해하는 교사 / 칠판 앞 | `teacher frustrated classroom`, `teacher writing blackboard` | 빈 교실 책상 와이드샷 |
| 10~17초 | 아이들이 모여 칠판/종이에 규칙 적는 장면 | `kids writing rules paper group`, `students collaborating classroom` | 종이에 손글씨 적히는 클로즈업 |
| 17~25초 | 비계(스캐폴딩) 이미지 / 건축 비계 거두는 장면 | `scaffolding construction removed`, `building scaffold timelapse` | 블록 쌓기, 손 떼는 장면 |
| 25~32초 | 부모-아이 보드게임 장면 (마주 앉음) | `parent child board game home`, `family playing game table` | 거실 카펫 위 카드게임 |
| 32~38초 | 종이에 규칙 적는 부모와 아이 손 | `family writing rules paper hands`, `parent child writing together` | 화이트보드에 함께 쓰기 |
| 38~45초 | 따뜻한 가족 분위기 + PDF 미리보기 오버레이 | `happy family playing together warm`, `family game night cozy` | 아이 웃는 얼굴 + 텍스트 오버레이 공간 |
