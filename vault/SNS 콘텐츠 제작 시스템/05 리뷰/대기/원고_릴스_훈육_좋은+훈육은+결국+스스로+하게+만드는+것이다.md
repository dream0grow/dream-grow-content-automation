# 릴스 스크립트: "비계는 철거되어야 한다"

---

## 1. 릴스 스크립트

**[0~3초] 후킹**
"아이가 말을 잘 듣는다고요? 그게 가장 위험한 신호일 수 있습니다."
*(화면: 부모가 시키자마자 즉시 행동하는 아이의 무표정한 얼굴 클로즈업)*

**[3~10초] 문제 공감**
"교사가 볼 때만 정리하는 아이. 교사가 없어도 정리하는 아이. 교실에서 매일 보는 두 모습이거든요."
*(화면: 교실에서 두 아이의 대비되는 모습 — 한 명은 어른 눈치 보고, 한 명은 자율적으로 행동)*

**[10~25초] 핵심 인사이트**
"좋은 훈육은 아이를 따르게 만드는 게 아닙니다. 점점 필요 없어지게 만드는 거예요. 비고츠키의 비계 이론 아시죠. 건물이 완성되면 비계는 반드시 철거되잖아요. 부모의 규칙도 똑같습니다. 외부 경계가 영구 구조물이 되는 순간, 그건 통제로 굳어버려요."
*(화면: 공사장 비계가 단계적으로 철거되는 타임랩스 → 완성된 건물 노출)*

**[25~38초] 실천법**
"두 가지만 기억하세요. 첫째, 떼쓰는 아이에게 '하지마'가 아니라 '속상했구나'부터. 감정이 수용돼야 아이의 각성이 가라앉고 전두엽이 다시 작동하거든요. 둘째, 결정권을 점진적으로 넘기세요. 1학년 땐 부모가, 3학년 땐 함께, 5학년 땐 아이가 먼저 제안하게."
*(화면: 부모가 아이 눈높이에서 대화하는 장면 → 아이가 직접 규칙을 적는 장면으로 전환)*

**[38~45초] 마무리 + CTA**
"부모의 목소리가 사라진 자리에서도 아이가 스스로 멈추는 것. 그게 훈육의 완성이에요. 이외에도 연령별 자율성 키우는 법 더 알고 싶으신 분들은 아무 댓글이나 남겨주세요. '연령별 훈육 단계 체크리스트' 자료 보내드릴게요. 아이가 건강하게 자라길 바랍니다."
*(화면: 리드마그넷 미리보기 이미지 + "댓글 남기면 무료 자료 전송" 텍스트 오버레이)*

## 2. 리드마그넷 제안

**이름:** 연령별 훈육 단계 체크리스트
**유형:** PDF 체크리스트 (1~6학년 6단계)
**내용 3줄:**
- 1~2학년: 부모 주도 규칙 설정 + 감정 인정 스크립트 모음
- 3~4학년: 함께 규칙 정하기 워크시트 + 협상 대화 예시
- 5~6학년: 아이 주도 규칙 제안 템플릿 + 자기조절 자기점검표

|---------|------|----------------------------|----------|
| 0~3초 | 무표정하게 부모 지시 따르는 아이 클로즈업 | `child obedient parent`, `kid blank face` | 어두운 톤의 아이 옆모습 |
| 3~10초 | 교실에서 정리하는 아이 vs 멍하니 있는 아이 | `classroom kids cleaning`, `students tidying` | 책상 정리 손 클로즈업 |
| 10~18초 | 공사장 비계 철거 타임랩스 | `scaffolding removal timelapse`, `construction scaffold` | 건물 완공 드론 샷 |
| 18~25초 | 완성된 건물 + 자유롭게 걷는 사람 | `finished building wide shot`, `architecture complete` | 자유롭게 노는 아이 |
| 25~32초 | 부모가 아이 눈높이에서 대화하는 장면 | `parent child eye level talk`, `mother listening kid` | 손 잡고 대화하는 부모 |
