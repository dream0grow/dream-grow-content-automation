# 릴스 스크립트 (총 45초)

**[0~3초] 후킹**
"책 백 권 읽어도 줄거리 못 말하는 아이, 진짜 문제는 따로 있거든요."
(화면: 책장 가득한 책들 빠른 줌인 → 멍하게 책장 넘기는 아이 클로즈업)

**[3~10초] 문제 공감**
"부모는 더 사 주고, 더 읽으라고 합니다. 근데 책을 덮자마자 머릿속에서 사라져요."
"독서량 비슷한 두 아이, 한 명은 '재밌었어요'가 끝. 다른 한 명은 주인공이 왜 화났는지까지 말합니다."
(화면: 부모가 책을 사 주는 모습 → 아이 두 명이 같은 책을 읽는 분리 화면)

**[10~25초] 핵심 인사이트**
"차이는 책을 덮은 뒤에 있었어요."
"같은 정보라도 자기 언어로 말하고 자기 경험에 연결할 때, 머릿속에 깊게 박히거든요."
"듣기만 하는 독서보다, 질문하고 답하는 독서에서 어휘가 훨씬 빠르게 자랍니다."
"책은 매개일 뿐이에요."
(화면: 책장을 덮는 손 슬로모션 → 뇌 일러스트에 단어들이 연결되는 모션그래픽 → 부모와 아이가 마주 앉아 대화하는 장면)

**[25~38초] 실천법**
"그래서 정답은 '한 권 더'가 아니라 '한 권을 더 깊이 말하기'입니다."
"매일 5분, 세 가지만 물어보세요."
"첫째, 가장 기억에 남는 장면. 둘째, 주인공 마음 추측해보기. 셋째, 너라면 어떻게 했을지."
"한 줄짜리 답에 부모가 살을 붙여 되돌려 주세요. 이게 전부예요."
(화면: 화면 위에 세 가지 질문 텍스트 순차 등장 → 식탁에서 부모-아이 대화 장면 → 5:00 타이머 그래픽)

**[38~45초] 마무리 + CTA**
"책은 입력이고, 대화는 저장입니다. 책을 덮은 뒤 5분이 한 시간 독서를 결정해요."
"이외에도 아이와 책으로 대화하는 방법 더 알고 싶으신 분들은, 아무 댓글이나 남겨주세요."
"PEER 대화법 실천 카드 자료 보내드릴게요."
(화면: 리드마그넷 미리보기 이미지 + "댓글 남기면 무료 자료 전송" 텍스트 오버레이)
"아이가 책을 사랑하는 사람으로 자라길 바랍니다."

# 리드마그넷 제안

**이름:** 초등 PEER 대화법 실천 카드 (책 덮은 뒤 5분 질문 가이드)

**유형:** PDF 카드형 자료 (A4 4페이지, 가정 출력용)

**내용 3줄:**
- PEER 4단계(질문-평가-확장-반복) 적용 예시 대본 8가지 (저학년/고학년 분리)
- 장르별 질문 카드 30종 (그림책·동화·지식책·전래·과학·역사)
- 책 덮은 뒤 5분 루틴 체크리스트 + 부모용 NG 반응 모음
