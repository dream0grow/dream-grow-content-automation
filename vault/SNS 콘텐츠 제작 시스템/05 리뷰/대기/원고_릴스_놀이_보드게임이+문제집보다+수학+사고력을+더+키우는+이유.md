# 릴스 스크립트 (45초)

**[0~3초] 후킹**
"수학 문제집 10권을 풀어도 못하는 아이, 보드게임 한 판으로 좋아하게 된 아이. 차이가 뭘까요?"
(화면: 책상 위 수북한 문제집 → 보드게임 테이블로 빠르게 전환)

**[3~10초] 문제 공감**
"부모는 수학 실력을 문제 푼 양으로 측정합니다.
그런데 교실에서 진짜 잘하는 아이들은 따로 있거든요.
'왜 이렇게 돼요?'라고 묻는 아이들이요."
(화면: 문제집 푸는 아이의 지친 표정, 손을 드는 다른 아이)

**[10~25초] 핵심 인사이트**
"비고츠키는 말했습니다. 놀이가 한 단계 높은 사고를 끌어낸다고요.
문제집은 정해진 답을 찾게 하잖아요.
보드게임은 매 순간 전략을 세우게 합니다.
사고의 결 자체가 다른 거죠.
초등 아이는 손에 잡히는 걸로 사고합니다.
루미큐브 숫자 타일이 추상으로 가는 다리예요."
(화면: 루미큐브 타일 클로즈업, 아이가 조합 맞추는 손)

**[25~38초] 실천법**
"주말 한 시간을 비우세요.
루미큐브로 숫자 조합, 블로커스로 공간 감각, 다빈치 코드로 추론.
중요한 건요. 부모가 지는 모습을 보여주는 겁니다.
이기는 아이가 아니라 생각하는 아이가 되거든요."
(화면: 가족이 보드게임 하는 장면, 아빠가 웃으며 "졌네" 하는 모습)

**[38~45초] 마무리 + CTA**
"수학의 토대는 정답률이 아니라 수에 대한 호감입니다.
이외에도 초등 수학을 놀이로 잡는 법 더 알고 싶으신 분들은
아무 댓글이나 남겨주세요.
'초등 수학 보드게임 추천 가이드' 자료 보내드릴게요."
(화면: 리드마그넷 미리보기 이미지 + "댓글 남기면 무료 자료 전송" 텍스트)
"아이가 수학을 즐기며 자라길 바랍니다."

**유형:** PDF (12페이지 시각 자료)
**내용 3줄:**
- 학년별(1~6학년) 수학 영역과 매칭되는 보드게임 추천 리스트
- 게임별로 자극되는 수학 사고(연산, 공간, 논리, 확률) 분류표
- 가정에서 부모가 활용하는 발문 예시 30개

# B-roll 장면 목록

**[0~3초]**
- 장면: 책상 위 쌓인 수학 문제집 → 보드게임 테이블 전환
- 키워드: `math workbook stack`, `board game table`
- 대체: `kid frustrated homework` → `family playing board game`

**[3~10초]**
- 장면: 지친 아이 표정, 손드는 활발한 아이
- 키워드: `tired child studying`, `child raising hand classroom`
- 대체: `child doing math worksheet`, `curious kid asking question`

**[10~25초]**
- 장면: 루미큐브/숫자 타일 클로즈업, 조합 맞추는 손
- 키워드: `rummikub close up`, `numbered tiles hands`, `kids playing strategy game`
- 대체: `colorful number blocks`, `wooden math tiles`

**[25~38초]**
- 장면: 가족이 함께 보드게임 하는 장면, 부모가 웃으며 패배 인정
- 키워드: `family board game weekend`, `parent child playing game laughing`
- 대체: `dad losing game smiling`, `family game night`

**[38~45초]**
- 장면: 리드마그넷 PDF 미리보기 + 댓글창 오버레이
- 키워드: `comment notification phone`, `pdf preview mockup`
- 대체: `child happy with math`, `parent child high five`
