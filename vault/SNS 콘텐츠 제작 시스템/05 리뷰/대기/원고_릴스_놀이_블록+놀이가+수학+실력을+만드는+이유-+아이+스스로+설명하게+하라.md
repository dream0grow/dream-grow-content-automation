# 릴스 스크립트

**[0~3초] 후킹**
(화면: 블록을 높이 쌓는 아이의 손 클로즈업)
"비싼 원목 블록, 그거 사주신다고 아이 수학 머리 자라지 않습니다."

**[3~10초] 문제 공감**
(화면: 거실 가득 쌓인 블록 / 부모가 흐뭇하게 바라보는 모습)
"블록 놀이가 수학에 좋다더라. 그 말 한마디에 원목 블록 들이신 부모님 많으시잖아요.
근데 교실에서 보면, 블록을 가장 높게 쌓는 아이가 꼭 수학을 잘하지는 않거든요."

**[10~25초] 핵심 인사이트**
(화면: 책상 위 색종이 접기 두 그룹 비교 / 한쪽은 조용히, 한쪽은 친구에게 설명)
"하버드 인지심리학자 브루너는 학습을 세 단계로 봤습니다.
직접 쌓는 손, 머릿속 그림, 숫자와 기호.
말 없는 블록 놀이는 1단계에서 멈춥니다.
2단계, 3단계로 끌어올리는 다리가 바로 '설명'이거든요."

**[25~38초] 실천법**
(화면: 엄마가 아이 작품 앞에 앉아 질문하는 모습 / 아이가 손짓하며 설명)
"오늘 저녁, 딱 한 마디만 물어봐 주세요.
'이거 어떻게 만든 거야?'
아이가 '이거랑 이거 붙였어' 하면, '왜 이렇게 붙였어?' 한 번 더 들어가세요.
정답을 받으려는 게 아닙니다. 사고를 말로 꺼내는 게 수학적 추론의 시작이거든요."

**[38~45초] 마무리 + CTA**
(화면: 아이가 작품을 들고 환하게 설명하는 모습 → 리드마그넷 미리보기 이미지 + "댓글 남기면 무료 자료 전송" 텍스트)
"블록 가격이 아니라 부모의 질문이 아이의 수학을 만듭니다.
이외에도 가정에서 수학 사고력 키우는 질문법 더 알고 싶으신 분들은
아무 댓글이나 남겨주세요.
'놀이 속 수학 질문 30선' 자료 보내드릴게요.
아이가 수학을 즐기며 자라길 바랍니다."

# 리드마그넷 제안

**이름:** 놀이 속 수학 질문 30선
**유형:** PDF 가이드 (10페이지)
**내용 3줄:**
- 블록·레고·종이접기·요리 등 일상 놀이별 수학 사고 유도 질문 30개
- 자기설명 효과를 끌어내는 '왜?' 질문 단계별 예시 (1단계 관찰 → 2단계 비교 → 3단계 추론)
- 부모-자녀 실제 대화 샘플 5세트 + 연령별 활용 팁

| 타임코드 | 장면 설명 | Pexels/Pixabay 검색 키워드 | 대체 옵션 |
|---------|----------|--------------------------|----------|
| 0~3초 | 아이가 블록을 높이 쌓는 손 클로즈업 | `child stacking blocks`, `kid building blocks closeup` | 원목 블록 정렬 샷, 레고 쌓는 손 |
| 3~7초 | 거실 가득 쌓인 원목 블록 / 장난감 정리된 모습 | `wooden blocks living room`, `kids toy room` | 블록 박스 언박싱, 상점 진열 |
| 7~10초 | 부모가 아이 노는 모습을 바라보는 장면 | `parent watching child play`, `mom observing kid` | 거실에서 함께 노는 가족 |
| 10~17초 | 손-머리-숫자 단계 시각화 (책상 위 블록 → 종이에 그림 → 숫자) | `child math learning`, `kid drawing shapes` | 칠판에 도형 그리는 손, 노트에 숫자 쓰기 |
| 17~25초 | 색종이 접기 두 그룹 비교 / 친구에게 설명하는 아이 | `kids origami classroom`, `children explaining` | 모둠 활동 교실, 친구와 대화하는 어린이 |
| 25~32초 | 엄마와 아이가 작품 앞에 마주 앉아 대화 | `mother child conversation`, `parent asking question kid` | 식탁에서 대화, 아이 눈높이 대화 |
| 32~38초 | 아이가 손짓하며 자기 작품 설명하는 모습 | `child explaining toy`, `kid pointing showing` | 발표하는 어린이, 자랑하는 아이 |
| 38~42초 | 아이가 환하게 웃으며 완성품 들어 보이기 | `happy child holding craft`, `proud kid showing creation` | 작품 들고 미소 짓는 아이 |
| 42~45초 | 리드마그넷 PDF 미리보기 + 댓글 안내 텍스트 오버레이 | (그래픽 디자인) | PDF 페이지 넘기는 모션, 휴대폰 알림 화면 |
