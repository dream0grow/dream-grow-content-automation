# ChatGPT를 활용한 동화책/그림책 만들기 강의

## 홍보 문구
"뒤쳐질 것인가, 뒤집을 것인가"

## 영상/강의 구조

### 1. ChatGPT 자유 활용
- 자유롭게 질문하고 원하는 답 얻기 (자유활용 시간)

### 2. ChatGPT 제대로 활용하기
- 원하는 정보를 얻는 방법 (실습)
  - 결과물 또는 문제 상황 제시
  - 어떤 질문을 해야 할까? 질문하고 간단히 답하기
  - 실행 -> 쉽게 답이 나오지 않음 -> 활용법 안내 -> 질 높은 정보 획득

### 3. 교과수업 연계
- 국어: 동화책 만들기
- 음악 만들기, 수학 문제 만들기, 코딩하기

### 4. AI 시대 교육의 방향
- 질문과 대화 능력 (타인조망능력, 질문의 기술)
- 창조성 (경험을 통해 키울 수 있다)
- 배움에 대한 열린 태도와 동기

### 5. 도구를 사용하는 인간
- 칼/불 -> 컴퓨터/핸드폰 -> AI
- DIKW 피라미드: 데이터 -> 정보 -> [[C_지식|지식]] -> 지혜(전이)
- AI가 전해주는 데이터/정보를 지식/지혜로 활용하는 사람이 되는 것
