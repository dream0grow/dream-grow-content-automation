# 릴스 스크립트 (약 42초)

**[0~3초] 후킹**
"공부 못하는 아이보다, 공부가 무서운 아이가 더 위험합니다."
(화면: 시험지 위에 떨리는 아이 손 클로즈업 / 자막 굵게)

**[3~10초] 문제 공감**
"교실에서 시험지를 받자마자 손이 떨리는 아이가 있어요.
점수는 중간이라 부모님은 몰라요. 성적표만 보면 멀쩡하거든요.
그런데 이 아이, 매년 공부에서 한 걸음씩 멀어지고 있습니다."
(화면: 성적표 위에 OK 표시 → 아이 뒷모습이 점점 멀어지는 컷)

**[10~25초] 핵심 인사이트**
"캐롤 드웩이 말한 마인드셋 이론이에요.
공부를 '내 능력을 검증하는 시험대'로 보는 아이는 실패를 자기 부정으로 받아들이거든요.
점수가 아니라 정체성이 무너지는 거죠.

6학년 한 아이가 떠올라요. 점수는 상위권인데 수학 시간만 되면 화장실을 갔어요.
저학년 때 들었던 '왜 이걸 못해' 한마디가 박혀 있었거든요.
점수가 아니라 감정이 학습을 결정합니다."
(화면: 책을 덮어버리는 아이의 손 / 교실 빈자리 / 화장실 문 클로즈업)

**[25~38초] 실천법**
"집에서 가장 먼저 할 일은 격려가 아니에요. 감정 인정입니다.
'할 수 있어'가 아니라 '문제집 펴는 게 무거웠겠다'라고 말해보세요.
그리고 어려운 문제 10개가 아니라, 풀 수 있는 문제 3개부터 시작하세요.
공포는 '내가 풀어냈다'는 경험으로만 녹거든요."
(화면: 부모가 아이 옆에 앉아 연필 짚어주는 장면 / 쉬운 문제 3개 동그라미 치는 손)

**[38~45초] 마무리 + CTA**
"공부를 못하는 아이는 방법을 배우면 되지만, 공부가 무서운 아이는 감정을 먼저 회복해야 합니다.
이외에도 공부 공포를 풀어주는 부모 말 한마디가 더 궁금하신 분들은
아무 댓글이나 남겨주세요.
'공부 공포 회복 부모 대화 스크립트' 자료 보내드릴게요.
아이가 공부 앞에서 작아지지 않길 바랍니다."
(화면: 리드마그넷 미리보기 이미지 + "댓글 남기면 무료 자료 전송" 텍스트 오버레이)

# 리드마그넷 제안

**이름:** 공부 공포 회복 부모 대화 스크립트
**유형:** PDF 가이드 (A4 6~8p)
**내용 3줄:**
1. 상황별 '감정 인정 → 자기 효능감 회복' 부모 대화 예시 12개 (시험 전/오답 직후/문제집 펴기 거부 등)
2. '할 수 있어' 대신 쓰는 7가지 대체 문장과 그 심리학적 근거 (드웩·브랜든 이론 기반)
3. '작은 성공' 설계 4단계 워크시트 — 오늘 풀 문제 3개를 부모와 함께 고르는 체크리스트

# B-roll 장면 목록

**[0~3초] 후킹**
- 장면: 시험지 위 떨리는 아이의 손, 연필 잡은 손가락 클로즈업
- Pexels/Pixabay 키워드: `child hand shaking pencil`, `kid taking exam close up`, `anxious student test paper`
- 대체: 빈 시험지를 응시하는 아이 뒷모습 클로즈업

**[3~10초] 문제 공감**
- 장면: 성적표 클로즈업 → 아이가 책상에서 멀어지는 뒷모습
- Pexels/Pixabay 키워드: `report card close up`, `child walking away school`, `student desk back view`
- 대체: 책가방을 무겁게 끄는 아이의 뒷모습

**[10~25초] 핵심 인사이트**
- 장면: 책을 덮어버리는 손 / 교실 빈자리 / 복도 화장실 문
- Pexels/Pixabay 키워드: `closing textbook hands`, `empty classroom seat`, `school hallway bathroom door`, `child hiding face`
- 대체: 수학 문제집 위 멈춰 있는 연필, 창밖을 응시하는 아이

**[25~38초] 실천법**
- 장면: 부모와 아이가 나란히 앉아 책상 위 문제집을 함께 보는 장면 / 쉬운 문제에 동그라미 치는 손
- Pexels/Pixabay 키워드: `parent helping child homework`, `mother teaching kid math`, `circle correct answer pencil`
- 대체: 아이가 문제 다 풀고 미소 짓는 클로즈업, 부모가 아이 어깨에 손 얹는 장면

**[38~45초] 마무리 + CTA**
- 장면: 따뜻한 톤의 아이-부모 대화 풀샷, 리드마그넷 PDF 표지 오버레이
- Pexels/Pixabay 키워드: `warm family conversation`, `parent child talking calm`, `mother son back light`
- 대체: 아이가 자신감 있게 책상에 앉아 공부 시작하는 장면
