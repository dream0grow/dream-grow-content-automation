# 릴스 스크립트 (45초)

**[0~3초] 후킹**
"아이가 책을 읽어도 내용을 모른다고요?
문해력 부족이 아닙니다. 질문이 잘못된 겁니다."
(화면: 책 덮은 아이의 무표정 얼굴 클로즈업 → 물음표 그래픽)

---

**[3~10초] 문제 공감**
"부모님, 이렇게 물어보시잖아요.
'재밌었어? 무슨 내용이야?'
아이는 '응, 주인공이 모험 가.' 끝.
이건 대화가 아니라 정보 확인이거든요."
(화면: 부모-아이 식탁 대화 시뮬레이션, 짧은 답변 자막)

---

**[10~25초] 핵심 인사이트**
"5학년 교실에서 같은 동화를 읽혔어요.
'주인공이 왜 그랬을까?' 절반이 입을 닫았습니다.
질문을 바꿨더니 손이 올라왔어요.
'네가 주인공이었다면 어떻게 했을까?'
사고는 질문이 만든 발판 위에서 자라거든요."
(화면: 교실에서 손드는 아이들 → 발판처럼 올라가는 그래픽 연출)

---

**[25~38초] 실천법 (3단 질문법)**
"가정에서 오늘부터 이렇게 해보세요.
첫째, '제일 기억에 남는 장면은?'
둘째, '왜 그렇게 됐을까?'
셋째, '너라면 어떻게 했을까?'
순서를 건너뛰면 무너집니다. 꼭 차례대로요."
(화면: 1·2·3 숫자 카드 순차 등장, 부모-아이 거실 대화 컷)

---

**[38~45초] 마무리 + CTA**
"책 한 권 더 읽히는 것보다 좋은 질문 하나가 더 깊습니다.
이외에도 아이 사고력을 키우는 독후 질문법 더 알고 싶으신 분들은
아무 댓글이나 남겨주세요.
'독후 대화 3단 질문법 카드' 자료 보내드릴게요."
(화면: 리드마그넷 미리보기 이미지 + "댓글 남기면 무료 자료 전송" 텍스트)
"아이가 책으로 사고하는 사람으로 자라길 바랍니다."

# 리드마그넷 제안

**이름:** 독후 대화 3단 질문법 카드
**유형:** PDF 카드 세트 (A6 사이즈, 인쇄용 6장)

**내용 3줄:**
1. 1단계 기억 / 2단계 분석 / 3단계 창조 - 단계별 질문 예시 각 5개
2. 학년별(저/중/고학년) 변형 질문 템플릿
3. 절대 피해야 할 닫힌 질문 vs 열린 질문 비교표
