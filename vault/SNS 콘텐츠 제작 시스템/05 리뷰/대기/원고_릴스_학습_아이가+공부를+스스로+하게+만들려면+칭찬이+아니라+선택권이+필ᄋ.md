# 릴스 스크립트 (45초)

**[0~3초] 후킹**
"잘했어"가 쌓일수록 아이는 공부에서 멀어집니다.
(화면: 부모가 아이에게 "잘했어" 칭찬하는 장면, 아이 표정은 무덤덤)

**[3~10초] 문제 공감**
칭찬 스티커도 붙이고, 안아도 주는데 왜 시키지 않으면 안 할까요.
교실에서 가장 자주 듣는 학부모 고민입니다.
(화면: 책상 앞에 멍하니 앉아있는 아이, 칭찬 스티커판 클로즈업)

**[10~25초] 핵심 인사이트**
원인은 노력이 아니라 구조에 있거든요.
아이의 행동을 결정하는 사람이 부모로 고정되어 있는 겁니다.
정해준 시간에, 정해준 분량을, 정해준 방식으로 공부하면
아이는 '내가 선택한 일'이라는 감각을 잃어버려요.
공부하는 아이를 만드는 건 칭찬이 아니라 선택권입니다.
(화면: 시계, 문제집, 빨간펜이 일렬로 놓인 통제된 책상 → 아이가 직접 책을 고르는 장면)

**[25~38초] 실천법**
가정에서 두 가지만 바꿔보세요.
첫째, "수학 30분 해"가 아니라 "수학과 독서 중 뭐 먼저 할래?"
둘째, "몇 점 받았어?"가 아니라 "어떤 부분이 어려웠어?"
선택한 사람이 책임을 가져가고, 과정 질문이 메타인지를 깨웁니다.
(화면: 부모가 아이에게 두 권의 책을 보여주며 선택 권하는 장면, 아이가 손가락으로 가리키는 모습)

**[38~45초] 마무리 + CTA**
부모가 평가자에서 동행자로 자리를 옮길 때 자기주도학습이 시작됩니다.
이외에도 자기주도학습 실천법 더 알고 싶으신 분들은
아무 댓글이나 남겨주세요.
'자기주도학습 부모 언어 가이드' 자료 보내드릴게요.
(화면: 리드마그넷 미리보기 PDF 표지 + "댓글 남기면 무료 자료 전송" 텍스트)
아이가 자기 공부의 주인으로 자라길 바랍니다.

# 리드마그넷 제안

**이름:** 자기주도학습 부모 언어 가이드

**유형:** PDF 가이드북 (10~12쪽)

**내용:**
1. 결정권을 옮기는 30가지 질문 카드 (시간/과목/방법별)
2. 결과 평가어 → 과정 질문어 변환 사전 (상황별 30개)
3. 일주일 실천 체크리스트 + 부모-자녀 대화 예시 5세트
