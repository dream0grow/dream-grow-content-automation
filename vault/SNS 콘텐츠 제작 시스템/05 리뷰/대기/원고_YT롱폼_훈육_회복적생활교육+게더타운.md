# 메타버스에서 배우는 회복적 생활교육 (게더타운 활용 연수)

## 핵심 구조
- 강의 내용을 동영상으로 제작 → 선생님들이 볼 수 있도록
- 핵심 내용만 명료하게 정리해서 안내
- 회복적 생활교육을 잘하면 수업이 잘되는 이유 (교실안정감 → 한 해가 편한 과학적 단계)
- 참고: 루이스 코졸리노 '애착 교실'

## 게더타운 활용 교육 활동
### 그림 복습 퀴즈
- 그룹핑 에어리어 만들기 → 순서대로 오기 → 단어 제시 → 되돌아가서 그림으로 설명

### 보물 찾기 응용
- 회복적 생활교육 연수 자료가 있는 방을 게더타운 내 아이템에서 찾기
- 방 이름(호수)은 회복적 생활교육 관련 단어들로 구성
- 게더타운용 방탈출 만들기도 가능

### 협업 도구 연동
- edupuzzle, liveworksheet 등 참고
- 선생님들이 함께 자료를 만드는 협업 활동

## 연수 론칭 전략
1. 포폴로 학교
2. 도내 교사 (에듀니티 초청 → 전국 홍보)
3. 전국 안내

### 과정 구성
- 회복적생활교육만 듣는 과정 / 회복적생활교육 + 게더타운 과정
- 게더타운 연수: 월 1회 x 3개월
- 에듀니티에 16차시 복습 강의 제공

## 콘텐츠 제작 아이디어
- 아이들과 대화하는 장면을 영상으로 촬영 → 편집
- 아이들/학부모 인터뷰
- 강의자료 읽고 인사이트 + 퀴즈 + 적용방법 생각하기
