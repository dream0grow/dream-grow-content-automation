# 릴스 스크립트 (약 43초)

**[0~3초] 후킹**
"그러면 안 돼" 이 한마디 때문에, 아이의 떼쓰기가 두 배로 길어집니다.
(화면: 마트 바닥에 누워 우는 아이, 당황한 부모 클로즈업)

**[3~10초] 문제 공감**
마트 바닥에 드러누운 아이.
"일어나, 사람들이 봐." 말할수록 아이는 더 크게 웁니다.
이유는 단순해요. 아이의 감정이 아직 단 한 사람에게도 가닿지 않았거든요.
(화면: 다급한 부모 표정, 더 크게 우는 아이의 얼굴)

**[10~25초] 핵심 인사이트**
심리학자 너새니얼 브랜든은 자존감을 두 축으로 봤습니다.
잘 해낼 수 있다는 자기 효능감, 그리고 존중받을 가치가 있다는 자기 가치감.
이 중 가치감은 내 감정이 누군가에게 받아들여질 때 자랍니다.
"그러면 안 돼"가 먼저 나가면, 훈육이 아니라 가치감부터 깎이는 거예요.
(화면: 두 축 그래픽 — '자기 효능감 / 자기 가치감')

**[25~38초] 실천법**
순서만 바꿔보세요. 세 단계입니다.
하나. 감정 번역. "장난감 못 사서 속상했구나."
둘. 잠시 기다리기. 아이 호흡이 가라앉을 때까지.
셋. 그때 규칙을 말합니다. "그래도 바닥에 눕는 건 안 돼."
감정은 받아들이되, 행동에는 경계를 둡니다.
(화면: 부모가 아이 눈높이로 앉아 천천히 말 거는 장면)

**[38~45초] 마무리 + CTA**
아이의 자존감은 칭찬이 아니라 감정 인정에서 시작됩니다.
이외에도 감정 인정 대화법 더 알고 싶으신 분들은
아무 댓글이나 남겨주세요.
"부모 감정 인정 대화법 카드" 자료 보내드릴게요.
(화면: 리드마그넷 미리보기 이미지 + "댓글 남기면 무료 자료 전송" 텍스트)
아이가 자신의 감정과 친해지며 자라길 바랍니다.

# 리드마그넷 제안

**이름:** 부모 감정 인정 대화법 카드 (3단계 실전 카드 14장)
**유형:** PDF 카드형 가이드 (인쇄해서 냉장고에 붙이기 가능)
**내용 3줄:**
- 상황별 감정 번역 문장 30개 (떼쓰기/형제다툼/등원거부 등)
- 감정 인정 → 기다림 → 규칙의 3단계 대화 스크립트
- 절대 하면 안 되는 부정 표현 vs 대체 표현 비교표

# B-roll 장면 목록

**[0~3초] 후킹**
- 필요 장면: 마트 바닥에 누워 우는 아이, 당황해 굳은 부모 얼굴 클로즈업
- 검색 키워드: `child tantrum supermarket`, `toddler crying floor`, `parent stressed shopping`
- 대체 옵션: 식당 바닥에서 우는 아이, 길거리 떼쓰기 장면

**[3~10초] 문제 공감**
- 필요 장면: 부모가 다급하게 말 거는 모습, 아이가 더 크게 우는 얼굴 컷
- 검색 키워드: `parent scolding child`, `crying child close up`, `frustrated mother toddler`
- 대체 옵션: 아빠가 아이를 일으키려는 장면, 주변 사람들이 쳐다보는 컷

**[10~25초] 핵심 인사이트**
- 필요 장면: 두 축 그래픽 (자기 효능감 / 자기 가치감), 책 페이지 넘기는 컷, 아이의 평온한 얼굴
- 검색 키워드: `self esteem diagram`, `psychology book pages`, `child peaceful face`
- 대체 옵션: 모션 그래픽으로 직접 제작, 너새니얼 브랜든 책 표지 이미지

**[25~38초] 실천법**
- 필요 장면: 부모가 무릎 꿇고 눈높이로 대화하는 장면, 아이를 안아주는 모습, 차분해진 아이
- 검색 키워드: `parent eye level child talk`, `mother comforting kid`, `gentle parenting moment`
- 대체 옵션: 1-2-3 숫자 자막 그래픽 + 각 단계별 짧은 컷

**[38~45초] 마무리 + CTA**
- 필요 장면: 웃으며 손잡고 걷는 부모-자녀, 리드마그넷 카드 PDF 미리보기 모션
- 검색 키워드: `parent child walking happy`, `family hand in hand`, `kid smiling outdoor`
- 대체 옵션: 노을 배경 실루엣, 아이가 부모 품에 안긴 정적 컷
