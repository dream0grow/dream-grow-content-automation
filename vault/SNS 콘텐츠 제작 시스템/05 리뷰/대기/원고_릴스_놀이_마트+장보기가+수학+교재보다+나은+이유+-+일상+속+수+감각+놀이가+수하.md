# 릴스 스크립트 (45초)

**[0~3초] 후킹**
대사: "수학 머리는 책상이 아니라 마트에서 만들어집니다."
(화면: 마트 카트를 미는 아이의 손 클로즈업 / 학습지 위에 연필 멈춘 손과 교차 컷)

**[3~10초] 문제 공감**
대사: "연산은 빠른데 문장제만 나오면 멈추는 아이. 너무 많잖아요. 계산은 되는데 수의 '의미'를 모르는 거거든요."
(화면: 문제집을 보며 한숨 쉬는 아이 / 지우개로 답을 지우는 손)

**[10~25초] 핵심 인사이트**
대사: "초등은 '구체적 조작기'예요. 손에 잡히는 사물로 사고하는 단계죠. 이때 종이 위 기호만 들이밀면 아이는 외울 수밖에 없어요. 교실에서 매년 봅니다. 구체물로 충분히 논 아이는 분수가 와도 흔들리지 않아요. 바로 추상으로 끌려간 아이는 5학년에서 흔들리기 시작합니다."
(화면: 블록·과일을 만지는 아이 손 / 분수 케이크를 자르는 장면 / 교실 칠판 앞에서 머리 긁는 학생)

**[25~38초] 실천법**
대사: "오늘부터 두 가지만 해보세요. 첫째, 마트에서 묻기. '우유 2개에 3천원이면 4개는?' 둘째, 요리할 때 계량컵 쥐여주기. 1/2컵 두 개가 1컵이 되는 걸 손으로 느끼는 게 분수의 시작이거든요."
(화면: 카트에 우유 담는 부모와 아이 / 영수증 같이 보는 장면 / 계량컵으로 밀가루 옮기는 아이 손)

**[38~45초] 마무리 + CTA**
대사: "교재 한 장보다 카트 한 번이 낫습니다. 이외에도 일상 속 수학 놀이 더 알고 싶으신 분들은 아무 댓글이나 남겨주세요. '일상 속 수학 놀이 30선' 자료 보내드릴게요. 아이가 수를 즐기며 자라길 바랍니다."
(화면: 리드마그넷 미리보기 이미지 + "댓글 남기면 무료 자료 전송" 텍스트 오버레이)

# 리드마그넷 제안

**이름:** 일상 속 수학 놀이 30선
**유형:** PDF 가이드 (10페이지)
**내용 3줄:**
- 마트·주방·산책 등 장소별 수학 대화 스크립트 30가지
- 학년별(1~6학년) 권장 활동과 개념 연결 표
- 부모가 바로 따라할 수 있는 질문 예시 + 아이 반응 대처법
