# 릴스 스크립트 (45초)

**[0~3초] 후킹**
"수학 1등급, 초등 5학년에 이미 갈립니다."
(화면: 칠판에 분수 문제 클로즈업 → '5학년' 자막 강조)

**[3~10초] 문제 공감**
"많은 부모가 학원부터 알아보시잖아요. 한 달 50, 100만원이 우습게 나갑니다.
근데 진짜 문제는 학원이 아닙니다."
(화면: 학원가 거리, 학원비 영수증, 한숨 쉬는 부모 손)

**[10~25초] 핵심 인사이트**
"순서를 건너뛰는 것이 문제거든요.
싱가포르 수학에는 CPA라는 원리가 있습니다. 구체물, 그림, 그다음이 숫자예요.
구체물을 건너뛰면 계산은 합니다. 이해는 못 합니다.
특히 분수에서 뚫린 구멍이 비와 비율까지 아이를 멈추게 해요."
(화면: 블록 교구 → 분수 그림 → 숫자 식으로 전환되는 모션 그래픽 / 분수 문제에서 X 표시)

**[25~38초] 실천법**
"가정에서 두 가지만 하세요.
첫째, 단원이 끝나면 아이가 직접 말로 설명하게 합니다. 막히면 이해 못 한 거예요.
둘째, 응용 문제 답지는 부모가 봅니다. 아이에겐 힌트만 주세요."
(화면: 아이가 식탁에서 부모에게 설명하는 모습 / 부모가 답지 보며 미소짓는 손)

**[38~45초] 마무리 + CTA**
"수학은 재능이 아니라 순서입니다.
이외에도 초등수학 영역별 학습 순서가 궁금하신 분들은 아무 댓글이나 남겨주세요.
초등수학 영역별 로드맵 자료 보내드릴게요.
아이가 수학을 즐기며 자라길 바랍니다."
(화면: '댓글 남기면 무료 자료 전송' 텍스트 + 로드맵 PDF 미리보기 이미지)

# 리드마그넷 제안

**이름:** 초등 1~6학년 수학 영역별 학습 로드맵
**유형:** PDF 가이드 (6~8페이지)
**내용 3줄:**
- 학년별 핵심 단원과 무너지기 쉬운 지점 정리
- 분수·소수·비율 연결 구조 한눈에 보는 도식
- 가정에서 단원별 구체물·그림으로 가르치는 실천 팁
