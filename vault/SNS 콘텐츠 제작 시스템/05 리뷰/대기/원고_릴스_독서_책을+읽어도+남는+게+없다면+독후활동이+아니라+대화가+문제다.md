# 릴스 스크립트 (45초)

**[0~3초] 후킹**
독후감 많이 쓴 아이가 더 많이 배운다는 건 착각이거든요.
(화면: 독후감 노트 위에 빨간 X 표시 오버레이, 빠른 줌인)

**[3~10초] 문제 공감**
"오늘 무슨 책 읽었어?" "재밌었어?" "응." 끝.
부모는 시켰고 아이는 글자를 봤는데, 머리에 남은 건 거의 없잖아요.
(화면: 식탁에서 짧게 끝나는 부모-아이 대화, 책장 덮는 손)

**[10~25초] 핵심 인사이트**
문제는 집중력이 아니라 대화의 구조입니다.
비고츠키는 말했어요. 아이는 혼자 읽을 때보다 능숙한 사람과 말을 나눌 때 더 깊이 생각한다고.
책은 입력, 대화는 출력입니다.
글로 정리하기 전에 입으로 꺼내는 과정이 먼저거든요.
(화면: 책 펼친 손 → 말풍선 두 개 오가는 모션그래픽 → "입력 / 출력" 자막)

**[25~38초] 실천법**
오늘 밤 5분, 이 세 가지만 물어보세요.
"제일 기억나는 장면 하나만."
"주인공이 너였다면 어떻게 했을 것 같아?"
"한 줄만 골라본다면?"
정답을 묻는 게 아니라 아이의 관점을 묻는 겁니다.
(화면: 자막 카드 3개 순차 등장, 침대 옆 스탠드 조명 아래 부모-아이 대화 컷)

**[38~45초] 마무리 + CTA**
독서의 깊이는 권수가 아니라 대화의 밀도예요.
이외에도 책 읽고 나누는 질문 더 알고 싶으신 분들은
아무 댓글이나 남겨주세요.
**'책 읽은 후 5분 대화 질문 카드 30선' 자료 보내드릴게요.**
(화면: 리드마그넷 PDF 미리보기 이미지 + "댓글 남기면 무료 자료 전송" 텍스트)
아이가 책을 사랑하는 사람으로 자라나길 바랍니다.

# 리드마그넷 제안

- **이름:** 책 읽은 후 5분 대화 질문 카드 30선
- **유형:** PDF 카드형 자료 (인쇄해서 책갈피로 사용 가능)
- **내용 3줄:**
  1. 장면·인물·문장 3축으로 분류한 30개 질문 (저학년·중학년·고학년 단계별)
  2. 정답 없는 열린 질문만 선별 — 부모가 먼저 답하는 예시 문장 동봉
  3. 책 장르별(동화/위인전/과학) 추천 질문 매칭표

# B-roll 장면 목록

| 타임코드 | 장면 | Pexels/Pixabay 검색 키워드 (영어) | 대체 옵션 |
|---|---|---|---|
| 0~3초 | 독후감 노트 클로즈업, X표시 오버레이 | `book report notebook`, `child writing homework` | 빨간 펜으로 줄 긋는 손 |
| 3~10초 | 식탁에서 부모-아이 짧은 대화, 책장 덮는 손 | `parent child dinner table`, `child closing book` | 거실 소파에서 책 들고 멍한 아이 |
| 10~15초 | 책 펼친 손 클로즈업 | `child reading book closeup`, `open book hands` | 도서관에서 책 고르는 아이 |
| 15~25초 | 말풍선 모션그래픽, "입력/출력" 자막 카드 | `speech bubble animation`, `conversation graphic` | 두 사람 대화하는 실루엣 |
| 25~38초 | 침대 옆 스탠드 조명 아래 부모-아이 대화 | `parent child bedtime reading`, `mother daughter talking bed` | 거실 러그 위에서 책 보며 이야기 |
| 38~45초 | 리드마그넷 PDF 미리보기 + CTA 텍스트 | (자체 제작 그래픽) | 책장에 책 정리하는 아이 손 |
