# 릴스 스크립트: 보드게임 반칙은 인지 발달의 신호

## 1. 릴스 스크립트 (45초)

**[0~3초] 후킹**
"반칙하지 마"라고 혼내고 계신가요? 그거 도덕 문제가 아니거든요.
*(화면: 아이가 보드게임 카드를 슬쩍 바꾸는 손 클로즈업, 빠른 줌인)*

**[3~10초] 문제 공감**
주사위를 다시 굴리고, 슬쩍 카드를 바꾸고. 분명 규칙을 알려줬는데도 말이죠. 답답하시죠.
*(화면: 부모가 한숨 쉬며 머리 짚는 모습, 흩어진 보드게임 말)*

**[10~25초] 핵심 인사이트**
근데 피아제의 도덕 발달 이론에서요. 7세 전후 아이는 규칙을 '어른이 정한 절대 명령'으로 믿거든요. 반칙은 거짓말이 아니라, "이걸 바꾸면 어떻게 되지?" 규칙의 본질을 시험하는 거예요. 여기서 막아버리면, 규칙은 영원히 외부에서 주어진 명령으로만 남습니다.
*(화면: 보드게임 말 위로 책 페이지 오버레이 'Piaget', 아이가 골똘히 고민하는 눈빛)*

**[25~38초] 실천법**
교실에서 할리갈리 다툼이 났을 때, 저는 한마디만 했어요. "그럼 너희끼리 규칙을 다시 정하고 시작해." 아이들은 5분 토론하고 새 규칙을 종이에 적었거든요. 가정에서도 게임 시작 전에 물어보세요. "이번엔 어떤 규칙으로 할까?"
*(화면: 종이에 규칙 적는 아이들 손, 가족이 둘러앉아 토론하는 모습)*

**[38~45초] 마무리 + CTA**
보드게임은 승부가 아니라 합의의 훈련장이에요. 이외에도 보드게임으로 사고력 키우는 법 더 알고 싶으신 분들은, 아무 댓글이나 남겨주세요. '보드게임 협상력 가이드' 자료 보내드릴게요. 아이가 규칙을 협상할 줄 아는 사람으로 자라길 바랍니다.
*(화면: 리드마그넷 미리보기 이미지 + "댓글 남기면 무료 자료 전송" 텍스트)*

## 2. 리드마그넷 제안

**이름:** 보드게임으로 키우는 협상력 가이드
**유형:** PDF 가이드 (8~10페이지)
**내용 3줄:**
- 연령별(5~12세) 추천 보드게임 + 변형 규칙 예시 모음
- "규칙 협상" 부모-자녀 대화 스크립트 (게임 시작/중간/끝)
- 아이가 만든 규칙 평가 체크리스트 (다음에 또 쓸 규칙 판별법)

|------|------|------------------------------|----------|
| 0~3초 | 아이가 카드를 슬쩍 옮기는 손 클로즈업 | `child hand board game`, `kid playing cards close up` | 주사위를 몰래 다시 굴리는 손 |
| 3~10초 | 부모가 한숨 쉬며 게임 보는 모습 | `parent frustrated game`, `family tension board game` | 아이와 부모 대치 장면 |
| 10~15초 | 보드게임 말, 주사위 텍스처 | `board game pieces macro`, `dice rolling slow motion` | 체스판 위 말 클로즈업 |
| 15~25초 | 아이가 골똘히 생각하는 눈빛 | `curious child thinking`, `kid eyes focused` | 책 페이지 위 'Piaget' 텍스트 오버레이 |
| 25~32초 | 교실에서 아이들이 토론하는 장면 | `kids classroom discussion`, `students talking group` | 할리갈리 종 치는 손 |

| 38~45초 | 행복하게 보드게임 하는 가족 + 리드마그넷 이미지 | `happy family board game`, `parent child laughing game` | PDF 미리보기 그래픽 |
