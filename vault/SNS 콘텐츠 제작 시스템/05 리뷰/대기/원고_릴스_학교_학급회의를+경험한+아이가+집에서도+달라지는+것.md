# 릴스 스크립트 (45초)

**[0~3초] 후킹**
"집에서 갑자기 말대답이 늘었다고요? 학급회의 때문일 수 있습니다."
(화면: 아이가 팔짱 끼고 부모 앞에서 입을 삐죽 내미는 장면, 클로즈업)

**[3~10초] 문제 공감**
"학교 다녀온 뒤로 버릇이 없어졌다고 느끼시죠.
그런데 그건 후퇴가 아니거든요.
자기 목소리를 내기 시작한 신호입니다."
(화면: 부모가 당황한 표정으로 아이를 바라보는 장면)

**[10~25초] 핵심 인사이트**
"퍼실리테이션은 답을 주는 기술이 아니라,
모든 목소리가 안전하게 들리는 환경을 만드는 기술입니다.
학급회의에서 교사는 한 발 물러섭니다.
말 많은 아이보다 침묵하는 아이를 먼저 살피고요.
1인 2분 발언, 말 끊지 않기.
이 안전장치 안에서 아이는 처음으로 자기 의견이 존중받는 경험을 합니다."
(화면: 교실에서 아이들이 둥글게 앉아 회의하는 장면, 교사는 뒤에서 지켜봄)

**[25~38초] 실천법**
"집에서도 가족회의를 해보세요.
단, 부모가 결론을 정해두면 안 됩니다.
아이가 불만을 말할 때 말대꾸로 막지 마시고
'무엇을 봤는지, 어떤 감정인지, 무엇이 필요한지' 되물어 주세요.
'엄마 미워'가 '약속을 못 지켜서 서운해'로 바뀝니다."
(화면: 식탁에서 부모와 아이가 마주 앉아 대화하는 장면, 엄마가 고개 끄덕임)

**[38~45초] 마무리 + CTA**
"자기 표현이 곧 자기 책임으로 자라납니다.
이외에도 가정에서 시민성을 기르는 법 더 알고 싶으신 분들은
아무 댓글이나 남겨주세요.
'가족회의 운영 가이드' 자료 보내드릴게요.
아이가 자기 목소리를 가진 사람으로 자라길 바랍니다."
(화면: 리드마그넷 미리보기 이미지 + "댓글 남기면 무료 자료 전송" 텍스트)

# 리드마그넷 제안

**이름**: 가족회의 운영 가이드
**유형**: PDF 체크리스트 + 대화 스크립트
**내용**:
1. 가족회의 5단계 진행 순서 (주제 정하기 → 1인 2분 발언 → 공감 → 합의 → 기록)
2. 비폭력대화 4요소(관찰-감정-욕구-부탁) 가정용 대화 예시 10개
3. 말대답을 자기표현으로 전환하는 되묻기 질문 카드 7개
