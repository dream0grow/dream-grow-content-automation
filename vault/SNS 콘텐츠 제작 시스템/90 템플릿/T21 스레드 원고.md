# 스레드 구조 설계

## 후킹 패턴 선택 (1/ 에 해당)
> 검증된 패턴 중 하나를 선택한다.
- [ ] 권위→충격→공감 : "[경력]입니다. [상식 뒤집는 주장]."
  - 예: "초등교사입니다. 초등 수학 공부, 문제지만 풀려서는 안 됩니다."
- [ ] 권위→충격→반전→과학근거 : "[경력]입니다. 제발, [역설적 조언]."
  - 예: "18년차 초등교사입니다. 제발, 아이를 좀 심심하게 키우세요."
- [ ] 자기서사→고난→전환→깨달음 : 실패 경험으로 시작, 핵심 깨달음으로 마무리
- [ ] 숫자→충격→궁금증 : "[놀라운 숫자]. [왜 그런지 설명하겠다]."
  - 예: "유튜브에서 구독자를 빠르게 늘리는 방법은 딱 하나."
- [ ] 공감→문제직면→해결 : 타겟의 고민을 정확히 짚는 것으로 시작
