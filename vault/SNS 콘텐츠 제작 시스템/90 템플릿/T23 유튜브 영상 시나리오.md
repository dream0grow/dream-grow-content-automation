# 사용 가이드
> 유튜브 롱폼(5분 이상) 전용 시나리오 템플릿. T09(본문 초안)가 범용이라면, 이 템플릿은 유튜브 영상 제작에 최적화되어 있다.
> 유튜브 롱폼 핵심 원칙:
> 1. 도입 구조 정석: 유인제시 -> 문제 제시 -> 해결책 유인
> 2. 시청 유지 장치를 블록마다 배치 ([[C_유머|유머]]/공감/사진전환/질문)
> 3. 썸네일+제목이 클릭을 만들고, 도입 30초가 이탈을 막는다
> 4. 각 블록에 반드시 근거/사례를 넣는다
