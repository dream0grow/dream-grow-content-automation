# 사용 가이드
> 하나의 원고를 여러 채널로 변환할 때 사용한다.
> 모든 채널을 채울 필요 없다. 발행할 채널만 작성한다.
> 리믹스(T20)와 차이: 리믹스는 "기존 콘텐츠 → 새 원고", 채널별 변환은 "완성 원고 → 각 채널 포맷".
