## 핵심 분석

### 1. 떡상 영상 비중과 구독자 성장의 상관관계
- 쇼컷: 전체 520만 조회 중 상위 4개 = 442만 (85%)
- 마세습: 전체 630만 조회 중 상위 4개 = 358만 (55%)
- **결론**: 소수의 떡상 영상이 채널 전체 성장을 견인

### 2. 기획 진화 패턴
- **신사임당**: 돈버는 정보 전달 -> 창업 다마고치 -> 인터뷰
- **총몇명**: 뮤직비디오 -> 기묘한 이야기 -> 혼밥러
- **코너**: 너프건 장난감 리뷰 -> 너프 전쟁 기획 -> 미스터비스트 벤치마킹(돈 걸고 미션)
- **결론**: 성공 채널은 한 포맷에 머무르지 않고 계속 기획을 진화시킨다

### 3. 정체 원인 분석
- 너진똑: 수십만에 머무름 = **속도** 문제
- 포리얼, 자청: 신사임당 벤치마킹 인터뷰 형식 = **식상함**
- 필미필미: **참여도** 하락

### 4. 떡상 요소 우선순위
1. 기획력 (새로운 참신함, 새로운 가치)
2. 영상 퀄리티 (재미/속도/참여)
3. 썸네일
4. 언어 (영어로만 해도 추가 성장 가능)

## 벤치마킹 적용 포인트
- 100만을 목표로 크게 그리되, 우선 떡상 영상 1개에 집중
- 표본이 넓은 주제(돈, 건강, 관계) 선택
- 기획의 진화를 미리 3단계까지 설계
- 영어 자막/더빙으로 글로벌 확장 가능성 열어두기
