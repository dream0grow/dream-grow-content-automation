# 유튜브 도입부 후킹 패턴 벤치마킹

## 도입부 구조의 정석
**유인제시 -> 문제 제시 -> 유인(해결책) 제시**

---

## 패턴 1: [공감]-[정의]-[설명] 구조
**출처**: 불안장애 바로 알기 (https://www.youtube.com/watch?v=J2V1AHu5wfU)

- 공감: "가슴이 답답하고 소화도 잘 안돼요. 매일 쫓기는 기분이 들어요."
- 정의: "불안 장애란 이유없이 불안을 느끼거나 불안의 정도가 지나친 정신질환입니다."
- 설명: "원래 불안과 공포는 위험한 상태를 대비하기 위한 생존신호로..."

---

## 패턴 2: 실제 사례 -> 궁금증 폭발
**출처**: 사기꾼의 심리학 (그림으로 보는 책 채널)

- 실제 사례: "사기꾼이 가짜 돈으로 450만원짜리 반지를 구매하는 실제 영상"
- 궁금증: "이는 실제 실험에서 일어난 일입니다. 어떻게 된 걸까요?"
- 공감+반론: "우리는 '나라면 절대 속지 않았을텐데' 하지만 피해자들은 '무언가에 홀린 것 같았다'"
- 유인: "사기꾼들의 심리 조종 방법을 알아보겠습니다"

---

## 패턴 3: 궁금증 유발 문구 템플릿

| 패턴 | 예시 |
|------|------|
| "도대체~" | "도대체 그는 어떻게 이 엄청난 일을 전부 해낼까요?" |
| "어떻게 하면~" | "어떻게 하면 불안을 낮추고 편해질 수 있을까요?" |
| "(통념)이 아니라 (새로운 정보)" | "이건 폰에 중독된 게 아니라 도파민에 중독된 거에요" |
| "모든 건 ...에서 시작됐다" | "모든 건 멕시코 출장에서 시작됐다. 충격적이었다" |
| "A가 아니라 B라고 해요" | "탐색도 중요하지만 정체성 자본을 쌓는 게 더 중요하다고 해요" |

## 패턴 4: 유인 제시 3가지 방법

### 4-1. 욕구 분석 후 나열
"대화할 때 볼륨이 너무 작은 분, 목소리가 빨리 쉬는 분, 목소리가 마음에 안드시는 분. 모두 다~ 하나로 고칠 수 있어요."
-> 권위 제시: "15년 전 2옥타브에서 현재 3옥타브 미까지"
-> "고민은 제가 해드릴게요. 여러분들은 따라만 오시면 됩니다."

### 4-2. 내용 나열
"이럴 때 어떻게 해야하는지 3가지 오늘 다뤄드릴거에요. 정말 마음이 변한 건지, 마음을 어떻게 돌려야하는지, 재회는 어떻게 하는지."

### 4-3. 쉬운 것 -> 어려운 것 연결
"오늘 저는 여러분에게 이 폰을 보는 의욕과 열정을 가지고 책을 보게끔 만들어 드릴거에요."

## 패턴 5: 문제제기 기법

| 기법 | 예시 |
|------|------|
| 통념 뒤집기 | "돈이 많은 사람들은 많지만, 하고 싶은 일 하면서 돈 많이 버는 사람은 거의 없다" |
| 문제 지속 경고 | "이 악순환을 끊어내지 않는다면 내년도 지금과 별 다를 게 없을 겁니다" |
| 위기 -> 기회 전환 | "저축 제로 세대 등장은 굉장히 큰 신호. 인구감소는 절대 악재가 아닙니다" |
| 세뇌 프레임 | "여러분이 전부 ...에 세뇌 당했기 때문입니다. 좀만 들어보시면 알거에요" |
| 보는 사람 공감+문제제기 | "이런 생각하는 사람이 있는데 애초에 사회가 잘못 가르친 거에요" |

## 패턴 6: 공감+주장 문구

- "별 거 아닌 것 같다고요? 아닙니다. 굉장히 중요합니다."
- "(내용)에 대해서 모른다면 당신은 반드시 후회합니다."
- "퇴사? 멋있는 게 아닙니다. 퇴사하고 더 잘되어야 멋있는 겁니다."
- "독보적인 사람이 있습니다." (권위 제시)
