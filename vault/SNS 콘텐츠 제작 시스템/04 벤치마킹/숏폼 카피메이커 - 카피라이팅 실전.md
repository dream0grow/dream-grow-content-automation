# 숏폼 카피메이커 - [[C_카피라이팅|카피라이팅]] 실전 분석

## 카피 핵심 = 파소나 법칙 → '오공문해' 변형
- 오잉? (호기심)
- 공감
- 문제제기
- 해결책 안내

## 실전 사례: 7번가 피자
- 문제점: "근본이 다른" → 소비자에게 어필 안 됨
- 분석: 7가지 맛의 비밀, 특히 곡물 도우 차별화
- 타겟팅: 가성비 (가격 대비 맛의 우수성)
- 개선 카피: "도우가 다른 7번가 피자", "곡물 도우 7번가 피자"
- 숏폼 시나리오: 기존 카피 문제 지적 → 핵심 차별화 → 개선 카피 제안

## 실전 사례: 여성 전용 헬스장
- 기존 문제: 싼 가격, 할인율만 어필 → 지루, 신뢰 없음
- 차별점: 여성, 동네
- 개선 카피 예시:
  - "병원 가면 백날 듣는 말, 운동하세요. 지겹지 않으세요?"
  - "요요 없이, 과식 없이 눈치 없이 운동할 수 있는 곳"

## 참신한 카피 분석
- 앵커링 효과 + 시각화: "14억 아렌시아, 피부에 형광등키는 옥풀세럼"
- 질문형 호기심: "뜯고 짜고 다해 보셨죠? 모공청소는 이렇게 하는 겁니다"
- 중의적 표현: "교육비를 막아주는 우산"
