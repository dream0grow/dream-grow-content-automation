# 도입부 핵심 문구 구조 모음

## 좋은 도입부 구조: [공감] - [정의 내리기] - [자세한 설명]

## 궁금증 제시 문구 패턴
1. **실제 사례 + 질문**: 실제 영상/사건 → "어떻게 된 걸까요?" (호기심 + 신뢰)
2. **"도대체~"**: "도대체 그는 어떻게 이 엄청난 일을 전부 해낼까요?"
3. **"어떻게 하면~"**: 중요한 일 앞두고 어떻게 불안을 낮출 수 있을까요?
4. **통념 깨기**: "이건 A한 게 아니라 B한 거에요" (예: 폰 중독이 아니라 도파민 중독)
5. **"모든 건...에서 시작됐다"**: 기원 이야기로 시작
6. **평범한 사람도 가능**: "그러면 처음 시작하는 사람은 이길 수 없는냐 그렇지 않다"
7. **"A가 아니라 B"**: 탐색도 중요하지만 정체성 자본을 쌓는 게 더 중요하다

## 유인 제시 문구 패턴
1. **욕구 분석 후 나열**: 대상의 문제 나열 → "모두 다~ 하나로 고칠 수 있어요"
2. **내용 나열**: "이럴 때 어떻게 해야하는지 3가지 오늘 다뤄드릴게요"
3. **쉬운 것으로 어려운 것**: "폰을 보는 의욕과 열정을 가지고 책을 보게끔 만들어 드릴게요"
4. **유인 → 문제 → 해결책**: 노화속도 4배 늦추는 습관 예시

## 공감 제시 문구 패턴
- **질문 시작**: 질문 → 공감사례 → 또 다른 공감사례로 주장 → 근거 → 유인제시

## 문제 제기 문구 패턴
1. **"(통념) 하지만 (상식밖)"**: "돈 많은 사람은 많지만, 하고 싶은 일 하면서 돈 버는 사람은 어렵다"
2. **"여러분 선택입니다. 하지만..."**: 악순환 끊지 않으면 내년도 지금과 같다
3. **위기를 기회로**: "저축 제로 세대 등장 = 큰 신호, 인구감소는 절대 악재가 아닙니다"
4. **"세뇌 당했기 때문입니다"**: 통념 깨기 강력 버전

## 공감 + 주장 문구
- "별 거 아닌 것 같다고요? 아닙니다. 굉장히 중요합니다."
- "퇴사? 멋있는 게 아닙니다. 퇴사하고 더 잘되어야 멋있는 겁니다."
