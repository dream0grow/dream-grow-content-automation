# 숏폼 카피메이커 분석

## 카피 핵심 = 파소나 법칙 → '오공 문해력'
- 오잉? (호기심)
- 공감
- 문제제기
- 해결책 안내

## 사례 분석: 7번가 피자
- 문제점: "근본이 다른" → 소비자에게 어필 안 됨
- 해결: 도우가 다른 7번가 피자 / 곡물 도우 / 쫄깃한 도우 → 구체적 카피로 변경
- 핵심: 카피는 소통이다

## 사례 분석: 여성 전용 헬스장
- 문제점: 싼 가격, 할인율만 어필 → 지루하고 신뢰 없음
- 해결: "병원 가면 백날 듣는 말, 운동하세요. 지겹지 않으세요?"
- 차별점: 여성, 동네 키워드 활용

## 참신한 [[C_카피라이팅|카피라이팅]] 기법
- 앵커링 효과 + 시각화 표현: "14억 아렌시아, 피부에 형광등키는 옥풀세럼"
- 질문형 호기심 유발: "뜯고 짜고 다해 보셨죠? 모공청소는 이렇게 하는 겁니다"
