# 영상 구조 - 하와이 대저택 퍼스트 히트

## 구조 한 줄 요약
유인 → 문제 → 주장 → 근거(사례 반복) → 반론 → 원리 → 결론 → 행동유발

## 도입 구조
- 첫 문장: 쉽게 크게 변할 수 있다 (권위적 연구진)
- 문제 제시 방식: "믿지 않는다" (BGM 도입으로 분위기 전환)
- 긴장감 만드는 방식: 권위적 연구진 언급 후 반전
- 약속 또는 보상 제시: 생각만으로 현실을 바꿀 수 있다

## 본문 구조
1. **주장**: 생각만으로 현실을 바꿀 수 있다
2. **근거1**: 간다 마사노리 이야기 (사례)
3. **근거2**: 성공한 사람들이 조용히 실천하는 이유
4. **반론 처리**: 구체적인 이야기로 해달라
5. **근거3**: 조 디스펜자 박사 회복 이야기 (사례)
6. **원리**: 기적 = 상식적으로 설명할 수 없는 일
7. **근거4**: 체내 시스템, 호르몬 (과학적)
8. **근거5**: 생각 → 도파민/펩티드 분비 → 현실화
9. **주장 반복**: 반복이 중요, 의식적 생각이 무의식이 됨

## 증거 방식
- 사례: 간다 마사노리, 조 디스펜자 박사
- 권위: 연구진 인용
- 과학적: 호르몬, 도파민, 펩티드

## 마무리 구조
- 요약: 생각만으로 현실을 만들 수 있다
- 행동 촉구: 구독과 알림 설정

## 재사용 가능한 뼈대
- 도입: 유인 (권위 + 쉬운 변화 약속)
- 본문 1: 주장 + 사례 근거
- 본문 2: 반론 처리 + 추가 근거
- 본문 3: 원리 설명 (과학적)
- 마무리: 주장 반복 + CTA

### 영상 뼈대 공식
유인 - 사례 - 문제 / 주장 - 근거 - 사례 - 원리 - 반론 - 대안
