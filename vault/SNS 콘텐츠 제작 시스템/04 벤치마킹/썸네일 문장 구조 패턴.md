# 썸네일 문장 구조 패턴

## 구조 1: (문제 상황 A)에 (쉬운 행동 B)하면 전부 해결됩니다
- 공감 가능한 문제 + 쉬워 보이는 해결책 = 클릭 유도

## 구조 2: (문제 현상 A) 겪는 사람들이 자주 (하는 행동 B)
- 자기 해당 여부 확인 심리 자극

## 구조 3: 모든 (결과 A) 문제는 (간단한 원인 B) 때문이다
- 복잡한 문제를 단순 원인으로 귀결 = 호기심

## 구조 4: (하려는 행동 A)전부 다 틀렸습니다
- 기존 노력 부정 → 강한 호기심 유발

## 구조 5: (하려는 행동 A)할 때 반드시 알아야 할 3가지
- 숫자 + 필수 정보 = 클릭 유도

## 구조 6: "(일반적인 문제 상황 A)" (문제 겪는 B)가 되는 과정
- 문제가 악화되는 과정 궁금증

## 구조 7: 모든 (잘하고 싶은 A)의 문제는 (평범한 B) 때문이다
- 의외의 원인 제시
