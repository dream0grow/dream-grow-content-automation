# 도입부 핵심 문구 모음

## 좋은 도입부 구조
[공감] → [정의 내리기] → [자세한 설명]

## 궁금증 제시 유형
1. **"도대체~"** - "도대체 그는 어떻게 이 엄청난 일을 전부 해낼까요?"
2. **"어떻게 하면~"** - "어떻게 하면 불안을 낮추고 편해질 수 있을까요?"
3. **"A가 아니라 B"** - "이건 폰에 중독된 게 아니라 도파민에 중독된 거에요."
4. **"모든 건 ...에서 시작됐다"** - "모든 건 멕시코 출장에서 시작됐다."
5. **실제 사례 제시** - 실제 영상/사례로 호기심 + 신뢰감 동시 확보

## 유인 제시 유형
1. **욕구 분석 후 나열하기** - 대상의 고민을 나열 → "모두 다 하나로 고칠 수 있어요"
2. **내용 나열하기** - "이럴 때 어떻게 해야하는지 3가지 오늘 다뤄드릴거에요"
3. **쉬운 것으로 어려운 것 연결** - "폰을 보는 의욕을 가지고 책을 보게끔 만들어 드릴거에요"

## 문제 제기 유형
- **통념 반박**: "하지만 (상식밖 이야기)를 하는 경우는 거의 없어요"
- **지속 경고**: "앞으로도 지금과 별로 다를 게 없을 겁니다"
- **위기→기회**: 위기 현상 짚기 → 기회로 보게 하기
- **세뇌 지적**: "여러분들이 전부 (......)에 세뇌 당했기 때문입니다"

## 도입부 마스터 구조
**유인제시 → 문제 제시 → 유인(해결책) 제시**
