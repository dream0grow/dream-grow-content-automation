# 논리적사고 콘텐츠 인덱스

나이절 워버턴 『논리적 생각의 핵심 개념들』(원제: *Thinking from A to Z*)의 4분류 체계와 약 200개 항목을 **부모-자녀 일상 사례**로 변환한 콘텐츠 소재 라이브러리.

## 4분류 (Warburton 분류)

| 분류 | 성격 | 부모 교육 적용 |
|------|------|----------------|
| 1. 논쟁 수단 | 토론에서 흔히 사용 | 부부 갈등, 자녀와의 다툼에서 자기 점검 |
| 2. 오류 | 무자각하게 빠지는 함정 | 일반화·인과 혼동·매몰비용 등 자기 검증 |
| 3. 설득·회피 기술 | 의도적 우회 | 자녀가 사용할 때 명명·되묻기 |
| 4. 심리적 방해 | 마음의 작용 | 부모 자신의 화·불안의 원인 진단 |

## 콘텐츠 변환 원칙

- **마케팅 후킹 금지**: 곤팀장 카피 529선과 달리 부드러운 언어 사용
- **A가 아니라 B 구조**: "이건 아이가 거짓말한 게 아니라 ___입니다" 구조 활용
- **교실 경험 1개 이상**: 추상적 논리 설명이 아니라 구체적 장면에서 출발
- **부모 자기 점검 우선**: 자녀를 비판하기 전에 부모 자신의 함정부터 다룸
- **이름 붙이기 효과**: 워버턴의 핵심 메시지 "이름이 있을 때 비로소 식별된다"

## 활용 채널

- **스레드**: 1개 노트 → 1개 스레드 (8~10문단)
- **릴스**: 1개 노트 → 1개 릴스 (도입 후킹: "이게 무슨 함정인지 아시나요")
- **뉴스레터**: 4분류를 묶어 4부작 시리즈로 활용
- **책 챕터**: 부모 글쓰기·대화법 챕터의 절(節) 단위 자료
