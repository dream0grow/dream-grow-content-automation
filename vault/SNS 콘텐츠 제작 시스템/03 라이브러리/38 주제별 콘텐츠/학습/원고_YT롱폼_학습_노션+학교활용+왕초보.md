# 단 1시간에 왕초보 떼자! 왕초보 노션 사용법 + 학교에서 노션 사용하는 방법

## 1. 노션이 인기인 이유
### 노트앱의 필요성
- 지금 불편한 온라인 과정 해결

### 노션이 좋은 이유
- 금방 배운다
- 예쁘고 자율성이 높다
- 협업에 유리하다 (개인적 사용 + 함께 사용)

### 노션 활용 사례
- 독서 메모
- 사적 모임 활동
- 사이트로 만들기
- 학교에서 활용

## 2. 노션! 바로 따라하기

### 가입 및 기본 설정
- 노션 가입 → 이름 만들기 → 언어 변경 (한글)

### 구조와 특징
#### 1) 페이지 구조
- 폴더가 아닌 페이지가 경로로 입력됨
- 페이지를 옮길 수 있음
- [실습] 페이지 여러개 만들기와 옮기기
- 다양한 뷰 만들기 (테이블, 보드, 캘린더 등)

#### 2) 마크다운 구조
- 핵심 키: / (마술지팡이)
- 가장 많이 사용하는 8가지:
  - Basic blocks: 글자 크기(헤드), 글자 색, 불렛/넘버, 인용구, 구분선, 콜아웃
  - Media: 파일, 이미지, 영상 업로드 (꿀팁: Win+Shift+S 캡처)
  - Database: inline vs Fullpage, 표 데이터베이스
  - Embed: 웹사이트 임베드, PDF 임베드, /synced block

### 노션으로 협업하기
- 워크스페이스 vs 개별 페이지 협업
- 사용 사례: 꿈들, 학교

## 3. 유의점 및 장점
- 데이터가 웹 기반
- 용량 확인 방법 + 요금제
- 꿀팁: 노션 템플릿 사용하기

### 학교에서 사용하는 이유
- 협업: 회의 내용 공유를 쉽게
- 사람이 바뀌어도 맥락 파악 가능한 회의 기록 시스템

## 고급편
- 개별 학생 평가 시스템 구축 (온라인/과정 중심)
- 학교 회의 시스템 아카이브
- 노션 백링크 사용법
