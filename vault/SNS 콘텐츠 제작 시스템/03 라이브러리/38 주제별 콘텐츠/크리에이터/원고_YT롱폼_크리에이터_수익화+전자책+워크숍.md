# 유튜브 수익화 파이프라인 - 전자책에서 워크숍까지

## 원본 스크립트 (November 2nd, 2022 기반)

### 오프닝: 잠자면서도 돈이 들어오는 시스템

직장인: (일한 시간 X 시간당 보수) = 시간적 한계 있음
부자: (상품 갯수 X 상품 한개당 이익) = 한계 없음

유튜브 수익 공식: 영상 1개 X 광고 조회수 X 1클릭당 이익
- 영상 10개면 수익 10배
- 영상 100개면 수익 100배
- 조회수도 무한대로 증가 가능
- 잠자면서도 돈이 들어옴

### 본론 1: 왜 인터넷이 돈을 벌기 쉽게 만드는가

- 인터넷 상점은 24시간 운영
- 이마트는 11시면 닫지만 네이버 스마트스토어는 새벽에도 운영
- "파이프라인의 우화" - 버킷 vs 파이프라인

### 본론 2: SNS가 신뢰를 만드는 창구

누군가가 내 정보를 보며:
1. '이렇게도 가능하다고?' 호기심
2. '나도 할 수 있을까?' 자기 삶에 대입
3. 전자책/강의 구매
4. 괜찮으면 또 구매

인스타 사례:
- 부자공장: 3개월 60개 글 -> 팔로워 1300명 -> 사업+수익
- 마인드피봇: 10개월 314개 게시글 -> 팔로워 1만7천명

핵심: 시간만 있으면 가능하다. 지속성이 관건.

### 본론 3: 유튜브 기반 수익 파이프라인

**상품 단계별 구조:**

| 단계 | 상품 | 가격 | 예상 수익 |
|------|------|------|-----------|
| 1단계 | 유튜브 구독자 늘리기 전자책 | 2만원 x 100명 | 연 최대 2000만원 |
| 2단계 | 유튜브 '습관' 만들기 전자책 | 2만원 x 100명 | 연 최대 2000만원 |
| 3단계 | 유튜브 온라인 강의 (나튜브) | 15만원 x 100명 | 연 최대 1.5억 |
| 4단계 | 한달 워크숍 | 20만원 x 20명/월 | 월 400만원 |
| 5단계 | 밀착 워크숍 5일 | 30만원 x 10명 x 2회/월 | 월 600만원 |

### 본론 4: 한달 워크숍 구체적 프로세스

- 카페에 1일~30일까지 매일 미션글 게시
- 참가자들이 미션을 실천하고 게시글 작성
- 참가자끼리 댓글로 피드백
- 운영자는 하루 3분(글 복사) + 1시간(댓글 피드백)으로 운영 가능

한달 워크숍 커리큘럼:
- 참가 이유 소개
- 1주차: 유튜브 채널 제작(채널 제목, 개설, 채널아트, 로고)
- 2~4주차: 영상 제작 방법 + 영상 1개 완성
  - 시나리오 작성 -> 촬영 -> BGM 준비 -> 편집 -> 썸네일 -> 업로드 -> 동료 피드백

### 결론

돈이 굴러들어오는 시스템은 요행이 아니다. 실행만 하면 가능하다.
