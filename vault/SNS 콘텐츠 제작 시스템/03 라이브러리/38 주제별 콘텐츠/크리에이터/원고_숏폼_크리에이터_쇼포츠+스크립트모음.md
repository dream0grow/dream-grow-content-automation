# 쇼포츠 스크립트 모음

스포츠 숏폼 채널 "쇼포츠"의 영상 스크립트 아이디어 모음.

## 핵심 전략
- 기부 이벤트와 연계한 구독 유도 (좋아요 수 x 10원 기부)
- 예상 못한 반전과 감동 요소 활용
- 댓글 유도 질문 삽입

## 완성 스크립트 목록
1. 샤라포바/조코비치 미션 - 테니스 컵 맞추기 반전
2. 음바페 VS 홀란드 - 발롱도르 예측 댓글 유도
3. 배구 리베로 - "영상 멈추고 댓글로 알려주세요"
4. 착한 사람만 보이는 줄넘기 - 구독 유도 기믹
5. 슈퍼맨 날다 - 장애 극복 감동
6. 메시 + 기부 - 좋아요 기부 이벤트
7. 어린이 농구 천재 - 재능 기부 이벤트

## 롱폼 아이디어
- 장대높이뛰기 대참사 모음집
- 감동 달리기 풀 영상 4분
- 여자 선수 실패 모음집
- 아기 영상 모음집
