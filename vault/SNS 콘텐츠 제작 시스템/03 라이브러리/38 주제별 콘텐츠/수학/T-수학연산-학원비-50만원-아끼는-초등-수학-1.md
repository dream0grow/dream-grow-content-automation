사실 부모가 최고의 수학 교사입니다.
 왜냐?
 1. '초등' 수학은 기본적인 것을 제대로 하는 것이 중요합니다.
2. '초등' 수학의 개념을 잘 이해하게 하는 것은
학원에 가지 않고도 가능합니다.
3. 응용, 심화 문제는 스스로 해결할 때 수학적 사고력이 향상 됩니다.
부모는 답지보며 조금씩 힌트만 주면 됩니다.
 그래서 엄마와 아이가 같이 수학공부하는 방법은 ...

아이가 수학공부가 처음이라면 2권 기본 개념과 응용 문제만 사세요.

문제지는  (기본) 'EBS 만점왕 수학'  또는 'EBS 수해력' 
(응용) '디딤돌 초등 수학 응용' 
(심화) '최상위 초등 수학 S' 

사면 됩니다. 

만약 현재 밀크 T, 마카다미아 같은 온라인 학습을 하고 있다면

 응용, 심화 문제만 사도 됩니다.

만약 부모님이 설명하기 어렵다면 
아이와 같이 강의를 들으면서 내용을 이해하면 됩니다. 

그리고 부모님이 이해한 것을 
아이에게 설명해 줍니다. 

여기까지는 감이 잡히실 겁니다. 

3단계가 핵심입니다.

5단계 : '오답 풀이 설명하기' + 노트로 정리하기 

며칠 후 틀린 문제를 다시 푸는 것은 매우 중요합니다. 

하지만 무작정 오답 노트를 쓰게 하는 것은 아이에게 또 다른 숙제가 될 뿐입니다.

일단 그 때 문제의 풀이 과정을 엄마에게 설명하라고 해보세요. 

그리고 오답노트 하기 전에 
왜 틀렸는지 아이 스스로 분석하게 하세요.

- 개념 부족: "아, 이 개념을 까먹었네."
- 문제 오독: " '모두' 구하는 건데 하나만 구했네."
- 조건 불충분 : "내가 지금 찾지 못한 조건이 이거구나." 
- 계산 실수: "뺄셈 해야하는데, 덧셈을 잘못했네." 

4가지를 중심으로 아이가 오답을 한 이유를 찾게 합니다.
