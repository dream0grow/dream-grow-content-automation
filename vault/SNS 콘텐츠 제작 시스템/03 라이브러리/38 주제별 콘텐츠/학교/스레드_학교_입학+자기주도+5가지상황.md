초등 입학하고 일단 지켜봐야 하는 상황 5가지

1. 스스로 책가방을 챙길 때
"빠진 거 없어? 엄마가 다시 볼게." (기회 박탈) 
-> "알림장보며 하나씩 챙겨봐!" (자기 주도)
2. 등하굣길 횡단보도를 건널 때
"차 조심해, 위험하니까 뛰지 마!" (불안 전달) -> "초록불! 좌우 살피고 건너!" (상황 인지)
3. 새로운 친구에게 다가갈 때
"친구들이랑 무조건 사이좋게 지내." (막연한 지시) -> "눈 마주치고! 안녕 하고 인사해!" (구체적 행동)
4. 숙제나 어려운 문제를 처음 마주했을 때
"모르겠어? 엄마가 그냥 알려줄게." (도전 차단) -> "천천히! 문제를 다시 읽어봐!" (문제 해결력)
5. 친구와 의견 충돌이 생겼을 때
"그냥 네가 양보하고 참아." (감정 억제) -> "너가 그렇게 말하니 나는 (감정)해" (자기 표현)

어려움을 대신 해결해주는 부모보다, 
낯선 환경을 스스로 관리하고 적응하는 법을 
가르치는 것이 필요합니다.
