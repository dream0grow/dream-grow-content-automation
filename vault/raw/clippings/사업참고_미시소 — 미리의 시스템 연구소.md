- 관리자 접근 권한 분리

- 회원 탈퇴를 통해 개인정보 삭제를 요청할 수 있습니다.
- 요청은 프로필 페이지 또는 이메일(miri@orot-ai.com)로 가능합니다.

## 9\. Chrome 확장 프로그램 (Notion Code Previewer)

미시소에서 제공하는 Chrome 확장 프로그램 "Notion Code Previewer"는 아래와 같이 개인정보를 처리합니다.

### 9-1. 수집 항목 및 저장 방식

| 수집 항목 | 저장 위치 | 목적 |
| --- | --- | --- |
| Notion OAuth 토큰 | chrome.storage.local (브라우저 로컬) | Notion API 인증 |
| GitHub OAuth 토큰 | chrome.storage.local (브라우저 로컬) | GitHub API 인증 |
| 노션 페이지 코드블록 내용 | 메모리 캐시 (브라우저 로컬) | 코드 프리뷰 렌더링 |

### 9-2. 외부 서버 전송

- 모든 데이터는 사용자의 브라우저 로컬에만 저장되며, 외부 서버로 전송되지 않습니다.
- GitHub 배포 기능: 사용자가 명시적으로 배포 버튼을 누를 때에만 GitHub API로 코드가 전송됩니다.

### 9-3. 외부 CDN 사용

코드 프리뷰 렌더링을 위해 다음 외부 CDN 리소스를 사용합니다: Tailwind CSS, React, Babel. 이 과정에서 개인정보는 전송되지 않습니다.

### 9-4. 데이터 삭제

확장 프로그램을 브라우저에서 제거하면 chrome.storage.local에 저장된 모든 데이터(OAuth 토큰 등)가 자동으로 삭제됩니다.

## 10\. Chrome 확장 프로그램 (Ring Ding Done Boost / 링딩던 부스트)

미시소에서 제공하는 Chrome 확장 프로그램 "Ring Ding Done Boost"(이하 "링딩던 부스트")는 노션 AI 보조 도구로, 아래와 같이 개인정보를 처리합니다.

### 10-1. 수집 항목 및 저장 방식

| 수집 항목 | 저장 위치 | 목적 |
| --- | --- | --- |
| Notion OAuth 액세스 토큰 | 미시소 서버 (Supabase, RLS 적용) | Notion API 인증 |
| Notion 워크스페이스 ID 및 이름 | 미시소 서버 (Supabase, RLS 적용) | 연결 상태 표시 |
| 노션 AI 대화 내용 (프롬프트·응답·사용된 도구·소요 시간) | 사용자 본인의 노션 워크스페이스 DB | 대화 자동 백업 및 맥락 이어쓰기 |
| 현재 노션 탭의 JS 힙 메모리 사용량 | 메모리 (전송 및 저장 없음) | 크래시 방지 배지 알림 |
| 사용자 설정 (사운드 종류, 볼륨, 자동 적재 활성화 여부) | chrome.storage.local (브라우저 로컬) | 확장 환경 설정 |

### 10-2. 데이터 전송 경로

- 노션 AI 대화 내용은 사용자 브라우저에서 Notion 공식 API(api.notion.com)로 직접 전송되어 사용자 본인의 노션 워크스페이스 DB에 저장됩니다. 미시소 서버를 경유하지 않으며, 미시소나 제3자가 대화 내용을 열람·저장하지 않습니다.
- 미시소 서버로는 OAuth 토큰 발급 및 연결 상태 조회 API만 호출됩니다.
- 노션 이외의 도메인으로는 어떠한 사용자 데이터도 전송되지 않습니다.

### 10-3. 원격 코드 및 외부 CDN

링딩던 부스트는 확장 번들 외부에서 JavaScript 코드를 동적으로 로드하거나 실행하지 않습니다. 모든 실행 코드는 확장 패키지 내부에 포함되어 있으며, 외부 CDN 리소스를 사용하지 않습니다.

### 10-4. 데이터 삭제 방법

- 확장 프로그램을 브라우저에서 제거하면 chrome.storage.local에 저장된 사용자 설정이 자동으로 삭제됩니다.
- 미시소 선물 페이지의 "재연결" 또는 노션 "설정 → 연결된 통합"에서 언제든지 연결을 해제할 수 있으며, 해제 즉시 미시소 서버에 저장된 OAuth 토큰 및 워크스페이스 정보가 삭제됩니다.
- 사용자 본인의 노션 워크스페이스 DB에 쌓인 대화 기록은 사용자가 직접 노션에서 삭제할 수 있습니다.

### 10-5. 요청 권한 및 사용 근거

- **storage**: 사용자 로컬 설정과 세션 상태 저장
- **scripting**: notion.so 페이지에 content script를 주입하여 노션 AI 응답 감지 및 맥락 요약 주입
- **tabs**: 노션 AI 세션 ID(\`?t=\` 파라미터) 추출 및 "채팅 새로 시작" 기능의 탭 식별
- **host\_permissions** (\*://\*.notion.so/\*, https://api.notion.com/\*, https://misiso.com/\*): 위 핵심 기능 동작 및 API 호출
