    - 스크랩 내용: 독서노트 만드는 노하우1 (`엘프화가` )
        - 저는 독서노트 이렇게 작성하는 편입니다.
        - `책이름 by 저자명` 으로 독서노트 만들기 - 이러면 저자 이름이 더 잘 기억에 남음. 남이랑 이야기할 때 좋음. (XX가 XX라는 책에서 이야기하길....)
        - 읽다가 기억해야겠다 싶은 내용(스몰토크용/지식/글쓰기용)은 수시로 메모
        - 비슷한 내용이 나오면 주제별로 묶기
        - 기억해야 할 지식 덩어리가 나오면 새 메모 만들기
        - 다른 책에서 비슷한 내용이 있을 경우.이미 관련 메모가 있으면 거기에 추가하고 책 이름 출처로 적기 / 아니면 새 메모 만들어서 양쪽 내용 같이 정리. 좀 더 지식이 명확해짐
        - 전문 용어가 나오면 인터넷으로 관련 자료 2~3개를 찾아 새 메모로 통합 정리. 이렇게 개념을 명확히 이해해 두면 책 읽어 나갈 때 훨씬 편함.
        - 책을 읽어나가면서 전반적인 내 느낌/책의 주제는 읽으면서 짬짬이 정리해 둠. 책 후기 쓰기 좋음.
        - 만들어진 관련 메모들은 독서노트 아래에 배치. 언제든 같이 읽을 수 있고, 더 기억에 잘 남게 됨
        - 이렇게 적으면 더 많이 기억에 남고, 스스로에게 도움이 되었던 것 같아요.
    - 스크랩 내용 : 독서노트 만드는 노하우2(`요즘애` )
        - 1. 이미 핵심내용등을 아는 분야 경우
            - 나만의 정리 목차를 만든다
            - 메모를 하면서 나만의 목차에 넣으면서 바로바로 정리한다.
        - 2. 새로운 분야 ex) 책인경우
            - 책 읽기 전에 목차를 보면서 궁금증을 기록해 놓는다.
            - 책을 읽는다.
            - 메모를 하면서 책의 목차말고, 내 흐름대로 책 안의 내용을 가지고 모으면서 그루핑을 한다. 나만의 흐름 대로 배치한다.(진짜 잘쓴책 아니고서는 어느정도 중구난방이기때문에)
            - 궁금증이 다 해결되었나 생각해본다. 핵심내용이 파악되었으면
            - 이것을 뛰어 넘는 범위 또는 구체적인 나만의 목적으로 하는 주제대로 목차를 만든다.
            - 정리된 자료를 보고 보강하면서 내가 만든 목차안으로 메모를 발전시키면서 잡아 넣는다. 
        - 3. 이제 이건 완벽한 내 지식이 된다

    - 독서노트 만들며 실천할 사항 
        - 1. 읽은 책에서 메모할 부분을 미리 목차형식으로 만들어도 괜찮겠다.
        - 2. 책을 읽으며 든 생각과 느낌도 독서노트에 넣아보자. 
        - 3. 제일 중요한 것은 독서노트의 내용을 영구메모로 만들어서 정리한다.
        - 4. 정리내용을 모아 하나의 글로 발행해보자.

        - 머니 블로그 글에 [[제텔카스텐]]에 대한 내용을 올리고, 생산성 확장과 창의성에 대한 이야기를 진행하기. 
        - 머니 블로그 작성 글 예시 생각하기
        - 머니 블로그를 잘 운영 방안에 대해 습관으로 어떻게 만들지 고민하기. 
        - 블로그와 영구메모, 인스타, 유튜브를 하나의 습관 고리로 창조하는 워크플로우 고민하기
        - 예) 영구메모 -> 인스타 발행 -> 블로그 글로 작성 -> 유튜브 내용으로 요약 정리
