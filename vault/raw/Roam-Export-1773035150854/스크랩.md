        - 제텔카스텐 - 숀케 아렌스 책을 통해 유명해짐. 가장 잘 알려주는 리소스
            - [Johannes Schmidt, “Niklas Luhmann‘s Card Index: Thinking Tool, Communication Partner, Publication Machine.”](https://pdfs.semanticscholar.org/88f8/fa9dfbc0c2b296758dd932b871917c5c775a.pdf?_ga=2.266886862.173722003.1654587701-103758878.1653900571)
        - 제텔카스텐은 2가지 의미이다. 
            - 단순 형태로서 메모 상자라는 의미 (물리적)
            - 메모를 활용해 창작물을 만들어내는 노트 방식 (루만 교수의 방법론이 내포된 의미)
                - 제텔카스텐은 세컨드 브렌인의 특정 용도에 맞춰져 있는 방법
        - 물리적으로 제텔카스텐은  여러 부속(Compartments)로 구성되어 있다
            - 영구 메모는 여러가지일 수 있다. 
                - ![](https://firebasestorage.googleapis.com/v0/b/firescript-577a2.appspot.com/o/imgs%2Fapp%2Fnormal_is_special%2FKAVzPNxhKm.14.01.png?alt=media&token=d5bbf3ea-89df-4e02-9285-e910717ee748)
                - Zettels 은 Main notes, Permanent notes, 인용문 + 나의 해석을 쓰는 것
                - Literature notes 은 Bibliographic Notes, Citation notes, Reference notes
                - Indexes
                - Structure Notes (Maps of Content, 디지털에서는 아날로그의 Folgezettel 즉 알파뉴메릭 주소를 대신할 수 있다)
            - 영구 메모 이외에는 프로젝트 노트, 임시(플리팅) 노트가 있다.
                - Project Notes (숑케 아렌스가 얘기하는 것. 또다른 종류의 Fleeting notes 나중에 얘기해 줄께)
                - Needs Finishing (제텔이 될 수도 있는 플리팅 노트)
                - Sleeping (다시 정리하지 않을 것 같은 플리팅 노트)
            - 제텔카스텐은 방법론을 지칭하기도 함
                - 노트하고 -> 정리하고 -> 연결하고 -> 글을 써라!
                - ![](https://firebasestorage.googleapis.com/v0/b/firescript-577a2.appspot.com/o/imgs%2Fapp%2Fnormal_is_special%2FJ3m4Alz9Ti.24.48.png?alt=media&token=37a1de1b-8e5f-41cb-9821-2631c1c36063)
                - 제텔카스텐은 두 가지 스텝으로 구성됨
                    - (1) 노트 만들기
                    - (2) 연결 하기
                - 효과
                    - 아이디어 간의 연결을 만들기
                    - 창의적인 결과물을 만들어 내는 지식을 향상 시킴
