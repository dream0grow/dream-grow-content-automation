    - 아침에 싸우지 않고 등교하기! 
        - 우리는 왜 아침에 자주 아이들과 싸우는가?
            - 할 일이 많다. 시간은 적다.
                - 나도 아침 준비  : 씻고, 아침밥 준비하고 설거지 
                - 아이도 챙겨야함 : 옷갈아입기, 아침밥, 양치, 세수, 머리묶기 
                - 마음이 분주 -> 아이에게 화를 내거나 짜증 이 많아짐. -> 아이도 울고 -> 달래주거나 더 화내거나 -> 시간이 부족 + 하루종일 마음이 답답하고 슬프기 후회 -> 사과, 
            - 목표하는 일을 아이가 제 시간에 하지 않는다.
                - 옷 입다가 갑자기 장난감으로 놀이
                - 밥을 느리게 먹기, 밥먹다가 눕는다.  
                - 화장실에서 놀이 
        -  어떻게 해결할까?
