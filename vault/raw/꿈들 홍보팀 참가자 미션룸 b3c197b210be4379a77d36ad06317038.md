# 꿈들 홍보팀 참가자 미션룸

## 🧭 페이지 목적

<aside>

</aside>

## 👋 시작 안내

꿈들 홍보팀에 오신 것을 환영합니다.

이 페이지에는 모든 강의 내용을 길게 넣지 않습니다.  

각 강의는 별도 페이지에서 보고, 미션을 작성하고, 서로의 답변을 확인합니다.

## 🧭 사용 방법

1. 아래 강의 목록에서 오늘 볼 강의를 선택합니다.
2. 해당 강의 페이지로 들어갑니다.
3. 영상을 끝까지 봅니다.
4. 강의 페이지 안에서 미션 질문에 답합니다.
5. 다른 참가자 1명의 답변을 읽습니다.
6. 공감 또는 피드백 댓글을 남깁니다.
7. 오픈톡방에 완료 인증을 남깁니다.

## ✅ 공통 참여 규칙

### 답변 규칙

- 완벽하게 쓰지 않아도 됩니다.
- 짧게 써도 됩니다.
- 정답은 없습니다.
- 나에게서 나온 문장이면 충분합니다.
- 다른 사람의 답변을 평가하지 않습니다.

### 댓글/피드백 규칙

- 먼저 공감합니다.
- 조언보다 발견한 점을 말합니다.
- “좋아요”만 쓰기보다 어떤 부분이 좋았는지 말합니다.
- 고치라는 말보다 더 궁금한 부분을 말합니다.
- 부담스럽게 길게 쓰지 않아도 됩니다.

### 댓글 예시

- 이 문장이 가장 공감됐어요.
- 이 이야기는 더 들어보고 싶어요.
- 이 소재는 인스타그램 글로 만들어도 좋을 것 같아요.
- 이 부분에서 대표님의 경험이 잘 느껴졌어요.
- 저도 비슷한 고민을 해본 적이 있어요.
