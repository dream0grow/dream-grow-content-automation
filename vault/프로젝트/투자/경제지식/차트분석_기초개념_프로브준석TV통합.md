# 핵심
프로브와 준석TV가 둘 다 강조하는 차트의 핵심 골격은 단 4가지: 지지·저항, 정배열·역배열, 신고가, 거래대금. 화려한 작도법은 모두 이 4가지의 변형이다.

# 두 저자가 공통으로 강조하는 것

## 1. 정배열 vs 역배열 (장기이평선)

**역배열**: 장기이평선이 차트 위에 위치 (5 < 20 < 60 < 224 위에 모두 차트가 있음)
- 가장 주가가 저렴한 자리
- 세력이 매집하는 시기

**정배열**: 장기이평선이 차트 아래 위치 (5 > 20 > 60 > 224 모두 차트보다 아래)
- 상승 추세 진행 중
- 5일선 → 20일선 → 60일선 → 112일선 → 224일선 단계별 돌파

**준석TV 5단계 모델**:
- 레벨 1: 5일선 터치 시 일시 하락
- 레벨 2: 20일선 터치 시 일시 하락 (5일선 지지)
- 레벨 3: 60일선 터치 시 일시 하락 (20일선 지지)
- 레벨 4: 112일선 터치 시 일시 하락 (60일선 지지)
- 레벨 5: 112일선 지지 시 신고가 도전

## 2. 거래대금 = 거래량 × 주가

**프로브 핵심**: 거래량은 속아도 거래대금은 안 속는다.
- 평균 100억 종목에 1,500억이 들어와야 진짜 큰돈
- 시장 환경별 기준: 평시 500억 / 유동성 장 1,000억 / 하락장 300억

**준석TV 핵심**: 매수 거래량 > 매도 거래량 = 고점/저점 상승. 반대면 하락.

## 3. 지지·저항 (자리)

**지지**: 주가가 떨어지다 멈추는 가격대
- 직전 저점, 장기이평선, 거래대금 폭발 봉의 종가

**저항**: 주가가 오르다 막히는 가격대
- 직전 고점, 신고가 부근, 매집봉 고점

**프로브**: 지지·저항이 명확한 자리에서만 진입 (1:3 손익비 가능)
**준석TV**: 공구리(바닥 지지), C자리(224일선 지지) 활용

## 4. 신고가의 의미

**프로브**: 신고가 돌파는 위에 매물대가 없다는 뜻 (저항 사라짐)
- 좋은 기준봉의 핵심 조건

**준석TV**: 52주 신고가 = 4번 자리 (정배열 이격 확대, 고수 영역)
- 텐버거 가능성 있으나 실패 시 -50% 가능

# 두 저자가 공통으로 경고하는 것

## 화려한 작도법은 함정

- 일목균형표, 엘리엇 파동, 추세선 작도 → 알고리즘이 이미 다 학습한 패턴
- 혼자 발견한 마법 같은 진입 신호는 거의 다 우연
- "단순한 4가지(지지/저항·정배열·신고가·거래대금)"가 실전에서 작동

## 단일 시그널은 작동 안 함

- 거래대금만 폭발: 노이즈 가능성
- 차트만 좋음: 재료/내러티브 없으면 못 오름
- 재료만 좋음: 차트 자리가 나쁘면 늦은 진입

→ **복합 시그널이 일치할 때만 진입**

# 프로브와 준석TV의 차이

| 구분 | 프로브 | 준석TV |
|------|--------|--------|
| 주력 시장 | 한국 + 미국 | 미국 단타 |
| 시점 | 단타 1~7일 | 단타 1일 이내 |
| 진입 자리 | 기준봉 + 내러티브 부합 | C자리 (224일선 부근 눌림) |
| 손절 룰 | 손익비 1:3, 지지선 이탈 | 매수가 = 공구리 라인 |
| 익절 룰 | 1:3 비율 또는 내러티브 변화 | 급등 폭의 50% 회복 |
| 차트 단위 | 일봉 + 5분봉 | 일봉 + 60분봉 + 1/5분봉 + 15틱 |
| 핵심 무기 | 매매일지 + 검색식 | 차트 패턴 4종 (밥그릇/V자/매집봉/공구리) |

# 활용 매뉴얼 (사용자용)

## 1단계 학습

1. **장기이평선 5개 화면 띄우기** (5/20/60/112/224)
2. **거래대금 차트 추가** (HTS 지표 설정)
3. **본인 보유/관심 종목 5개의 정배열-역배열 상태 확인**

## 2단계 시그널 매칭

1. 정배열 회복 시작 종목 발굴
2. 거래대금이 평소 대비 5~10배인지 확인
3. 시장 내러티브 부합 여부 점검
4. 손익비 1:3 가능한 자리인지 검토
5. 손절 라인을 매수가로 잡을 수 있는지 확인

## 3단계 매매 실행

분할 매수 → 분할 익절 → 매매일지 기록
