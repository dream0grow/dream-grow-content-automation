# 핵심
빚투(신용·미수)는 이자 비용이 아니라 멘탈의 주도권을 시장에 빼앗기는 행위다. 반대매매 공포가 이성을 마비시킨다.

# 왜 중요한가

신용·미수로 매매하면 손절 룰이 무너진다. -5%면 잘라야 할 자리에서 "조금만 더 기다리면 회복할 텐데"라고 버티게 된다. 이유는 단순하다. 손절하면 빚을 그대로 안고 가야 하기 때문이다.

빚투의 진짜 리스크는 "이자율 4~6%"가 아니다. **시장이 흔들릴 때 합리적 판단이 불가능해지는 정신 상태**다.

# 빚투의 4단계 함정

1. **자신감 단계**: "단타는 빨리 끝나니까 이자 부담 없다"
2. **물림 단계**: 한 번 -10% 물리면 빚 갚으려고 무리한 매매 반복
3. **반대매매 공포 단계**: -20% 가까이 가면 손과 발이 떨림. 잘라야 할 자리에서 못 자름
4. **깡통 단계**: 반대매매 후 빚 + 자금 손실 동시 발생

# 본업 = 가장 완벽한 헷지 수단

- 본업 월급은 손실을 메꾸는 게 아니라 **시장에서 빠져나올 수 있는 선택권**을 준다
- 단타로 돈 잃어도 다음 달 월급으로 다시 시드 만들 수 있음
- 빚투는 이 선택권을 미리 시장에 넘기는 것

# 구체적 행동 원칙

1. **신용·미수 절대 사용 금지** (특히 단타)
2. **본업 그만두고 전업 트레이더 전환은 누적 1억 이상 + 3년 이상 일관된 수익 후 검토**
3. **시드는 잃어도 생활에 영향 없는 금액만**
4. **레버리지 ETF(2배·3배)도 빚투와 같은 메커니즘** — 변동성 + 시간 가치 손실로 깡통

# 함정

- "한 번만 신용 쓰자"는 가장 위험한 생각
- 신용 한도가 높아질수록 그 한도를 다 쓰게 됨 (앵커링 효과)
- 본업 월급 받자마자 통째로 시장에 넣는 것도 빚투와 다를 게 없음 (생활 자금 필요 시 손절 강요됨)
