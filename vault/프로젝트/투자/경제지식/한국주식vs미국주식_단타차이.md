# 핵심
같은 단타라도 시장이 다르면 룰이 다르다. 한국은 정보 비대칭과 시간 제약이 핵심, 미국은 시간대 분할과 변동성이 핵심이다.

# 시장 구조 차이

| 구분 | 한국 | 미국 |
|------|------|------|
| 거래 시간 (한국 시간) | 09:00~15:30 | 22:30~05:00 (본장) |
| 부가 시장 | 시간외 단일가 | 프리장 5.5h + 에프터마켓 4h |
| 변동성 제한 | 상하한 ±30% (가격제한폭) | 무제한 (서킷브레이커만) |
| 거래대금 의미 | 평균 200~500억 / 단타 후보 | 평균 1,000억+ / 진입 자유 |
| 정보 출처 | 증권사 리포트, 뉴스, 블로그 | 트위터(X), 레딧, CNBC |
| 알고리즘 비중 | 약 30% | 약 60~70% |

# 단타 전략 차이

## 한국 단타 (프로브 스타일)

**핵심**: 차트 + 내러티브 + 거래대금 검색식

**일과**:
1. 09:00 시초가 베팅 (전날 기준봉 출현 종목)
2. 09:30~11:00 주도주 추격
3. 13:00~15:00 종가 베팅 (다음날 갭 노림)
4. 15:30 이후 검색식 돌리며 다음날 후보 추출

**핵심 종목군**:
- 시가총액 300억~5,000억 (단타 가능 호가창)
- 글로벌 피어 그룹 추종 (HD현대일렉트릭 등)
- 정책 수혜·실적 시즌 테마주

**상하한 제한 활용**:
- 상한가 가면 마감까지 안 풀리는 경우 많음
- 상한가 다음날 갭 상승 → 시초가 매도
- 하한가 풀린 직후 반발 매수

## 미국 단타 (준석TV 스타일)

**핵심**: 차트 패턴 + 시간대 + 224일선

**일과** (한국 시간):
1. 17:00 (서머타임) 프리장 시작 → 종목 순위 모니터링 (50% 급등주 발생)
2. 22:30 본장 시작 → 반대 베팅 (시작하자마자 떨어지면 매수)
3. 00:00~05:00 본장 후반 → C자리 진입
4. 05:00~09:00 에프터마켓 (참고용)

**핵심 종목군**:
- 시가총액 1억~100억 달러 (변동성 큼)
- 바이오, AI, 전기차, 우주, 양자컴퓨팅 테마
- 텐버거 후보 (한 달 1~2종목)

**무제한 변동성 활용**:
- 시가 대비 +100~300% 가능
- C자리 진입 시 -40% 후 +25% 회복 패턴
- 텐버거 종목은 한 달 1~2개

# 한국 직장인 관점 (사용자 시점)

## 한국 주식의 장단점

**장점**:
- 거래 시간이 직장인 이른 시간/점심에 부분 가능
- 모국어 정보 접근 용이
- 글로벌 피어 추종으로 미국 시장 활용 가능
- 변동성 제한으로 한 번에 큰 손실 막힘

**단점**:
- 풀타임 모니터링 어려움 (직장 시간대)
- 알고리즘이 패턴 학습한 상태
- 시간외 단일가 노이즈

## 미국 주식의 장단점

**장점**:
- 한국 시간 저녁~새벽 (직장 후 가능)
- 변동성 큼 → 단타 수익 폭 큼
- 프리장 + 본장 + 에프터마켓 활용 가능
- 자동 손절 주문으로 잘 때도 관리 가능

**단점**:
- 수면 부족 → 판단력 저하
- 환율 리스크 (수익 후 환차손)
- 변동성 제한 없어 한 번에 -50% 가능
- 정보 비대칭 (영어 자료 위주)

# 통합 매매 모델 제안

## 직장인용 하이브리드

| 시간대 | 시장 | 활동 |
|-------|------|------|
| 출근 전 (07:00~08:00) | 미국 본장 후반 | 잔고 점검, 익절·손절 |
| 점심 시간 (12:00~13:00) | 한국 점심장 | 보유 종목 점검 |
| 퇴근 후 (18:00~22:00) | 미국 프리장 | 종목 순위, 매수 후보 발굴 |
| 취침 전 (23:00) | 미국 본장 시작 | 자동 주문 걸고 취침 |

## 학습 우선순위

1. 한국 시장 매매일지 + 검색식 (프로브 방식, 진입 장벽 낮음)
2. 미국 본장 첫 30분 반대 베팅 (준석TV, 단순 룰)
3. 미국 C자리 매매 (준석TV, 224일선 활용)

# 적용 시 주의

- 한국에서 익숙한 룰을 미국에 그대로 적용하면 변동성에 깡통 빠르다
- 미국 룰을 한국에 적용하면 상하한 막혀서 의도와 다르게 작동
- 두 시장 동시 운영은 자금/멘탈 분산 → 1년 한 시장 마스터 후 확장 권장
