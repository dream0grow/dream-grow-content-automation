# 세일즈퍼널의 이해

- 세일즈 = 잠재고객의 구매 과정, 퍼널 = 깔때기
- 잠재고객이 구매하는 과정을 깔때기로 표현한 것 (깔때기는 고객의 수를 의미)
- 세일즈 퍼널을 통해 얻을 수 있는 인사이트: 고객이 구매하는 데까지 고민하는 과정을 거친다는 것
- 각 과정에 효과적으로 마케팅을 하면 구매전환률이 올라간다

## 돈 버는 공식
- 트래픽을 상품 구매로 전환하는 구조
- 월 50만원 계속 만드는 구조 = 트래픽 → 상품 구매 전환
- 조회수 60회로도 30만원 프로그램 2명 신청 = 60만원 수익 가능
