# 핵심
C자리는 급등 후 일시적 눌림 자리다. 급등 종목이 224일선 부근까지 한 번 빠진 후 반등할 때가 가장 안전한 매수 자리다.

# 왜 중요한가

급등주를 처음 오를 때 잡으면 늦거나 비싸게 산다. 한 번 급등 후 일시적으로 빠질 때(C자리)에서 매수하면 짧은 시간에 다시 반등하므로, 손실은 적고 수익률은 높다.

# 구체적 실행 방법

## C자리 정의

- A자리: 첫 급등 시점 (위험, 추격매수)
- B자리: 급등 중 눌림 (애매)
- **C자리: 급등 후 224일선 부근까지 일시적으로 내려와 매집 후 재반등하는 자리**

## 매수·익절 기준

1. **급등 종목 선별**: 시가 대비 50~100% 급등한 종목 (텐버거는 한 달 1~2개)
2. **224일선 터치 대기**: 90% 이상 일시적으로 224일선까지 내려옴
3. **5분봉 매집 확인 후 매수**: 224일선 부근에서 거래량 줄며 횡보하다 양봉 출현
4. **익절 폭**: 급등 폭의 50% 정도 회복 시 익절
   - 예: 1.0 → 2.0(+100%) → 1.2(-40%)로 빠진 후 1.5(+25%) 회복 시 익절
   - 통상 20~30% 익절이 안전. 30% 초과 욕심내면 못 팔게 됨.
5. **손절**: 224일선 이탈하면 즉시 손절

## 시간대 활용 (미국 주식)

- **프리장 (서머타임 적용 시 17:00~22:30)**: 급등주 50%가 여기서 나옴
- **본장 시작 직후**: 거래량 폭발, 7할 확률로 첫 움직임 반대로 돌아감
- **새벽 3, 4, 5시 정각**: 60분봉 전환점, 급등 자주 발생

# 적용 시 주의점

- "100% 정확"은 절대 아니다. 평균 70% 적중. 손절 룰 없으면 적중률은 의미 없음.
- 매수 시 분할 매수 필수. 몰빵은 가장 치명적인 실수.
- 1분봉/5분봉만 보면 흐름 못 본다. 일봉 + 60분봉 + 1분/5분봉 모두 봐야 함.
- 매집봉이 5개 이상 나온 종목은 세력 도주 가능성. 진입 자제.
- 미국 주식 시장이 약할 땐 어떤 자리도 작동 안 한다. 시장 자체부터 본다.
