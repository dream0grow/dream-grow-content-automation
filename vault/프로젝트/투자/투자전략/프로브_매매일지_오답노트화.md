# 핵심
매매일지는 학습의 오답 노트다. 기록 없는 매매는 같은 실수를 반복하고, 기록 있는 매매는 1년 뒤 다른 사람을 만든다.

# 왜 중요한가

프로브 저자가 500만 원 → 1.3억(177%) 만든 본질은 화려한 기법이 아니다. 매일 시나리오 → 실행 → 복기를 1년 넘게 반복한 습관이다. 같은 패턴의 실수가 3번 반복되면 행동을 바꾼다.

# 구체적 실행 방법

## 매매일지 5필드

1. **종목·날짜·시각**
2. **진입 이유** (내러티브, 차트 자리, 거래대금, 피어 상태 4축으로)
3. **목표가·손절가** (1:3 손익비 계산 결과)
4. **실제 결과** (익절가/손절가, %, 금액)
5. **차이 발생 원인** (계획 대비 어긋난 지점)

## 주간 복기 (일요일 30분)

- 이번 주 매매 전체 리스트업
- 패턴별 분류: 돌파매매 / 눌림매매 / 충동매매
- 손실의 80%가 어떤 패턴에서 나왔는가
- 다음 주에 안 할 행동 1가지 결정

## 월간 정리

- 승률·손익비 계산
- '계획 대비 잘 지킨 매매'와 '충동매매' 비율
- 카테고리별 수익률 (테마주 / 우량주 / 단타 / 스윙)

# 적용 시 주의점

- 수익 난 매매도 기록한다. "운으로 번 돈"인지 "원칙대로 번 돈"인지 구분이 안 되면 다음에 재현 못함.
- 감정 메모 1줄 필수 ("FOMO로 들어감", "확신 없이 들어감"). 행동을 바꾸는 건 숫자가 아니라 감정 패턴 인식이다.
- 일지를 미루면 한 달치 한꺼번에 쓰게 되는데, 그건 일지가 아니라 소설이다. 매매 직후 5분 룰.
