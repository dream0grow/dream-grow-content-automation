# 핵심
미국 주식 단타는 시간대 자체가 신호다. 프리장-본장 시작-에프터마켓 각 구간마다 세력 행동 패턴이 다르다.

# 왜 중요한가

급등주 50%는 프리장에서 발생한다. 같은 차트 모양이어도 본장 9시에 등장한 것과 프리장 19시에 등장한 것은 결과가 다르다. 시간대를 모르면 같은 자리에서도 적중률이 떨어진다.

# 구체적 실행 방법

## 시간대 매매 전략 (한국 시간 기준)

### 프리장 (서머타임 적용 시 17:00~22:30, 미적용 시 18:00~23:30)
- **하루 급등주의 50%가 발생**
- 거래량 적어 변동성 큼 → 진입 시 분할 매수 필수
- **80% 룰**: 프리장에서 급등하면 본장에서 오른 만큼 떨어짐 → 익절 후 재매수 대기

### 본장 시작 직후 (22:30~23:00)
- 거래량 폭발, 첫 움직임 반대로 돌아갈 확률 70%
- **반대 베팅**: 시작하자마자 떨어지면 매수 → 11시(0:00 한국시간) 전 매도
- **반대 베팅**: 시작하자마자 오르면 그 시점이 마지막 고점, 매도 타이밍

### 본장 후반 (00:00~05:00)
- 추세 안정화 구간
- 매집/배포 끝난 종목 정리되는 시간

### 에프터마켓 (05:00~09:00)
- **새벽 3, 4, 5, 6시 정각에 급등 자주 발생** (60분봉 전환점)
- 거래량 적어 신뢰도 낮음, 보조 지표로만 활용

## 추천 분봉

| 용도 | 분봉 |
|------|------|
| 급등주 큰 흐름 | 일봉 |
| 매수 타이밍 디테일 | 60분봉 |
| 단타 진입·청산 | 1분봉, 5분봉 |
| 호가 디테일 | 15틱 |

## 종목 순위 모니터링

- 삼성증권 HOME → 실시간 랭킹 → 나스닥 → 더보기 → 전일대비율
- 50~100% 급등주뿐 아니라 **5~10% 오른 종목**도 관찰 (매수세 들어오는 시작점)
- 하루 50~100번 순위 체크가 기본

# 적용 시 주의점

- 70% 적중이지 100%가 아니다. 손절 룰 없으면 무의미.
- 한국 직장인이 본장 풀타임 모니터링은 불가능. 프리장 + 자동 손절 주문 활용.
- 80% 룰은 통계적 평균. 강력한 호재가 있으면 본장에서도 오른다 (텐버거 후보).
- 새벽 매매는 수면 부족으로 판단력 저하. 자동 손절 주문 걸어두고 자는 게 안전.
