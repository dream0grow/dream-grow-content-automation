# 유튜브 수익화 3가지 방법

1. **Google 애드센스** - 광고 수익
2. **외주 광고** - 기업 PPL, 협찬
3. **사업형 유튜브** - 유튜브를 통해 개인 브랜딩 -> 자체 상품 판매
   - 예: 작심만일처럼 유튜브 -> 클래스유 강의 판매, 전자책 판매 등

핵심: 유튜브 광고 수익보다 **퍼널을 통한 자체 상품 판매**가 진짜 수익이다.

## 사업형 유튜브의 수익 구조
- 유튜브 -> 신뢰 확보 -> 팬층 형성 -> 자체 상품(강의, 전자책, 코칭) 판매
- 전자책은 유튜브와 선순환: 영상 주제 -> 전자책 -> 유튜브 홍보 -> 판매
