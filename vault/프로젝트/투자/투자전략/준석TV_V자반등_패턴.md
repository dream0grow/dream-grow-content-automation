# 핵심
V자반등은 밥그릇기법 2번에서 3번으로 가기 직전 세력이 한 번 더 악재로 공포를 띄워 개미를 털어내는 페이크다. V자 바닥에서 매집이 끝나면 1~3일 만에 급등이 나온다.

# 왜 중요한가

매집이 원활하지 않을 때 세력은 "더 떨어진다"는 공포를 만들어 개미 매도를 유발한다. 이 구간을 모르면 손절하고 나간 직후 급등을 놓치게 된다. V자 패턴을 알면 공포의 바닥을 매수 자리로 활용할 수 있다.

# 구체적 실행 방법

## V자반등 흐름

1. 밥그릇기법 2번 자리(횡보 매집) 진행 중
2. 매집이 충분하지 않아 3번을 시작 못함
3. 세력이 추가 악재 띄우며 주가를 한 번 더 강하게 내림 (공포 유발)
4. 개미 매도세 폭발 → 세력이 낮은 가격에 마지막 물량 흡수
5. 매집 완료 즉시 1~3일 내 급등 시작 (V자 모양)

## 진입 체크리스트

1. **밥그릇기법 2번 자리에서 형성되는가** (이미 정배열 종목은 V자가 아니라 단순 조정)
2. **공포 구간 거래량이 평소 대비 폭발하는가** (개미 투매 신호)
3. **장기이평선 224일선 부근까지 내려왔는가**
4. **급락 후 양봉 출현 + 거래량 유지** (반등 시작 신호)
5. **공구리 패턴(바닥 지지라인) 형성되는가**

## 매수·익절·손절

- **매수**: V자 바닥 확인 후 양봉 출현 시 분할 진입
- **1차 익절**: 5일선 또는 20일선 닿을 때 (절반 정리)
- **2차 익절**: 60일선 또는 직전 매집봉 고가
- **손절**: V자 바닥 이탈 시 즉시 손절 (보통 -5~7%)

# 적용 시 주의점

- V자처럼 보이는 단순 데드캣 바운스(반등 후 재하락)와 구분해야 한다. 차이는 거래량.
- 개미 공포가 정점일 때 매수해야 하므로 심리적으로 가장 어려운 자리. 자동 주문 활용.
- V자가 한 번 실패하면 더 깊은 W자로 갈 수 있다. 손절 룰 절대 무시 금지.
- 작전주 특성상 V자 후 급등이 텐버거가 될 수도, 단발 5%에 그칠 수도 있음.
