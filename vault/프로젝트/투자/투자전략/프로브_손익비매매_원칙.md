# 핵심
승률이 아니라 손익비가 생존을 결정한다. 1:3 손익비를 만들 수 있는 자리에서만 진입한다.

# 왜 중요한가

승률 90%여도 단 한 번의 큰 손실로 마이너스가 되고, 승률 30%여도 1:3 손익비를 지키면 플러스가 된다.

- **하수 패턴 (승률 90%, 결과 -1만 원)**: 9번 1만 원씩 이긴 뒤(+9만), 마지막 1번 손절 못해 10만 원 손실
- **고수 패턴 (승률 30%, 결과 +8만 원)**: 7번 1만 원씩 손절(-7만), 3번 5만 원씩 익절(+15만)

1:1 손익비는 수수료·세금 고려 시 사실상 손실. 최소 1:2~1:3을 만들지 못하는 자리는 진입 자체를 안 한다.

# 구체적 실행 방법

1. **진입 전 손절선부터 정한다.** 매수가 -1~2% 이내에서 깨질 지지선이 명확한 자리만 본다.
2. **목표가는 손절 폭의 3배.** -1% 손절이면 +3% 이상 익절 자리까지 그림이 그려져야 매수.
3. **지지·저항이 보이지 않는 어설픈 구간(추세 중간)은 패스.** 거기서 사면 -15%까지 견뎌야 하고, 손익비가 무너진다.
4. **손절 3가지 기준**:
   - 설정한 지지라인을 깼을 때
   - 시장 외부 변수(전쟁, 환율 급변 등) 발생 시
   - 물타기 후 평단 회복 시 반드시 비중 덜어내기

# 적용 시 주의점

- 빚투(신용·미수)는 손익비 계산을 무력화한다. 반대매매 공포가 손절을 마비시킨다.
- 한 종목 비중이 너무 크면 -2% 손절도 못한다. 비중 자체가 손익비를 결정한다.
- "이번엔 다를 것"이라는 생각이 드는 순간 손익비 원칙은 이미 깨졌다.
