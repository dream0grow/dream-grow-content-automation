# 핵심
프로브 저자가 500만 원 → 누적손익 1.3억(177%) 만든 본질은 화려한 기법이 아니라 매일 시나리오 → 실행 → 복기를 1년 넘게 반복한 습관이다.

# 사례 데이터

## 시작 시드와 결과
- 시드: 500만 원 + 100만 원 (다양한 시장)
- 누적손익: 1.7억
- 본업 누적 수익률: 177%
- 활용 시장: 국내주식 + 코인 선물 + 에어드랍

## 구조 (11장 가이드 기반)

1. **자기객관화** — 1.5세 이상의 매매 경력자 vs 초보자 구분
2. **HTS 화면 세팅** — 시야 절반은 화면이 결정한다
3. **투자 vs 트레이딩** — 다른 게임이라는 인식
4. **주식공부** — 책 + 영상 + 직접 매매 3축
5. **매크로** — 환율, 금리, 리스크 선호도
6. **재무제표** — 주가는 미래 실적의 그림자
7. **차트** — 보조 도구, 본질이 아님
8. **섹터·테마** — 4단계 생애 주기
9. **내러티브** — 시장이 듣고 싶어 하는 이야기
10. **계좌관리** — 비중 베팅 vs 분산
11. **금기와 습관** — 빚투 금지, 매매일지 필수

# 학습 포인트 (5가지)

## 1. 본업 = 가장 완벽한 헷지
- 본업 그만두지 않은 상태로 시드 만들고 실력 쌓음
- 손실이 나도 다음 달 월급으로 복구 가능
- 빚투 절대 금지 (멘탈 주도권 시장에 빼앗김)

## 2. 화려한 지표 거부
- 224일선·일목균형표·엘리엇 파동은 알고리즘 먹잇감
- **5가지 본질만 매일 확인**: 시장 인정, 재료, 거래대금, 신고가, 정배열

## 3. 매매일지 = 오답 노트
- 매일 실행 → 복기 → 패턴 학습
- 같은 실수 3번 반복되면 행동 변경
- 감정 메모 1줄 필수

## 4. 비중 베팅 vs 백화점 분산
- 30종목 분산 시 1종목 100% 떠도 전체 3% 수익
- 매크로·실적·재료·차트 4축에서 확신 시 1~3종목 집중
- 단, 확신 없는 집중은 도박

## 5. 글로벌 피어 후행성 활용
- 한국 증시는 글로벌 트렌드 후행 지표
- 일본 후지쿠라(전력망), 무라타(MLCC), 미국 이튼이 먼저 움직이면 한국 추종주 따라옴
- HD현대일렉트릭 텐배거(3만 → 120만 원)가 대표 사례

# 사용자 적용 가이드

## 학습 단계 (3개월)
1. 매일 매매일지 5필드 작성 (종목·이유·목표·결과·차이)
2. 5가지 본질만 추적: 시장 인정, 재료, 거래대금, 신고가, 정배열
3. 신용·미수 절대 사용 금지

## 실험 단계 (3~6개월)
1. 1:3 손익비 자리만 진입
2. 1~3종목 비중 베팅 (단, 매크로·실적·재료·차트 4축 확신 시에만)
3. 글로벌 피어 매핑표 작성

## 복기 단계 (반복)
- 주간 복기 (일요일 30분): 패턴별 분류, 충동매매 비율
- 월간 정리: 승률·손익비·카테고리별 수익률

# 함정 (저자가 강조한 금기)

- 리딩방·고액 강의·인큐베이팅 가짜 전문가
- "이번엔 다를 것" 충동매매
- FOMO 진입 (대중이 떠들기 시작하면 이미 끝물)
- 평단 회복 후 익절 안 하고 욕심
- 차트 자리 없는 추격매수
