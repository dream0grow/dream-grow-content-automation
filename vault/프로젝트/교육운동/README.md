# 프로젝트: 교육운동

> **타겟 독자: 교사그룹** — 드림그로우(학부모 대상)와 완전히 다른 대상이다.
> 이 프로젝트의 모든 글·기록은 동료 교사에게 말하는 글이며, 학부모용 채널에 섞지 않는다.

## 무엇을 하는 프로젝트인가

이한결이 참여하는 교육운동 활동을 기록하고, 그 경험을 **교사 대상 블로그·페이스북 글**로
발신한다. 발행은 사람이 한다(초안까지만 자동).

| 활동 | 성격 |
|---|---|
| 꿈들 | 교육성장네트워크 활동 (홍보팀 등) |
| 새넷 | 교사 네트워크 활동 |
| 전교조 | 조합 활동, 교육 이슈 대응 |

## 자동 파이프라인과의 연결

플라우드로 녹음한 활동 회의·강연·성찰이 재료다. `vault_pipeline`이 매일 밤:

1. 녹음에서 활동(꿈들/새넷/전교조/수업/일반)을 판별하고
2. 교육운동 관련 녹음이면 **활동기록 노트**를 `활동기록/`에 남기며
3. 교사그룹에게 말할 가치가 있는 주제면 블로그+페이스북 초안을
   `발행/블로그_교사/`·`발행/페이스북_교사/`에 생성한다 (frontmatter `타겟: 교사그룹`).

## 폴더

```
프로젝트/교육운동/
├── README.md      ← 이 파일
└── 활동기록/       ← 녹음별 활동 요약 (author: AI, 출처: plaud:<id>)
```

글의 최종 벽돌(주장·의견)은 제텔카스텐에 쌓인다. 여기는 활동의 사실 기록과
발행 이력을 관리하는 작업장이다.
