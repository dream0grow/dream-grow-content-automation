# S_구매로 직결하는 카피 529선

곤팀장(한성곤, 네블스쿨/머니북 대표)이 영업사원 시절 1,000장 이상의 계약서 작성과 9,900명+ 수강생 교육 경험을 바탕으로 정리한 카피 529문장 사전. 46페이지 PDF, 28,858자 분량.

## 한 줄 요약

온라인 수익화의 90%는 글쓰기에서 나온다는 전제 아래, 구매 전환을 만들어내는 17가지 목적별 문장 패턴 529개를 카탈로그화한 실전 카피 사전. 빈칸(00, 000) 형식의 템플릿이라 어떤 분야에도 끼워 넣어 사용 가능.

## 17가지 카피 분류

| 분류 | 목적 | 핵심 효과 |
|------|------|-----------|
| 1. 121가지 핵심단어 | 단어 자체로 클릭 유도 | 후킹의 최소 단위 |
| 2. 신뢰성 3초 상승 | 작성자 권위 즉시 형성 | "n년간 ___ 경험" 류 |
| 3. 공감대·친밀감 | 독자와 나의 거리 좁힘 | "저도 처음엔 ___였습니다" |
| 4. 즉각적 큰 이득 | 구매 시 받을 가치 명시 | "오늘 끝낼 수 있다" |
| 5. 복잡함을 명료하게 | 정리·요약으로 집중 만듦 | "결국 N가지면 끝" |
| 6. 확언 | 강력한 확신 전달 | "유일한 길입니다" |
| 7. '쉽다' 반복 | 대중성 확보 | 진입장벽 낮춤 |
| 8. 드라마틱 강조 | 중요성 부각 | "100번 강조해도 모자란" |
| 9. 선기빙(先 give) | 선한 이미지 구축 | 무료 자료 먼저 제공 |
| 10. 상식 벗어난 이야기 | 스팟성 참신함 | "남들 다 ___할 때 저는" |
| 11. 한정 효과 | 즉각 액션 유도 | "오늘까지만" |
| 12. 감정 터칭 | 마음을 움직임 | 스토리·미래 시각화 |
| 13. 멱살 잡기 | 끝까지 읽게 만듦 | 호기심 격차(curiosity gap) |
| 14. 결제 부담감 낮추기 | 가격을 저렴하게 인지 | "커피 한 잔 값" 류 |
| 15. 공포감 조성 | 경각심 유도 | "이거 모르면 평생 ___" |
| 16. 경쟁사 단점 → 내 장점 | 차별화 | 비교 후킹 |
| 17. 바쁨으로 인기도 직감 | 사회적 증명 | "예약 마감 임박" |

## Dream_Grow 활용 가능 카피 (선별)

스레드/릴스 후킹과 부모 교육 콘텐츠에 변형해 쓸 수 있는 형태:

**복잡함을 명료하게 (분류 5)**:
- "결국 ___을(를) 진짜 간단하게 생각하자면 N가지라고 할 수 있습니다."
- "한방에 하는 건 어렵지만, 하나씩 하면 쉽습니다."
- "사고의 전환 한끗차이가 전부입니다."

**감정 터칭 (분류 12)**:
- "저도 ___ 시절이 있었습니다. 그때 알았다면 좋았을 텐데..."
- "지금 바로 ___을(를) 시작하지 않으면, 1년 뒤 후회합니다."

**멱살 잡고 읽게 만들기 (분류 13)**:
- "마지막 줄까지 읽지 않으면 오해할 수 있습니다."
- "여기까지 읽었다면 이미 상위 1%입니다."

**상식 벗어난 이야기 (분류 10)**:
- "[A를 흔히 권하는데] 저는 그렇게 하지 말라고 합니다."  ← Dream_Grow의 "A가 아니라 B" 구조와 호환

## 구조적 특징

- 빈칸 템플릿 형식: 00/000으로 영역과 무관한 응용 가능
- 한 분류당 20~35개 변형 → 같은 메시지를 다양한 어조로 변주 가능
- 저자 본인이 영업사원 → 온라인 사업가 전환을 강조 → 카피의 본질이 신뢰 형성과 욕구 자극이라는 일관된 관점

## 평가

- **장점**: 즉시 사용 가능한 형태. 카피 한 줄을 만들기 위해 30분 고민하던 시간을 5분으로 단축.
- **한계**: 통계·연구·교실 경험 같은 근거 없이 카피 자체에 의존. Dream_Grow의 '가짜 통계 금지', '교실 경험 기반' 규칙과 결합해 사용해야 안전.
- **윤리적 주의**: 분류 11(한정 효과), 분류 15(공포감) 류는 부모 대상 콘텐츠에서 과용 시 신뢰 훼손. 필요 시 사용하되 사실 기반과 결합 필요.

## 활용 가능 영역

- **스레드 도입부**: 17가지 분류 중 본문 주제에 맞는 후킹 분류 선택 → 변형
- **릴스 첫 3초**: 분류 13(멱살 잡기) + 분류 4(즉각적 이득) 결합
- **리드마그넷 랜딩**: 분류 9(선기빙) + 분류 16(경쟁사 비교)
- **뉴스레터 제목**: 분류 5(복잡함을 명료하게) 변형
- **유튜브 썸네일/제목**: 분류 10(상식 벗어난) 변형
