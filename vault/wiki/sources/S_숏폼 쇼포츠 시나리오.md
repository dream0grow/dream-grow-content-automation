# 숏폼 쇼포츠 시나리오

스포츠 숏폼 채널 '쇼포츠'의 콘텐츠 시나리오 모음. 스포츠 하이라이트 + 기부 이벤트를 결합한 28개 이상의 숏폼/롱폼 콘텐츠 기획.

[출처:: [[raw/Roam-Export-1773035150854/숏폼_쇼포츠.md]]]

## 콘텐츠 패턴

### 1. 스포츠 미션/챌린지형
- 샤라포바/조코비치 미션, 프리다이빙, 스카이다이빙 등
- 구조: 상황 설명 → 긴장감 조성 → 결과 반전

### 2. 기부 연동형 (구독/좋아요 이벤트)
- "좋아요 1개당 10원 기부" 형식
- 메시+기부, 어린이 농구선수, 소방관 기부 등
- 구독자 참여를 유도하면서 선한 영향력 전파

### 3. VS 비교형
- 음바페 vs 홀란드, aguskafree 등
- 댓글 참여 유도

### 4. 감동/역전 스토리형
- 슈퍼맨(장애 극복), 여자아이 스케이트보드, 감동 달리기

### 5. 참여 유도형
- "착한 사람만 보이는 줄넘기" - 구독 유도 장치
