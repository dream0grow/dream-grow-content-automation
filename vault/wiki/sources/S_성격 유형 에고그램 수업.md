# 성격 유형 에고그램 수업

에고그램을 활용한 성격 유형 수업 설계 노트이다. 유형별 인터뷰, 성격 특성을 깨는 질문, 자기개선과 관계개선의 두 방향을 다룬다.

[출처:: [[raw/Roam-Export-1773035150854/성격 유형.md]]]

## 수업 흐름

1. **유형별 인터뷰와 특성 정리**: 성격 특성 종이 만들기
2. **발표와 인터뷰**
3. **성격 특성을 깨는 질문**
   - AC(순응적 아이)에게: "당신은 항상 소심한가요?"
   - FC(자유로운 아이)에게: "당신은 항상 자유분방하게 행동하나요?"
   - "아니요"라는 대답 유도 → "그럼 언제는 소심하고 언제는 대담한가요?"
4. **자기 개선 방향 만들기**: 어떤 상황에서 성장하고 싶은가?
5. **관계개선의 관점 도입**
   - 자기 개선만이 아니라 상대 성격을 알아주는 대화로 문제 해결
   - CP, NP 등 성격 특성에 따라 맞춰서 대화하는 방식
6. **성격과 나, 성격과 상대를 정리하기**
