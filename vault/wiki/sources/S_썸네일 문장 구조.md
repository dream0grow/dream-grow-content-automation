# 썸네일 문장 구조

유튜브 썸네일에 사용되는 후킹 문장의 4가지 핵심 구조를 정리한 문서. 실제 썸네일 이미지 예시 다수 포함.

[출처:: [[raw/Roam-Export-1773035150854/썸네일 문장 구조.md]]]

## 4가지 썸네일 문장 구조

### 구조 1: 문제 + 쉬운 해결
"(문제 상황 A)에 (쉬운 행동 B)하면 전부 해결됩니다"

### 구조 2: 문제 + 흔한 행동
"(문제 현상 A) 겪는 사람들이 자주 (하는 행동 B)"

### 구조 3: 결과 + 간단한 원인
"모든 (결과 A) 문제는 (간단한 원인 B) 때문이다"

### 구조 4: 행동 부정
"(하려는 행동 A) 전부 다 틀렸습니다"
"(하려는 행동 A)할 때 반드시 알아야 할 3가지"

### 추가 구조
- "(일반적인 문제 상황 A)" (문제 겪는 B)가 되는 과정
- "모든 (잘하고 싶은 A)의 문제는 (평범한 B) 때문이다"
