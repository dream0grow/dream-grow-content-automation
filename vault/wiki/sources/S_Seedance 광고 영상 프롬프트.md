# Seedance 2.0 광고 영상 프롬프트 (Slowflow Forest Calming)

15초 화장품 브랜드 필름 광고를 위한 Seedance 2.0 초별 프롬프트 템플릿. 비디오 생성 AI를 활용한 브랜드 광고 제작 워크플로우 사례.

## 구조

15초를 6개 컷으로 분할하여 각 구간마다 (1) 비주얼 프롬프트 (2) 사운드 디자인을 별도 명시.

| 시간 | 장면 | 핵심 비주얼 |
|------|------|------------|
| 0~2s | OPEN · 제품 전체샷 | 라탄 바구니, 녹차잎, 카모마일 |
| 2~4s | 녹차잎 클로즈업 | 모닝 듀, 슬로모션 0.5x, 황금 백라이트 |
| 4~6s | 손 · 세럼 어플리케이션 | 우아한 손, 슬로모션 한 방울 |
| 6~8s | 카모마일 클로즈업 | 꽃잎 개화, 꿀빛 백라이트 |
| 8~11s | 어깨라인 · 크림 바르기 | 노쇼울더, 친밀한 일상 의례 |
| 11~13s | 제품 + 자연 정물 | 탑다운 플랫레이, 줌아웃 |
| 13~15s | LOGO + COPY | "자연이 건네는 조용한 위로" |

## 프롬프트 작성 원칙 — [[C_AI 영상 프롬프트 구조화]]

- 9:16 세로 비율 명시
- 35mm 필름룩, 슬로모션 배율, 백라이트 구체화
- 사운드를 비주얼과 별도 레이어로 작성 (ASMR, 무음 페이드, 악기 진입 시점)
- 나레이션 톤·발화 속도·자막 영문 동시 명시
- 모델 패션 가이드(상의/손/전체 톤) 표로 부속

## 활용 맥락

Dream_Grow 또는 10x인생 채널의 브랜드 컬래버 광고/리드마그넷 트레일러에 응용 가능한 템플릿.
