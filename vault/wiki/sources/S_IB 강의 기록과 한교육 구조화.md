# IB 강의 기록과 한교육 구조화

2022년 1월 21일 IB교육 강의를 기록한 노트. 온라인 참여 형식으로 패들렛, 줌 소그룹 토론, 마인드맵을 활용한 강의 구조를 분석하고, 이를 한교육 시스템에 적용하는 아이디어를 정리했다.

## 핵심 내용
- IB교육의 구조: 교육 목표(학습자상) → 미션(교육 선언문) → 모듈/목표표
- 한교육 적용: 비전(홍익인간) → 미션(관계성을 키우는 한 인간으로서의 성장) → 핵심 역량(자기 사랑, 문제해결, 공감적 대화) → 역량별 기술 구조화
- 기술에서 예술로의 발전: Skill → Sight → Doing → Being → Art
