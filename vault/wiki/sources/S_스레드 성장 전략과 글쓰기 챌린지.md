# S_스레드 성장 전략과 글쓰기 챌린지

스레드 플랫폼에서 팔로워를 빠르게 성장시키는 전략과 글쓰기 챌린지 운영 방법을 종합한 페이지.

## 스레드 성장 사례

- 6개월간 팔로워 100명 → 새 계정으로 1달 만에 6000명 달성
- 기존 실패: 글 써도 좋아요 1개, 조회수 100~300회

## 변화 포인트 3가지

1. **후킹(hooking)** 기술 습득
2. 타겟층을 좁히지 않고 **파이를 넓힘**
3. 꾸준한 글쓰기 + **피드백 루프**

## 글쓰기 챌린지 4주 과정

| 주차 | 주제 |
|------|------|
| 1주차 | 후킹 연습 |
| 2주차 | 좋은 글 벤치마킹 |
| 3주차 | 체류시간 늘리는 글쓰기 |
| 4주차 | 타겟층 꿀주제 탐색 |

- 핵심 규칙: "글을 올리면 반드시 1명은 피드백 해준다!"
- 매달 4주 간격으로 반복하여 핵심 기술 실천
