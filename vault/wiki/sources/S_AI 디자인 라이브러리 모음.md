# AI 디자인 라이브러리 모음

웹/AI 디자인 작업에 활용 가능한 컴포넌트 라이브러리 링크 클리핑.

## 라이브러리

- **shadcn/ui** — https://ui.shadcn.com/ (React 컴포넌트 라이브러리, 코드 복붙 방식)
- **Aceternity UI** — https://ui.aceternity.com/ (애니메이션 강조 컴포넌트)
- **React Bits** — https://www.reactbits.dev/ (재사용 가능한 React 패턴)
- **Design.md** — https://getdesign.md/ (마크다운 기반 디자인 시스템)

## 활용 맥락

콘텐츠 자동화 시스템의 발행 페이지, 리드마그넷 랜딩페이지, 내부 대시보드 UI 구축 시 참고.
