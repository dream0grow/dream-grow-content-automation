# AI 사진 감성 손그림 프롬프트

업로드한 사진(아이 + 놀이터)에 손그림 주석을 덧입혀 인스타그램 스토리풍으로 변환하는 이미지 생성 프롬프트.

## 프롬프트 구조

1. **기준 이미지 사용 선언**: 업로드 사진을 메인 기준으로 하여 자연스럽게 반영
2. **손그림 주석 요청**: 흰 펜으로 그린 듯한 러프한 선
3. **요소별 주석**: 인물 외곽선, 화살표, 시선 흐름
4. **메모 톤**: 한글 귀여운 손글씨, 혼잣말·감성적 톤
5. **콘텐츠 가이드**: 아이는 "날씨 좋은 날", 기구는 "어린이집 하원길", 전체 한마디 요약, 멘트 4개
6. **분위기**: 잡지 메모풍, 인스타 스토리풍, 여백 살림

## 활용 맥락

Dream_Grow 인스타그램 콘텐츠 중 사진 기반 일상 메모형 게시물에 활용 가능. 감성적 느낌을 유지하면서 메시지를 시각적으로 전달.
