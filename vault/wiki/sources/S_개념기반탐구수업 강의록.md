# 개념기반탐구수업 강의록

2024년 6월 15일 제주도교육청 연수 내용. 개념이 왜 중요한지, 개념기반 탐구수업의 이론과 실습을 다룬다.

## 핵심 내용
- 지식의 구조: 사실 → 개념 → 원리(일반화) → 전이
- OECD 2030 학습 나침반의 변혁적 역량: 새로운 가치 창출, 갈등과 딜레마 조정, 책임 의식
- See-Think-Wonder 전략: Think는 See가 근거가 되어야 한다
- 프레이어 모델을 활용한 개념 정의하기
- 탐구수업 설계는 백워드 설계를 기반으로 한다
