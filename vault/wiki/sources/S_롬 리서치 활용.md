# 롬 리서치 활용

Roam Research의 그래프 뷰 발전 과정, 마크다운 활용법, 다이어그램, 템플릿, 플러그인(오토태그) 사용법을 정리한 실전 가이드.

[출처:: [[raw/Roam-Export-1773035150854/롬 리서치 활용.md]]]

## 핵심 내용

### 그래프 뷰 발전 3단계
1. 시작: 데일리 노트만 존재 (1줄)
2. 중간: 백링크 시작 → 노트 연결 (2줄)
3. 활용: 제대로 연결 → 4~5층위 네트워크

### 노트 연결 방법
- 페이지 개념 이해 → 페이지 내 글쓰기 → 다른 글을 백링크로 연결

### 마크다운 활용
- 하이라이트, 인용구, 앨리어스, 구분선, 블록 임베드

### 플러그인 - 오토태그
- 동의어 자동 연결 (인간 = 사람)
- 분야 → 지식의 구조(KDC)로 자동 연결

### 핵심 용어 구분
- **키워드**: 메모의 핵심 개념어
- **태그**: 검색 시 사용할 단어
- **카테고리**: 지식의 구조에서의 분류 영역
