# 블로그 발행 컨텐츠

유튜브 초기 성장 전략과 블로그 키워드 발굴, 유튜브 수익 구조 분석, 끌어당김 워크숍 모집글 등을 정리한 콘텐츠 기획 문서.

[출처:: [[raw/Roam-Export-1773035150854/블로그 발행 컨텐츠.md]]]

## 핵심 내용

### 1. 블로그 키워드 발굴
- 유튜브 수익 계산기, 세이노의 가르침, 그레이트 마인드셋 등 검색 키워드 기반 콘텐츠 기획

### 2. 블로그 글 템플릿 - 문제제기형
- 문제 제기 → 빠른 해결책 제시 → 문제 재부각 → 근거 제시 → 해결책 제시 → 암시/보상

### 3. 유튜브 초기 성장 핵심
- 구글은 새 채널/영상을 검증한다: **시청시간 + 반응도(좋아요, 댓글)**가 핵심
- 조회수가 높아서 반응도가 높은 것이 아니라, 반응도가 높아서 조회수가 올라간다
- 유튜브 스튜디오 3대 지표: 조회수, 시청시간, 구독자

### 4. 유튜브 수익 구조
- 구글 애드센스 광고 수익: 조회수당 약 2~5원
- 진짜 수익은 [[C_세일즈퍼널]]을 통한 자체 상품 판매 (광고 수익의 20~100배)
- 사례: 부업하는 부부 채널 - 광고 180만원 vs 크몽 전자책 3500만원~1.7억원

### 5. 끌어당김 워크숍 모집
- 목표 100번 쓰기 모임
- 끌어당김의 과학적 근거 + 실천 환경 제공
