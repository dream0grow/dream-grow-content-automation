# 숏폼 만들기 - 카피메이커

카피라이팅 분석을 숏폼으로 제작하는 '카피메이커' 콘텐츠 기획. 실제 브랜드/광고의 카피를 분석하고 개선안을 제시하는 구조.

[출처:: [[raw/Roam-Export-1773035150854/숏폼 만들기 - 카피메이커.md]]]

## 카피 핵심 공식
- **파소나 법칙** 수정 → '오공문해' (오잉? → 공감 → 문제제기 → 해결책 안내)

## 사례 분석

### 7번가 피자
- 기존 카피 "근본이 다른 7번가 피자" → 소비자에게 어필 안됨
- 차별점: 곡물 도우, 7가지 맛
- 개선안: "도우가 다른 7번가 피자", "곡물 도우 7번가 피자"
- 핵심: **카피는 소통이다** (추상적 → 구체적)

### 여성 전용 헬스장
- 기존: 할인율만 어필 → 지루함, 신뢰성 없음
- 개선안: "병원 가면 백날 듣는 말, 운동하세요. 지겹지 않으세요?"

### 참신한 카피 분석
- 앵커링 효과 + 시각화 표현 (14억 아렌시아, 피부에 형광등키는)
- 질문형식 호기심 유발 ("뜯고 짜고 다해 보셨죠?")
- 중의적 표현 (교육비 → 비 → 우산)
- 대비를 통한 재미
