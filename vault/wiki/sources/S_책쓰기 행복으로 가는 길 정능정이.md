# 책 쓰기 - 행복으로 가는 길, 한국인의 4가지 덕

정능정이(情能正利) 모델을 중심으로 한국적 행복(복)의 구조를 설명하는 책 기획 문서. 4가지 덕목의 조화로운 발전이 진정한 복(福)이라는 메시지.

[출처:: [[raw/Roam-Export-1773035150854/책 쓰기 - 행복으로 가는 길, 한국인의 4가지 덕을 중심으로.md]]]

## 정능정이(情能正利) 4덕 모델

| 덕목 | 의미 | 내용 |
|------|------|------|
| 정(情) | 관계의 덕 | 가족애, 정서적 유대감, 공감과 배려 |
| 능(能) | 실력의 덕 | 전문성, 자기계발, 창의적 문제해결 |
| 정(正) | 바름의 덕 | 도덕성, 가치관 확립, 사회적 책임감 |
| 이(利) | 경제의 덕 | 물질적 풍요, 경제적 안정, 가치 창출 |

## 덕목 결합의 시너지
- 정+능 = 협력, 정+정 = 공헌, 능+이 = 성과, 정+이 = 가치
- 3덕 결합: 하나가 빠지면 특정 문제 발생 (외로움, 경제적 불안 등)
- **4덕 조화 = 복(福)**: 중용과 균형이 핵심

## 보완 방향
- 역사적 맥락: 조선시대 → 근대 → 현대의 '복' 변천
- 동서양 철학 비교: 유교 수기치인, 불교 중도, 아리스토텔레스 유데모니아
- 문화적 사례: 응답하라 1988(정), 기생충(능+이 불균형), 미생(능+이)
- 심리학적 통찰과 사회학적 관점 통합

## 책 목차 (1안 기준)
- 1부: 한국인과 복 (복의 의미, 동서양 철학)
- 2부: 네 가지 덕으로 가는 길 (정/능/정/이 각론)
- 3부: 덕의 만남과 조화 (시너지, 중용)
- 4부: 나만의 행복 만들기 (진단, 실천, 확산)
