# 천재적 통찰 도출 공식

AI 프롬프트 또는 사고 프레임워크로 활용 가능한 공식 모음.

## 1. Genius Insight Formula
GI = (O x C x P x S) / (A + B)
- O: 관찰의 깊이, C: 연결의 독창성, P: 패턴 인식 능력, S: 종합적 사고
- A: 고정관념 수준, B: 편향 정도
- 적용: 고정관념과 편향을 최소화하면서 관찰-연결-패턴-종합 순서로 사고 전개

## 2. 다차원적 분석 프레임워크
MDA = 합계[Di x Wi x Ii]
- 시간적 차원 (과거-현재-미래)
- 공간적 차원 (로컬-글로벌-우주적)
- 추상적 차원 (구체-중간-추상)
- 인과적 차원 (원인-과정-결과)
