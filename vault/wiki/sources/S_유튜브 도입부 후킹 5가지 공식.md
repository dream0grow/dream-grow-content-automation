# 유튜브 도입부 후킹 5가지 공식

## 핵심 요약

유튜브 영상의 처음 5~15초에 시청자 이탈을 막는 5가지 도입부 공식을 체계화한 자료. PAS(문제-자극-해결), 공감형, 통계 충격형, 스토리텔링형, 위기-기회 전환형으로 구분하며, 각각 다른 심리적 메커니즘으로 시청 지속을 유도한다.

## 상세 내용

### 1. PAS (Problem-Agitate-Solution)

마케팅 카피라이팅의 고전 공식을 영상 도입부에 적용한 구조.

- **Problem(문제 제기)**: 시청자가 겪는 구체적 문제를 짚는다. "아이가 공부를 안 하려고 해서 걱정이시죠?"
- **Agitate(자극/심화)**: 문제를 방치했을 때 벌어질 상황을 증폭한다. "이대로 두면 중학교 가서 더 힘들어집니다."
- **Solution(해결 암시)**: 영상에서 해결책을 줄 것임을 약속한다. "오늘 이 방법 하나만 바꾸면 달라집니다."

적합한 장르: 교육, 자기계발, 부모교육, 건강 콘텐츠

### 2. 공감형

시청자의 감정/경험에 동일시하며 시작하는 구조.

- **패턴**: "저도 처음에는..." / "혹시 이런 경험 있으신가요?"
- **원리**: 미러링 효과. 자신의 상황을 정확히 묘사해주면 '이 사람은 나를 이해한다'는 신뢰가 형성됨
- **확장**: 공감 후 반전("그런데 제가 이걸 하고 나서 완전히 달라졌습니다")
- **주의점**: 공감만 하고 끝나면 시청 지속이 안 됨. 반드시 해결의 힌트를 함께 제시

적합한 장르: 브이로그, 경험담, 초등교육 학부모 콘텐츠

### 3. 통계 충격형

수치나 데이터로 인지 부조화를 일으키는 구조.

- **패턴**: "한국 초등학생의 70%가 스마트폰을 하루 3시간 이상 씁니다" / "이 방법을 쓴 채널의 88%가 3개월 안에 구독자 2배가 됐습니다"
- **원리**: 기대를 벗어나는 숫자는 뇌의 주의 자원을 자동 할당함 (놀람 반응)
- **조건**: 출처가 있어야 신뢰 유지. 과장 숫자는 역효과
- **변형**: "단 1%만 알고 있는..." (희소성 + 숫자)

적합한 장르: 정보성 콘텐츠, 뉴스형 분석, 교육 팩트체크

### 4. 스토리텔링형

구체적 장면이나 사건으로 시작하는 구조.

- **패턴**: "지난주에 학부모 상담을 했는데요, 어머니가 갑자기 울기 시작하셨습니다" / "모든 건 3년 전 그 한마디에서 시작됐다"
- **원리**: 서사는 뇌의 디폴트 모드 네트워크를 활성화시켜 몰입을 유도함. 인간은 스토리에 대한 완결 욕구가 있음 (자이가르닉 효과)
- **구조**: 인물 + 갈등/사건 + 미해결(영상 후반에 해결)
- **주의점**: 도입부에서 결론을 주면 안 됨. 궁금증 유지가 핵심

적합한 장르: 교육 사례, 경험 공유, 다큐 스타일

### 5. 위기-기회 전환형

현재의 위기 상황을 제시한 뒤, 기회로 프레이밍하는 구조.

- **패턴**: "(위기 현상 제시) 하지만 이걸 아는 사람은 오히려 이 상황을 기회로 바꿉니다" / "앞으로도 지금과 별로 다를 게 없을 겁니다. 이걸 모른다면요."
- **원리**: 손실 회피(loss aversion)를 자극한 뒤, 해결 가능성을 제시하여 시청 동기 생성
- **변형**: 통념 뒤집기("세뇌 당했기 때문입니다"), 권위 제시("15년 현직 교사로서 말씀드립니다")
- **강화**: "모른다면 당신은 반드시 후회합니다" (FOMO 자극)

적합한 장르: 트렌드 분석, 시사, 투자, 교육 정책

### 공식별 선택 기준

| 공식 | 핵심 심리 | 적합 상황 |
|---|---|---|
| PAS | 문제 인식 → 고통 증폭 → 희망 | 솔루션이 명확한 주제 |
| 공감형 | 동일시 → 신뢰 | 감정적 주제, 학부모 대상 |
| 통계 충격 | 인지 부조화 → 호기심 | 정보성/팩트 기반 주제 |
| 스토리텔링 | 서사 몰입 → 완결 욕구 | 사례/경험 기반 주제 |
| 위기-기회 | 손실 회피 → 기회 탐색 | 트렌드/변화 주제 |
