# S_유튜브 수익화 12주 과정

유튜브 수익화를 위한 12주 과정의 핵심 구조와 전략을 정리한 페이지.

## 전체 구조

- **1주차**: 수익화 구조 기초 (풀링-키-랜딩-판매)
- **2주차**: 수익화 기획 (무엇을 팔지)
- **3주차**: 키콘텐츠 기획 (현상-욕구-계획-행동-보상)
- **4주차**: 풀링 콘텐츠 기획

## 풀링 콘텐츠 4가지 종류

에버그린 / 시즌성 / 데일리 / 히어로

## 유튜브 성공 4요소

1. 들어오게 하기 (클릭률)
2. 계속 보게 하기 (시청지속시간)
3. 다시 보게 하기 (구독)
4. 수익화 (판매)

## 콘텐츠 가치 체계

- **콘텐츠 가치**: 정량적(클릭률/시청지속시간) + 정성적(정보/공감/재미/흥미)
- **개인 가치**: 캐릭터 롤모델 설정, 15가지 질문
- **채널 가치**: 히트 콘텐츠 시리즈화, 관점 정하기, 결 일치

## 원고 구조

공감 → 정보 → 개인가치 → 행동버튼 → 결론
