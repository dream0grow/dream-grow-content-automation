# 유튜브 100일 성장 로드맵

## 핵심 요약

유튜브 채널을 처음 시작하는 크리에이터를 위한 100일 단계별 성장 전략. 처음 30일은 루틴 형성과 기본기, 31~60일은 데이터 기반 최적화, 61~100일은 수익화 기반 구축에 집중하는 3단계 구조.

## 상세 내용

### Phase 1: 30일 루틴 형성 (Day 1~30)

**목표**: 영상 제작 근육을 만들고, 완벽주의를 버린다.

- **주 2회 이상 업로드** 루틴 확립 (최소 8~12개 영상)
- 10문장 뼈대로 시작: 완벽한 대본 대신 핵심 흐름만 잡고 촬영
- 장비 최소화: 스마트폰 + 자연광 + 무료 편집앱으로 충분
- **벤치마킹 3채널 선정**: 나와 비슷한 주제의 성공 채널 분석
  - 썸네일 구조, 제목 패턴, 도입부 구조, 영상 길이 기록
- 매 영상마다 1가지씩 실험 (제목/썸네일/도입부 중 하나)
- **핵심 마인드셋**: "못생긴 영상 10개가 안 만든 완벽한 영상 1개보다 낫다"

#### 30일 체크리스트
- [ ] 채널 아트, 프로필, 소개 설정 완료
- [ ] 벤치마킹 3채널 분석 시트 작성
- [ ] 최소 8개 영상 업로드
- [ ] 10문장 뼈대 템플릿 자기 것으로 만들기
- [ ] 촬영-편집 루틴 시간 측정 및 단축

### Phase 2: 60일 최적화 (Day 31~60)

**목표**: 데이터를 읽고, 클릭률과 시청 지속시간을 올린다.

- **유튜브 스튜디오 분석 루틴화**
  - 클릭률(CTR): 5% 미만이면 썸네일/제목 개선
  - 평균 시청 지속시간: 전체 길이의 40% 미만이면 구조 개선
  - 이탈 구간 분석: 어디서 빠지는지 파악
- **후킹 공식 적용**: 5가지 후킹 공식을 돌아가며 테스트
- **실험이론 적용**: 가설 -> 실험 -> 검증 사이클
  - "이번 영상은 통계 충격형 도입부를 쓰면 시청 지속시간이 올라갈 것이다"
  - 결과 기록 -> 다음 영상에 반영
- **시리즈화 시작**: 조회수 높은 주제를 시리즈로 확장
- 댓글 100% 답변 + 커뮤니티 탭 활용 시작

#### 60일 체크리스트
- [ ] 주간 데이터 분석 루틴 (매주 월요일)
- [ ] 후킹 공식 5가지 각 1회 이상 테스트
- [ ] 실험 기록 시트 운영 (가설-결과-인사이트)
- [ ] 시리즈 1개 이상 기획 (3편+)
- [ ] 총 누적 20개+ 영상

### Phase 3: 100일 수익화 기반 (Day 61~100)

**목표**: 구독자 1,000명 + 시청시간 4,000시간 달성 기반 마련, 수익화 파이프라인 1단계 준비.

- **콘텐츠 믹스 최적화**
  - 풀링 콘텐츠 60%: 검색/추천으로 신규 유입
  - 키 콘텐츠 30%: 채널 정체성, 구독 유도
  - 커뮤니티 콘텐츠 10%: 기존 구독자 유지
- **전자책 기획 시작**: 인기 영상 5~10개 원고를 전자책으로 묶기
- **리드 마그넷 설계**: 무료 PDF/체크리스트 -> 이메일 수집 시작
- **협업/크로스 프로모션**: 비슷한 규모 채널과 콜라보
- 채널 트레일러 영상 제작 (채널 가치 30초 요약)

#### 100일 체크리스트
- [ ] 콘텐츠 믹스 비율 적용
- [ ] 전자책 목차 초안 완성
- [ ] 리드 마그넷 1개 제작 완료
- [ ] 총 누적 30개+ 영상
- [ ] 구독자/시청시간 목표 대비 진척률 점검

### 100일 이후: 지속 성장 원칙

- 떡상은 운이지만, 꾸준함은 실력이다
- 20만 구독자까지는 운의 영역이 크다 (레벨이론)
- 알고리즘보다 시청자에게 집중
- 주 2회 업로드 루틴은 최소 1년 유지
