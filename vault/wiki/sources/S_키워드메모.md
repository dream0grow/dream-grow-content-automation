# 키워드메모

제텔카스텐 키워드 페이지의 실제 사용 예시. 성장, 창의성, 확언, 유머, 아이스브레이킹 5개 키워드에 대해 육하원칙(What/Why/How) 구조로 메모를 분류한 문서.

[출처:: [[raw/Roam-Export-1773035150854/키워드메모.md]]]

## 포함된 키워드

### 성장
- What: 체-지-덕 3가지 종류, 새로운 행동 선택지를 만드는 것
- How: 꾸준한 시간 노력

### 창의성
- 정의: 기존의 것을 새롭게 해석하고 재창조하는 능력 (시골의사 박경철)
- What: 이질적 아이디어를 융합하는 것 (제텔카스텐)
- 창의성 논문 스크랩: 14가지 신화 vs 사실 정리

### 확언
- 정의: 원하는 목표를 무의식에 믿음으로 심는 방법
- How: 확언 규칙을 지키며 실천

### 유머
- How: 통념/예상에서 벗어난 이야기 = 웃김
- 연상어구 활용 (맥락에 맞는 비틀기)
- 공감 특징 극대화
- 자기 낮춤으로 상대 높이기

### 아이스브레이킹
- 분위기를 푸는 5분이 50% 이상 효율적인 회의를 만든다
