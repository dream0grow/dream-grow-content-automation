# 세컨드 브레인과 제텔카스텐 강의 종합

제레미강의 제텔카스텐, 데본씽크 사용 사례 강의, 안티넷 제텔카스텐(아날로그) 등을 종합 정리한 노트.

## 핵심 내용
- 제텔카스텐 소개: 루만 교수의 메모 시스템, 원자성의 원칙이 핵심
- 메모 7종류: 임시 → 문헌 → 영구보관 → 키워드 → 의견 → 주장 → 두번째 뇌
- 4가지 능력: 건축가(구조), 정원사(아이디어), 사서(수집), 경찰관(연결)
- 12가지 원리 중 원자성이 가장 중요
- 워크플로우 CODE: Capture → Organize → Distill → Express
- 한국십진분류표/듀이십진분류표로 인덱스를 만들어 지식 구조를 먼저 세운다
- 기존 메모 탐색 5가지 방법: 무한 서랍 읽기, 키워드별, 분야별, 검색, 그래프뷰
- 아날로그 방법: 연필 → 검정펜 → 빨간펜(수정) → 파란펜(인용) → 포스트잇(추가/궁금)
