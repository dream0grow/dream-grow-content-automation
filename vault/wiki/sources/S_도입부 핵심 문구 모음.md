# 도입부 핵심 문구 모음

유튜브 영상 도입부(훅)에서 시청자를 붙잡는 5가지 핵심 구조와 실전 카피 문구를 체계적으로 정리한 문서.

[출처:: [[raw/Roam-Export-1773035150854/도입부 핵심 문구 모음.md]]]

## 도입부 5대 구조

### 1. 공감-정의-설명 구조
- 공감 파트: 시청자의 고통/경험을 나열
- 정의 파트: 주제를 한 문장으로 정의
- 설명 파트: 배경 지식 제공

### 2. 궁금증 유발 문구
- 실제 사례 제시 (궁금증 + 신뢰감)
- "도대체~", "어떻게 하면~"
- "(통념)한 게 아니라 (상식 밖 이야기)한 거에요"
- "모든 건 ...에서 시작됐다"
- "A가 아니라 B라고 해요" (공감하는 정보 → 새로운 정보)

### 3. 유인 제시 문구
- 욕구 분석 후 나열하기
- 내용 나열하기 ("3가지 오늘 다뤄드릴거에요")
- 쉬운 것으로 어려운 것 해결 연결
- 유인제시 → 문제 제시 → 해결책 제시

### 4. 문제 제기 문구
- "(통념) 하지만 (상식밖 이야기)"
- "앞으로도 지금과 별로 다를 게 없을 겁니다"
- 위기 현상 → 기회로 전환
- "세뇌 당했기 때문입니다"

### 5. 공감 + 주장/문제 문구
- "별 거 아닌 것 같다고요? 아닙니다. 굉장히 중요합니다."
- "모른다면 당신은 반드시 후회합니다"
- 권위 제시: "독보적인~"
