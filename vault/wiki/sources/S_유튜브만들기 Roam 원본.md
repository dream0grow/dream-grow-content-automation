# 유튜브만들기 Roam 원본

## 핵심 내용 요약

- **주제 선정**: 상급주제 + 내 소재 엮기, 키워드 3개
- **시나리오**: 10문장 뼈대 -> 서론(문제공감) -> 본론(해결책) -> 결론(가치)
- **후킹**: 도입부 5초 안에 이탈 방지, 반전/질문/숫자/권위 활용
- **성장 전략**: 벤치마킹 -> 떡상 영상 공식 -> 시리즈화 -> 커뮤니티
- **수익화**: 광고 -> 전자책 -> 강의 -> 워크숍 퍼널
