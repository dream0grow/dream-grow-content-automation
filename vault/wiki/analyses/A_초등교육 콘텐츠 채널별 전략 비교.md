# 초등교육 콘텐츠 채널별 전략 비교

## 핵심 요약

동일한 초등교육 전문성을 스레드, 릴스(숏폼), 유튜브(롱폼), 블로그 4개 채널에 각각 최적화하여 배포하는 전략을 비교 분석한다. 각 채널의 알고리즘 특성, 콘텐츠 형태, 성장 속도, 수익화 경로가 다르므로 동일 메시지를 채널에 맞게 변형하는 것이 핵심이다.

## 상세 내용

### 채널별 특성 비교

| 항목 | 스레드 | 릴스(숏폼) | 유튜브(롱폼) | 블로그 |
|---|---|---|---|---|
| **형태** | 텍스트 (500자) | 영상 (15~90초) | 영상 (5~20분) | 텍스트 (1000~3000자) |
| **핵심 지표** | 도달/공유 | 조회수/저장 | 시청시간/구독 | 검색유입/체류시간 |
| **후킹 단위** | 첫 문장 (10~20자) | 첫 프레임 (1~3초) | 도입부 (5~15초) | 제목 + 첫 문단 |
| **성장 속도** | 빠름 (바이럴) | 빠름 (추천) | 느림 (누적형) | 느림 (검색 누적) |
| **수명** | 24~48시간 | 3~7일 | 6개월~수년 | 수년 (에버그린) |
| **수익화** | 간접 (브랜딩) | 간접 (광고/브랜딩) | 직접 (광고+상품) | 간접 (애드센스+유입) |
| **관계 깊이** | 얕고 넓음 | 얕고 넓음 | 깊고 좁음 | 중간 |

### 채널별 콘텐츠 전략

#### 스레드 (@dream_grow_lee)

- **역할**: 유입 채널 + 브랜드 인지도. 가장 빠른 도달 확보.
- **콘텐츠 형태**: 교육 인사이트 1개를 500자 이내로 압축. 리스트형, 대비형, 스토리형.
- **후킹**: 첫 문장에 핵심 갈등/놀라움 배치 ("아이가 떼쓸 때 절대 하면 안 되는 말")
- **빈도**: 일 1회 이상
- **전환 경로**: 스레드 -> 프로필 링크 -> 유튜브/전자책
- **강점**: 88K 조회수 콘텐츠 가능. 빠른 피드백으로 소재 검증 용도로도 활용.

#### 릴스(숏폼)

- **역할**: 발견 채널. 스레드보다 시각적 임팩트가 강한 소재에 적합.
- **콘텐츠 형태**: 1분 기승전결. 자막 필수. 얼굴 노출 or 텍스트+이미지.
- **후킹**: 첫 3초에 텍스트 오버레이 + 강렬한 시각 요소
- **빈도**: 주 3~5회
- **전환 경로**: 릴스 -> 프로필 -> 유튜브/스레드
- **강점**: 알고리즘 추천 비중이 높아 비팔로워 도달이 큼

#### 유튜브(롱폼)

- **역할**: 깊이 채널 + 수익화 허브. 신뢰 구축의 핵심.
- **콘텐츠 형태**: 5~15분 심층 콘텐츠. 10문장 뼈대 기반 제작.
- **후킹**: 5가지 후킹 공식 적용. 첫 15초가 승부.
- **빈도**: 주 1~2회
- **전환 경로**: 유튜브 -> 전자책/강의/코칭 (수익화 파이프라인)
- **강점**: 시청시간 누적. 검색 기반 에버그린 콘텐츠. 직접 수익화 가능.

#### 블로그

- **역할**: 검색 채널 + 에버그린 자산. 유튜브 원고의 텍스트 재활용.
- **콘텐츠 형태**: SEO 최적화 글. 유튜브 영상 원고를 블로그 글로 변환.
- **후킹**: 키워드 중심 제목 + 첫 문단 요약
- **빈도**: 주 1~2회
- **전환 경로**: 검색유입 -> 블로그 -> 유튜브 임베드 -> 구독
- **강점**: 구글/네이버 검색으로 장기 유입. 유튜브와 시너지.

### 동일 소재의 4채널 변환 예시

**소재**: "구체물 없이 수학 문제지만 풀면 안 되는 이유"

| 채널 | 변환 |
|---|---|
| 스레드 | "아이가 수학 문제지만 풀고 있다면 지금 당장 멈추세요. 연산 100점인데 응용문제를 못 푸는 아이, 이유가 있습니다..." (500자) |
| 릴스 | 60초. 바둑돌로 덧셈하는 아이 영상 + 자막 "문제지만 푸는 아이 vs 구체물로 배우는 아이" |
| 유튜브 | 10분. CPA이론 설명 + 가정에서 할 수 있는 구체물 활동 3가지 + 학년별 가이드 |
| 블로그 | "초등 수학, 구체물이 먼저입니다 - CPA이론으로 본 수학교육 가이드" (2000자) |

### 채널 우선순위 전략

1. **스레드**: 소재 검증 + 즉시 도달 (현재 주력)
2. **유튜브**: 깊이 + 수익화 (성장 중)
3. **블로그**: 검색 자산 + 유튜브 시너지 (보조)
4. **릴스**: 시각적 소재 한정 활용 (선택)

원칙: 1콘텐츠 4채널 배포. 원본은 유튜브 수준으로 깊이 있게 만들고, 나머지 채널로 압축/변형.
