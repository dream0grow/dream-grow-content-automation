# 생산적 실패 (Productive Failure)

학생이 정답을 알지 못한 채 문제와 씨름하는 초기 실패 경험이, 이후 개념 학습의 깊이를 강화한다는 학습 이론. 싱가포르 NIE(국립교육연구원) 소속이었고 현재 ETH 취리히 교수인 마누 카푸르(Manu Kapur)가 2008년 Cognition and Instruction 논문에서 처음 제시했다.

카푸르는 분산·평균 등 통계 개념 수업에서, 먼저 직접 가르친 집단보다 학생이 스스로 문제를 풀려다 실패한 뒤 나중에 정식 설명을 들은 집단의 전이(transfer) 점수가 더 높음을 반복 검증했다. 실패가 기존 지식 활성화, 지식 격차 인식, 변별 가능한 표상 생성, 개념적 주의 집중을 유발하기 때문이다.

교실 함의: 실수를 즉시 교정하는 대신, 충분한 탐색 시간을 보장하고 실패 경험을 학습 자원으로 다루어야 한다. '실수해도 괜찮아'라는 정서적 메시지의 학습과학적 근거다.
