# 비폭력대화(NVC)

비폭력대화(Nonviolent Communication, NVC)는 마셜 로젠버그(Marshall Rosenberg)가 1960년대에 정립한 의사소통 모델이다. 평가가 아닌 관찰, 생각이 아닌 느낌, 전략이 아닌 욕구, 강요가 아닌 부탁의 4단계 구조로 대화를 구성한다.

핵심은 '평가와 관찰의 분리'다. "너는 게을러"는 평가지만 "30분째 책상에 앉아 있지 않구나"는 관찰이다. 부모-자녀 대화에서 평가 언어는 방어를, 관찰 언어는 대화를 연다.

느낌과 생각도 구분한다. "무시당한 것 같아"는 생각(타인의 행동 해석)이고, "속상해"가 느낌이다. 느낌 뒤에는 항상 충족되지 않은 욕구가 있다는 전제 위에 작동한다.

교육 현장에서는 교사-학생, 부모-자녀 갈등 중재, 형제 갈등 중재의 기본 틀로 활용된다. 감정코칭과 결합되어 한국 부모교육에서 널리 보급되었다.
