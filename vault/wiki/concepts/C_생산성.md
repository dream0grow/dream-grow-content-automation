# C_생산성

원하는 결과물을 효율적으로 만들어내는 능력. 생산성 = 집중력 x 절제된 환경. 집중력이 아무리 높아도 환경이 방해하면 역효과가 난다.

## 분야

#0000자기계발/0500창의적사고·정보와시간관리

## 핵심 공식

- **생산성 = 집중력 x 환경**
- 100(집중력) x -1(환경) = -100 → 높은 집중력이라도 환경이 나쁘면 역효과

## How: 생산성 높이는 방법

### 환경 개선
- 스마트폰 방해금지 모드
- 카페에서 작업
- 브라우저 홈페이지를 구글로 변경

### 집중력 향상
- 작업과 무관한 생각을 노트에 즉시 손으로 적기
- [[C_제텔카스텐]]의 원자성과 연결 원리 활용
