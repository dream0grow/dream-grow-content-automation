# 상세페이지 전환율 구조

상세페이지 전환율은 디자인 품질이 아니라 **기획 구조**가 결정한다. 고객이 결제까지 도달하도록 의심을 단계적으로 줄이는 흐름이 핵심이다.

## 핵심 명제

> "상세페이지는 디자인이 아니라 전략이다. 기획이 90%다."

전환이 안 나오는 페이지의 본질적 문제는 촌스러운 디자인이 아니라 **소비자의 구매 흐름이 끊겨 있다**는 것이다. 따라서 리뉴얼 = 디자인 변경이 아니라 **전환율 구조 재설계**다.

## 7-Section 표준 구조

| Section | 핵심 메시지 | 작동 원리 |
|---------|------------|----------|
| 1. 후킹 | 검증된 결과 + 운영자 변화 + 들어야 하는 이유 | 첫 3초에 의심 무너뜨리기 |
| 2. 문제 정의 | 고객의 마음속 질문을 그대로 노출 | 공감 → 자기 대입 |
| 3. 해결책 | "재능이 아닌 구조" 강조 | 진입장벽 제거 |
| 4. 차별성 | 타사 vs 자사 비교 | USP 시각화 |
| 5. 사회적 증명 | 숫자 + 톡 + 배지 3중 | 신뢰 누적 |
| 6. 가격/혜택 | 정가 → 할인가 → 추가 혜택 | 가치 인식 강화 |
| 7. CTA + FAQ | 망설임 제거 + 행동 유도 | 결제 진입 |

## 작동 원리: 의심의 누적 제거

전환율은 **고객이 결제 직전까지 가지는 N개의 의심을 페이지가 얼마나 제거하느냐**의 함수다.

각 섹션은 특정 의심 1~2개를 명확히 무너뜨리는 역할:

- 후킹 → "이게 진짜야?" 의심 제거
- 문제 정의 → "이게 내 문제야?" 의심 제거
- 해결책 → "이걸로 해결돼?" 의심 제거
- 차별성 → "왜 굳이 이거여야 해?" 의심 제거
- 사회적 증명 → "남들도 그래?" 의심 제거
- 가격 → "이 값어치 해?" 의심 제거
- FAQ → "이런 경우엔 어떻게?" 의심 제거

## 16% 전환율을 만드는 5요소 (post3 핵심)

1. **성과 증명 우선** — 설명보다 결과 수치 먼저
2. **'재능'이 아닌 '구조' 강조** — 누구나 따라할 수 있는 패턴
3. **Before/After 서사** — 부족했던 시점부터 보여줘 자기 대입
4. **실행 → 결과 흐름 시각화** — 구매 후 미래 장면
5. **망설임 사전 제거** — FAQ + 보상 장치

## 실패 패턴 (디자인은 예쁜데 전환이 안 되는 이유)

1. 기획 없이 디자인만 한 페이지
2. 고객 문제보다 제품 기능만 나열
3. 스크롤 이탈 높은 복잡한 구조
4. 리뷰·인증·근거 부재
5. CTA가 끝에만 한 번 등장

## 사회적 증명 3중 레이어

신뢰는 한 가지 요소로 만들어지지 않는다. 다음 셋이 동시에 있어야 한다:

1. **누적 수치** — 거래/판매/구독자/팔로워
2. **진짜 대화** — 카카오톡/이메일/DM 캡처 (빨간 박스 강조)
3. **권위 배지** — 자격증/수상/인증/유명인 추천

## 디자인 위계 (시각적 hierarchy)

1. 핵심 숫자 (가장 큰 폰트, 단독 배치)
2. 핵심 키워드 (포인트 컬러로 강조)
3. 본문 (흰색·검정 기본)
4. POINT XX 배지로 섹션 구분
5. 비교표/체크리스트로 정보 구조화

## 적용 분야

- 강의/코칭 (1순위 적용 — 16% 모델 검증된 영역)
- 디지털 상품 (PDF/VOD)
- 쇼핑몰 제품 (식품/뷰티/공구)
