# 측정과 판단의 분리

LLM 운영 품질 관리에서 채점은 외부 스크립트가, 해석과 수정은 LLM이 담당하는 역할 분리 원칙.

## 문제 인식

LLM에게 "너 스스로 평가해봐"라고 시키면 자기가 만든 결과물을 자기가 채점하는 구조가 된다. 결과 점수는 항상 후하게 나오고, 진짜 문제는 가려진다.

## 해법 구조

| 역할 | 담당 | 산출물 |
|------|------|--------|
| 측정 | 결정론적 스크립트 | 9축 점수표 |
| 해석 | LLM | 우선순위 + 수정 제안 |
| 승인 | 사람 | 무엇을 고칠지 결정 |

## Kaizen Loop 9축 예시

- AQI 6축: 위임 정확도, 역할 집중도, 제약 준수율, 산출물 품질, 컨텍스트 효율, 비용 효율
- 인프라 3축: Hook 건강도, 규칙 파일 건강도, 운영 병목

## 적용 가능성

- 콘텐츠 자동화 파이프라인 산출물의 외부 점수화
- 에이전트 팀 작업의 위반 항목 자동 검출
- launchd 정기 작업 Hook 모니터링
