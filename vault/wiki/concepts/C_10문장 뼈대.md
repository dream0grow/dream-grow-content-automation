# 10문장 뼈대

## 핵심 요약

완벽한 대본 대신 10개 핵심 문장으로 영상의 뼈대를 잡는 콘텐츠 제작 방법론. 초보 크리에이터가 완벽주의에 빠져 영상을 못 만드는 문제를 해결하며, 자연스러운 말투와 빠른 제작 속도를 동시에 달성한다.

## 상세 내용

### 개념 정의

10문장 뼈대는 영상 시나리오를 작성할 때 전체 대본을 쓰는 대신, 영상의 흐름을 결정하는 핵심 문장 10개만 미리 정하고 나머지는 촬영 중 자연스럽게 채우는 방식이다.

### 기본 구조

1. **도입부 후킹** (1~2문장): 시청자를 잡는 첫 마디
2. **문제 공감** (2~3문장): 시청자의 고통/상황 묘사
3. **해결책 제시** (3~4문장): 본론의 핵심 포인트
4. **가치 요약 + CTA** (1~2문장): 핵심 정리와 행동 유도

### 왜 10문장인가

- **완벽한 대본의 함정**: 대본을 다 쓰면 읽는 느낌이 나서 오히려 부자연스러움
- **빈 화면의 공포 제거**: 아무것도 없이 시작하면 막막하지만, 10문장이 있으면 방향이 잡힘
- **최소 유효 구조(MVS)**: 영상의 논리적 흐름을 유지하는 최소한의 장치
- **제작 속도**: 대본 작성 시간 70% 이상 단축

### 실전 예시 (초등교육 콘텐츠)

```
1. "아이가 수학 문제지만 풀고 있다면, 지금 당장 멈추세요."
2. 많은 학부모가 연산 반복이 수학 실력이라고 착각한다.
3. 하지만 수학의 본질은 양감과 수 감각이다.
4. 구체물 조작 -> 그림/반구체물 -> 추상(숫자)의 순서가 핵심이다.
5. 예시: 바둑돌로 덧셈을 먼저 체험시키기.
6. 이걸 건너뛰면 3학년부터 수학을 포기한다.
7. 가정에서 할 수 있는 구체물 활동 3가지.
8. (활동1, 활동2, 활동3 간략히)
9. 핵심은 "손으로 먼저, 머리는 나중에".
10. 다음 영상에서 학년별 구체물 활용법을 다루겠다.
```
