# 유튜브 콘텐츠 제작 시스템

Dream_Grow의 유튜브 콘텐츠 제작 워크플로우. Roam 원본에서 추출.

## 영상 제작 7단계

1. **주제 선정** - 상급주제 + 내 소재, 키워드 3개
2. **시나리오 작성** - 10문장 뼈대, 서론-본론-결론
3. **촬영 준비** - 장비, 조명, 배경
4. **촬영** - B롤, 인서트, 오디오
5. **편집** - 컷편집, 자막, 효과음, BGM
6. **썸네일/제목** - 클릭율 높이는 구조
7. **업로드/최적화** - SEO, 설명란, 태그

## 후킹 공식 5가지

1. 반전형: "절대 하지 마세요"
2. 질문형: "이거 아세요?"
3. 숫자형: "5가지 비밀"
4. 권위형: "전문가로서 말합니다"
5. 공감형: "저도 처음에는..."

## 스레드 -> 영상 전환

[[06 스레드→영상 전환 가이드]]에 따라 고성과 스레드를 영상으로 변환.
- 조회수 10만+ 스레드 우선
- 스레드 구조를 시나리오 뼈대로 활용
- 본문 확장 + 시각 자료 추가
