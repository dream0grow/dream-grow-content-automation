# C_장기이평선 5/20/60/112/224

준석TV 차트 분석의 핵심 도구. 단기 5/20일선과 장기 60/112/224일선을 결합해 주가의 큰 사이클을 5단계로 구분한다.

## 5단계 모델

| 레벨 | 이평선 | 의미 |
|------|--------|------|
| 1 | 5일선 | 닿으면 일시 하락. 가장 약한 저항/지지 |
| 2 | 20일선 | 닿으면 일시 하락 (5일선 지지) |
| 3 | 60일선 | 닿으면 일시 하락 (20일선 지지) |
| 4 | 112일선 | 닿으면 일시 하락 (60일선 지지) |
| 5 | 224일선 (1년선) | 지지 시 신고가 도전 가능 |

레벨이 올라갈수록 지지·저항이 강해 주가 상승 난이도 증가.

## 정배열 vs 역배열

- **역배열**: 모든 이평선이 차트보다 위 (가장 저렴, 매집 진행)
- **정배열**: 모든 이평선이 차트보다 아래 (상승 추세, 5→20→60→112→224 단계 돌파 진행)

## 224일선의 특별한 의미

- 1년선이라 부르며, 가장 큰 저항·지지
- 급등주는 99% 224일선 터치 시 며칠 안에 빠르게 하락 시도
- 우량주(테슬라·애플)는 224일선 위에서 지속 지지하며 꾸준히 상승
- C자리 매매의 핵심: 급등 후 224일선까지 빠진 자리에서 매수

## 차트 보는 봉 단위 추천

| 단위 | 용도 |
|------|------|
| 일봉 | 큰 흐름, 밥그릇 단계 판단 |
| 60분봉 | 추세 전환, 매수 타이밍 디테일 |
| 5분봉 | 단타 진입·청산 |
| 1분봉 | 호가 디테일 |
| 15틱 | 주문 체결 디테일 |

## 적용

준석TV는 5분봉 5/20/60/112/224 라인을 모두 띄우는 것을 기본으로 함. 단기 단타에서도 장기 흐름 무시 금지.
