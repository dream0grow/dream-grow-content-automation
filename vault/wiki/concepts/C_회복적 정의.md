# 회복적 정의

응보적 정의(가해자 처벌 중심)의 한계를 넘어, 피해자의 욕구 회복에 초점을 맞추는 정의 패러다임.

## 핵심 원리

- **응보적 정의의 한계**: 가해자 처벌은 피해자의 고통을 회복시키지 못함. 가해자는 벌을 줄이는 데만 집중하게 됨
- **회복적 접근**: 피해자의 감정과 욕구를 먼저 듣고, 가해자가 책임을 인식하도록 대화의 장을 마련
- **정의의 여신상 재해석**: 칼(처벌)이 아닌 저울(균형 회복)에 초점

## 교육 현장 적용

- 학급 내 갈등 해결: 처벌이 아닌 대화와 회복의 서클
- 교실 문화 형성: 문제 학생이 아닌 환경이 아이를 키운다
- [[C_비폭력대화]]와 연계한 소통 구조

## 기원

- 엘마이라 사건(1974, 캐나다): 두 청소년이 22가구 재산 파괴 → 판사가 피해자 직접 만남 명령 → 회복적 정의의 시초
