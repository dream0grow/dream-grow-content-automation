# CPA이론

## 핵심 요약

CPA(Concrete-Pictorial-Abstract)는 수학 교육에서 구체물(실물 조작) -> 그림/반구체물(시각적 표현) -> 추상(숫자/기호)의 순서로 개념을 도입하는 교수법 원리. 제롬 브루너(Jerome Bruner)의 표상 이론에 기반하며, 싱가포르 수학 교육의 핵심 원리로 채택되었다.

## 상세 내용

### 3단계 구조

#### C: Concrete (구체물)
- 실제 물건을 손으로 조작하여 수학적 개념을 체험
- 예: 바둑돌로 덧셈, 블록으로 분수, 자로 길이 측정
- 핵심: 신체 감각을 통해 양감(quantity sense)을 형성

#### P: Pictorial (그림/반구체물)
- 구체물 경험을 그림, 도표, 다이어그램으로 표현
- 예: 수 막대 그림, 원 모델, 바 모델(bar model)
- 핵심: 실물에서 벗어나되 아직 추상이 아닌 중간 단계

#### A: Abstract (추상)
- 숫자, 기호, 수식으로 표현
- 예: 3 + 4 = 7, x + 2 = 5
- 핵심: C와 P 단계를 충분히 거친 후에야 의미 있는 추상화 가능

### 왜 순서가 중요한가

- 구체물 단계를 건너뛰고 바로 추상(문제지 풀기)으로 가면:
  - 수학이 "의미 없는 기호 조작"이 됨
  - 절차는 외우지만 개념을 이해하지 못함
  - 학년이 올라갈수록 수학 포기율 급증 (소위 "수포자")
- 연산 100점인데 응용문제를 못 푸는 아이 = C-P 단계 부족의 전형

### 가정에서의 적용

1. **저학년 덧셈/뺄셈**: 바둑돌, 과자, 동전 등으로 먼저 해보기
2. **분수**: 피자, 케이크를 실제로 나눠보기 -> 그림 그리기 -> 숫자로 표현
3. **곱셈**: 같은 수의 물건 묶음 만들기 -> 배열 그리기 -> 구구단

### 교육적 의의

- 비고츠키의 [[C_비계(scaffolding)]]와 연결: 구체물은 추상적 사고에 이르는 비계
- 피아제의 구체적 조작기(7~11세) 이론과도 일치: 초등학생은 구체적 경험 기반 사고가 중심
