# C_밥그릇기법

차트가 밥그릇 모양(급락 → 횡보 → 급등)으로 움직이는 패턴. 대주주와 세력의 작전 구조를 4단계로 도식화한 단타 매매 기법.

## 4단계 구조

1. **1번 자리 (급락)**: 대주주가 악재로 주가 반토막
2. **2번 자리 (횡보 매집)**: 세력이 시간을 들여 개미 물량 흡수, 224일선 이격 확대
3. **3번 자리 (상승 시작)**: 매집 완료 후 정배열 회복하며 상승 — **매수 타이밍**
4. **4번 자리 (52주 신고가)**: 정배열 이격 확대, 고수 영역

## 핵심 원리

- 작전 동기: 회사 파산 직전 대주주와 세력이 마지막 수익 추출
- 9/10은 상장폐지 (저자 언급) → 단타 수익 후 즉시 익절 필요
- 1~7일 단기 매매로 장기 투자 대안

## 보조 신호

- **매집봉**: 2번 자리에서 30~100% 일시 급등 (개미 물량 테스트)
- **공구리**: 3번 자리 시작 시 바닥 지지라인
- **V자반등**: 2번 → 3번 사이 한 번 더 악재로 공포 유발

## 한계

- 매집봉 5개 이상이면 작전 포기 신호 (세력 도주 가능성)
- 작전주 특성상 재무·뉴스 기반 판단 어려움 → 차트 패턴에 의존
