# C_곤팀장 카피·블로그 4종 시스템

곤팀장(한성곤, 네블스쿨)이 4개의 자료에 걸쳐 일관되게 제시한 글쓰기 시스템. 카피·첨삭·블로그·제안서 4축이지만 본질은 한 가지: **결론부터 + 후킹 3종 + MECE 논리 + 감성 마무리**.

## 4종 자료의 공통 골격

| 자료 | 다루는 영역 | 핵심 도구 |
|------|-------------|-----------|
| 카피 529선 | 단어·문장 단위 | 17개 분류, 빈칸 템플릿 |
| 컨설팅·첨삭 모음집 | 글 전체 구조 | 컨설팅 3축(제목·표현·논리), 첨삭 3축(인트로·본문·디자인) |
| 블로그 상위노출 A-Z | 알고리즘·플랫폼 | 5단계 공식, 체류시간 |
| 제안서 템플릿 | B2B 협업 | 7단계 구조, 사회적 증명 |

## 공통 원리 5가지

### 1. 결론부터 (두괄식)
- 모든 자료가 "거두절미"를 강조
- 첫 3~10초가 90% 이탈 결정
- 제안서·컨설팅·인트로 모두 동일

### 2. 후킹 3종
모든 자료에서 반복 등장:
- (a) 독자의 기존 생각 파괴
- (b) 공포감 부여
- (c) 구체적 이득 제시

### 3. MECE 논리 + 3의 법칙
- 주장 1개에 근거 3개
- 중복(겹침)·착오(놓침)·비약(건너뜀) 제거
- 근거끼리 상호 연결되어야 함

### 4. 비유의 영역 확장
- 자기 업계와 교차하는 4~5개 다른 영역에서 공통 속성을 끌어와 비유
- 다재다능한 인상 + 무한한 글감

### 5. 감성으로 마무리
- 본론에서 논리·데이터를 녹였다면 결론은 감성으로 매듭
- 카피·블로그·제안서 모두 동일

## Dream_Grow 적용 매트릭스

| 채널 | 구조 | 활용 자료 |
|------|------|-----------|
| 스레드 도입부 | 결론 선언 + 후킹 1개 | 카피 529선 + 블로그 A-Z |
| 스레드 본문 | MECE 3축 + 교실 경험 | 컨설팅 첨삭 모음집 |
| 스레드 마무리 | 감성 + 브랜드 서명 | 컨설팅 첨삭 모음집 |
| 릴스 첫 3초 | 멱살잡기 + 즉각 이득 | 카피 529선 분류 13·4 |
| 뉴스레터 제목 | 복잡함을 명료하게 | 카피 529선 분류 5 |
| 협업 제안 | 7단계 구조 | 제안서 템플릿 |

## 윤리적 변형 필요

곤팀장 자료는 마케팅·세일즈 맥락이 강하다. Dream_Grow의 부모 교육 톤에 적용 시 변형 필요:

- **공포감 후킹**: "이거 모르면 우리 아이 망친다" → "많은 부모님이 놓치고 계신 부분이 있습니다" (강도 낮춤)
- **한정 효과**: 마감 임박 식 강압 금지. 대신 "이 글은 한 번 읽고 끝내지 마세요"식 행동 유도.
- **사회적 증명**: "수강생 9,900명" 식 숫자 자랑 대신 "교실에서 만난 학생 ___명" 식 교사 권위 활용.

## 한계

- **저자 글쓰기 철학의 한계**: 부드러운 정서 콘텐츠보다 강한 설득에 최적화. 부모 교육의 절반은 "조급해하지 마세요" 같은 톤이 필요한데, 곤팀장 자료는 정반대.
- **MECE의 함정**: 3축으로 정리하다 보면 실제 복잡성이 단순화됨. 자녀 교육처럼 변수가 많은 영역에서는 MECE를 도구로만 쓰고 결론 단순화는 경계해야.
- **저자 인용 시 주의**: '고두의 예'(맹신) 함정. 곤팀장 자료가 효과적이라고 그가 다른 분야에서 한 발언까지 진리로 받지 않는다.
