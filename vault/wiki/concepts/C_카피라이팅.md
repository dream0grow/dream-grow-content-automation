# 카피라이팅

독자/시청자의 행동을 유도하기 위해 심리적 트리거를 활용하여 글을 작성하는 기술. 마케팅 글쓰기의 핵심이자 유튜브/숏폼 콘텐츠의 도입부(훅) 설계 기반.

## 핵심 공식

### 파소나 법칙 → 오공문해
1. **오잉?** (호기심 유발)
2. **공감** (시청자 경험 공유)
3. **문제제기** (통념 뒤집기)
4. **해결책 안내**

### 유튜브 도입부 5대 구조
1. 공감-정의-설명 구조
2. 궁금증 유발 ("도대체~", "A가 아니라 B")
3. 유인 제시 (욕구 분석 후 나열)
4. 문제 제기 ("세뇌 당했기 때문")
5. 공감+주장 ("별거 아닌 것 같다고요? 아닙니다")

### 썸네일 문장 4구조
1. "(문제 A)에 (쉬운 B)하면 전부 해결됩니다"
2. "(문제 A) 겪는 사람들이 자주 (하는 행동 B)"
3. "모든 (결과 A) 문제는 (간단한 원인 B) 때문이다"
4. "(하려는 행동 A) 전부 다 틀렸습니다"

### 카피 심리 기법
- 앵커링 효과 (숫자로 기준점 설정)
- 시각화 표현 ("피부에 형광등 키는")
- 질문형 호기심 유발
- 중의적 표현과 대비
