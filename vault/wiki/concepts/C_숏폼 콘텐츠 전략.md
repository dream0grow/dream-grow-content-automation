# 숏폼 콘텐츠 전략

1분 이내 짧은 영상으로 시청자의 관심을 끌고 구독/참여를 유도하는 콘텐츠 전략. 초반 5초 후킹이 핵심이며, 짧은 시간 안에 기승전결을 완성해야 한다.

## 핵심 원칙
1. **초반 5초**: 호기심과 계속 보고 싶은 마음 전달
2. **1분 기승전결**: 짧은 시간 안에 스토리텔링 완성
3. **참여 유도**: 댓글, 좋아요, 구독 요청을 콘텐츠에 자연스럽게 녹이기

## 숏폼 유형별 전략

### 정보형 (카피메이커)
- 실제 브랜드/광고 분석 → 개선안 제시
- 오공문해 공식: 오잉? → 공감 → 문제제기 → 해결책

### 스포츠형 (쇼포츠)
- 미션/챌린지, VS 비교, 감동 역전, 기부 연동
- 구독자 참여 + 선한 영향력

### 스토리텔링형 (부자 시리즈)
- 영화/드라마 캐릭터로 경제 원리 설명
- "예상치 못한 부자" 유머 코드

### AI 활용형 (샷건)
- ChatGPT로 시나리오 자동 생성
- 강렬한 캐릭터 컨셉
