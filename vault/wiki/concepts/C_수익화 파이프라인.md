# 수익화 파이프라인

## 핵심 요약

콘텐츠를 돈으로 전환하는 단계적 구조. 무료 콘텐츠로 신뢰를 쌓고, 저가 상품(전자책)에서 고가 상품(코칭/워크숍)으로 자연스럽게 이동시키는 설계. 광고 수익에 의존하지 않고 자체 상품/서비스를 통해 지속 가능한 수익을 만드는 것이 핵심이다.

## 상세 내용

### 개념 정의

수익화 파이프라인은 콘텐츠 크리에이터가 무료 콘텐츠 발행에서 출발하여 단계적으로 유료 상품/서비스로 전환하는 비즈니스 구조다. 제조업의 공급망처럼, 콘텐츠 비즈니스에도 인지 -> 신뢰 -> 구매 -> 반복구매의 흐름이 있다.

### 4단계 구조

1. **광고 수익** (기반): 플랫폼 광고로 최소 수익 확보. 브랜드 구축이 진짜 목적.
2. **디지털 상품** (저가): 전자책, 템플릿, 체크리스트 등. 가격 저항이 낮아 첫 구매 유도에 적합.
3. **온라인 강의** (중가): 깊이 있는 교육 콘텐츠. 한 번 만들면 반복 판매 가능(패시브 인컴).
4. **코칭/워크숍** (고가): 직접 상호작용. 규모는 작지만 단가가 높아 수익 극대화.

### 세일즈퍼널과의 관계

- [[C_세일즈퍼널]]이 고객의 심리적 여정(인지 -> 관심 -> 결정 -> 행동)을 설계하는 것이라면, 수익화 파이프라인은 그 여정에 맞는 상품 라인업을 배치하는 것.
- 퍼널의 각 단계에 적합한 상품이 파이프라인의 각 단계에 대응한다.

### 핵심 원칙

- 무료에서 유료로의 전환은 **가치의 점진적 상승**이어야 한다
- 각 단계가 다음 단계의 자연스러운 다리가 되어야 한다
- 신뢰 없이 판매하면 실패한다: "먼저 줘라, 그다음에 팔아라"
