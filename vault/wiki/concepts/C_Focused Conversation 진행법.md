# C_Focused Conversation 진행법

Jo Nelson, *The Art of Focused Conversation for Schools* Chapter 5의 진행자 매뉴얼. 잘 설계된 대화도 진행 도중 어긋날 수 있다는 인정 위에서, 진행자의 책임과 9가지 troubleshooting 매트릭스를 제공한다.

## 출발 명제

> "While the Focused Conversation method follows a natural thinking pattern, systematic thinking requires discipline."

ORID는 인간 사고의 자연 흐름이지만, 그 자연 흐름이 집단 안에서 그대로 흐르는 일은 거의 없다. 일부는 발언을 독점하고 일부는 침묵하며, 대화는 곁가지로 빠진다. 진행자는 이를 막는 사람이다.

## 진행 10단계

### 1. Set a Suitable Environment — 환경 설정
- 모든 참가자가 서로 보고 들을 수 있는 배치
- 반원형 또는 직사각형 추천
- 권위적 위치(교탁 뒤, 책상 머리)에 진행자가 서지 말 것

### 2. Invitation — 초대
- 그룹이 자리를 잡고 안정될 시간을 줌
- 비공식적 시간을 짧게 두는 것이 더 좋음

### 3. Opening — 시작 발언
- Chapter 4 Step 7에서 사전 작성한 도입부 그대로 진입
- 대화의 목적, 주제, 역사적 중요성, 결과 활용 방법 명시

### 4. The First Questions — 첫 질문
- 답하기 쉬운 객관 질문으로 분위기 풀기
- 15명 이상이면 진행 방식 조정 (소그룹 분할 등)
- 응답은 짧고 자연스럽게
- "틀린 답은 없다" 분위기 명시
- 첫 객관 질문은 "round" 형식(돌아가며 한 명씩)으로 모두의 목소리를 안전하게 꺼내기

### 5. Subsequent Questions — 후속 질문
- 두 번째 질문부터는 전체 그룹 대상
- 순서 없이 자유롭게 답할 수 있도록

### 6. Getting off the Topic — 주제 이탈 대응
- 주제 이탈은 규율 문제가 아니라 인간의 연상적 사고 때문에 자연스러움
- 인정하면서도 정중히 다시 가져오기: "재미있는 이야기네요. 다시 우리 주제로 돌아오면..."

### 7. Wrong Answers — 틀린 답
- **사실 오류**: 부드럽게 정정, 학생 위축시키지 않기
- **공격적 발언**: 의도와 분위기 파악, 다른 표현 요청, ground rules 재확립
- **의도적 이상한 답변**: 단순히 넘기거나 "왜 그렇게 생각하나요?"로 자존감 다치지 않게 처리

### 8. Long or Abstract Answers — 길거나 추상적 답변
- 구체 질문이 구체 답변을 끌어냄
- "더 간단히 말씀해 주시겠어요?"
- "구체적인 예를 들어주시겠어요?"
- 모호한 진술이 명확해지지 않으면 더 깊은 통찰 차단됨

### 9. Arguments — 논쟁 발생 시
- 핵심: 찬반 토론이 아닌 응답을 명확히 하고 설명하는 것
- 선언적 발언에 근거 묻기
- 반대 의견도 "다른 관점"으로 수용
- 방해하는 참가자는 다른 사람 발언 후 의견 제시하도록 유도

### 10. Closing — 마무리
- 그룹의 결론 요약
- 참가자에게 감사 표시
- 메모 활용 방법 공지

## 9가지 흔한 문제 + 해결책 (책의 Troubleshooting 매트릭스)

### 1. Group Isn't Focusing
| 원인 | 해결책 |
|------|--------|
| 혼란스럽거나 불친절한 환경 | 조용한 공간 재배치, 상호 시선 가능 착석 |
| 그룹의 주의를 끌지 못함 | 비공식 시간 계획, 정중한 주의 집중 요청, 정시 시작 |
| 맥락 불명확 | 회의 목적·방법 충분히 설명 |
| 주제가 그룹과 무관 | 기획 단계 재검토 |

### 2. Group Doesn't Respond to Questions
| 원인 | 해결책 |
|------|--------|
| 진행자 신뢰 부족 | 따뜻한 말로 시작, 경청 시연, "정답 없음" 명시 |
| 주제 초점 흐림 | 구체 목표 명확히 |
| 추상적 시작점 | 구체적 시작점 재명시, 객관 질문 구체화 |
| 질문 순서 오류 | "잠깐, 다른 질문 먼저 하겠습니다"로 즉시 변경 |
| 질문이 폐쇄형, 결론 유도로 느껴짐 | "What is..." 형식으로 재구성 |
| 큰 그룹에서 침묵 선호 | "아직 말 안 한 분?", 이름 불러 호명, 소그룹 활용 |
| 권위자 존재로 안전감 부족 | 그룹 구성 재편, 동질 소그룹 |
| 수줍음 | 첫 객관 질문은 round 형식 |
| 너무 격식 | 앉아서 진행 등 비공식화 |
| "감정 이야기 알레르기" | 감정 단어 회피 R 질문: "어느 부분이 가장 마음에 걸렸나요?" |

### 3. Group Gives Wrong Answers
| 원인 | 해결책 |
|------|--------|
| 진행자에게 숨은 의도 있음 | 진행자 역할 자기 점검 |
| 질문 순서 오류 | 순서 재조정 |
| 질문 모호 | 샘플 답변 제시 |

### 4. Group Doesn't Give Truthful Answers
| 원인 | 해결책 |
|------|--------|
| 안전감 부족 | 덜 개인적 주제로 시작 |
| 질문 구체성 부족 | 사전 질문 테스트 |
| 참가자에게 숨은 의도 | 의도 드러나는 구체 질문 |

### 5. Some Participants Dominate
| 원인 | 해결책 |
|------|--------|
| 충분히 들리지 않았다고 느낌 | 진지한 경청, 적극적 듣기 시연, 정중한 차단: "주요 포인트 잘 들었습니다. 쉬는 시간에 더 듣고 싶네요. 이제 다른 분?" |
| 책임자가 그룹을 신뢰 못 함 | 따로 만나 우려 청취, 독점의 결과 알림 |
| 대화 스타일 너무 개방적 | 일부 질문은 round 형식, 짧은 응답 요청 |
| 조용한 사람과 표현 좋아하는 사람 불균형 | 소그룹 혼합 후 결과 보고 |

### 6. Group Goes Off on Tangents
| 원인 | 해결책 |
|------|--------|
| 주제·책임 회피 욕구 | 그룹과 함께 ORID로 메타 성찰 |
| 주제 비초점·비관련 | 정중하지만 단호하게 복귀 |
| 진행자가 너무 우유부단 | "facilitating은 인기 얻기가 아니라 그룹이 자기 관심사를 다루도록 돕는 것" 자기 상기 |

### 7. Not Getting Useful Results
| 원인 | 해결책 |
|------|--------|
| 주제 비관련 | (1번 참조) |
| 주제 너무 큼 | 부분으로 나눈 여러 대화 설계 |
| 질문이 유용한 정보 못 끌어냄 | 필요 결과에서 역산하여 각 단계 데이터 재설계 |

### 8. Arguments Break Out
| 원인 | 해결책 |
|------|--------|
| 상호 존중 맥락 미형성 | "다차원 현실" 강한 맥락 수립; 첫 발언자 직후 다른 관점 질문; 패턴 질문 |
| 진행자가 편견 보임 | 모든 답변 받고 있는지 자기 점검; "나는 호기심이 있다" 자기 상기 |
| 학생들이 논쟁 능력으로 교사에게 인상 남기려 함 | 사려 깊은 경청·참여에 점수 있음을 상기 |

### 9. Group Challenges the Facilitator
| 원인 | 해결책 |
|------|--------|
| 자신감 전달 못 함 | 신중하게 설계된 과정임을 처음에 알리기, 답 강요 안 함을 명시 |
| 전문가-질문자 혼재 메시지 | 개방성 확보를 위한 질문 점검 |
| 진행자 권력 과민 반응 | 질문을 보이는 곳에 작성, 다른 사람이 진행하도록 역할 분리 |

## Facilitating Is Not Teaching Content — 진행자 정체성

저자가 가장 강조하는 진행자 자기 인식:

- 교사는 답을 가지고 있어도 진행 중에는 답을 주지 않는다
- 발표(presenter) 모자와 진행(facilitator) 모자를 명확히 구분
- 발표는 대화 전·후에 별도로 한다
- 칠판/플립차트에 질문을 적으면 "사람 vs 의견" 분리에 도움
- John Kloepfer의 *docta ignorantia* (소크라테스적 열린 무지) 체화

> "Without a real openness, no real insight is possible."

## 한국 학급·가정 적용 시 추가 주의

- **Step 4 첫 질문**: 한국 학급에서 첫 발언이 가장 어려움. 반드시 round 형식으로 모두 목소리 한 번씩 꺼내기.
- **Step 7 wrong answers**: 한국 교실 문화에서 "틀렸다"는 피드백은 위축이 큼. "재미있는 관점이네요. 그런데 ___ 부분은..."처럼 받기.
- **Problem 5 발언 독점**: 한국 회의에서 권위자(교장, 부장)가 자주 독점. 진행자가 미리 "오늘은 모두의 의견을 듣는 자리입니다"로 ground rule을 깔고, 직접 호명으로 균형 잡기.
- **Problem 8 arguments**: 한국 교사 회의에서 논쟁이 인신공격으로 번지기 쉬움. "패턴 질문"으로 일반화하여 개인 공격에서 빠져나오게 하기.
