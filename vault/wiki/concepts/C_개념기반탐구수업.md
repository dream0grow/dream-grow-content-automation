# C_개념기반탐구수업

사실에서 개념을 추출하고, 개념을 연결하여 전이 가능한 지식을 만드는 수업 방식. 지식의 구조(사실 → 개념 → 원리/일반화 → 전이)와 과정의 구조에서 개념을 뽑아내는 것이 핵심이다. 사실만 아는 것이 아니라 개념을 이해하면 실제 상황에 전이할 수 있다.

## 분야

#2000사회과학/2100교육학/2120교육과정·수업설계

## 핵심 원리

1. **지식의 구조: 사실 → 개념 → 원리 → 전이** — 사실의 수집에서 개념을 추출하고, 개념을 연결하여 원리(일반화)를 발견하고, 이를 새로운 상황에 전이한다.
2. **개념은 사실에서 귀납적으로 뽑아낸 공통점이다** — 개념은 전이가 가능한 지식이며, [[C_비판적 사고]]를 통한 검증이 필수이다.
3. **변혁적 역량** — OECD 학습 나침반 2030이 제시하는 새로운 가치 창출, 갈등과 딜레마 조정, 책임 의식.
4. **탐구 중심 학습** — 교사가 지식을 전달하는 것이 아니라 학생이 스스로 탐구하여 개념을 발견하게 한다.
5. **백워드 설계(Backward Design)** — 최종 도달점(전이)에서 출발하여 역순으로 수업을 설계한다.

## 관련 연구/이론

- **린 에릭슨(H. Lynn Erickson)의 개념기반 교육과정**: 사실-개념-일반화의 지식 구조를 체계화하고, 개념적 렌즈를 통한 탐구수업을 제안.
- **위긴스 & 맥타이(Wiggins & McTighe)의 백워드 설계(UbD)**: 이해를 위한 설계. 최종 평가에서 출발하여 수업을 역순으로 설계하는 방법론.
- **OECD 학습 나침반 2030**: 학생 주도성(Student Agency)과 변혁적 역량을 교육의 방향으로 제시.
- **IB 교육과정**: 탐구 중심 + 개념 이해 중심 교육과정의 실제 적용 사례. [[C_한교육]]이 벤치마킹하는 모델.

## 실천 방법

- **See-Think-Wonder 전략**: 관찰(See) → 사고(Think, See가 근거) → 궁금증(Wonder)으로 관찰 기반 사고를 촉진한다.
- **프레이어 모델(Frayer Model)**: 개념의 정의, 특징, 예시, 비예시를 네 칸에 정리하여 개념을 명확히 정의한다.
- **백워드 설계로 수업 설계**: 최종 전이 목표 → 평가 과제 → 학습 활동 순으로 설계한다.
- **조사 시 4단계**: 출처 신뢰성 확인 → 모르는 단어 확인 → 핵심 찾기 → 자신의 말로 정리.
- **교육과정 재구성**: 교사의 철학을 담아 아이들의 이야기를 펼칠 수 있는 방향으로 교육과정을 재구성한다.
