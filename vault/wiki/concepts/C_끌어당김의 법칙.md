# 끌어당김의 법칙

생각과 감정이 현실을 만든다는 원리. 목표를 명확히 정의하고 긍정적 사고 회로를 만들어 무의식에 믿음을 심는 실천 방법론.

[출처:: [[raw/Roam-Export-1773035150854/전자책(무료배포용) - 끌어당김 실행 홈 가이드 북 (워크북).md]]]

## 실천 단계
1. **자기 인식**: 의구심(브레이크) 제거 → 진정한 열망 파악
2. **목표 설정**: 만다라트 72개 아이디어 → 직감으로 10개 선택 → 검증
3. **긍정 사고 회로**: 시각화 + 긍정 문장 반복 + 감정 기록
4. **시각화/명상**: 목표 달성 모습을 상상하며 명상
5. **피드백**: 습관 만들기 + 일기 쓰기 + 주간 실천

## 핵심 원리
- 무의식이 95%를 지배하며, 직감은 무의식의 소리
- [[C_자아상]]의 변화가 현실 변화로 이어진다
- [[확언]]과 결합하여 효과 극대화
