# 유튜브 알고리즘

탐색 기반 추천 시스템으로, 시청자의 행동 데이터를 분석해 홈 화면과 추천 영상을 결정하는 메커니즘.

## 핵심 요소 (4가지)

1. **시청 시간**: 영상을 얼마나 오래 보는가
2. **클릭률(CTR)**: 노출 대비 클릭 비율 — 썸네일과 제목이 결정
3. **시청 지속률**: 영상의 어느 지점까지 보는가
4. **참여도**: 좋아요, 댓글, 공유, 저장

## 떡상의 메커니즘

- 홈 화면 노출이 떡상의 핵심
- 알고리즘이 소규모 표본에 먼저 노출 → 반응이 좋으면 확대 노출
- 초기 반응(24~48시간)이 영상의 운명을 결정
