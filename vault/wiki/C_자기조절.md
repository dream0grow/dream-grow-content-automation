# 자기조절 (Self-Regulation)

자기조절은 아이가 외부 통제 없이 스스로 감정·행동·주의를 관리하는 능력이다. 발달심리학자 클레어 코프(Claire Kopp)는 자기조절을 외적 통제 → 자기통제 → 자기조절의 3단계 발달로 설명했다.

월터 미셸의 마시멜로 실험은 만족 지연(delayed gratification) 능력이 자기조절의 핵심임을 보여준다. 단, 미셸 본인이 강조하듯 이는 의지력이 아니라 '주의 전환 전략'의 문제이다.

훈육의 최종 목표는 부모가 시키지 않아도 아이가 스스로 행동하는 상태, 즉 자기조절의 내면화이다. 외적 보상·처벌에 의존하는 훈육은 자기조절을 약화시키고, 이유를 설명하고 선택권을 주는 훈육은 자기조절을 강화한다.
