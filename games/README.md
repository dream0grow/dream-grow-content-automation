# 🪲 곶자왈 곤충 찾기 타워

아이 컨셉으로 만든 **단일 HTML 파일 게임**(Canvas). 라이브러리·빌드 없음 → 브라우저에서 바로 열림, PC·휴대폰 모두 동작.

## 바로 해보기
`gotjawal-tower.html` 파일을 더블클릭(또는 브라우저로 열기)하면 끝.

## 게임 규칙
- 🪲 **물장군**이 되어 타워를 점프해서 올라간다 (로블록스 타워 점프 게임 벤치마킹).
- 🌋 시간이 지나면 **용암이 차오른다** — 멈추지 말고 올라가야 함.
- ❤️ 목숨 5개. 폭탄/용암/추락 시 1개 잃고, 용암 위 안전한 발판에서 부활.
- 💣 **폭탄**을 건드리면 펑! / 🧱 **금 간 블록**은 밟으면 사라짐.
- ❓ **물음표 블록**에서 곶자왈 곤충 퀴즈 → 맞히면 곤충 획득.
- 🏆 곶자왈 곤충 5마리(물장군·왕사마귀·애기뿔소똥구리·제주홍단딱정벌레·애기사마귀)를 다 모으면 **도감 완성 승리**.

## 조작
- PC: `←` `→` 이동, `Space`/`↑` 점프
- 폰: 화면 아래 터치 버튼(◀ ▶ 점프)

## 배포 (공유 URL 만들기)
단일 파일이라 어디든 올리면 끝:
- **GitHub Pages**: repo Settings → Pages → 이 폴더 게시 → `.../games/gotjawal-tower.html`
- **Netlify / Cloudflare Pages**: 파일 드래그&드롭 한 번이면 URL 발급
- 발급된 URL로 QR코드를 만들면 아이가 친구·가족과 바로 공유 가능

## 검증
게임 로직은 Node 환경에서 자동 테스트로 확인:
- 핵심 메커닉 10종 단위 테스트 통과(퀴즈 획득/오답/승리/깨지는 블록/폭탄/용암/부활/게임오버)
- 렌더 경로 무오류, 봇 플레이 기준 클리어 가능(난이도 균형 확인)

> 도감 곤충은 제주 곶자왈에서 볼 수 있는 실제 곤충들이며, 퀴즈는 초등 저학년 눈높이로 작성했습니다.
