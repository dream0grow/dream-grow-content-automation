# 곶자왈 곤충 탐험대 🐛 (스테이지 1)

초등 저학년용 **탭(클릭) 곤충 찾기 게임**. 단일 HTML 파일이라 더블클릭하면 바로 플레이됩니다.

## 플레이 방법
1. `index.html`을 브라우저로 엽니다 (또는 휴대폰으로 열기).
2. **돌**을 톡 누르면 곤충이 나오거나, 어두운 **숨골**에 빠져 목숨이 줄어요.
3. **푸른 나무(❓)**를 누르면 퀴즈 → 맞히면 **채집통/잠자리채**를 만들어요.
4. **기울어진 죽은 나무**는 건드리면 쓰러져 위험!
5. 곤충은 알맞은 도구가 있어야 잡을 수 있어요. 3마리를 모두 모으면 성공!

## 게임 규칙(설계 논리)
- **모든 위험에는 단서가 있다** — 곤충 흔적은 ✨ 반짝, 숨골은 어두운 구멍, 죽은 나무는 ⚠️ 기울어짐.
- **퀴즈(지식) → 도구 → 채집** 순환 — 배움이 곧 실력.
- **곤충마다 도구가 다르다** — 물장군·넓적사슴벌레는 🪣채집통, 왕사마귀는 🥅잠자리채.
- 목숨 3개로 시작, 숨은 ❤️ 하트로 목숨을 더 얻을 수 있어요.

## 곤충 사진 출처 (자유 라이선스)
실제 곤충 모습이 중요하므로 **위키미디어 공용(Wikimedia Commons)** 이미지를 직링크로 사용합니다.
인터넷이 연결돼야 사진이 보이며, 사진 로딩에 실패하면 이모지+이름 폴백이 표시됩니다.

- 넓적사슴벌레: `Lucanidae - Dorcus titanus-002.JPG`
- 왕사마귀: `Tenodera sinensis (Chinese praying mantis) (Newark, Ohio, USA) 1 (31503867486).jpg`
- 물장군: `Lethocerus deyrollei.jpg`

> 더 정확한 사진으로 바꾸려면 `index.html`의 `INSECTS` 객체 안 `img` 주소만 교체하면 됩니다.
