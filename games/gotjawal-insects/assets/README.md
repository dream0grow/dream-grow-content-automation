# 사진 넣는 곳 (assets)

여기에 아래 **파일명 그대로** 사진을 넣으면 게임에 바로 나타납니다.
(`.jpg`가 없으면 `.png`도 자동으로 시도해요. 둘 다 없으면 이모지로 임시 표시됩니다.)

## 꼭 필요한 파일
| 파일명 | 내용 |
|---|---|
| `배경.jpg` | 곶자왈 숲 배경 사진 (가로로 넓은 사진 추천) |
| `왕사마귀.jpg` | 왕사마귀 사진 |
| `두점박이사슴벌레.jpg` | 두점박이사슴벌레(붉은빛, 긴 턱) 사진 |
| `뚱보귀뚜라미.jpg` | 뚱보귀뚜라미 사진 |

## 다음 스테이지용 (지금은 없어도 됨)
| 파일명 | 내용 |
|---|---|
| `애기뿔소똥구리.jpg` | 검은 소똥구리 사진 |
| `메뚜기.jpg` | 메뚜기 사진 |

> 곤충을 바꾸거나 추가하려면 `index.html`의 `INSECTS` / `STAGE1` 부분만 고치면 됩니다.
> 파일명에는 **곤충 이름**이 그대로 들어갑니다 (한글 파일명 OK).
