그 사진이 바로 뜹니다. (정사각형에 가까울수록 예쁘게 보여요. 가로 400px 이상 권장)

## 동작 방식 (우선순위)
1. **이 폴더의 사진 파일** (있으면 최우선)
2. 없으면 → 코드에 들어있는 **공개 사진 URL**(위키미디어)을 브라우저에서 로드
3. 그것도 실패하면 → **자동 일러스트**로 대체 (게임은 항상 정상 동작)

> 사진은 저작권을 확인하고 사용하세요. 직접 찍은 사진이나 CC/공공누리 등 사용 가능한 사진을 권장합니다.
> URL을 바꾸고 싶으면 `gotjawal-tower.html`의 `INSECTS` 배열에서 `img:[...]`를 수정하면 됩니다.
