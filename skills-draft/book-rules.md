# 책쓰기 프로젝트 규칙

## 절대 금지 (SNS 습관 차단)

1. **SNS 브랜딩 문구 금지** — "아이와 부모의 꿈을 키웁니다. -Dream_Grow-", 해시태그, CTA 일체 삽입 금지
2. **SNS식 마무리 금지** — "~바랍니다" 기원문 + 서명으로 끝내지 않는다. 챕터 마무리는 여운과 다음 챕터 연결로
3. **이모지/이모티콘 금지** (SNS와 동일)
4. **출처 없는 통계 금지** (SNS와 동일)
5. **챕터 간 동일 에피소드 중복 사용 금지** — 같은 교실 이야기를 두 챕터에서 쓰지 않는다
6. **선언적 단정 남용 금지** — "~이다"로만 끝나는 문장이 3개 연속되면 안 된다. 리듬을 바꿔야 한다

## 필수 구조

### 챕터 구조
- 도입: 교실 에피소드(장면 묘사)로 시작. 시간/장소/인물이 있는 구체적 장면
- 전환: 부모의 일상으로 연결 ("부모도 같은 경험을 한다")
- 분석: 왜 이런 일이 일어나는가 (이론/연구 근거)
- 대비: 'A가 아니라 B이다' 논증. 기존 상식 vs 실제
- 실천: 가정에서 바로 쓸 수 있는 구체적 방법 (단계별)
- 대화 예시: 부모-자녀 대화 before/after (최소 1세트)
- 여운: 챕터 핵심을 한 문장으로 각인 + 다음 챕터 예고

### 챕터 분량
- 8,000~15,000자 (원고지 40~75매, 15~25페이지)
- 소제목 4~6개
- 교실 에피소드 2~3개 (도입 1개 + 본론 1~2개)

## 글쓰기 톤 (SNS와 다른 점)

### SNS 톤 (하지 말 것)
- "결론부터 말한다." (선언적 후킹)
- "이것만 기억하세요." (명령형)
- "~해야 한다." (단정형 반복)

### 책 톤 (해야 할 것)
- 독자와 함께 생각하는 어투: "~를 생각해보자", "~일지도 모른다", "여기서 질문이 생긴다"
- 사색과 관찰의 언어: "교실에서 오래 지켜보면 ~가 보인다"
- 짧은 문장과 긴 문장의 교차로 리듬 만들기
- 단정이 필요할 때는 확실하게, 나머지는 독자의 판단에 여지를 남기기

### 문장 리듬 규칙
- 짧은 문장(10자 이내)과 긴 문장(30자 이상)을 3:7 비율로 섞는다
- 짧은 문장은 강조/전환/각인에 사용한다 ("이유는 간단하다." "여기서 멈춰야 한다.")
- 한 문단 안에서 같은 어미가 3번 연속 반복되지 않도록 한다

## 챕터 마무리 규칙

- SNS처럼 "~바랍니다" + 서명으로 끝내지 않는다
- 대신 다음 중 하나를 사용:
  1. **각인 문장** — 챕터 핵심을 한 문장으로 압축 ("부모에게 필요한 것은 더 큰 목소리가 아니라, 더 많이 연습할 시간이다.")
  2. **열린 질문** — 독자가 스스로 생각하게 ("그렇다면 아이가 정말 원하는 것은 무엇일까.")
  3. **다음 챕터 브릿지** — 자연스럽게 다음 주제로 연결 ("훈육이 행동의 경계를 긋는 일이라면, 감정은 그 경계 안에서 아이가 살아가는 방식이다.")

## 근거와 깊이

- SNS: 교실 에피소드 1개면 충분
- 책: 에피소드 2~3개 + 이론적 근거(뇌과학, 발달심리학 등) 필수
- 인용 시 "~에 따르면" 형식. 출처를 각주 또는 참고문헌으로 명시
- wiki/ 와 제텔카스텐/에서 이론 근거를 가져올 것

## 품질 체크리스트 (챕터 완성 시)

- [ ] SNS 브랜딩 문구가 없는가
- [ ] 소리 내어 읽었을 때 리듬이 자연스러운가
- [ ] 에피소드가 시간/장소/인물이 있는 장면으로 그려지는가
- [ ] 한 챕터가 하나의 핵심 주장을 관통하는가
- [ ] 독자가 밑줄 치고 싶은 문장이 2~3개 있는가
- [ ] 부모-자녀 대화 예시가 최소 1세트 있는가
- [ ] 다음 챕터로 자연스럽게 이어지는가
- [ ] 다른 챕터와 에피소드/논점이 중복되지 않는가
- [ ] 이론 근거가 최소 1개 포함되어 있는가
- [ ] 이모지가 없는가

## 모델 설정

- 책 원고 작성: **Opus 4.6** (사고의 깊이 + 문체의 자연스러움 최대화)
- 구조 설계/목차/자료 정리: Sonnet (유틸리티)
