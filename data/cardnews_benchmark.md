# 카드뉴스 벤치마킹 (갱신: 2026-07-07 08:03 KST, 신뢰도: low)

## 지금 뜨는 흐름
- 표지에 텍스트 한 줄만 크게 넣고 나머지는 사진으로 채우는 '한 문장 후킹형' 캐러셀
- 체크리스트/자가진단 형식('이 중 3개 이상 해당되면...')으로 참여를 유도하는 포맷
- 전문가·현직자 화법('OO 10년차가 말하는', '교실에서 보니')을 앞세운 경험 기반 정보성 카드뉴스
- 표지는 질문, 마지막 장은 요약·저장 유도 문구로 끝맺는 '질문→답' 구조
- 긴 설명 대신 3~5개 키워드만 큼직하게 배치하는 미니멀 타이포 캐러셀
- 손글씨체·메모지 느낌의 아날로그 톤 디자인(과한 그래픽보다 친근함 강조)
- 부모 반응·댓글을 캡처해 다음 장에 삽입하는 '실제 사례 인용형' 구성
- 짧은 영상(릴스)과 카드뉴스를 같은 주제로 동시에 올려 상호 유입시키는 크로스포맷 운영
## 후킹(표지) 패턴
- 단정 대신 질문형: '우리 아이가 이런 말 자주 하나요?'
- 역설/반전 제시: '혼내지 않았는데 아이가 더 짜증 낼 때'
- 구체적 상황 묘사로 시작: '학교 다녀와서 가방을 던지듯 내려놓는다면'
- 숫자·목록 예고: '초등 저학년이 유독 힘들어하는 3가지 순간'
- 공감형 부정문: '이거 하지 마세요, 저도 예전엔 몰랐어요' 식의 자기 경험 고백
- 대사 인용형: 아이가 실제로 할 법한 말을 큰 따옴표로 표지에 배치
- 저장 유도 예고: '끝까지 보면 오늘 저녁에 바로 써먹을 수 있어요'
## 전개 구조 패턴
- 문제 제기(1장) → 흔한 오해·잘못된 대응(2~3장) → 반전 관점 제시(4장) → 구체적 실천법(5~6장) → 요약·저장 유도(마지막장)
- 질문 던지기(1장) → 원인 설명(2~3장) → 사례(4장) → 대안 행동 3단계(5~7장) → 한 줄 정리(마지막장)
- 체크리스트형: 표지에 목록 예고 → 각 장에 항목 하나씩 → 마지막 장에 '몇 개 해당되나요' 재확인
- 짧은 장수(5~6장) 구성이 완독률에 유리하다는 경향이 최근 언급되나, 정확한 수치는 출처 확인 필요
## 시각 요소 패턴
- 실사 인물 사진 위에 굵은 산세리프 타이포를 얹는 구성(과도한 일러스트보다 사실감 선호)
- 장마다 배경색을 통일하되 포인트 컬러 1개만 사용해 시리즈 일관성 부여
- 핵심 키워드에만 색을 넣고 나머지는 무채색 처리해 시선 집중
- 손글씨 폰트나 메모지 이미지를 활용해 '정보'보다 '개인의 기록'처럼 보이게 하는 톤
- 각 장 하단에 페이지 번호(1/6 등)를 넣어 완독 유도
## 캡션·CTA 패턴
- '저장해두고 오늘 저녁에 한번 해보세요' 식의 행동 유도형 마무리
- 캡션에 카드뉴스 내용 요약을 다시 텍스트로 반복해 검색·공유 노출 확대
- '우리 집은 어떤가요? 댓글로 알려주세요' 식의 댓글 유도 질문
- 팔로우보다 '저장'을 명시적으로 요청하는 문구가 최근 더 자주 보이는 경향(정확한 효과 수치는 미확인)
## 피할 요소
- '이렇게 안 하면 아이 망칩니다' 류의 공포·죄책감 유발 문구
- 출처 없는 '전문가에 따르면', '연구 결과' 식 통계 인용
- 과도한 이모지·비교문구('문제아' 등 낙인성 표현)
- 지나치게 많은 장수(10장 이상)로 완독률이 떨어지는 구성
- 특정 인물·학교가 식별 가능한 실제 사례 노출
